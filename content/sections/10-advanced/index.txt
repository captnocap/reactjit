=== OVERVIEW ===
This section covers the internals of ReactJIT for developers who need to understand
what happens beneath the component API. Topics include the complete Lua runtime module
inventory, focus management and controller navigation, the Lua-owned interaction pattern
for zero-bridge-traffic primitives, networking (HTTP, WebSockets, proxies), Tor integration,
the event dispatch system, performance optimization strategies, debugging tools, rendering
internals, and how to create new render targets.

These topics assume familiarity with the rendering pipeline (see 02-architecture/) and
the component API (see 05-components/).
