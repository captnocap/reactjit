=== OVERVIEW ===
useTransition automatically transitions to a target value whenever it changes, using timing-based animation with a configurable duration and easing curve. Unlike useSpring, which determines its own timing from physics, useTransition gives you precise control over how long the animation takes.

The hook skips animation on the first render and jumps straight to the target value. On subsequent changes, it starts a timing animation from the current position to the new target.

useTransition is best for:
- Animations that need a specific, predictable duration
- Progress bars and loading indicators
- Opacity fades with precise timing
- Any value that should arrive at its target in exactly N milliseconds

For physics-based motion with natural overshoot, use useSpring instead. For Lua-driven transitions that produce zero JS re-renders, use style.transition in your style objects.

=== API / SYNTAX ===
```tsx
import { useTransition } from '@reactjit/core';

const currentValue = useTransition(targetValue, config?);
```

Parameters:
| Parameter   | Type   | Description                                |
|-------------|--------|--------------------------------------------|
| targetValue | number | The value to transition toward             |
| config      | object | Optional timing configuration              |

Config:
| Field    | Type           | Default          | Description                     |
|----------|----------------|------------------|---------------------------------|
| duration | number         | 300              | Duration in milliseconds        |
| easing   | EasingFunction | Easing.easeInOut | Easing curve for the transition |

Returns: number — the current interpolated value. Updates each frame during the transition.

=== EXAMPLES ===
Example 1: Smooth progress bar
---
```tsx
import { useState, useEffect } from 'react';
import { Box, Text, useTransition, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function SmoothProgress({ percent }: { percent: number }) {
  const c = useThemeColors();
  const width = useTransition(percent, {
    duration: 500,
    easing: Easing.easeOut,
  });

  return (
    <Box style={{ height: 8, backgroundColor: c.surface, borderRadius: 4 }}>
      <Box style={{
        height: 8,
        width: `${Math.max(0, Math.min(100, width))}%`,
        backgroundColor: c.primary,
        borderRadius: 4,
      }} />
    </Box>
  );
}
```

Example 2: Fade between states
---
```tsx
import { useState } from 'react';
import { Box, Pressable, Text, useTransition, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function FadeToggle() {
  const c = useThemeColors();
  const [active, setActive] = useState(false);
  const opacity = useTransition(active ? 1 : 0.3, {
    duration: 400,
    easing: Easing.easeInOut,
  });

  return (
    <Pressable onPress={() => setActive(v => !v)}>
      <Box style={{
        opacity,
        padding: 16,
        backgroundColor: c.surface,
        borderRadius: 8,
      }}>
        <Text style={{ color: c.text, fontSize: 14 }}>
          {`${active ? 'Active' : 'Inactive'}`}
        </Text>
      </Box>
    </Pressable>
  );
}
```

Example 3: Animated sidebar width
---
```tsx
import { useState } from 'react';
import { Box, Pressable, Text, useTransition, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function CollapsibleSidebar() {
  const c = useThemeColors();
  const [open, setOpen] = useState(true);
  const sidebarWidth = useTransition(open ? 240 : 48, {
    duration: 250,
    easing: Easing.easeInOut,
  });

  return (
    <Box style={{ flexDirection: 'row', width: '100%', height: '100%' }}>
      <Box style={{
        width: sidebarWidth,
        backgroundColor: c.bgElevated,
        overflow: 'hidden',
      }}>
        <Pressable onPress={() => setOpen(v => !v)}>
          <Box style={{ padding: 12 }}>
            <Text style={{ color: c.text, fontSize: 14 }}>
              {`${open ? 'Collapse' : '>'}`}
            </Text>
          </Box>
        </Pressable>
      </Box>
      <Box style={{ flexGrow: 1, backgroundColor: c.bg }} />
    </Box>
  );
}
```

=== CRITICAL RULES ===
- useTransition skips animation on the first render — the component starts at the target value instantly.
- The component re-renders every frame during the transition. For visual-only effects, prefer Lua-driven style.transition which produces zero JS re-renders.
- If the target value changes mid-transition, the animation restarts from the current interpolated position to the new target.
