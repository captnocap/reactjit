=== OVERVIEW ===
useAnimation is the primary hook for consuming animated values in a component. It creates an AnimatedValue, subscribes to its changes, and triggers re-renders when the value updates. The AnimatedValue persists across re-renders via a ref.

Returns a tuple of [AnimatedValue, currentValue]. Use the AnimatedValue to start timing or spring animations. Use currentValue in your JSX to apply the animated number to styles.

The listener fires once per Lua frame during active animation — no batching is needed because updates are already synchronized to Love2D's frame rate via tickAnimations().

On unmount, the hook automatically unsubscribes and stops any active animation.

=== API / SYNTAX ===
```tsx
import { useAnimation } from '@reactjit/core';

const [animatedValue, currentValue] = useAnimation(initialValue);
```

Parameters:
| Parameter    | Type   | Description                              |
|--------------|--------|------------------------------------------|
| initialValue | number | The starting value of the animation      |

Returns:
| Index | Type          | Description                                      |
|-------|---------------|--------------------------------------------------|
| 0     | AnimatedValue | The mutable animation value (use to start/stop)  |
| 1     | number        | The current numeric value (use in JSX)            |

=== EXAMPLES ===
Example 1: Fade-in on mount
---
```tsx
import { useEffect } from 'react';
import { Box, Text, useAnimation, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function FadeIn({ children }: { children: React.ReactNode }) {
  const c = useThemeColors();
  const [anim, opacity] = useAnimation(0);

  useEffect(() => {
    const fadeIn = anim.timing({
      toValue: 1,
      duration: 600,
      easing: Easing.easeOut,
    });
    fadeIn.start();
    return () => fadeIn.stop();
  }, [anim]);

  return (
    <Box style={{ opacity }}>
      {children}
    </Box>
  );
}
```

Example 2: Toggle between two values
---
```tsx
import { useState, useEffect } from 'react';
import { Box, Pressable, Text, useAnimation, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function ToggleBox() {
  const c = useThemeColors();
  const [expanded, setExpanded] = useState(false);
  const [anim, height] = useAnimation(40);

  useEffect(() => {
    const target = expanded ? 200 : 40;
    const a = anim.timing({
      toValue: target,
      duration: 300,
      easing: Easing.easeInOut,
    });
    a.start();
    return () => a.stop();
  }, [expanded, anim]);

  return (
    <Box>
      <Pressable onPress={() => setExpanded(e => !e)}>
        <Box style={{ padding: 8, backgroundColor: c.surface, borderRadius: 6 }}>
          <Text style={{ color: c.text, fontSize: 14 }}>
            {`${expanded ? 'Collapse' : 'Expand'}`}
          </Text>
        </Box>
      </Pressable>
      <Box style={{
        height,
        backgroundColor: c.bgElevated,
        borderRadius: 6,
        overflow: 'hidden',
      }}>
        <Text style={{ color: c.muted, fontSize: 12, padding: 8 }}>
          {'Content goes here'}
        </Text>
      </Box>
    </Box>
  );
}
```

Example 3: Interpolation for color and scale
---
```tsx
import { useEffect } from 'react';
import { Box, useAnimation } from '@reactjit/core';

export function ProgressIndicator({ progress }: { progress: number }) {
  const [anim, value] = useAnimation(0);

  useEffect(() => {
    anim.timing({ toValue: progress, duration: 400 }).start();
  }, [progress, anim]);

  const color = anim.interpolate({
    inputRange: [0, 0.5, 1],
    outputRange: [
      'rgba(239, 68, 68, 1)',
      'rgba(245, 158, 11, 1)',
      'rgba(16, 185, 129, 1)',
    ],
    extrapolate: 'clamp',
  });

  const width = anim.interpolate({
    inputRange: [0, 1],
    outputRange: [0, 300],
    extrapolate: 'clamp',
  });

  return (
    <Box style={{ height: 8, width: 300, backgroundColor: 'rgba(255,255,255,0.1)', borderRadius: 4 }}>
      <Box style={{
        height: 8,
        width: width as number,
        backgroundColor: color as string,
        borderRadius: 4,
      }} />
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The AnimatedValue is created once and persisted in a ref. The initialValue is only used on the first render.
- Always call anim.stop() in the useEffect cleanup to prevent animations from running after unmount.
- The currentValue updates synchronously with the Lua frame loop — no additional batching or throttling is needed.
