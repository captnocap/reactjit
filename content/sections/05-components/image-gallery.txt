=== OVERVIEW ===
ReactJIT provides four image gallery components for different layout needs. All share the same ImageViewerModal lightbox for full-screen viewing with keyboard navigation and filmstrip thumbnails.

ImageGallery is the general-purpose gallery. It displays images in either a wrapping row layout (fixed thumbnail widths) or a columns layout (equal-width columns that fill available space). Clicking a thumbnail opens the ImageViewerModal.

BentoImageGallery arranges images in a Pinterest-style masonry grid with three columns of varying proportions. It uses 8 tile slots with different heights, creating a visually dynamic layout without any configuration.

HoverPreviewRowsGallery shows a large preview image at the top with a grid of thumbnails below. Hovering or tapping a thumbnail swaps the preview. Clicking opens the full viewer.

ImageViewerModal is the shared lightbox. It renders in a Modal with prev/next navigation, counter, captions, and a scrollable filmstrip. Supports keyboard navigation (left/right arrows, Home/End) and optional looping.

=== API / SYNTAX ===
```tsx
import { ImageGallery, BentoImageGallery, HoverPreviewRowsGallery, ImageViewerModal } from '@reactjit/core';
```

=== EXAMPLES ===
Example 1: Basic wrap gallery
---
```tsx
const images = [
  { src: 'photos/sunset.jpg', title: 'Sunset' },
  { src: 'photos/ocean.jpg', title: 'Ocean' },
  { src: 'photos/forest.jpg', title: 'Forest' },
];

<ImageGallery images={images} />
```

```tsx
<ImageGallery
  images={photos}
  layout="columns"
  columns={4}
  gap={8}

/>
```

=== PLATFORM NOTES ===
Love2D:
  All gallery components use Pressable for hover and press states. The ImageViewerModal renders inside a Modal (position:absolute, high zIndex). Keyboard navigation (left/right/Home/End) uses Love2D key names. The filmstrip renders in a horizontal ScrollView.

=== CRITICAL RULES ===
- BentoImageGallery uses 8 tile slots. If you provide fewer than 8 images, slots wrap around using modulo indexing. Provide at least 8 images for best results.
- Columns are clamped to 1-8. Values outside this range are silently clamped.
- All galleries include an ImageViewerModal automatically. Do not wrap them in another Modal.
