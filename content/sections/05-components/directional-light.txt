=== OVERVIEW ===
DirectionalLight simulates a distant light source such as a sun, with rays that are parallel across the entire scene. The direction prop points toward the light source (not away from it). It drives diffuse shading and Phong specular highlights on all lit meshes. Only one DirectionalLight per Scene is currently supported. It renders nothing on web targets.

=== API / SYNTAX ===
```tsx
import { DirectionalLight } from '@reactjit/3d';
```

Props:
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| direction | [number, number, number] | — | Direction toward the light source as [x, y, z] — the vector is normalized automatically |
| color | string | '#ffffff' | Light color as a hex string |
| intensity | number | 1.0 | Intensity multiplier applied to the color |

=== EXAMPLES ===
Example 1: Standard white directional light from above and to the side
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function LitScene() {
  return (
    <Scene style={{ width: 800, height: 600 }}>
      <Camera position={[0, -5, 3]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#1a1a2e" intensity={0.15} />
      <DirectionalLight direction={[1, -1, 1]} color="#ffffff" intensity={1.0} />
      <Mesh geometry="sphere" color="#89b4fa" specular={32} />
    </Scene>
  );
}
```

```tsx

      <DirectionalLight direction={[1, -0.5, 0.3]} color="#fffbe6" intensity={1.4} />
      <Mesh
        geometry="sphere"
        texture="planet"
        seed={12}
        specular={16}
        fresnel={4}
        opacity={0.95}
      />

  );
}
```

      <AmbientLight color="#050510" intensity={0.05} />

        geometry="box"
        color="#222244"
        edgeColor="#000000"
        edgeWidth={0.025}

      />
    </Scene>
  );
}

=== CRITICAL RULES ===
- DirectionalLight renders nothing on web targets and returns null — it is Love2D only
- Only one DirectionalLight per Scene is currently supported — placing multiple will produce undefined behavior
- DirectionalLight must be a direct child of Scene, not nested inside other elements
- The direction prop points TOWARD the light, not away from it — [1, -1, 1] means light comes from the positive-X, negative-Y, positive-Z direction
- The direction vector is normalized automatically; you do not need to provide a unit vector
- DirectionalLight with intensity set to 0 disables directional shading; only AmbientLight will remain
- Specular highlights on Mesh are driven by DirectionalLight — adjusting Mesh's specular prop changes highlight tightness

=== SEE ALSO ===
- Scene — the 3D viewport that DirectionalLight must be placed inside
- AmbientLight — provides base fill light to complement DirectionalLight
- Mesh — the geometry that receives directional shading and specular highlights
- Camera — the viewpoint whose angle relative to DirectionalLight affects specular intensity
