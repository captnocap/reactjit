=== OVERVIEW ===
Spacer is a layout utility component that creates space between elements. When used without props, it uses flexGrow to fill all available space in its parent container. When given a size prop, it creates a fixed-height gap. Spacer is the idiomatic way to push elements apart without manual margin calculations.

=== API / SYNTAX ===
import { Spacer } from '@reactjit/core';

Props:
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| size | number | undefined | Fixed height in pixels. If omitted, Spacer uses flexGrow: 1 to fill available space |

=== EXAMPLES ===
Example 1: Push footer to bottom
---
<Box style={{ width: '100%', height: '100%', flexDirection: 'column' }}>
  <Text fontSize={24}>Header</Text>
  <Spacer />
  <Text fontSize={16}>Footer</Text>
</Box>
---
Platforms: All
Description: The Spacer fills all vertical space between header and footer, pushing the footer to the bottom of the container.

Example 2: Fixed gap between elements
---
<Box style={{ flexDirection: 'column' }}>
  <Text fontSize={18}>First Item</Text>
  <Spacer size={20} />
  <Text fontSize={18}>Second Item</Text>
</Box>
---
Platforms: All
Description: Creates a precise 20-pixel vertical gap between two text elements.

Example 3: Multiple spacers distribute space
---
<Box style={{ width: '100%', height: 200, flexDirection: 'column' }}>
  <Text fontSize={16}>Top</Text>
  <Spacer />
  <Text fontSize={16}>Middle</Text>
  <Spacer />
  <Text fontSize={16}>Bottom</Text>
</Box>
---
Platforms: All
Description: Two Spacers with flexGrow: 1 each split the available space equally, distributing elements evenly.

Example 4: Horizontal spacing in row
---
<Box style={{ width: '100%', flexDirection: 'row', alignItems: 'center' }}>
  <Text fontSize={16}>Left</Text>
  <Spacer />
  <Text fontSize={16}>Right</Text>
</Box>
---
Platforms: All
Description: Spacer works in row containers too, filling horizontal space to push elements to the edges.

=== CRITICAL RULES ===
- Spacer with no size prop requires a parent with known dimensions — flexGrow only works when there's space to fill
- In row containers, Spacer fills horizontal space (width); in column containers, it fills vertical space (height)
- Fixed-size Spacers (with size prop) behave like Box with explicit height — they don't grow
- Never use Spacer inside auto-sized containers — flexGrow collapses to zero when there's no available space
- Multiple flexGrow Spacers in the same container share available space equally
- Prefer Spacer over manual marginTop/marginBottom for cleaner, more maintainable layouts
- Spacer is effectively a shorthand for Box with either flexGrow: 1 or a fixed height — use Box directly if you need additional styling
