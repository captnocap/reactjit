=== OVERVIEW ===
Spacer is a layout utility that either fills remaining space or inserts a fixed-size gap between siblings.

Without a `size` prop, Spacer renders a Box with `flexGrow: 1` that expands to absorb all available space in the parent's main axis. This is the most common use — pushing siblings to opposite ends of a row or column.

With a `size` prop, Spacer renders a Box with a fixed `height` in pixels. Use this for consistent spacing between elements when `gap` on the parent is not appropriate (e.g., you need different spacing between specific siblings).

=== API / SYNTAX ===
```tsx
import { Spacer } from '@reactjit/core';
```

Props:
| Prop | Type   | Default   | Description                                              |
|------|--------|-----------|----------------------------------------------------------|
| size | number | undefined | Fixed height in pixels. Omit for flexible grow behavior. |

=== EXAMPLES ===
Example 1: Push items to opposite ends
---
```tsx
const colors = useThemeColors();

<FlexRow align="center" style={{ width: '100%', padding: 12 }}>
  <Text style={{ color: colors.text, fontSize: 14 }}>Logo</Text>
  <Spacer />
  <Text style={{ color: colors.muted, fontSize: 12 }}>Settings</Text>
</FlexRow>
```
---
Platforms: love2d

Example 2: Fixed spacing
---
```tsx
const colors = useThemeColors();

<FlexColumn>
  <Text style={{ color: colors.text, fontSize: 18 }}>Section One</Text>
  <Spacer size={24} />
  <Text style={{ color: colors.text, fontSize: 18 }}>Section Two</Text>
  <Spacer size={12} />
  <Text style={{ color: colors.text, fontSize: 18 }}>Section Three</Text>
</FlexColumn>
```
---
Platforms: love2d

Example 3: Header with title and actions
---
```tsx
const colors = useThemeColors();

<FlexRow align="center" style={{ padding: 12, backgroundColor: colors.bgElevated }}>
  <Text style={{ color: colors.text, fontSize: 16, fontWeight: 'bold' }}>Dashboard</Text>
  <Spacer />
  <Pressable onPress={() => refresh()} style={{ padding: 8 }}>
    <Text style={{ color: colors.primary, fontSize: 12 }}>Refresh</Text>
  </Pressable>
</FlexRow>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- Without `size`, Spacer uses `flexGrow: 1`. The parent must have a defined size (explicit or from its own parent) for grow to work.
- With `size`, Spacer sets a fixed `height`. In a row, this means the spacer adds vertical height, not horizontal width. For horizontal fixed spacing in a row, use `gap` on the parent or a Box with explicit `width`.
