=== OVERVIEW ===
Mesh is the primary renderable object in a 3D scene. It supports four built-in geometry types (box, cube, sphere, plane) as well as externally loaded OBJ model files. Materials are defined through flat color, image textures, or procedural textures. Mesh supports a UV-based GLSL edge shader for wireframe borders, Phong specular highlights, Fresnel rim glow, and alpha transparency. Position, rotation, and scale are reactive and update each frame without reconciliation overhead.

=== API / SYNTAX ===
```tsx
import { Mesh } from '@reactjit/3d';
```

Props:
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| geometry | 'box' \| 'cube' \| 'sphere' \| 'plane' | — | Built-in geometry type to render |
| model | string | — | Path to an OBJ model file |
| color | string | — | Flat color as a hex string — used when no texture is provided |
| texture | string | — | Path to an image file, or a procedural name such as 'planet' |
| seed | number | — | Seed for procedural textures — different seeds produce different terrain |
| position | [number, number, number] | [0, 0, 0] | Position in 3D world space as [x, y, z] |
| rotation | [number, number, number] | [0, 0, 0] | Euler angles in radians as [rx, ry, rz] |
| scale | number \| [number, number, number] | 1 | Uniform scale as a number, or per-axis scale as [sx, sy, sz] |
| edgeColor | string | — | Hex color for edge/wireframe borders drawn on face edges |
| edgeWidth | number | 0.03 | Edge line width as a fraction of face UV space |
| wireframe | boolean | — | Draw longitude/latitude grid lines — makes spheres visibly 3D |
| gridLines | number | 8 | Number of grid divisions per axis when wireframe is enabled |
| opacity | number | 1 | Opacity from 0 to 1 — values below 1 enable alpha blending |
| specular | number | 32 | Specular shininess power — higher values produce tighter highlights |
| fresnel | number | 0 | Fresnel rim power — 0 disables it, 3–5 produces atmosphere glow |
| unlit | boolean | — | Skip lighting and render with flat texture color only |
| onClick | (event: LoveEvent) => void | — | Click handler (raycasting, Phase 5) |
| onPointerEnter | (event: LoveEvent) => void | — | Pointer enter handler (raycasting, Phase 5) |
| onPointerLeave | (event: LoveEvent) => void | — | Pointer leave handler (raycasting, Phase 5) |

=== EXAMPLES ===
Example 1: Colored box with edge borders
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function ColoredBox() {
  return (
    <Scene style={{ width: 800, height: 600 }}>
      <Camera position={[0, -4, 2]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#1a1a2e" intensity={0.15} />
      <DirectionalLight direction={[1, -1, 1]} color="#ffffff" intensity={1.0} />
      <Mesh
        geometry="box"
        color="#89b4fa"
        edgeColor="#000000"
        edgeWidth={0.03}
        position={[0, 0, 0]}
      />
    </Scene>
  );
}
```

Example 2: Spinning sphere with procedural planet texture
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';
import { useAnimationFrame } from '@reactjit/core';

function SpinningPlanet() {
  const [rotation, setRotation] = React.useState<[number, number, number]>([0, 0, 0]);

  useAnimationFrame((dt) => {
    setRotation(([rx, ry, rz]) => [rx, ry + dt * 0.3, rz]);
  });

  return (
    <Scene style={{ width: 800, height: 600 }} stars backgroundColor="#0a0a14">
      <Camera position={[0, -5, 2]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#1a1a2e" intensity={0.1} />
      <DirectionalLight direction={[1, -0.5, 0.5]} color="#fffbe6" intensity={1.2} />
      <Mesh
        geometry="sphere"
        texture="planet"
        seed={7}
        rotation={rotation}
        fresnel={4}
        specular={16}
        wireframe
        gridLines={12}
      />
    </Scene>
  );
}
```

Example 5: Transparent mesh with Fresnel atmosphere glow
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function AtmosphereEffect() {
  return (
    <Scene style={{ width: 800, height: 600 }} stars backgroundColor="#020208">
      <Camera position={[0, -5, 0]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#000814" intensity={0.05} />
      <DirectionalLight direction={[1, -1, 0.3]} color="#fffbe6" intensity={1.5} />
      {/* Solid planet */}
      <Mesh geometry="sphere" texture="planet" seed={3} specular={24} />
      {/* Transparent atmosphere shell */}
      <Mesh
        geometry="sphere"
        color="#7eb3ff"
        scale={1.05}
        opacity={0.25}
        fresnel={5}
        unlit
      />
    </Scene>
  );
}
```

=== CRITICAL RULES ===
- Mesh renders nothing on web targets and returns null — it is Love2D only
- Provide either geometry or model, not both; the behavior when both are set is undefined
- color is ignored when texture is set — texture takes precedence for the surface material
- edgeColor and wireframe are independent features: edgeColor draws borders on cube/box faces via a UV shader; wireframe draws lat/lon grid lines and is most useful on spheres
- The fresnel prop controls alpha at glancing angles — it requires opacity to be less than 1 to be visible as transparency
- scale as a single number applies uniformly to all axes; scale as [sx, sy, sz] scales each axis independently
- rotation values are in radians, not degrees
- onClick, onPointerEnter, and onPointerLeave are Phase 5 raycasting features and may not be available in all versions

=== SEE ALSO ===
- Scene — the 3D viewport that Mesh must be placed inside
- Camera — defines the viewpoint that sees Mesh geometry
- AmbientLight — provides base illumination for Mesh shading
- DirectionalLight — provides directional shading and specular highlights on Mesh
