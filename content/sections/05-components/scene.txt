=== OVERVIEW ===
Scene is the root container for all 3D rendering in ReactJIT. It creates an off-screen canvas that the Love2D painter composites into the 2D layout at the node's computed position. Children of Scene must be 3D elements: Camera, Mesh, AmbientLight, and DirectionalLight. On web targets, Scene renders a placeholder div because 3D support is Love2D only.

=== API / SYNTAX ===
```tsx
import { Scene } from '@reactjit/3d';
```

Props:
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| style | Style | — | Style applied to the viewport box in the 2D layout (width, height, etc.) |
| backgroundColor | string | '#12121b' | Background color of the 3D viewport as a hex string |
| stars | boolean | — | Render procedural stars in the background |
| orbitControls | boolean | — | Enable Lua-side orbit controls — drag to rotate all meshes with zero latency |
| children | ReactNode | — | 3D child elements: Camera, Mesh, AmbientLight, DirectionalLight |

=== EXAMPLES ===
Example 1: Minimal scene with a box mesh
---
```tsx
import { Scene, Camera, Mesh } from '@reactjit/3d';

function App() {
  return (
    <Box style={{ width: '100%', height: '100%' }}>
      <Scene style={{ width: '100%', height: 400 }}>
        <Camera position={[0, -4, 2]} lookAt={[0, 0, 0]} />
        <Mesh geometry="box" color="#89b4fa" />
      </Scene>
    </Box>
  );
}
```

Platforms: love2d

Example 2: Scene with starfield background and orbit controls
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function SpaceScene() {
  return (
    <Scene
      style={{ width: 800, height: 600 }}
      backgroundColor="#0a0a14"
      stars
      orbitControls
    >
      <Camera position={[0, -5, 3]} lookAt={[0, 0, 0]} fov={Math.PI / 3} />
      <AmbientLight color="#1a1a2e" intensity={0.15} />
      <DirectionalLight direction={[1, -1, 1]} color="#ffffff" intensity={1.0} />
      <Mesh geometry="sphere" texture="planet" seed={42} />
    </Scene>
  );
}
```

Platforms: love2d

Example 3: Scene embedded in a 2D layout
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function Dashboard() {
  return (
    <Box style={{ width: '100%', height: '100%', flexDirection: 'row' }}>
      <Box style={{ width: 300, height: '100%', backgroundColor: '#1e1e2e' }}>
        <Text fontSize={16}>Controls</Text>
      </Box>
      <Scene style={{ flexGrow: 1, height: '100%' }} backgroundColor="#12121b">
        <Camera position={[3, -3, 3]} lookAt={[0, 0, 0]} />
        <AmbientLight intensity={0.2} />
        <DirectionalLight direction={[1, -2, 1]} />
        <Mesh geometry="box" color="#cba6f7" position={[0, 0, 0]} />
      </Scene>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Scene is a surface node — without explicit dimensions it uses the proportional fallback (1/4 of parent). For precise viewport control, set width and height explicitly or use flexGrow
- All direct children must be 3D elements (Camera, Mesh, AmbientLight, DirectionalLight); mixing 2D Box/Text children inside Scene is not supported
- Scene renders nothing useful on web targets — it shows a placeholder div with text
- Only one Scene can own a given region of the layout; nested Scenes are not supported
- orbitControls operates on the Lua side with zero JS round-trip latency; rotating via orbitControls does not fire React state updates

=== SEE ALSO ===
- Camera — defines the viewpoint inside a Scene
- Mesh — renderable 3D geometry or model inside a Scene
- AmbientLight — uniform base illumination inside a Scene
- DirectionalLight — sun-like directional light inside a Scene
