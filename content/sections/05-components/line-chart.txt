=== OVERVIEW ===
LineChart renders a line graph as a Lua `Chart2D` node. Data points are connected by lines, with optional dot markers and area fill. The chart is rendered entirely by `lua/chart.lua`.

When `interactive` is enabled, hovering near a point shows a tooltip. Press events are available via `onPointPress`.

LineChart is a surface element. If no explicit dimensions are set, it falls back to proportional surface sizing (1/4 of parent). Set `width`/`height` or use `flexGrow` for precise control.

=== API / SYNTAX ===
```tsx
import { LineChart } from '@reactjit/core';
```

Props:
| Prop         | Type                                                        | Default   | Description                              |
|--------------|-------------------------------------------------------------|-----------|------------------------------------------|
| data         | LineChartPoint[]                                            | required  | Array of data points                     |
| width        | number                                                      | undefined | Chart width in pixels                    |
| height       | number                                                      | undefined | Chart height in pixels                   |
| color        | Color                                                       | undefined | Line and dot color                       |
| showDots     | boolean                                                     | undefined | Show dot markers at each point           |
| showArea     | boolean                                                     | undefined | Fill the area below the line             |
| areaOpacity  | number                                                      | undefined | Opacity of the area fill (0-1)           |
| interactive  | boolean                                                     | undefined | Enable hover tooltips and press events   |
| onPointHover | (index: number \| null, point: LineChartPoint \| null) => void | undefined | Called on point hover/unhover         |
| onPointPress | (index: number, point: LineChartPoint) => void              | undefined | Called on point press                    |
| style        | Style                                                       | undefined | Full style object                        |

LineChartPoint:
| Field | Type   | Default   | Description                          |
|-------|--------|-----------|--------------------------------------|
| x     | string | undefined | Label for the x-axis position        |
| value | number | required  | Y-axis value                         |

=== EXAMPLES ===
Example 1: Basic line chart
---
```tsx
const colors = useThemeColors();

<LineChart
  data={[
    { x: 'Mon', value: 10 },
    { x: 'Tue', value: 25 },
    { x: 'Wed', value: 18 },
    { x: 'Thu', value: 32 },
    { x: 'Fri', value: 22 },
  ]}
  color={colors.primary}
  width={300}
  height={150}
  showDots
/>
```
---
Platforms: love2d

Example 2: Line chart with area fill
---
```tsx
const colors = useThemeColors();

<LineChart
  data={revenueData}
  color={colors.primary}
  width={400}
  height={200}
  showArea
  areaOpacity={0.3}
/>
```
---
Platforms: love2d

Example 3: Interactive with press handler
---
```tsx
const colors = useThemeColors();

<LineChart
  data={temperatureReadings}
  color="#ef4444"
  height={180}
  showDots
  interactive
  onPointPress={(index, point) => setSelectedDay(point.x)}
  style={{ flexGrow: 1 }}
/>
```
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Rendered by lua/chart.lua as a Chart2D surface. Lines are drawn with love.graphics.line, dots with love.graphics.circle. Area fill uses a polygon from the line points to the bottom of the chart area. All scaling and coordinate mapping is handled in Lua.
