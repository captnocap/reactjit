=== OVERVIEW ===
HorizontalBarChart renders bars horizontally, with optional labels on the left and values on the right. Useful for ranked lists, category comparisons, or any data where horizontal orientation reads better than vertical.

Built with Box and Text primitives (not a Chart2D surface). Each bar is scaled proportionally to the maximum value in the dataset. When `interactive` is enabled, hovering highlights the bar and shows a descriptive tooltip with label, value, and percentage.

=== API / SYNTAX ===
```tsx
import { HorizontalBarChart } from '@reactjit/core';
```

Props:
| Prop       | Type                                                              | Default   | Description                       |
|------------|-------------------------------------------------------------------|-----------|-----------------------------------|
| data       | HorizontalBarChartBar[]                                          | required  | Array of bars to render           |
| width      | number                                                           | 280       | Total chart width in pixels       |
| barHeight  | number                                                           | 20        | Height of each bar                |
| gap        | number                                                           | 6         | Vertical gap between bars         |
| showLabels | boolean                                                          | true      | Show labels to the left of bars   |
| showValues | boolean                                                          | false     | Show values to the right of bars  |
| labelWidth | number                                                           | 60        | Width reserved for labels         |
| color      | Color                                                            | '#3b82f6' | Default bar color                 |
| interactive| boolean                                                          | false     | Enable hover highlight and tooltips|
| onBarHover | (index: number \| null, bar: HorizontalBarChartBar \| null) => void | undefined | Called on hover/unhover       |
| onBarPress | (index: number, bar: HorizontalBarChartBar) => void             | undefined | Called on bar press                |
| style      | Style                                                            | undefined | Full style object                 |

HorizontalBarChartBar:
| Field | Type   | Default   | Description                |
|-------|--------|-----------|----------------------------|
| label | string | required  | Label text for the bar     |
| value | number | required  | Numeric value              |
| color | Color  | undefined | Per-bar color override     |

=== EXAMPLES ===
Example 1: Basic horizontal bar chart
---
```tsx
const colors = useThemeColors();

<HorizontalBarChart
  data={[
    { label: 'TypeScript', value: 45 },
    { label: 'Lua', value: 30 },
    { label: 'Python', value: 18 },
    { label: 'Rust', value: 12 },
  ]}
  color={colors.primary}
  showValues
/>
```
---
Platforms: love2d

Example 2: Per-bar colors with interaction
---
```tsx
const colors = useThemeColors();

<HorizontalBarChart
  data={[
    { label: 'Critical', value: 3, color: '#ef4444' },
    { label: 'Warning', value: 12, color: '#f59e0b' },
    { label: 'Info', value: 28, color: '#3b82f6' },
    { label: 'Debug', value: 45, color: '#6b7280' },
  ]}
  width={320}
  barHeight={16}
  showValues
  interactive
  onBarPress={(i, bar) => console.log(bar.label)}
/>
```
---
Platforms: love2d

Example 3: Compact without labels
---
```tsx
const colors = useThemeColors();

<HorizontalBarChart
  data={rankings}
  color={colors.primary}
  showLabels={false}
  barHeight={12}
  gap={4}
  width={200}
/>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- The bar area width is computed as `width - labelWidth - valueWidth`. If `showLabels` is true, the default `labelWidth` of 60 is subtracted. If `showValues` is true, 40px is subtracted for the value text.
- An empty `data` array renders an empty Box at the specified width.
- Not a Chart2D surface -- built from Box/Text primitives. Width defaults to 280px; height is determined by the number of bars, barHeight, and gap.
