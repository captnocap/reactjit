=== OVERVIEW ===
HorizontalBarChart renders a series of labeled bars growing left-to-right, each scaled relative to the maximum value in the dataset. Labels appear on the left, the bar fills proportionally, and an optional value readout follows on the right. When interactive mode is enabled, hovering a bar dims all others and reveals a tooltip showing the label, raw value, and percentage of the maximum.

=== API / SYNTAX ===
```tsx
import { HorizontalBarChart } from '@reactjit/core';
```

Data shape:
```tsx
interface HorizontalBarChartBar {
  label: string;   // Row label shown on the left
  value: number;   // Numeric value driving bar width
  color?: Color;   // Per-bar override; falls back to the chart-level `color` prop
}
```

Props:
| Prop         | Type                                                              | Default      | Description                                                         |
|--------------|-------------------------------------------------------------------|--------------|---------------------------------------------------------------------|
| data         | HorizontalBarChartBar[]                                           | required     | Array of bars to render                                             |
| width        | number                                                            | 280          | Total width of the chart in pixels                                  |
| barHeight    | number                                                            | 20           | Height of each bar row in pixels                                    |
| gap          | number                                                            | 6            | Vertical gap between bar rows in pixels                             |
| showLabels   | boolean                                                           | true         | Whether to render the label column on the left                      |
| showValues   | boolean                                                           | false        | Whether to render the numeric value after each bar                  |
| labelWidth   | number                                                            | 60           | Fixed pixel width reserved for the label column                     |
| color        | Color                                                             | '#3b82f6'    | Default bar color used when a bar has no per-item `color`           |
| style        | Style                                                             | undefined    | Additional styles applied to the outer container Box                |
| interactive  | boolean                                                           | false        | Enable hover highlighting and tooltips                              |
| onBarHover   | (index: number \| null, bar: HorizontalBarChartBar \| null) => void | undefined  | Called when the pointer enters or leaves a bar                      |
| onBarPress   | (index: number, bar: HorizontalBarChartBar) => void               | undefined    | Called when a bar is clicked (requires interactive: true)           |

=== EXAMPLES ===
Example 1: Basic labeled chart
---
```tsx
import { HorizontalBarChart } from '@reactjit/core';

const languages = [
  { label: 'TypeScript', value: 84 },
  { label: 'Lua',        value: 61 },
  { label: 'Python',     value: 43 },
  { label: 'Rust',       value: 28 },
];

export default function App() {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <HorizontalBarChart data={languages} width={320} />
    </Box>
  );
}
```

Example 3: Interactive with hover callback
---
```tsx
import { useState } from 'react';
import { HorizontalBarChart } from '@reactjit/core';

const scores = [
  { label: 'Alice', value: 92 },
  { label: 'Bob',   value: 74 },
  { label: 'Carol', value: 88 },
  { label: 'Dave',  value: 55 },
];

export default function App() {
  const [hovered, setHovered] = useState<string | null>(null);

  return (
    <Box style={{ width: '100%', height: '100%', padding: 20, gap: 12 }}>
      <HorizontalBarChart
        data={scores}
        width={300}
        interactive
        onBarHover={(_, bar) => setHovered(bar ? bar.label : null)}
      />
      {hovered && (
        <Text style={{ fontSize: 12, color: '#94a3b8' }}>
          {`Hovering: ${hovered}`}
        </Text>
      )}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The `width` prop controls the total chart width; bar fill width is computed from the remaining space after subtracting the label column (`labelWidth + 8`) and value column (40px) when those are shown.
- When `showLabels` is false, `labelWidth` has no effect but the space is still not reserved — set `showLabels={false}` to reclaim that space.
- Per-bar `color` takes priority over the chart-level `color` prop.
- `onBarPress` only fires when `interactive` is also `true`; setting `onBarPress` alone without `interactive` has no effect.
- Empty `data` arrays render a sized empty Box with no error — safe to pass live data that may start empty.
- Do not use Unicode symbols in any `<Text>` within this component or in labels passed as data — use Box-based geometry for visual indicators instead.
