=== OVERVIEW ===
ReactJIT provides a small set of primitive components that map directly to nodes in the Lua layout engine. There is no DOM. No `<div>`, no `<span>`, no CSS. Every component you see here becomes a Lua node with position, size, color, and text — rendered by Love2D's OpenGL painter.

All components import from `@reactjit/core`:

```tsx
import { Box, Text, Image, Pressable, ScrollView, Modal, FlatList, Video, VideoPlayer, Portal, PortalHost } from '@reactjit/core';
```

React's job is to declare the layout tree and diff it. Lua does the rendering, input handling, and runtime work. Components are thin — most are one-liners that pass props through to the reconciler.

Primitives:
- Box — universal container, the building block for all layouts
- Text — text rendering with font metrics
- Image — image display from file paths or URLs
- Pressable — interactive element with press/hover/focus states
- ScrollView — scrollable container with clipping
- Modal — full-screen overlay with backdrop
- FlatList — virtualized scrollable list
- Video / VideoPlayer — video playback via libmpv
- Portal / PortalHost — render children in a different part of the tree

Every component accepts a `style` prop using the shared `Style` type. Layout uses flexbox with three sizing tiers: explicit dimensions, content auto-sizing, and proportional surface fallback. See the Layout System section for details.
