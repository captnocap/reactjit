=== OVERVIEW ===
Sparkline renders a compact inline chart from an array of numbers. Each data point is represented as a thin vertical bar whose height is proportional to the value's position between the dataset's minimum and maximum. This creates a quick visual representation of trends or distributions within a small footprint.

Unlike BarChart, Sparkline is designed for inline use where labels and axes are unnecessary. It automatically computes bar widths to fill the given width and normalizes heights against the data range. When interactive mode is enabled, hovering a bar dims all others and shows a ChartTooltip with the raw value and index.

=== API / SYNTAX ===
```tsx
import { Sparkline } from '@reactjit/core';
```

Props:
| Prop          | Type                                                    | Default    | Description                                                       |
|---------------|---------------------------------------------------------|------------|-------------------------------------------------------------------|
| data          | number[]                                                | required   | Array of numeric data points                                      |
| width         | number                                                  | 80         | Total chart width in pixels                                       |
| height        | number                                                  | 24         | Total chart height in pixels                                      |
| color         | Color                                                   | '#3b82f6'  | Bar color                                                         |
| style         | Style                                                   | undefined  | Additional styles applied to the container Box                    |
| interactive   | boolean                                                 | false      | Enable hover highlighting and per-bar ChartTooltip                |
| onPointHover  | (index: number \| null, value: number \| null) => void  | undefined  | Called when the pointer enters or leaves a bar                    |
| onPointPress  | (index: number, value: number) => void                  | undefined  | Called when a bar is clicked (requires interactive: true)         |

=== EXAMPLES ===
Example 1: Basic sparkline
---
```tsx
import { Sparkline } from '@reactjit/core';

export default function App() {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <Sparkline data={[10, 25, 15, 30, 20, 35, 28]} />
    </Box>
  );
}
```

Example 3: Inline with text and interactive hover callback
---
```tsx
import { useState } from 'react';
import { Sparkline } from '@reactjit/core';

export default function App() {
  const [hovered, setHovered] = useState<number | null>(null);

  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <Box style={{ flexDirection: 'row', alignItems: 'center', gap: 8 }}>
        <Text style={{ fontSize: 11, color: '#e2e8f0' }}>CPU</Text>
        <Sparkline
          data={cpuData}
          width={60}
          height={16}
          color="#ef4444"
          interactive
          onPointHover={(_, value) => setHovered(value)}
        />
        <Text style={{ fontSize: 11, color: '#94a3b8' }}>
          {hovered !== null ? `${hovered}%` : '72%'}
        </Text>
      </Box>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The `data` prop is required. An empty array renders an empty placeholder Box at the specified width and height.
- Sparkline normalizes to the data's own min/max range, not 0-based. If all values are equal, the range defaults to 1, rendering all bars at minimum height (1px).
- The component has a fixed size based on `width` and `height`. It does not grow or shrink with its container. Pass `style` to override if needed.
- Bar widths are computed as `Math.max(1, (width - (data.length - 1) * barGap) / data.length)` where `barGap` is 1px. Very long data arrays in a narrow width may compress bars to 1px minimum.
- For real-time data, pass a new array reference to `data` on each update. Sparkline re-renders whenever the data reference changes.
- `onPointPress` only fires when `interactive` is also `true`.
- Hover events require pointer support — not available on all targets.
