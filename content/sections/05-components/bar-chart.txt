=== OVERVIEW ===
BarChart renders a vertical bar chart from an array of data points. Each bar's height is proportional to its value relative to the maximum value in the dataset. Bars can have individual colors or share a default color.

The component renders entirely with Box and Text primitives, so it works on all targets. An optional label row appears below the bars, and values can be shown above each bar. When barWidth is omitted, bars auto-size to fill the parent container evenly using flexGrow. When interactive mode is enabled, hovering a bar dims all other bars and reveals a ChartTooltip showing the label, raw value, and percentage of the maximum.

=== API / SYNTAX ===
```tsx
import { BarChart } from '@reactjit/core';
```

Data shape:
```tsx
interface BarChartBar {
  label: string;  // Bar label shown below the chart
  value: number;  // Numeric value driving bar height
  color?: Color;  // Per-bar color override; falls back to the chart-level `color` prop
}
```

Props:
| Prop        | Type                                                        | Default    | Description                                                      |
|-------------|-------------------------------------------------------------|------------|------------------------------------------------------------------|
| data        | BarChartBar[]                                               | required   | Array of bar data points                                         |
| height      | number                                                      | 120        | Chart area height in pixels                                      |
| barWidth    | number                                                      | undefined  | Fixed width per bar in pixels; omit for auto-sizing via flexGrow |
| gap         | number                                                      | 8          | Horizontal gap between bars in pixels                            |
| showLabels  | boolean                                                     | true       | Show label text below each bar                                   |
| showValues  | boolean                                                     | false      | Show numeric value above each bar                                |
| color       | Color                                                       | '#3b82f6'  | Default bar color; overridden per bar via BarChartBar.color      |
| style       | Style                                                       | undefined  | Additional styles applied to the outer container Box             |
| interactive | boolean                                                     | false      | Enable hover highlighting and per-bar ChartTooltip               |
| onBarHover  | (index: number \| null, bar: BarChartBar \| null) => void   | undefined  | Called when the pointer enters or leaves a bar                   |
| onBarPress  | (index: number, bar: BarChartBar) => void                   | undefined  | Called when a bar is clicked (requires interactive: true)        |

=== EXAMPLES ===
Example 1: Basic bar chart with auto-sizing
---
```tsx
import { BarChart } from '@reactjit/core';

export default function App() {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <BarChart
        data={[
          { label: 'Q1', value: 40 },
          { label: 'Q2', value: 65 },
          { label: 'Q3', value: 52 },
          { label: 'Q4', value: 78 },
        ]}
      />
    </Box>
  );
}
```

Example 3: Interactive chart with hover callback
---
```tsx
import { useState } from 'react';
import { BarChart } from '@reactjit/core';

const weekly = [
  { label: 'Mon', value: 32 },
  { label: 'Tue', value: 58 },
  { label: 'Wed', value: 47 },
  { label: 'Thu', value: 71 },
  { label: 'Fri', value: 55 },
];

export default function App() {
  const [active, setActive] = useState<string | null>(null);

  return (
    <Box style={{ width: '100%', height: '100%', padding: 20, gap: 12 }}>
      <BarChart
        data={weekly}
        height={100}
        interactive
        onBarHover={(_, bar) => setActive(bar ? bar.label : null)}
      />
      {active && (
        <Text style={{ fontSize: 12, color: '#94a3b8' }}>
          {`Hovering: ${active}`}
        </Text>
      )}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The `data` prop is required. An empty array renders an empty placeholder Box at the specified height.
- Bar heights are relative to the maximum value in the data. If all values are equal, all bars render at full height.
- If `maxValue` is 0 (all values are 0), bars are normalized against 1, producing minimum-height (1px) bars.
- Per-bar `color` overrides the chart-level `color` prop for that specific bar only.
- BarChart does not handle negative values. All values should be non-negative.
- When `showValues` is true, 14px of vertical space is reserved above bars for value labels — factor this into the `height` prop.
- When `barWidth` is omitted, bars use `flexGrow: 1` and fill the parent container. The parent must have an explicit width for this to work.
- When `barWidth` is set, the chart computes its own total width as `data.length * barWidth + (data.length - 1) * gap`.
- `onBarPress` only fires when `interactive` is also `true`.
- Hover events require pointer support — not available on all targets. Check platform capabilities before enabling `interactive`.
