=== OVERVIEW ===
PieChart renders a circular chart as a Lua `Chart2D` node. Each segment represents a labeled value with a color. The segment angle is proportional to its share of the total.

Set `innerRadius` to a value greater than 0 to create a donut chart. When `interactive` is enabled, a table-layout tooltip shows each segment's label, value, and percentage.

PieChart is a surface element. The `size` prop sets both width and height to create a square bounding box. If no `size` is set, proportional surface fallback applies.

=== API / SYNTAX ===
```tsx
import { PieChart } from '@reactjit/core';
```

Props:
| Prop        | Type              | Default   | Description                              |
|-------------|-------------------|-----------|------------------------------------------|
| data        | PieChartSegment[] | required  | Array of segments                        |
| size        | number            | undefined | Diameter in pixels (sets width + height) |
| innerRadius | number            | undefined | Inner radius for donut mode (0 = full)   |
| interactive | boolean           | undefined | Enable hover tooltip                     |
| style       | Style             | undefined | Full style object                        |

PieChartSegment:
| Field | Type   | Default  | Description               |
|-------|--------|----------|---------------------------|
| label | string | required | Segment label             |
| value | number | required | Numeric value             |
| color | Color  | required | Segment color             |

=== EXAMPLES ===
Example 1: Basic pie chart
---
```tsx
<PieChart
  data={[
    { label: 'Desktop', value: 55, color: '#3b82f6' },
    { label: 'Mobile', value: 35, color: '#22c55e' },
    { label: 'Tablet', value: 10, color: '#f59e0b' },
  ]}
  size={120}
/>
```
---
Platforms: love2d

Example 2: Donut chart with interaction
---
```tsx
<PieChart
  data={[
    { label: 'Used', value: 68, color: '#ef4444' },
    { label: 'Free', value: 32, color: '#1e293b' },
  ]}
  size={100}
  innerRadius={30}
  interactive
/>
```
---
Platforms: love2d

Example 3: Dashboard usage
---
```tsx
const colors = useThemeColors();

<Box style={{ flexDirection: 'row', alignItems: 'center', gap: 16 }}>

  />
  <Box>
    <Text style={{ color: colors.text, fontSize: 14 }}>Budget</Text>
    <Text style={{ color: colors.muted, fontSize: 12 }}>{`${total} total`}</Text>
  </Box>
</Box>
```
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Rendered by lua/chart.lua as a Chart2D surface. Segments are drawn using love.graphics.arc. The tooltip uses table layout to align label/value/percentage columns. Inner radius creates a donut by drawing a second arc in the background color.
