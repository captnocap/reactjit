=== OVERVIEW ===
PieChart rasterizes a pie chart into a 2x2-pixel cell grid using polar coordinate math, then renders each run of same-colored cells as a single Box. This approach requires no canvas or SVG — it works on any target that supports Box with backgroundColor. Setting innerRadius greater than zero hollows the center to produce a donut chart. When interactive mode is enabled, hovering the chart reveals a legend tooltip listing every segment's label, raw value, and percentage.

=== API / SYNTAX ===
```tsx
import { PieChart } from '@reactjit/core';
```

Data shape:
```tsx
interface PieChartSegment {
  label: string;  // Segment name shown in the hover tooltip legend
  value: number;  // Numeric value; proportions are computed from the total
  color: Color;   // Fill color for this segment; required per segment
}
```

Props:
| Prop          | Type                   | Default    | Description                                                                     |
|---------------|------------------------|------------|---------------------------------------------------------------------------------|
| data          | PieChartSegment[]      | required   | Array of segments; values are proportional, not percentages                     |
| size          | number                 | 120        | Width and height of the chart square in pixels                                  |
| innerRadius   | number                 | 0          | Hollow center radius in pixels; 0 = solid pie, >0 = donut ring                 |
| interactive   | boolean                | false      | Enable hover to show the full segment legend tooltip                            |
| style         | Style                  | undefined  | Additional styles applied to the outer container Box                            |

=== EXAMPLES ===
Example 1: Basic pie chart
---
```tsx
import { PieChart } from '@reactjit/core';

const marketShare = [
  { label: 'Chrome',  value: 65, color: '#3b82f6' },
  { label: 'Safari',  value: 19, color: '#22c55e' },
  { label: 'Firefox', value: 9,  color: '#f59e0b' },
  { label: 'Other',   value: 7,  color: '#64748b' },
];

export default function App() {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <PieChart data={marketShare} size={160} />
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The chart is rendered as a pixel grid at 2px cell resolution. `size` must be even for the grid to divide cleanly; odd values are floored internally via `Math.floor(size / cellSize)`.
- All segment `color` values are required — there is no fallback color. Omitting `color` from a segment will render that segment transparently.
- `innerRadius` is specified in the same pixel units as `size`, not as a fraction. A `size={120}` chart with `innerRadius={60}` produces a ring exactly half the radius.
- The tooltip appears to the right of the chart (`left: '100%'`). Ensure the chart container has enough horizontal space for the tooltip or it will be clipped.
- Rasterization cost scales with `size`. Very large values (>300) will produce a high Box count; prefer sizes in the 80–200px range.
- Segments with a value of 0 are included in the legend but produce no visible arc — this is safe.
- Do not use Unicode symbols in segment `label` strings.
