=== OVERVIEW ===
StackedBarChart renders a column of vertical bars where each bar is subdivided into colored segments representing multiple data series. Series heights are proportional to the maximum column total across all bars, so every bar uses the full chart height at most. Optional column labels appear below the bars. When interactive mode is enabled, hovering a column dims all others and shows a tooltip with the column total and per-series breakdown.

=== API / SYNTAX ===
```tsx
import { StackedBarChart } from '@reactjit/core';
```

Data shape:
```tsx
interface StackedBarChartSeries {
  label: string;   // Series name shown in the hover tooltip
  color: Color;    // Color for this series' segment
  data: number[];  // One value per bar column; must be the same length across all series
}
```

Props:
| Prop        | Type                      | Default   | Description                                                                            |
|-------------|---------------------------|-----------|----------------------------------------------------------------------------------------|
| series      | StackedBarChartSeries[]   | required  | Array of data series; each series provides one segment per bar column                 |
| labels      | string[]                  | undefined | Optional column labels rendered below the bars (one per bar column)                   |
| height      | number                    | 120       | Chart height in pixels; the tallest bar fills this exactly                             |
| barWidth    | number                    | undefined | Fixed pixel width per bar; when omitted, bars flex to fill the container              |
| gap         | number                    | 8         | Horizontal gap between bars in pixels                                                  |
| showLabels  | boolean                   | true      | Whether to render the column label row beneath the bars                                |
| style       | Style                     | undefined | Additional styles applied to the outer container Box                                   |
| interactive | boolean                   | false     | Enable hover highlighting and per-column tooltips                                      |

=== EXAMPLES ===
Example 1: Quarterly revenue by product line
---
```tsx
import { StackedBarChart } from '@reactjit/core';

const series = [
  { label: 'Hardware', color: '#3b82f6', data: [42, 55, 38, 61] },
  { label: 'Software', color: '#22c55e', data: [28, 30, 35, 40] },
  { label: 'Services', color: '#f59e0b', data: [15, 18, 22, 19] },
];

export default function App() {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <StackedBarChart
        series={series}
        labels={['Q1', 'Q2', 'Q3', 'Q4']}
        height={140}
        barWidth={32}
        gap={10}
      />
    </Box>
  );
}
```

=== CRITICAL RULES ===
- All series must have `data` arrays of equal length. The number of bars is taken from `series[0].data.length`; extra values in shorter series are ignored and missing values in shorter series default to 0.
- Segments are rendered bottom-to-top by reversing the series array internally. The first series in your array will appear at the top of each bar visually.
- When `barWidth` is omitted, bars use `flexGrow: 1` and fill the parent container width. The parent container must have an explicit width for this to produce visible output.
- When `barWidth` is set, the chart computes its own total width as `numBars * barWidth + (numBars - 1) * gap`. You do not need to set `width` on the chart itself.
- `showLabels` only renders text if `labels` is also provided. Providing `labels` without `showLabels` does render them; set `showLabels={false}` to suppress the label row entirely.
- Segments with a computed height of 0 are omitted from the render tree — this is safe and avoids invisible 0-height boxes.
- Do not use Unicode symbols in any `<Text>` within this component or in labels passed as data.
