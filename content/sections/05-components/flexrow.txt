=== OVERVIEW ===
FlexRow is a convenience wrapper around Box that sets `flexDirection: 'row'`. It provides shorthand props for the most common horizontal layout needs: gap between children, main-axis justification, cross-axis alignment, and wrapping.

Use FlexRow when you want a horizontal container without writing `style={{ flexDirection: 'row' }}` on a Box. For simple cases, the `Row` primitive from Box also works. FlexRow is best when you need the explicit `gap`, `justify`, `align`, and `wrap` props as a self-documenting API.

=== API / SYNTAX ===
```tsx
import { FlexRow } from '@reactjit/core';
```

Props:
| Prop     | Type                          | Default   | Description                              |
|----------|-------------------------------|-----------|------------------------------------------|
| gap      | number                        | undefined | Space between children in pixels         |
| justify  | Style['justifyContent']       | undefined | Main-axis alignment (start, center, end, space-between, space-around, space-evenly) |
| align    | Style['alignItems']           | undefined | Cross-axis alignment (start, center, end, stretch) |
| wrap     | boolean                       | undefined | Enable flex wrapping                     |
| style    | Style                         | undefined | Additional style overrides               |
| children | ReactNode                     | required  | Child elements                           |

=== EXAMPLES ===
Example 1: Basic horizontal layout
---
```tsx
const colors = useThemeColors();

<FlexRow gap={12}>
  <Box style={{ width: 48, height: 48, backgroundColor: colors.primary, borderRadius: 8 }} />
  <Box style={{ width: 48, height: 48, backgroundColor: colors.surface, borderRadius: 8 }} />
  <Box style={{ width: 48, height: 48, backgroundColor: colors.surface, borderRadius: 8 }} />
</FlexRow>
```
---
Platforms: love2d

Example 2: Centering children vertically
---
```tsx
const colors = useThemeColors();

<FlexRow gap={8} align="center">
  <Box style={{ width: 40, height: 40, backgroundColor: colors.primary, borderRadius: 20 }} />
  <Text style={{ color: colors.text, fontSize: 16 }}>Username</Text>
</FlexRow>
```
---
Platforms: love2d

Example 3: Spacing children apart
---
```tsx
const colors = useThemeColors();

<FlexRow justify="space-between" align="center" style={{ width: '100%', padding: 12 }}>
  <Text style={{ color: colors.text, fontSize: 14 }}>Left</Text>
  <Text style={{ color: colors.muted, fontSize: 14 }}>Right</Text>
</FlexRow>
```
---
Platforms: love2d

Example 4: Wrapping layout
---
```tsx
const colors = useThemeColors();
const tags = ['React', 'Lua', 'Love2D', 'OpenGL', 'QuickJS'];

<FlexRow gap={8} wrap>
  {tags.map(tag => (
    <Box key={tag} style={{ paddingLeft: 10, paddingRight: 10, paddingTop: 4, paddingBottom: 4, backgroundColor: colors.surface, borderRadius: 12 }}>
      <Text style={{ color: colors.text, fontSize: 11 }}>{tag}</Text>
    </Box>
  ))}
</FlexRow>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- FlexRow sets `flexDirection: 'row'` on the underlying Box. Do not also set `flexDirection` in the `style` prop.
- Rows auto-wrap by default in the layout engine unless any child has `flexGrow > 0`. Use the `wrap` prop to force wrapping when children have flexGrow.
- The `style` prop merges after the shorthand props. If you set both `gap={12}` and `style={{ gap: 8 }}`, the style prop wins.
