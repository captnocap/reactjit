=== OVERVIEW ===
FlexRow is a convenience component that wraps Box with flexDirection='row' pre-configured. It provides simplified props for common horizontal layout tasks: gap spacing, horizontal justification, vertical alignment, and wrapping. Use FlexRow when you need to arrange children horizontally without manually setting flexDirection on every Box.

=== API / SYNTAX ===
import { FlexRow } from '@reactjit/core';

Props:
| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| gap | number | No | undefined | Spacing between children (pixels) |
| justify | 'flex-start' \| 'flex-end' \| 'center' \| 'space-between' \| 'space-around' \| 'space-evenly' | No | undefined | Horizontal distribution (justifyContent) |
| align | 'flex-start' \| 'flex-end' \| 'center' \| 'stretch' \| 'baseline' | No | undefined | Vertical alignment (alignItems) |
| wrap | boolean | No | false | Allow children to wrap to next line (flexWrap: 'wrap') |
| style | Style | No | {} | Additional style properties (merged after defaults) |
| children | React.ReactNode | Yes | - | Child elements to lay out horizontally |

=== EXAMPLES ===
Example 1: Basic horizontal layout with spacing
---
import { FlexRow, Text } from '@reactjit/core';

<FlexRow gap={16}>
  <Text fontSize={16}>First</Text>
  <Text fontSize={16}>Second</Text>
  <Text fontSize={16}>Third</Text>
</FlexRow>
---
Platforms: All

Example 2: Centered content with justify and align
---
import { FlexRow, Text, Box } from '@reactjit/core';

<FlexRow
  justify="center"
  align="center"
  style={{ width: '100%', height: 100, backgroundColor: '#f0f0f0' }}
>
  <Box style={{ width: 50, height: 50, backgroundColor: 'blue' }} />
  <Text fontSize={16}>Centered</Text>
</FlexRow>
---
Platforms: All

Example 3: Space-between distribution (toolbar pattern)
---
import { FlexRow, Text } from '@reactjit/core';

<FlexRow justify="space-between" style={{ width: '100%', padding: 8 }}>
  <Text fontSize={14}>Left</Text>
  <Text fontSize={14}>Center</Text>
  <Text fontSize={14}>Right</Text>
</FlexRow>
---
Platforms: All

Example 4: Wrapping horizontal grid
---
import { FlexRow, Box } from '@reactjit/core';

<FlexRow wrap gap={8} style={{ width: 300 }}>
  {Array.from({ length: 10 }).map((_, i) => (
    <Box key={i} style={{ width: 60, height: 60, backgroundColor: 'green' }} />
  ))}
</FlexRow>
---
Platforms: All

Example 5: Combining with custom styles
---
import { FlexRow, Text } from '@reactjit/core';

<FlexRow
  gap={12}
  align="baseline"
  style={{
    padding: 16,
    backgroundColor: '#333',
    borderRadius: 8
  }}
>
  <Text fontSize={24} style={{ color: 'white' }}>Title</Text>
  <Text fontSize={14} style={{ color: '#aaa' }}>Subtitle</Text>
</FlexRow>
---
Platforms: All

=== PLATFORM NOTES ===
No platform-specific differences. FlexRow behaves identically on all targets (Love2D, web, terminal, ComputerCraft, Neovim, Hammerspoon, AwesomeWM). The underlying flex layout engine is consistent across all renderers.

=== CRITICAL RULES ===
- FlexRow requires explicit width for justify to work properly. Row containers have no intrinsic width (only Text nodes do). Without a width, justifyContent='center' or 'space-between' will have no effect. Always set width: '100%' or a fixed width when using justify.
- Do not use flexGrow on children without setting a width on FlexRow. Percentage-based children need explicit parent dimensions.
- Gap spacing is applied between children, not around the edges. Use padding in style prop for outer spacing.
- The style prop is merged last, so you can override gap, justify, and align by passing conflicting properties in style (though this defeats the purpose of the convenience props).
- FlexRow auto-sizes to fit content if no width/height is specified. Height is determined by the tallest child. Width is sum of children widths plus gaps.
- Use wrap={true} when you want a responsive horizontal grid that breaks to multiple rows. Requires explicit width on FlexRow.
- align='stretch' only works if children do not have explicit heights. Children will stretch to match the tallest sibling.
