title: AmbientLight
description: Uniform ambient illumination that prevents shadowed areas from going fully black

=== OVERVIEW ===
AmbientLight provides a base level of non-directional illumination for all meshes in a Scene. It ensures shadowed surfaces receive some light instead of rendering as pure black. AmbientLight has no position or direction — it contributes equally from all angles. Only one AmbientLight per Scene is supported. It renders nothing on web targets.

=== API / SYNTAX ===
```tsx
import { AmbientLight } from '@reactjit/3d';
```

Props:
| Prop | Type | Default | Description |
|------|------|---------|-------------|
| color | string | '#1a1a2e' | Ambient light color as a hex string |
| intensity | number | 0.15 | Intensity multiplier applied to the color |

=== EXAMPLES ===
Example 1: Default ambient light for a dark space scene
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function SpaceScene() {
  return (
    <Scene style={{ width: 800, height: 600 }} backgroundColor="#0a0a14" stars>
      <Camera position={[0, -5, 2]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#1a1a2e" intensity={0.15} />
      <DirectionalLight direction={[1, -1, 1]} color="#ffffff" intensity={1.0} />
      <Mesh geometry="sphere" texture="planet" seed={4} />
    </Scene>
  );
}
```
---
Platforms: love2d

Example 2: Warm tinted ambient light for an interior scene
---
```tsx
import { Scene, Camera, Mesh, AmbientLight, DirectionalLight } from '@reactjit/3d';

function WarmScene() {
  return (
    <Scene style={{ width: 800, height: 600 }} backgroundColor="#1a0e00">
      <Camera position={[3, -3, 2]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#3d1f00" intensity={0.4} />
      <DirectionalLight direction={[0, -1, 1]} color="#ffcc88" intensity={0.8} />
      <Mesh geometry="box" color="#c0a060" edgeColor="#2a1500" edgeWidth={0.02} />
    </Scene>
  );
}
```

Example 3: High ambient for a flat, evenly lit scene
---
```tsx
import { Scene, Camera, Mesh, AmbientLight } from '@reactjit/3d';

function FlatLitScene() {
  return (
    <Scene style={{ width: 800, height: 600 }}>
      <Camera position={[0, -4, 2]} lookAt={[0, 0, 0]} />
      <AmbientLight color="#ffffff" intensity={0.9} />
      <Mesh geometry="sphere" color="#89b4fa" />
    </Scene>
  );
}
```

=== CRITICAL RULES ===
- AmbientLight renders nothing on web targets and returns null — it is Love2D only
- Only one AmbientLight per Scene is supported — placing multiple will produce undefined behavior
- AmbientLight must be a direct child of Scene, not nested inside other elements
- intensity is a multiplier on the color — setting intensity to 0 effectively disables ambient lighting
- AmbientLight alone without a DirectionalLight will cause all surfaces to be shaded with only the flat ambient color, losing all depth cues

=== SEE ALSO ===
- Scene — the 3D viewport that AmbientLight must be placed inside
- DirectionalLight — directional sun-like light that works alongside AmbientLight for full shading
- Mesh — the geometry that receives ambient illumination
- Camera — the viewpoint used to render the scene
