=== OVERVIEW ===
Card is a styled container that provides a rounded, bordered surface with optional header (title and subtitle) and padded body. It renders as a Box with background color, border, border radius, and overflow hidden.

When `title` or `subtitle` is provided, Card renders a header section separated from the body by a bottom border. The body always receives 12px padding by default.

Use Card to group related content into visually distinct sections. Both `headerStyle` and `bodyStyle` allow overriding the default styling of those regions.

=== API / SYNTAX ===
```tsx
import { Card } from '@reactjit/core';
```

Props:
| Prop        | Type      | Default   | Description                              |
|-------------|-----------|-----------|------------------------------------------|
| title       | string    | undefined | Header title text                        |
| subtitle    | string    | undefined | Header subtitle text (smaller, muted)    |
| children    | ReactNode | required  | Body content                             |
| style       | Style     | undefined | Outer container style overrides          |
| headerStyle | Style     | undefined | Header section style overrides           |
| bodyStyle   | Style     | undefined | Body section style overrides             |

=== EXAMPLES ===
Example 1: Card with title
---
```tsx
const colors = useThemeColors();

<Card title="System Status">
  <Text style={{ color: colors.text, fontSize: 12 }}>All services operational</Text>
</Card>
```
---
Platforms: love2d

Example 2: Card with title and subtitle
---
```tsx
const colors = useThemeColors();

<Card title="CPU Usage" subtitle="Last 60 seconds">
  <Text style={{ color: colors.text, fontSize: 24, fontWeight: 'bold' }}>42%</Text>
</Card>
```
---
Platforms: love2d

Example 3: Card without header
---
```tsx
const colors = useThemeColors();

<Card>
  <FlexRow gap={12} align="center">
    <Box style={{ width: 40, height: 40, backgroundColor: colors.primary, borderRadius: 20 }} />
    <FlexColumn gap={2}>
      <Text style={{ color: colors.text, fontSize: 14 }}>User Name</Text>
      <Text style={{ color: colors.muted, fontSize: 11 }}>Online</Text>
    </FlexColumn>
  </FlexRow>
</Card>
```
---
Platforms: love2d

Example 4: Custom styled card
---
```tsx
const colors = useThemeColors();

<Card
  title="Alerts"
  style={{ borderColor: colors.primary }}
  headerStyle={{ padding: 16 }}
  bodyStyle={{ padding: 16, gap: 8 }}
>
  <Text style={{ color: colors.text, fontSize: 12 }}>No new alerts</Text>
</Card>
```
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Card uses hardcoded dark theme colors as defaults (bg: #1e293b, border: #334155). To integrate with the theme system, pass colors through the `style` prop: `style={{ backgroundColor: colors.bgElevated, borderColor: colors.border }}`.
