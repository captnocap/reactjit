=== OVERVIEW ===
The Card component is a pre-styled container that provides a visual boundary around related content. It automatically includes dark theme styling (slate colors), rounded corners, borders, and proper padding. Cards can optionally display a header section with a title and subtitle, separated from the body content by a horizontal divider. The component auto-sizes to its content unless explicit dimensions are provided via the `style` prop.

=== API / SYNTAX ===
```tsx
import { Card } from '@reactjit/core';

<Card
  title="Optional Title"
  subtitle="Optional Subtitle"
  style={{ width: 300 }}
  headerStyle={{ backgroundColor: '#2d3748' }}
  bodyStyle={{ gap: 8 }}
>
  {children}
</Card>
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | (required) | Content rendered in the card body section |
| `title` | `string` | `undefined` | Optional title displayed in the header (bold, 14px, light gray) |
| `subtitle` | `string` | `undefined` | Optional subtitle below title (11px, muted gray) |
| `style` | `Style` | `{}` | Overrides for the outer card container (dimensions, margins, etc.) |
| `headerStyle` | `Style` | `{}` | Overrides for the header section (padding, background, borders) |
| `bodyStyle` | `Style` | `{}` | Overrides for the body section (padding, gap, alignment) |

**Default Styling:**
- Background: `#1e293b` (slate-800)
- Border: 1px `#334155` (slate-700)
- Border radius: 8px
- Header padding: 12px
- Body padding: 12px
- Header border-bottom: 1px when title or subtitle present

=== CRITICAL RULES ===
- All `<Text>` children inside Card must have explicit `fontSize` (linter enforces this)
- Card auto-sizes to content unless explicit `width`/`height` provided in `style`
- Header section only renders if `title` or `subtitle` is provided
- Do not use Unicode symbols in Text children (see layout rules)
- For full-viewport cards, use `style={{ width: '100%', height: '100%' }}`
- Use `bodyStyle={{ gap: X }}` instead of manual spacing between children
- `headerStyle` and `bodyStyle` merge with defaults (use to override padding, colors, etc.)
- Card uses `overflow: 'hidden'` to clip content at rounded corners
