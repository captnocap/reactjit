=== OVERVIEW ===
Select renders a dropdown picker for choosing from a list of options. It emits a `Select` host element -- Lua handles the dropdown popup rendering, hover tracking, keyboard navigation, opening/closing, and selection highlighting.

React is a declarative wrapper that passes the options array and receives selection callbacks. The dropdown menu auto-positions within the window bounds.

The options are serialized as JSON to cross the bridge. Each option has a `label` (display text) and a `value` (selection identifier).

=== API / SYNTAX ===
```tsx
import { Select } from '@reactjit/core';
```

Props:
| Prop          | Type                        | Default      | Description                        |
|---------------|-----------------------------|--------------|------------------------------------|
| value         | string                      | undefined    | Controlled selected value          |
| defaultValue  | string                      | undefined    | Initial value for uncontrolled mode|
| onValueChange | (value: string) => void     | undefined    | Fires when selection changes       |
| options       | SelectOption[]              | required     | Array of selectable options        |
| placeholder   | string                      | 'Select...'  | Placeholder text when no selection |
| disabled      | boolean                     | false        | Disable interaction                |
| style         | Style                       | undefined    | Container style                    |
| color         | Color                       | '#3b82f6'    | Accent color                       |

SelectOption:
| Field | Type   | Description                   |
|-------|--------|-------------------------------|
| label | string | Display text shown in dropdown|
| value | string | Value returned on selection   |

=== EXAMPLES ===
Example 1: Basic dropdown
---
```tsx
const colors = useThemeColors();
const [country, setCountry] = useState('');

<Select
  value={country}
  onValueChange={setCountry}
  placeholder="Choose country"
  color={colors.primary}
  options={[
    { label: 'United States', value: 'us' },
    { label: 'Canada', value: 'ca' },
    { label: 'United Kingdom', value: 'uk' },
  ]}
/>
```

=== PLATFORM NOTES ===
Love2D:
  The dropdown popup is rendered by Lua as an overlay. It supports keyboard navigation (arrow keys to move, Enter to select, Escape to dismiss) and mouse hover highlighting. The popup auto-positions to stay within window bounds -- it opens downward by default but flips upward if there is not enough space below.

  Options are serialized as JSON via `JSON.stringify()` on the React side and parsed by Lua. Keep option arrays reasonably sized (hundreds of items are fine; thousands may cause a parsing delay on first open).

=== CRITICAL RULES ===
- The `options` prop is required. An empty array renders a disabled-looking Select with just the placeholder.
- Each option's `value` must be unique within the array. Duplicate values cause undefined selection behavior.
- Select has a default `minWidth: 160` and `height: 40`. Override via the `style` prop if needed.
