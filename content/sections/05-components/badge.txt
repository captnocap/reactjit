=== OVERVIEW ===
Badge renders a small pill-shaped label with colored background and text. It supports five semantic variants: default, success, warning, error, and info. Each variant provides a pre-defined background and text color pair.

Badge uses `alignSelf: 'start'` by default so it shrink-wraps to its label text rather than stretching to fill the parent width.

=== API / SYNTAX ===
```tsx
import { Badge } from '@reactjit/core';
```

Props:
| Prop    | Type         | Default   | Description                                     |
|---------|--------------|-----------|-------------------------------------------------|
| label   | string       | required  | Text content of the badge                       |
| variant | BadgeVariant | 'default' | Color variant: 'default', 'success', 'warning', 'error', 'info' |
| style   | Style        | undefined | Style overrides for the outer container         |

BadgeVariant colors:
| Variant | Background | Text    |
|---------|------------|---------|
| default | #334155    | #e2e8f0 |
| success | #166534    | #bbf7d0 |
| warning | #854d0e    | #fef08a |
| error   | #991b1b    | #fecaca |
| info    | #1e40af    | #bfdbfe |

---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Badge uses hardcoded variant colors. To customize colors beyond the five variants, use the `style` prop to override `backgroundColor` and wrap a `Text` with your own color.
