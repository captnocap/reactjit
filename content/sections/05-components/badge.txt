=== OVERVIEW ===
The Badge component renders a compact, pill-shaped label with semantic color variants. It automatically sizes to fit the label text and uses `alignSelf: 'start'` to prevent unwanted stretching in flex layouts. Badges are ideal for status indicators, category tags, notification counts, or any small textual label that needs visual emphasis. The component provides five built-in variants with carefully chosen background and text color pairings optimized for readability.

=== API / SYNTAX ===
```tsx
import { Badge } from '@reactjit/core';

<Badge
  label="Active"
  variant="success"
  style={{ marginLeft: 8 }}
/>
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | (required) | Text content displayed inside the badge |
| `variant` | `BadgeVariant` | `'default'` | Color scheme: `'default'`, `'success'`, `'warning'`, `'error'`, `'info'` |
| `style` | `Style` | `{}` | Overrides for the badge container (margins, explicit dimensions, etc.) |

**Variant Color Schemes:**

| Variant | Background | Text | Use Case |
|---------|-----------|------|----------|
| `default` | `#334155` (slate-700) | `#e2e8f0` (slate-200) | Neutral labels, defaults |
| `success` | `#166534` (green-800) | `#bbf7d0` (green-200) | Active, online, completed, success |
| `warning` | `#854d0e` (yellow-800) | `#fef08a` (yellow-200) | Caution, pending, warnings |
| `error` | `#991b1b` (red-800) | `#fecaca` (red-200) | Error, failed, critical, offline |
| `info` | `#1e40af` (blue-800) | `#bfdbfe` (blue-200) | Information, tips, metadata |

**Default Styling:**
- Border radius: 10px (pill shape)
- Padding: 2px vertical, 8px horizontal
- Font size: 11px
- Font weight: bold
- Alignment: `alignSelf: 'start'` (no stretch)

Example 8: Dynamic Variant Based on State
---
```tsx
function StatusBadge({ status }: { status: 'online' | 'offline' | 'away' }) {
  const variantMap = {
    online: 'success',
    offline: 'error',
    away: 'warning',
  } as const;

  return <Badge label={status} variant={variantMap[status]} />;
}

// Usage:
<StatusBadge status="online" />
```

=== CRITICAL RULES ===
- Do not use Unicode symbols in the `label` prop (see layout rules)
- Badge auto-sizes to label text width (no explicit width needed)
- Uses `alignSelf: 'start'` by default to prevent stretching in flex containers
- For horizontal alignment with text, wrap in a Box with `flexDirection: 'row'` and `alignItems: 'center'`
- The internal Text component uses `fontSize: 11` and `fontWeight: 'bold'` (not customizable without forking)
- Variant colors are hardcoded; for custom colors, fork Badge or use a styled Box + Text
- Badge has no onClick handler; wrap in Pressable for interactivity
