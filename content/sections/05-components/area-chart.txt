=== OVERVIEW ===
AreaChart is a convenience wrapper around LineChart with `showArea` enabled, `showDots` disabled, and `areaOpacity` set to 0.4. It accepts all the same props as LineChart -- any prop you pass overrides the defaults.

Rendered by `lua/chart.lua` as a Chart2D surface, identical to LineChart with area fill.

=== API / SYNTAX ===
```tsx
import { AreaChart } from '@reactjit/core';
```

Props: Same as LineChart. The following defaults are overridden:
| Prop        | AreaChart Default | LineChart Default |
|-------------|-------------------|-------------------|
| showArea    | true              | undefined         |
| showDots    | false             | undefined         |
| areaOpacity | 0.4               | undefined         |

See LineChart for the full props table.

=== EXAMPLES ===
Example 1: Basic area chart
---
```tsx
const colors = useThemeColors();

<AreaChart
  data={[
    { x: 'Jan', value: 100 },
    { x: 'Feb', value: 120 },
    { x: 'Mar', value: 90 },
    { x: 'Apr', value: 140 },
    { x: 'May', value: 160 },
  ]}
  color={colors.primary}
  width={300}
  height={150}
/>
```
---
Platforms: love2d

Example 2: With dots re-enabled
---
```tsx
const colors = useThemeColors();

<AreaChart
  data={memoryUsage}
  color="#22c55e"
  width={400}
  height={180}
  showDots
  areaOpacity={0.2}
  interactive
/>
```
---
Platforms: love2d

Example 3: Filling available space
---
```tsx
const colors = useThemeColors();

<Box style={{ width: '100%', height: '100%' }}>
  <Text style={{ color: colors.text, fontSize: 16 }}>Network Traffic</Text>
  <AreaChart
    data={trafficData}
    color={colors.primary}
    style={{ flexGrow: 1 }}
  />
</Box>
```

Platforms: love2d
