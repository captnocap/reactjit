=== OVERVIEW ===
Breadcrumbs renders a horizontal trail of navigation items separated by a configurable separator character. All items except the last are pressable links. The last item is rendered as bold text representing the current location.

Colors are theme-aware via `useThemeColorsOptional()`. Non-current items display at reduced opacity and brighten on hover.

=== API / SYNTAX ===
```tsx
import { Breadcrumbs } from '@reactjit/core';
```

Props:
| Prop      | Type                   | Default | Description                            |
|-----------|------------------------|---------|----------------------------------------|
| items     | BreadcrumbItem[]       | required| Array of breadcrumb items              |
| onSelect  | (id: string) => void   | undefined | Callback when a non-current item is pressed |
| separator | string                 | '/'     | Character between items                |
| style     | Style                  | undefined | Outer container style overrides       |

BreadcrumbItem:
| Field | Type   | Description                |
|-------|--------|----------------------------|
| id    | string | Unique item identifier     |
| label | string | Display text               |

=== EXAMPLES ===
Example 1: Basic breadcrumbs
---
```tsx
<Breadcrumbs
  items={[
    { id: 'home', label: 'Home' },
    { id: 'docs', label: 'Docs' },
    { id: 'components', label: 'Components' },
  ]}
  onSelect={(id) => navigate(id)}
/>
```
---
Platforms: love2d

Example 2: Custom separator
---
```tsx
<Breadcrumbs
  items={[
    { id: 'root', label: 'Root' },
    { id: 'src', label: 'src' },
    { id: 'app', label: 'App.tsx' },
  ]}
  separator=">"
  onSelect={(id) => openFolder(id)}
/>
```
---
Platforms: love2d

Example 3: Breadcrumbs in a page header
---
```tsx
const colors = useThemeColors();

<FlexColumn gap={8} style={{ padding: 16 }}>
  <Breadcrumbs
    items={[
      { id: 'dashboard', label: 'Dashboard' },
      { id: 'users', label: 'Users' },
      { id: 'profile', label: 'John Doe' },
    ]}
    onSelect={setPage}
  />
  <Text style={{ color: colors.text, fontSize: 20, fontWeight: 'bold' }}>John Doe</Text>
</FlexColumn>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- The last item in the `items` array is always rendered as static text (bold, non-pressable). It represents the current page.
- `onSelect` only fires for items that are not the last item. If the array has one item, nothing is pressable.
