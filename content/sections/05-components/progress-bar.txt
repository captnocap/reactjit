=== OVERVIEW ===
ProgressBar displays a horizontal fill bar representing progress from 0 to 1. The fill width is rendered as a percentage of the track width. An optional label shows the percentage or custom text overlay, and the `animated` prop uses `useSpring` to smoothly interpolate value changes.

The component uses Box and Text primitives, making it compatible with all targets. The track and fill both use pill-shaped border radii derived from the height. When interactive mode is enabled, hovering the bar reveals a ChartTooltip showing the percentage and exact fill fraction.

=== API / SYNTAX ===
```tsx
import { ProgressBar } from '@reactjit/core';
```

Props:
| Prop        | Type    | Default    | Description                                                              |
|-------------|---------|------------|--------------------------------------------------------------------------|
| value       | number  | required   | Progress value from 0 to 1; clamped internally to [0, 1]                 |
| color       | Color   | '#3b82f6'  | Fill bar color                                                           |
| trackColor  | Color   | '#1e293b'  | Background track color                                                   |
| height      | number  | 8          | Bar height in pixels                                                     |
| showLabel   | boolean | false      | Show percentage label overlaid on the bar (requires height >= 14)        |
| label       | string  | undefined  | Custom label text; defaults to the computed percentage (e.g. "65%")     |
| animated    | boolean | false      | Animate value changes with a spring (stiffness: 120, damping: 20)       |
| interactive | boolean | false      | Enable hover to show a ChartTooltip with percentage and fill fraction    |
| style       | Style   | undefined  | Additional styles applied to the outer container Box                     |

=== EXAMPLES ===
Example 1: Basic progress bar
---
```tsx
import { ProgressBar } from '@reactjit/core';

export default function App() {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <ProgressBar value={0.42} style={{ width: '100%' }} />
    </Box>
  );
}
```

Example 3: Animated progress with custom label and interactive tooltip
---
```tsx
import { ProgressBar } from '@reactjit/core';

export default function App({ downloaded, total }: { downloaded: number; total: number }) {
  return (
    <Box style={{ width: '100%', height: '100%', padding: 20 }}>
      <ProgressBar
        value={downloaded / total}
        animated
        label={`${downloaded} / ${total} MB`}
        showLabel
        height={18}
        color="#f59e0b"
        trackColor="#1e1e24"
        interactive
        style={{ width: '100%' }}
      />
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The `value` prop is required and is clamped to [0, 1] internally. Values outside this range are silently clamped — no error is thrown.
- ProgressBar needs a known `width` for the fill bar to render correctly. Use explicit width, '100%', or flexGrow to ensure the track has resolved dimensions.
- The `showLabel` prop is silently ignored when `height < 14` — the text physically does not fit within the bar at that height.
- When `animated` is true, the fill width is driven by `useSpring` targeting `value * 100`. The hook is called unconditionally, so animated cannot be toggled conditionally at runtime.
- Label text is positioned absolutely and centered over the full track width. It is always white (`color: '#ffffff'`) with `fontWeight: 'bold'`.
- Label font size is `Math.max(9, height - 4)` — it scales with bar height but never goes below 9px.
- When both `label` and `showLabel` are provided, the custom `label` string is shown instead of the computed percentage.
- `interactive` adds a ChartTooltip showing the percentage and the raw clamped fraction (e.g. "0.75 / 1.00"). Hover requires pointer support.
