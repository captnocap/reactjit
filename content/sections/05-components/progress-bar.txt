=== OVERVIEW ===
ProgressBar displays a horizontal fill indicator for a 0-to-1 value. The fill width is rendered as a percentage of the track. Values outside 0-1 are clamped.

Built with Box and Text primitives (not a Chart2D surface). When `showLabel` is true and height is at least 14px, the percentage or custom label text is overlaid on the bar. The `animated` prop enables a spring transition on the fill width.

When `interactive` is enabled, hovering shows a descriptive tooltip with the label, percentage, and raw value.

=== API / SYNTAX ===
```tsx
import { ProgressBar } from '@reactjit/core';
```

Props:
| Prop        | Type    | Default   | Description                                |
|-------------|---------|-----------|--------------------------------------------|
| value       | number  | required  | Progress value from 0 to 1                 |
| color       | Color   | '#3b82f6' | Fill color                                 |
| trackColor  | Color   | '#1e293b' | Background track color                     |
| height      | number  | 8         | Bar height in pixels                       |
| showLabel   | boolean | false     | Show label text (requires height >= 14)    |
| label       | string  | undefined | Custom label (defaults to percentage)      |
| animated    | boolean | false     | Animate width changes with spring easing   |
| interactive | boolean | false     | Show tooltip on hover                      |
| style       | Style   | undefined | Full style object (applied to track)       |

=== EXAMPLES ===
Example 1: Basic progress bar
---
```tsx
const colors = useThemeColors();

<ProgressBar value={0.65} color={colors.primary} />
```
---
Platforms: love2d

Example 2: With label and animation
---
```tsx
const colors = useThemeColors();

<ProgressBar
  value={progress}
  color={colors.primary}
  height={20}
  showLabel
  animated
/>
```

---
Platforms: love2d

Example 3: Custom label and styling
---
```tsx
const colors = useThemeColors();

<ProgressBar
  value={downloaded / total}
  color="#22c55e"
  trackColor={colors.surface}
  height={24}
  showLabel
  label={`${downloaded} / ${total} MB`}
  style={{ borderRadius: 4 }}
/>
```
---
Platforms: love2d

Example 4: Interactive with tooltip
---
```tsx
const colors = useThemeColors();

<ProgressBar
  value={0.82}
  color={colors.primary}
  trackColor={colors.surface}
  height={12}
  interactive
  label="Upload"
/>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- The `value` prop must be between 0 and 1. Values outside this range are clamped.
- The label overlay only appears when `showLabel` is true AND `height` is at least 14px. At smaller heights the text would not fit.
- ProgressBar is not a surface element -- it is built from Box primitives. It does not get proportional surface fallback. Its height defaults to 8px; width comes from the parent container.
