=== OVERVIEW ===
Image renders an image file as a Lua `Image` node. It accepts a `src` prop pointing to a file path (relative to the Love2D filesystem) or a URL.

Image is a surface element — if no explicit dimensions are set and there are no children, it falls back to 1/4 of parent size (proportional surface fallback). Set `w`/`h` or `style.width`/`style.height` for precise control.

Image also integrates with the built-in icon registry. When `src` is a bare name with no path separators or file extension (e.g. `"chevron-right"`), Image looks it up in the icon registry and renders it as vector stroke paths instead of a raster image.

=== API / SYNTAX ===
```tsx
import { Image } from '@reactjit/core';
```

Props:
| Prop    | Type                       | Default   | Description                         |
|---------|----------------------------|-----------|-------------------------------------|
| src     | string                     | required  | File path, URL, or icon name        |
| w       | number \| string           | undefined | Width shorthand                     |
| h       | number \| string           | undefined | Height shorthand                    |
| radius  | number                     | undefined | Border radius shorthand             |
| style   | Style                      | undefined | Full style object                   |
| onClick | (event: LoveEvent) => void | undefined | Click handler                       |
| onWheel | (event: LoveEvent) => void | undefined | Wheel scroll handler                |

Relevant style properties:
| Property  | Type                                      | Description                  |
|-----------|-------------------------------------------|------------------------------|
| objectFit | 'fill' \| 'contain' \| 'cover' \| 'none' | How the image fits its box   |
| width     | number \| string                          | Element width                |
| height    | number \| string                          | Element height               |

=== EXAMPLES ===
Example 1: Basic image with explicit size
---
```tsx
<Image src="assets/logo.png" w={200} h={200} />
```
---
Platforms: love2d

Example 2: Image with object-fit
---
```tsx
<Image
  src="photos/landscape.jpg"
  style={{
    width: '100%',
    height: 300,
    objectFit: 'cover',
    borderRadius: 12,
  }}
/>
```
---
Platforms: love2d

Example 3: Icon from the registry
---
```tsx
const colors = useThemeColors();

<Image
  src="chevron-right"
  w={24}
  h={24}
  style={{ color: colors.text }}
/>
```
---
Platforms: love2d

Example 4: Clickable image
---
```tsx
<Image
  src="assets/thumbnail.png"
  w={120}
  h={120}
  radius={8}
  onClick={() => setSelected(true)}
/>
```
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Image files are resolved relative to the Love2D filesystem root (the project directory). Supported formats include PNG, JPEG, BMP, and other formats supported by Love2D's image decoder. When src is a bare name (no `/`, `\`, or `.`), the icon registry is checked and the result renders as vector stroke paths colored by the `color` style property.
