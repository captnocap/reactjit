=== OVERVIEW ===
The weather hooks wrap the OpenWeatherMap API (data/2.5 endpoint). Authentication uses an API key passed as a query parameter — get one at https://openweathermap.org/api. Pass the key as the first argument to every hook.

Location can be specified either as a city name string (e.g. "London,UK") or as latitude/longitude coordinates. If neither is provided the request is skipped.

The package also exports a weatherIcon() utility for building OpenWeatherMap icon image URLs from the icon code returned in API responses.

=== API / SYNTAX ===

--- useWeatherCurrent ---

Fetch current weather conditions for a city or coordinate pair.

  import { useWeatherCurrent } from '@reactjit/apis/weather';

  const result: APIResult<WeatherCurrent> = useWeatherCurrent(apiKey, opts);

Parameters:
  apiKey   string | null
  opts     object (required — at least one of city or lat/lon must be provided)
    city    string                          City name, optionally with country code: "Paris,FR"
    lat     number                          Latitude (use with lon)
    lon     number                          Longitude (use with lat)
    units   'metric' | 'imperial'           Default: 'metric'

Returns: APIResult<WeatherCurrent>

WeatherCurrent fields:
  name       string   City name as returned by the API
  coord      { lat: number; lon: number }
  weather    WeatherCondition[]
  main       WeatherMain
  wind       { speed: number; deg: number; gust?: number }
  clouds     { all: number }   Cloud coverage percent
  visibility number   Visibility in metres
  sys        { sunrise: number; sunset: number; country: string }   Unix timestamps
  dt         number   Unix timestamp of data calculation
  timezone   number   Timezone offset in seconds from UTC

WeatherCondition fields:
  id           number   Condition code
  main         string   Group: 'Rain', 'Snow', 'Clear', etc.
  description  string   Human-readable description
  icon         string   Icon code — use with weatherIcon()

WeatherMain fields:
  temp       number   Temperature in the requested units
  feels_like number
  temp_min   number
  temp_max   number
  humidity   number   Percent
  pressure   number   hPa

--- useWeatherForecast ---

Fetch a 5-day / 3-hour forecast for a city or coordinate pair.

  import { useWeatherForecast } from '@reactjit/apis/weather';

  const result: APIResult<WeatherForecast> = useWeatherForecast(apiKey, opts);

Parameters:
  apiKey   string | null
  opts     object (same shape as useWeatherCurrent)
    city    string
    lat     number
    lon     number
    units   'metric' | 'imperial'   Default: 'metric'

Returns: APIResult<WeatherForecast>

WeatherForecast fields:
  list   WeatherForecastItem[]   Up to 40 entries (5 days x 8 per day)
  city   { name: string; country: string; coord: { lat: number; lon: number } }

WeatherForecastItem fields:
  dt      number             Unix timestamp
  main    WeatherMain        Same as WeatherCurrent.main
  weather WeatherCondition[]
  wind    WeatherCurrent['wind']
  clouds  { all: number }
  pop     number             Probability of precipitation (0–1)
  dt_txt  string             Human-readable datetime string

--- useGeocode ---

Resolve a city name or address query to latitude/longitude coordinates.

  import { useGeocode } from '@reactjit/apis/weather';

  const result: APIResult<GeoLocation[]> = useGeocode(apiKey, query, opts?);

Parameters:
  apiKey   string | null
  query    string | null   City name or address. Pass null to skip.
  opts     object (optional)
    limit  number   Maximum results to return. Default: 5.

Returns: APIResult<GeoLocation[]>

GeoLocation fields:
  name     string
  lat      number
  lon      number
  country  string   Two-letter country code
  state    string (optional)

--- weatherIcon ---

Build an OpenWeatherMap icon URL from the icon code returned in any WeatherCondition.

  import { weatherIcon } from '@reactjit/apis/weather';

  const url: string = weatherIcon(icon, size?);

Parameters:
  icon   string   Icon code from WeatherCondition.icon (e.g. '10d')
  size   '1' | '2' | '4'   Scale multiplier. Default: '2' (50x50 px at 2x).

Returns: string — full https URL to the PNG icon image.

=== EXAMPLES ===

-- Current conditions widget --

Display temperature, description, and wind speed for a fixed city:

  import { useWeatherCurrent } from '@reactjit/apis/weather';
  import { Box, Text } from '@reactjit/core';

  const API_KEY = process.env.OWM_KEY ?? null;

  function WeatherWidget({ city }: { city: string }) {
    const { data, loading, error } = useWeatherCurrent(API_KEY, {
      city,
      units: 'metric',
    });

    if (loading) return <Text fontSize={13}>Loading weather...</Text>;
    if (error || !data) return <Text fontSize={13}>Weather unavailable</Text>;

    const condition = data.weather[0];

    return (
      <Box padding={16} gap={6} width={200}>
        <Text fontSize={20}>{data.name}</Text>
        <Text fontSize={36}>{Math.round(data.main.temp)}°C</Text>
        <Text fontSize={14}>{condition.description}</Text>
        <Text fontSize={12}>Feels like {Math.round(data.main.feels_like)}°C</Text>
        <Text fontSize={12}>Humidity {data.main.humidity}%</Text>
        <Text fontSize={12}>Wind {data.wind.speed} m/s</Text>
      </Box>
    );
  }

-- 5-day forecast strip --

Show daily high/low temperatures for the next 5 days by sampling one entry per day:

  import { useWeatherForecast } from '@reactjit/apis/weather';
  import { Box, Text } from '@reactjit/core';

  function ForecastStrip({ city, apiKey }: { city: string; apiKey: string }) {
    const { data, loading } = useWeatherForecast(apiKey, { city, units: 'metric' });

    if (loading || !data) return <Text fontSize={12}>Loading forecast...</Text>;

    // Pick one entry per calendar day (the noon-ish entry)
    const daily = data.list.filter((_, i) => i % 8 === 4).slice(0, 5);

    return (
      <Box flexDirection="row" gap={12}>
        {daily.map(entry => {
          const date = new Date(entry.dt * 1000);
          const day = date.toLocaleDateString('en', { weekday: 'short' });
          return (
            <Box key={entry.dt} alignItems="center" gap={4} width={48}>
              <Text fontSize={12}>{day}</Text>
              <Text fontSize={14}>{Math.round(entry.main.temp_max)}°</Text>
              <Text fontSize={12}>{Math.round(entry.main.temp_min)}°</Text>
              <Text fontSize={11}>{entry.weather[0].main}</Text>
            </Box>
          );
        })}
      </Box>
    );
  }
