=== OVERVIEW ===
Hooks for the OpenWeatherMap API. Auth uses an API key passed as a query parameter. Get one at https://openweathermap.org/api.

Location can be specified by city name or lat/lon coordinates. Units default to metric (Celsius, m/s). Set `units: 'imperial'` for Fahrenheit and mph.

The `weatherIcon()` helper constructs icon URLs from the icon codes returned by the API.

=== API / SYNTAX ===
```tsx
import {
  useWeatherCurrent,
  useWeatherForecast,
  useGeocode,
  weatherIcon,
} from '@reactjit/apis';
```

---- useWeatherCurrent

```tsx
function useWeatherCurrent(
  apiKey: string | null,
  opts: { city?: string; lat?: number; lon?: number; units?: 'metric' | 'imperial' }
): APIResult<WeatherCurrent>
```

Fetch current weather conditions. Specify location by city name or lat/lon. At least one of `city` or `lat` must be provided.

---- useWeatherForecast

```tsx
function useWeatherForecast(
  apiKey: string | null,
  opts: { city?: string; lat?: number; lon?: number; units?: 'metric' | 'imperial' }
): APIResult<WeatherForecast>
```

Fetch 5-day / 3-hour forecast. Same location options as useWeatherCurrent.

---- useGeocode

```tsx
function useGeocode(
  apiKey: string | null,
  query: string | null,
  opts?: { limit?: number }
): APIResult<GeoLocation[]>
```

Geocode a place name to lat/lon coordinates. Returns up to `limit` results (default 5). Pass null as query to disable.

---- weatherIcon

```tsx
function weatherIcon(icon: string, size?: '1' | '2' | '4'): string
```

Constructs an OpenWeatherMap icon URL. Size defaults to '2' (2x). Returns URLs like `https://openweathermap.org/img/wn/10d@2x.png`.

---- Types

WeatherCurrent:
| Field      | Type                  | Description                         |
|------------|-----------------------|-------------------------------------|
| name       | string                | City name                           |
| coord      | { lat, lon }          | Coordinates                         |
| weather    | WeatherCondition[]    | Array of weather conditions         |
| main.temp  | number                | Temperature (units-dependent)       |
| main.feels_like | number           | Feels-like temperature              |
| main.temp_min   | number           | Minimum temperature                 |
| main.temp_max   | number           | Maximum temperature                 |
| main.humidity   | number           | Humidity percentage                 |
| main.pressure   | number           | Atmospheric pressure (hPa)         |
| wind.speed | number                | Wind speed (m/s or mph)             |
| wind.deg   | number                | Wind direction in degrees           |
| visibility | number                | Visibility in meters                |
| sys.sunrise | number               | Sunrise time (Unix timestamp)       |
| sys.sunset  | number               | Sunset time (Unix timestamp)        |
| sys.country | string               | Country code                        |

WeatherCondition:
| Field       | Type   | Description                             |
|-------------|--------|-----------------------------------------|
| id          | number | Condition ID                            |
| main        | string | Group ('Rain', 'Clear', 'Clouds', etc.) |
| description | string | Detailed description                    |
| icon        | string | Icon code (e.g., '10d')                 |

WeatherForecast:
| Field     | Type                  | Description                   |
|-----------|-----------------------|-------------------------------|
| list      | WeatherForecastItem[] | 3-hour interval forecast data |
| city.name | string                | City name                     |
| city.country | string             | Country code                  |
| city.coord   | { lat, lon }       | Coordinates                   |

WeatherForecastItem:
| Field   | Type               | Description                    |
|---------|--------------------|--------------------------------|
| dt      | number             | Forecast time (Unix timestamp) |
| main    | object             | Same shape as WeatherCurrent.main |
| weather | WeatherCondition[] | Conditions                     |
| wind    | object             | Same shape as WeatherCurrent.wind |
| pop     | number             | Probability of precipitation (0-1) |
| dt_txt  | string             | Human-readable datetime        |

GeoLocation:
| Field   | Type    | Description       |
|---------|---------|-------------------|
| name    | string  | Place name        |
| lat     | number  | Latitude          |
| lon     | number  | Longitude         |
| country | string  | Country code      |
| state   | string? | State/region name |

=== EXAMPLES ===
Example 1: Current weather display
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useWeatherCurrent, weatherIcon } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function CurrentWeather({ apiKey, city }: { apiKey: string; city: string }) {
  const colors = useThemeColors();
  const { data, loading } = useWeatherCurrent(apiKey, { city });

  if (loading || !data) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  const condition = data.weather[0];

  return (
    <Box style={{ flexDirection: 'row', gap: 12, alignItems: 'center' }}>
      <Image src={weatherIcon(condition.icon)} style={{ width: 48, height: 48 }} />
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontSize: 24, fontWeight: 'bold' }}>
          {`${Math.round(data.main.temp)}\u00B0C`}
        </Text>
        <Text style={{ color: colors.muted, fontSize: 12 }}>
          {`${data.name} | ${condition.description}`}
        </Text>
      </Box>
    </Box>
  );
}
```

Example 2: 5-day forecast
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useWeatherForecast } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function Forecast({ apiKey, city }: { apiKey: string; city: string }) {
  const colors = useThemeColors();
  const { data } = useWeatherForecast(apiKey, { city });

  const daily = data?.list.filter((_, i) => i % 8 === 0) ?? [];

  return (
    <Box style={{ gap: 4 }}>
      {daily.map(item => (
        <Box key={item.dt} style={{ flexDirection: 'row', gap: 8 }}>
          <Text style={{ color: colors.muted, fontSize: 12, width: 80 }}>
            {item.dt_txt.slice(5, 10)}
          </Text>
          <Text style={{ color: colors.text, fontSize: 12 }}>
            {`${Math.round(item.main.temp)}\u00B0 | ${item.weather[0].main}`}
          </Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 3: Geocode then fetch weather
---
```tsx
import { useState, useEffect } from 'react';
import { Box, Text } from '@reactjit/core';
import { useGeocode, useWeatherCurrent } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function WeatherBySearch({ apiKey }: { apiKey: string }) {
  const colors = useThemeColors();
  const [search, setSearch] = useState<string | null>('London');
  const { data: locations } = useGeocode(apiKey, search);
  const loc = locations?.[0];

  const { data: weather } = useWeatherCurrent(
    apiKey,
    { lat: loc?.lat, lon: loc?.lon },
  );

  if (!weather) return <Text style={{ color: colors.muted }}>Searching...</Text>;

  return (
    <Text style={{ color: colors.text, fontSize: 14 }}>
      {`${weather.name}: ${Math.round(weather.main.temp)}\u00B0C`}
    </Text>
  );
}
```

=== CRITICAL RULES ===
- You must provide either `city` or both `lat` and `lon`. If neither is set, the hook is disabled.
- Units default to 'metric'. Set `units: 'imperial'` for Fahrenheit and mph.
- The free tier allows 60 calls per minute and 1,000,000 calls per month.
- weatherIcon returns a PNG URL. The icon code comes from `data.weather[0].icon`.
