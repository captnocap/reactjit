=== OVERVIEW ===
The Hue hooks communicate with a Philips Hue Bridge on the local network. Authentication uses an API key embedded in the URL path (no Bearer token). All read hooks accept the bridge IP and API key as separate arguments.

To get an API key, press the physical link button on the bridge, then POST to http://<bridge-ip>/api with {"devicetype":"reactjit#app"}. The response contains the username (API key).

useHueControl provides mutation functions for setting light state, group actions, and activating scenes. The hueXYToHex utility converts Hue's CIE xy color space to hex strings for display.

=== API / SYNTAX ===
```tsx
import {
  useHueLights,
  useHueLight,
  useHueGroups,
  useHueSensors,
  useHueScenes,
  useHueControl,
  hueXYToHex,
} from '@reactjit/apis';
```

---- useHueLights

```tsx
function useHueLights(
  bridgeIp: string | null,
  apiKey: string | null,
  opts?: { interval?: number }
): APIResult<Record<string, HueLight>>
```

Returns a map of light ID to HueLight. Supports polling via the interval option (milliseconds).

---- useHueLight

```tsx
function useHueLight(
  bridgeIp: string | null,
  apiKey: string | null,
  lightId: string | null,
  opts?: { interval?: number }
): APIResult<HueLight>
```

Fetches a single light by ID. All three parameters must be non-null.

---- useHueControl

```tsx
function useHueControl(bridgeIp: string | null, apiKey: string | null): {
  setLight: (lightId: string, state: Partial<HueLight['state']>) => Promise<any>;
  setGroup: (groupId: string, action: Partial<HueLight['state']>) => Promise<any>;
  activateScene: (sceneId: string, groupId?: string) => Promise<any>;
  loading: boolean;
  error: Error | null;
}
```

Light/group state properties: on (boolean), bri (0-254), hue (0-65535), sat (0-254), ct (153-500 mirek), xy ([x, y] CIE color), effect ("none" | "colorloop"), alert ("none" | "select" | "lselect").

---- hueXYToHex

```tsx
function hueXYToHex(x: number, y: number, bri: number): string
```

Converts CIE xy color + brightness to a hex color string. Useful for rendering light colors in the UI.

---- Types

HueLight:
| Field            | Type    | Description                                  |
|------------------|---------|----------------------------------------------|
| state.on         | boolean | Whether the light is on                      |
| state.bri        | number  | Brightness (0-254)                           |
| state.hue        | number  | Hue (0-65535, maps to 0-360 degrees)         |
| state.sat        | number  | Saturation (0-254)                           |
| state.ct         | number  | Color temperature in mirek (153-500)         |
| state.xy         | [x, y]  | CIE color coordinates                        |
| state.colormode  | string  | Active color mode ('hs', 'xy', or 'ct')      |
| state.reachable  | boolean | Whether the bridge can communicate with it   |
| name             | string  | User-assigned light name                     |
| type             | string  | Light type (e.g. "Extended color light")     |
| modelid          | string  | Hardware model identifier                    |

HueGroup:
| Field        | Type     | Description                              |
|--------------|----------|------------------------------------------|
| name         | string   | Group name                               |
| lights       | string[] | Array of light IDs in this group         |
| type         | string   | Group type (Room, Zone, etc.)            |
| state.all_on | boolean  | Whether all lights in the group are on   |
| state.any_on | boolean  | Whether any light in the group is on     |
| action       | object   | Last-sent action state for the group     |

=== EXAMPLES ===
Example 1: Light status dashboard
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useHueLights, hueXYToHex } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function LightDashboard({ bridgeIp, apiKey }: { bridgeIp: string; apiKey: string }) {
  const colors = useThemeColors();
  const { data: lights, loading } = useHueLights(bridgeIp, apiKey, { interval: 5000 });

  if (loading) return <Text style={{ color: colors.muted }}>Connecting to bridge...</Text>;

  return (
    <Box style={{ gap: 6 }}>
      {lights && Object.entries(lights).map(([id, light]) => {
        const lightColor = light.state.on && light.state.xy
          ? hueXYToHex(light.state.xy[0], light.state.xy[1], light.state.bri)
          : undefined;
        return (
          <Box key={id} style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
            <Box style={{ width: 12, height: 12, borderRadius: 6, backgroundColor: light.state.on ? (lightColor ?? colors.primary) : colors.muted }} />
            <Text style={{ color: colors.text }}>{light.name}</Text>
            <Text style={{ color: colors.muted }}>{light.state.on ? `${Math.round(light.state.bri / 254 * 100)}%` : 'Off'}</Text>
          </Box>
        );
      })}
    </Box>
  );
}
```

Example 2: Toggle a light
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { useHueLight, useHueControl } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function LightToggle({ bridgeIp, apiKey, lightId }: { bridgeIp: string; apiKey: string; lightId: string }) {
  const colors = useThemeColors();
  const { data: light, refetch } = useHueLight(bridgeIp, apiKey, lightId, { interval: 2000 });
  const { setLight } = useHueControl(bridgeIp, apiKey);

  const toggle = async () => {
    if (!light) return;
    await setLight(lightId, { on: !light.state.on });
    refetch();
  };

  return (
    <Pressable onPress={toggle}>
      <Box style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
        <Text style={{ color: colors.text }}>{light?.name ?? 'Loading...'}</Text>
        <Text style={{ color: light?.state.on ? colors.primary : colors.muted }}>
          {light?.state.on ? 'ON' : 'OFF'}
        </Text>
      </Box>
    </Pressable>
  );
}
```

=== CRITICAL RULES ===
- The bridge must be on the same local network. Hue API is HTTP-only (no HTTPS) on the local bridge IP.
- Brightness ranges from 0-254, not 0-255. Hue ranges from 0-65535. Saturation from 0-254.
- Color temperature is in mirek (153 = coldest/6500K, 500 = warmest/2000K). This is the inverse of Kelvin.
- setLight sends state, setGroup sends action. The API endpoints differ.
- activateScene defaults to group "0" (all lights) if no groupId is specified.
