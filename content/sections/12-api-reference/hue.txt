=== OVERVIEW ===
The Hue hooks talk directly to the local Hue bridge over HTTP — no cloud required. All reads use the reactive `useAPI` base hook (polling, loading/error state, manual refetch). Writes go through `useHueControl`, which wraps `useAPIMutation` and exposes three imperative methods: set a single light state, set an entire group action, or activate a named scene.

Auth: Press the physical link button on your bridge, then POST to `http://<bridgeIp>/api` with `{"devicetype":"myapp"}`. The bridge returns a username string — use that as your `apiKey` in every call below.

=== API / SYNTAX ===

--- useHueLights ---

Fetch all lights registered on the bridge. Returns a dictionary keyed by light ID string.

  import { useHueLights } from '@reactjit/apis/hue';

  const result: APIResult<Record<string, HueLight>> = useHueLights(
    bridgeIp: string | null,
    apiKey:   string | null,
    opts?:    { interval?: number },
  );

Parameters:
  bridgeIp         string | null   Local IP of the Hue bridge (e.g. '192.168.1.10'). Pass null to disable.
  apiKey           string | null   API key (username) obtained from the bridge. Pass null to disable.
  opts.interval    number          Polling interval in milliseconds. Omit to fetch once.

Returns: APIResult<Record<string, HueLight>>
  data     Record<string, HueLight> | null   Map of light ID → HueLight
  loading  boolean
  error    Error | null
  refetch  () => void

HueLight fields:
  name              string
  type              string
  modelid           string
  manufacturername  string
  productname       string
  uniqueid          string
  state.on          boolean
  state.bri         number                       Brightness 0–254
  state.hue         number                       Hue 0–65535
  state.sat         number                       Saturation 0–254
  state.ct          number                       Color temperature in mireds
  state.xy          [number, number]             CIE xy color space coordinates
  state.colormode   'hs' | 'xy' | 'ct'
  state.reachable   boolean
  state.alert       string
  state.effect      string

--- useHueLight ---

Fetch a single light by its ID.

  import { useHueLight } from '@reactjit/apis/hue';

  const result: APIResult<HueLight> = useHueLight(
    bridgeIp: string | null,
    apiKey:   string | null,
    lightId:  string | null,
    opts?:    { interval?: number },
  );

Parameters:
  bridgeIp         string | null   Local bridge IP. Pass null to disable.
  apiKey           string | null   Bridge API key. Pass null to disable.
  lightId          string | null   Light ID as a string (e.g. '3'). Pass null to disable.
  opts.interval    number          Polling interval in milliseconds.

Returns: APIResult<HueLight>
  (same fields as described under useHueLights)

--- useHueGroups ---

Fetch all groups (rooms and zones) on the bridge.

  import { useHueGroups } from '@reactjit/apis/hue';

  const result: APIResult<Record<string, HueGroup>> = useHueGroups(
    bridgeIp: string | null,
    apiKey:   string | null,
  );

Parameters:
  bridgeIp  string | null
  apiKey    string | null

Returns: APIResult<Record<string, HueGroup>>
  data  Record<string, HueGroup> | null   Map of group ID → HueGroup

HueGroup fields:
  name           string
  lights         string[]               Light IDs belonging to this group
  type           string
  class          string | undefined      Room class (e.g. 'Living room', 'Bedroom')
  state.all_on   boolean                True if every light in the group is on
  state.any_on   boolean                True if at least one light is on
  action         HueLight['state']      The last action applied to the group

--- useHueScenes ---

Fetch all saved scenes on the bridge.

  import { useHueScenes } from '@reactjit/apis/hue';

  const result: APIResult<Record<string, HueScene>> = useHueScenes(
    bridgeIp: string | null,
    apiKey:   string | null,
  );

Parameters:
  bridgeIp  string | null
  apiKey    string | null

Returns: APIResult<Record<string, HueScene>>
  data  Record<string, HueScene> | null   Map of scene ID → HueScene

HueScene fields:
  name         string
  type         string
  group        string | undefined   Group ID this scene belongs to
  lights       string[]             Light IDs included in the scene
  lightstates  Record<string, Partial<HueLight['state']>> | undefined

--- useHueControl ---

Imperative mutation hook for sending commands to the bridge. Returns methods for setting individual light states, group actions, and activating scenes.

  import { useHueControl } from '@reactjit/apis/hue';

  const ctrl = useHueControl(
    bridgeIp: string | null,
    apiKey:   string | null,
  );

Parameters:
  bridgeIp  string | null
  apiKey    string | null

Returns: object
  setLight      (lightId: string, state: Partial<HueLight['state']>) => Promise<any>
                  Send a partial state update to a single light (e.g. { on: true, bri: 200 })
  setGroup      (groupId: string, action: Partial<HueLight['state']>) => Promise<any>
                  Apply an action to every light in a group simultaneously
  activateScene (sceneId: string, groupId?: string) => Promise<any>
                  Activate a saved scene. groupId defaults to '0' (all groups)
  loading       boolean   True while any mutation request is in flight
  error         Error | null

--- hueXYToHex (utility) ---

Convert a Hue light's CIE xy coordinates and brightness to an approximate sRGB hex string. Useful for rendering a color swatch from live light state.

  import { hueXYToHex } from '@reactjit/apis/hue';

  const hex: string = hueXYToHex(x: number, y: number, bri: number);

Parameters:
  x    number   CIE x coordinate (from state.xy[0])
  y    number   CIE y coordinate (from state.xy[1])
  bri  number   Brightness 0–254 (from state.bri)

Returns: string   Hex color string, e.g. '#f5a623'

=== EXAMPLES ===

-- Live light list with on/off toggle --

Poll every 3 seconds and render each light as a pressable row. Tap a row to toggle it.

  import { useHueLights, useHueControl, hueXYToHex } from '@reactjit/apis/hue';
  import { Box, Text } from '@reactjit/core';

  const BRIDGE_IP = '192.168.1.10';
  const API_KEY   = 'your-bridge-api-key';

  function LightList() {
    const { data, loading, error } = useHueLights(BRIDGE_IP, API_KEY, { interval: 3000 });
    const ctrl = useHueControl(BRIDGE_IP, API_KEY);

    if (loading && !data) return <Text fontSize={14}>Connecting to bridge...</Text>;
    if (error) return <Text fontSize={14}>Error: {error.message}</Text>;

    const lights = Object.entries(data ?? {});

    return (
      <Box width="100%" height="100%" padding={16} gap={8}>
        {lights.map(([id, light]) => {
          const color = light.state.colormode === 'xy'
            ? hueXYToHex(light.state.xy[0], light.state.xy[1], light.state.bri)
            : '#ffffff';

          return (
            <Box
              key={id}
              flexDirection="row"
              alignItems="center"
              gap={12}
              padding={10}
              backgroundColor={light.state.on ? '#1e1e1e' : '#111'}
              onPress={() => ctrl.setLight(id, { on: !light.state.on })}
            >
              {/* Color swatch */}
              <Box
                width={24}
                height={24}
                backgroundColor={light.state.on ? color : '#333'}
              />
              <Text fontSize={14}>{light.name}</Text>
              <Text fontSize={12}>{light.state.on ? 'On' : 'Off'}</Text>
            </Box>
          );
        })}
      </Box>
    );
  }

-- Activate a scene from a scene picker --

Fetch all scenes and render them as buttons. Tapping one activates it on group '1'.

  import { useHueScenes, useHueControl } from '@reactjit/apis/hue';
  import { Box, Text } from '@reactjit/core';

  const BRIDGE_IP = '192.168.1.10';
  const API_KEY   = 'your-bridge-api-key';
  const GROUP_ID  = '1';

  function ScenePicker() {
    const { data } = useHueScenes(BRIDGE_IP, API_KEY);
    const ctrl = useHueControl(BRIDGE_IP, API_KEY);

    const scenes = Object.entries(data ?? {}).filter(([, s]) => s.group === GROUP_ID);

    return (
      <Box width="100%" flexDirection="row" flexWrap="wrap" gap={8} padding={12}>
        {scenes.map(([id, scene]) => (
          <Box
            key={id}
            backgroundColor="#2a2a2a"
            padding={10}
            onPress={() => ctrl.activateScene(id, GROUP_ID)}
          >
            <Text fontSize={13}>{scene.name}</Text>
          </Box>
        ))}
      </Box>
    );
  }
