=== OVERVIEW ===
The Home Assistant hooks wrap the Home Assistant local REST API. Authentication uses a long-lived access token (Bearer) — generate one in Settings > Profile > Long-Lived Access Tokens. You also need the base URL of your Home Assistant instance (e.g. http://homeassistant.local:8123).

Both baseUrl and token are required by every hook. Passing null for either skips the request. The polling hooks (useHAStates, useHAEntity) accept an optional interval parameter so entity state stays live on screen.

useHACallService is a mutation hook — it returns a call() function rather than reactive data. Use it to trigger automations, toggle lights, set thermostat values, and any other service action.

=== API / SYNTAX ===

--- useHAStates ---

Fetch all entity states from Home Assistant. Optionally polls on an interval.

  import { useHAStates } from '@reactjit/apis/homeassistant';

  const result: APIResult<HAState[]> = useHAStates(baseUrl, token, opts?);

Parameters:
  baseUrl   string | null   Home Assistant base URL, e.g. 'http://homeassistant.local:8123'
  token     string | null   Long-lived access token.
  opts      object (optional)
    interval  number   Polling interval in ms. No polling if omitted.

Returns: APIResult<HAState[]>

HAState fields:
  entity_id     string   e.g. 'light.living_room', 'sensor.temperature'
  state         string   Entity state value, e.g. 'on', '21.5', 'unavailable'
  attributes    Record<string, any>   Entity-specific attributes
  last_changed  string   ISO 8601 timestamp
  last_updated  string   ISO 8601 timestamp

--- useHAEntity ---

Fetch a single entity's state by entity ID. Optionally polls on an interval.

  import { useHAEntity } from '@reactjit/apis/homeassistant';

  const result: APIResult<HAState> = useHAEntity(baseUrl, token, entityId, opts?);

Parameters:
  baseUrl    string | null
  token      string | null
  entityId   string | null   Entity ID to fetch, e.g. 'sensor.bedroom_temperature'. Pass null to skip.
  opts       object (optional)
    interval  number   Polling interval in ms.

Returns: APIResult<HAState>   Same HAState shape as above.

--- useHAConfig ---

Fetch the Home Assistant instance configuration (location, timezone, version, unit system).

  import { useHAConfig } from '@reactjit/apis/homeassistant';

  const result: APIResult<HAConfig> = useHAConfig(baseUrl, token);

Parameters:
  baseUrl   string | null
  token     string | null

Returns: APIResult<HAConfig>

HAConfig fields:
  location_name  string
  latitude       number
  longitude      number
  elevation      number
  unit_system    Record<string, string>   e.g. { temperature: '°C', length: 'km' }
  time_zone      string   IANA timezone string
  version        string   Home Assistant version

--- useHAServices ---

Fetch the list of all available service domains and their service definitions.

  import { useHAServices } from '@reactjit/apis/homeassistant';

  const result: APIResult<HAService[]> = useHAServices(baseUrl, token);

Parameters:
  baseUrl   string | null
  token     string | null

Returns: APIResult<HAService[]>

HAService fields:
  domain    string   e.g. 'light', 'switch', 'climate'
  services  Record<string, { description: string; fields: Record<string, any> }>

--- useHAHistory ---

Fetch the state history for a single entity over a recent time window.

  import { useHAHistory } from '@reactjit/apis/homeassistant';

  const result: APIResult<HAState[][]> = useHAHistory(baseUrl, token, entityId, opts?);

Parameters:
  baseUrl    string | null
  token      string | null
  entityId   string | null   Pass null to skip.
  opts       object (optional)
    hours   number   How many hours back to fetch. Default: 24.

Returns: APIResult<HAState[][]>   Outer array has one entry per entity matched (usually one). Inner arrays are time-ordered HAState snapshots.

--- useHALogbook ---

Fetch the logbook (human-readable event log) for an optional entity filter.

  import { useHALogbook } from '@reactjit/apis/homeassistant';

  const result: APIResult<HALogEntry[]> = useHALogbook(baseUrl, token, opts?);

Parameters:
  baseUrl   string | null
  token     string | null
  opts      object (optional)
    entityId  string   Filter log to a single entity.
    hours     number   How many hours back to fetch. Default: 24.

Returns: APIResult<HALogEntry[]>

HALogEntry fields:
  name       string
  message    string
  entity_id  string (optional)
  when       string   ISO 8601 timestamp

--- useHACallService ---

Imperative hook for calling Home Assistant services (toggle lights, set climate, trigger automations, etc.).

  import { useHACallService } from '@reactjit/apis/homeassistant';

  const ha = useHACallService(baseUrl, token);

Parameters:
  baseUrl   string | null
  token     string | null

Returns:
  call     (domain: string, service: string, data?: Record<string, any>) => Promise<any>
  loading  boolean   True while the service call is in flight.
  error    Error | null

=== EXAMPLES ===

-- Light toggle panel --

Display the state of a light entity and provide a toggle button:

  import { useHAEntity, useHACallService } from '@reactjit/apis/homeassistant';
  import { Box, Text } from '@reactjit/core';

  const HA_URL = 'http://homeassistant.local:8123';
  const TOKEN = process.env.HA_TOKEN ?? null;

  function LightControl({ entityId }: { entityId: string }) {
    const { data } = useHAEntity(HA_URL, TOKEN, entityId, { interval: 5000 });
    const ha = useHACallService(HA_URL, TOKEN);

    const isOn = data?.state === 'on';
    const brightness = data?.attributes?.brightness;
    const label = data?.attributes?.friendly_name ?? entityId;

    return (
      <Box padding={12} gap={8} width={200}>
        <Text fontSize={15}>{label}</Text>
        <Text fontSize={13}>{isOn ? 'On' : 'Off'}</Text>
        {brightness != null && (
          <Text fontSize={12}>Brightness: {Math.round((brightness / 255) * 100)}%</Text>
        )}
        <Box
          padding={8}
          backgroundColor={isOn ? '#FFC107' : '#333'}
          onPress={() => ha.call('light', 'toggle', { entity_id: entityId })}
        >
          <Text fontSize={13}>{ha.loading ? '...' : 'Toggle'}</Text>
        </Box>
      </Box>
    );
  }

-- Sensor dashboard --

Display a grid of sensor readings with live polling:

  import { useHAStates } from '@reactjit/apis/homeassistant';
  import { Box, Text } from '@reactjit/core';

  const SENSORS = [
    'sensor.living_room_temperature',
    'sensor.bedroom_humidity',
    'sensor.front_door_battery',
    'sensor.electricity_usage',
  ];

  function SensorDashboard({ baseUrl, token }: { baseUrl: string; token: string }) {
    const { data, loading } = useHAStates(baseUrl, token, { interval: 10000 });

    if (loading && !data) return <Text fontSize={13}>Connecting...</Text>;

    const sensors = SENSORS.map(id =>
      data?.find(s => s.entity_id === id)
    );

    return (
      <Box flexDirection="row" flexWrap="wrap" gap={12} padding={16}>
        {sensors.map((sensor, i) => (
          <Box key={SENSORS[i]} padding={12} width={140} gap={4} backgroundColor="#1a1a1a">
            <Text fontSize={11}>{sensor?.attributes?.friendly_name ?? SENSORS[i]}</Text>
            <Text fontSize={22}>{sensor?.state ?? '--'}</Text>
            {sensor?.attributes?.unit_of_measurement && (
              <Text fontSize={12}>{sensor.attributes.unit_of_measurement}</Text>
            )}
          </Box>
        ))}
      </Box>
    );
  }
