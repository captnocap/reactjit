=== OVERVIEW ===
The Todoist hooks in @reactjit/apis use the Todoist REST API v2. Authentication is a single Bearer token found under Settings > Integrations > Developer in the Todoist web app.

Read hooks (useTodoistTasks, useTodoistProjects, etc.) are reactive — they fetch on mount and expose a refetch() method. The useTodoistMutation hook returns imperative functions for write operations: creating tasks, marking them complete, reopening, updating, and deleting.

All read hooks accept null as the token argument to suppress fetching, which is the standard pattern for conditional data loading in ReactJIT.

=== API / SYNTAX ===

--- useTodoistTasks ---

Fetch tasks with optional filtering by project, label, or Todoist filter expression.

  import { useTodoistTasks } from '@reactjit/apis';

  const result: APIResult<TodoistTask[]> = useTodoistTasks(
    token: string | null,
    opts?: { projectId?: string; label?: string; filter?: string },
  );

Parameters:
  token  string | null   Todoist API token
  opts   object | undefined
    projectId  string    Filter to a specific project ID
    label      string    Filter to a specific label name
    filter     string    Todoist filter query string (e.g. 'today | overdue')

Returns: APIResult<TodoistTask[]>

TodoistTask fields:
  id             string
  content        string              Task title
  description    string              Additional notes
  is_completed   boolean
  priority       1 | 2 | 3 | 4      1 = normal, 4 = urgent
  due            object | null
    date         string              Due date in YYYY-MM-DD format
    string       string              Human-readable due string
    datetime     string | undefined  ISO 8601 if time is set
    recurring    boolean
  labels         string[]            Label names
  project_id     string
  section_id     string | undefined
  parent_id      string | null       ID of parent task if nested
  order          number              Sort order within project
  created_at     string              ISO 8601
  url            string              Direct link to the task

--- useTodoistComments ---

Fetch comments for a task or project.

  import { useTodoistComments } from '@reactjit/apis';

  const result: APIResult<TodoistComment[]> = useTodoistComments(
    token: string | null,
    opts: { taskId?: string; projectId?: string },
  );

Parameters:
  token  string | null   Todoist API token
  opts   object          Exactly one of taskId or projectId must be provided
    taskId     string    Fetch comments for this task
    projectId  string    Fetch comments for this project

Returns: APIResult<TodoistComment[]>

TodoistComment fields:
  id          string
  content     string
  posted_at   string              ISO 8601
  task_id     string | undefined
  project_id  string | undefined

--- useTodoistMutation ---

Get imperative operations for creating and managing tasks.

  import { useTodoistMutation } from '@reactjit/apis';

  const mutation = useTodoistMutation(token: string | null);

Parameters:
  token  string | null  Todoist API token

Returns: object
  createTask    (content: string, opts?) => Promise<any>
                Create a new task. opts fields:
                  projectId   string    Target project ID (defaults to Inbox)
                  priority    number    1–4 where 4 is urgent
                  dueString   string    Natural language due date (e.g. 'tomorrow at 9am')
                  labels      string[]  Label names to apply
  completeTask  (taskId: string) => Promise<any>
                Mark a task complete (close it).
  reopenTask    (taskId: string) => Promise<any>
                Reopen a previously completed task.
  deleteTask    (taskId: string) => Promise<any>
                Permanently delete a task.
  updateTask    (taskId: string, updates: Partial<Pick<TodoistTask, 'content' | 'description' | 'priority' | 'labels'>>) => Promise<any>
                Update task fields. Only provided keys are changed.
  loading       boolean        True while any mutation is in flight
  error         Error | null   Last error, or null

=== EXAMPLES ===

-- Today's task list --

Show tasks due today or overdue, with a tap-to-complete interaction:

  import { useTodoistTasks, useTodoistMutation } from '@reactjit/apis';
  import { Box, Text } from '@reactjit/core';

  const TOKEN = process.env.TODOIST_TOKEN;

  function TodayTasks() {
    const { data, loading, refetch } = useTodoistTasks(TOKEN, { filter: 'today | overdue' });
    const { completeTask } = useTodoistMutation(TOKEN);

    if (loading) return <Text fontSize={13}>Loading...</Text>;

    const tasks = data ?? [];

    return (
      <Box gap={4} width="100%">
        <Text fontSize={16}>Today ({tasks.length})</Text>
        {tasks.map((task) => (
          <Box
            key={task.id}
            flexDirection="row"
            gap={10}
            padding={10}
            backgroundColor="#161622"
            onPress={async () => {
              await completeTask(task.id);
              refetch();
            }}
          >
            <Box width={16} height={16} borderRadius={8} backgroundColor="#555" />
            <Box flex={1}>
              <Text fontSize={13}>{task.content}</Text>
              {task.due?.string && (
                <Text fontSize={11}>{task.due.string}</Text>
              )}
            </Box>
            {task.priority === 4 && (
              <Box width={8} height={8} borderRadius={4} backgroundColor="#D32F2F" />
            )}
          </Box>
        ))}
      </Box>
    );
  }

-- Quick-add task form --

Create a new task with a priority and natural-language due date:

  import { useTodoistMutation, useTodoistProjects } from '@reactjit/apis';
  import { Box, Text } from '@reactjit/core';
  import { useState } from 'react';

  const TOKEN = process.env.TODOIST_TOKEN;

  function QuickAdd() {
    const [content, setContent] = useState('');
    const { data: projectsData } = useTodoistProjects(TOKEN);
    const { createTask, loading, error } = useTodoistMutation(TOKEN);

    // Find the inbox project
    const inbox = projectsData?.find((p) => p.is_inbox_project);

    async function handleAdd() {
      if (!content.trim() || !inbox) return;
      await createTask(content, {
        projectId: inbox.id,
        dueString: 'today',
        priority: 2,
      });
      setContent('');
    }

    return (
      <Box gap={8}>
        {/* TextInput would be wired to setContent here */}
        <Box onPress={handleAdd} padding={10} backgroundColor="#DB4035">
          <Text fontSize={13}>{loading ? 'Adding...' : 'Add Task'}</Text>
        </Box>
        {error && <Text fontSize={11}>{error.message}</Text>}
      </Box>
    );
  }
