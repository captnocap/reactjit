=== OVERVIEW ===
The Spotify hooks wrap the Spotify Web API. Authentication is handled via OAuth2 Bearer tokens — get one from the Spotify Developer Dashboard at https://developer.spotify.com. Pass the token as the first argument to every hook. Hooks that accept a null token skip the request entirely and return { data: null, loading: false, error: null }, so conditional rendering is safe without any extra guards.

All hooks return APIResult<T>, a stable object containing data, loading, error, and refetch. useSpotifyNowPlaying polls on a configurable interval (default 5 s). useSpotifyPlayback is a mutation hook — it returns action callbacks rather than reactive data.

=== API / SYNTAX ===

--- useSpotifyNowPlaying ---

Poll the currently-playing track. Re-fetches on a timer so the display stays live.

  import { useSpotifyNowPlaying } from '@reactjit/apis/spotify';

  const result: APIResult<SpotifyNowPlaying> = useSpotifyNowPlaying(token, opts?);

Parameters:
  token   string | null   OAuth2 Bearer token. Pass null to skip the request.
  opts    object (optional)
    interval  number   Polling interval in ms. Default: 5000.

Returns: APIResult<SpotifyNowPlaying>
  data     SpotifyNowPlaying | null
  loading  boolean
  error    Error | null
  refetch  () => void

SpotifyNowPlaying fields:
  is_playing               boolean
  progress_ms              number
  item                     SpotifyTrack | null
  currently_playing_type   string

--- useSpotifyTopTracks ---

Fetch the authenticated user's top tracks over a time range.

  import { useSpotifyTopTracks } from '@reactjit/apis/spotify';

  const result: APIResult<SpotifyTopItem<SpotifyTrack>> = useSpotifyTopTracks(token, opts?);

Parameters:
  token   string | null
  opts    object (optional)
    timeRange  'short_term' | 'medium_term' | 'long_term'   Default: none (API default applies)
    limit      number                                         Max results. Default: none.

Returns: APIResult<SpotifyTopItem<SpotifyTrack>>
  data.items   SpotifyTrack[]
  data.total   number

--- useSpotifyTopArtists ---

Fetch the authenticated user's top artists over a time range.

  import { useSpotifyTopArtists } from '@reactjit/apis/spotify';

Parameters:
  token   string | null
  opts    object (optional)
    timeRange  'short_term' | 'medium_term' | 'long_term'
    limit      number

Returns: APIResult<SpotifyTopItem<SpotifyArtist>>
  data.items   SpotifyArtist[]
  data.total   number

--- useSpotifyRecentTracks ---

Fetch the authenticated user's recently played tracks.

  import { useSpotifyRecentTracks } from '@reactjit/apis/spotify';

  const result: APIResult<{ items: RecentItem[] }> = useSpotifyRecentTracks(token, opts?);

Parameters:
  token   string | null
  opts    object (optional)
    limit  number   Max results. Default: 20.

Returns: APIResult<{ items: Array<{ track: SpotifyTrack; played_at: string }> }>

--- useSpotifyPlaylists ---

Fetch the authenticated user's saved playlists.

  import { useSpotifyPlaylists } from '@reactjit/apis/spotify';

  const result: APIResult<SpotifyPaginated<SpotifyPlaylist>> = useSpotifyPlaylists(token, opts?);

Parameters:
  token   string | null
  opts    object (optional)
    limit  number   Default: 50.

Returns: APIResult<SpotifyPaginated<SpotifyPlaylist>>
  data.items   SpotifyPlaylist[]
  data.total   number
  data.limit   number
  data.offset  number
  data.next    string | null

SpotifyPlaylist fields:
  id           string
  name         string
  description  string
  images       SpotifyImage[]
  tracks       { total: number }
  owner        { display_name: string }

--- useSpotifySearch ---

Search the Spotify catalog by query string.

  import { useSpotifySearch } from '@reactjit/apis/spotify';

  const result: APIResult<SearchResults> = useSpotifySearch(token, query, opts?);

Parameters:
  token   string | null
  query   string | null   Pass null to skip the request (e.g. while the user is typing).
  opts    object (optional)
    type   string   Comma-separated Spotify object types. Default: 'track,artist'.
    limit  number   Default: 20.

Returns: APIResult<{ tracks?: SpotifyPaginated<SpotifyTrack>; artists?: SpotifyPaginated<SpotifyArtist> }>

--- useSpotifyPlayback ---

Imperative playback control. Returns action callbacks rather than reactive data.

  import { useSpotifyPlayback } from '@reactjit/apis/spotify';

  const controls = useSpotifyPlayback(token);

Parameters:
  token   string | null

Returns:
  play      () => Promise<any>   Resume playback
  pause     () => Promise<any>   Pause playback
  next      () => Promise<any>   Skip to next track
  previous  () => Promise<any>   Skip to previous track
  loading   boolean              True while a control command is in flight
  error     Error | null

=== EXAMPLES ===

-- Now playing display with playback controls --

Show the current track and allow the user to skip or pause:

  import { useSpotifyNowPlaying, useSpotifyPlayback } from '@reactjit/apis/spotify';
  import { Box, Text } from '@reactjit/core';

  const TOKEN = process.env.SPOTIFY_TOKEN ?? null;

  function NowPlaying() {
    const { data, loading } = useSpotifyNowPlaying(TOKEN, { interval: 5000 });
    const controls = useSpotifyPlayback(TOKEN);

    if (loading && !data) {
      return <Text fontSize={14}>Loading...</Text>;
    }

    if (!data?.item) {
      return <Text fontSize={14}>Nothing playing</Text>;
    }

    const { item, is_playing, progress_ms } = data;
    const progress = Math.round((progress_ms / item.duration_ms) * 100);

    return (
      <Box width="100%" padding={16} gap={8}>
        <Text fontSize={16}>{item.name}</Text>
        <Text fontSize={13}>{item.artists.map(a => a.name).join(', ')}</Text>
        <Text fontSize={12}>{item.album.name}</Text>

        {/* Progress bar represented as two boxes */}
        <Box flexDirection="row" width="100%" height={4} backgroundColor="#333">
          <Box width={`${progress}%`} height={4} backgroundColor="#1DB954" />
        </Box>

        <Box flexDirection="row" gap={12}>
          <Box padding={8} backgroundColor="#222" onPress={() => controls.previous()}>
            <Text fontSize={12}>Prev</Text>
          </Box>
          <Box padding={8} backgroundColor="#222" onPress={() => is_playing ? controls.pause() : controls.play()}>
            <Text fontSize={12}>{is_playing ? 'Pause' : 'Play'}</Text>
          </Box>
          <Box padding={8} backgroundColor="#222" onPress={() => controls.next()}>
            <Text fontSize={12}>Next</Text>
          </Box>
        </Box>
      </Box>
    );
  }

-- Top artists chart --

Display the user's top 5 artists for the past month:

  import { useSpotifyTopArtists } from '@reactjit/apis/spotify';
  import { Box, Text } from '@reactjit/core';

  function TopArtists({ token }: { token: string }) {
    const { data, loading, error } = useSpotifyTopArtists(token, {
      timeRange: 'short_term',
      limit: 5,
    });

    if (loading) return <Text fontSize={13}>Loading...</Text>;
    if (error) return <Text fontSize={13}>Error: {error.message}</Text>;

    return (
      <Box gap={6}>
        <Text fontSize={15}>Top Artists This Month</Text>
        {data?.items.map((artist, i) => (
          <Box key={artist.id} flexDirection="row" gap={8} alignItems="center">
            <Text fontSize={13}>{i + 1}.</Text>
            <Text fontSize={13}>{artist.name}</Text>
            {artist.genres?.[0] && (
              <Text fontSize={11}>{artist.genres[0]}</Text>
            )}
          </Box>
        ))}
      </Box>
    );
  }
