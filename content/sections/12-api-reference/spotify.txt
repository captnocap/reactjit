=== OVERVIEW ===
Hooks for the Spotify Web API. All hooks require an OAuth2 Bearer token. Get one at https://developer.spotify.com.

Pass the token as the first argument. Pass null to disable fetching (e.g., when the key is not yet loaded). Every hook returns `{ data, loading, error, refetch }`.

useSpotifyNowPlaying polls by default every 5 seconds. Other hooks fetch once unless you provide an interval via useAPI options.

=== API / SYNTAX ===
```tsx
import {
  useSpotifyNowPlaying,
  useSpotifyTopTracks,
  useSpotifyTopArtists,
  useSpotifyRecentTracks,
  useSpotifyPlaylists,
  useSpotifySearch,
  useSpotifyPlayback,
} from '@reactjit/apis';
```

---- useSpotifyNowPlaying

```tsx
function useSpotifyNowPlaying(
  token: string | null,
  opts?: { interval?: number }
): APIResult<SpotifyNowPlaying>
```

Returns the currently playing track. Polls every 5 seconds by default.

SpotifyNowPlaying:
| Field                   | Type               | Description                    |
|-------------------------|--------------------|--------------------------------|
| is_playing              | boolean            | Whether playback is active     |
| progress_ms             | number             | Current position in ms         |
| item                    | SpotifyTrack\|null | The track, null if none        |
| currently_playing_type  | string             | 'track', 'episode', etc.       |

---- useSpotifyTopTracks

```tsx
function useSpotifyTopTracks(
  token: string | null,
  opts?: { timeRange?: 'short_term' | 'medium_term' | 'long_term'; limit?: number }
): APIResult<SpotifyTopItem<SpotifyTrack>>
```

Returns the user's top tracks. timeRange defaults to medium_term (roughly 6 months).

---- useSpotifyTopArtists

```tsx
function useSpotifyTopArtists(
  token: string | null,
  opts?: { timeRange?: 'short_term' | 'medium_term' | 'long_term'; limit?: number }
): APIResult<SpotifyTopItem<SpotifyArtist>>
```

Returns the user's top artists. Same timeRange options as TopTracks.

---- useSpotifyRecentTracks

```tsx
function useSpotifyRecentTracks(
  token: string | null,
  opts?: { limit?: number }
): APIResult<{ items: Array<{ track: SpotifyTrack; played_at: string }> }>
```

Returns recently played tracks. Limit defaults to 20.

---- useSpotifyPlaylists

```tsx
function useSpotifyPlaylists(
  token: string | null,
  opts?: { limit?: number }
): APIResult<SpotifyPaginated<SpotifyPlaylist>>
```

Returns the user's playlists. Limit defaults to 50.

---- useSpotifySearch

```tsx
function useSpotifySearch(
  token: string | null,
  query: string | null,
  opts?: { type?: string; limit?: number }
): APIResult<{ tracks?: SpotifyPaginated<SpotifyTrack>; artists?: SpotifyPaginated<SpotifyArtist> }>
```

Search Spotify. Pass null as query to disable. Type defaults to 'track,artist'. Limit defaults to 20.

---- useSpotifyPlayback

```tsx
function useSpotifyPlayback(
  token: string | null
): { play: () => Promise; pause: () => Promise; next: () => Promise; previous: () => Promise; loading: boolean; error: Error | null }
```

Imperative playback controls. Returns mutation functions for play/pause/next/previous.

---- Types

SpotifyTrack:
| Field       | Type             | Description                     |
|-------------|------------------|---------------------------------|
| id          | string           | Spotify track ID                |
| name        | string           | Track name                      |
| artists     | SpotifyArtist[]  | Array of artists                |
| album       | SpotifyAlbum     | Album containing the track      |
| duration_ms | number           | Duration in milliseconds        |
| preview_url | string \| null   | 30-second preview URL           |
| explicit    | boolean          | Whether the track is explicit   |
| uri         | string           | Spotify URI (spotify:track:...) |

SpotifyArtist:
| Field  | Type              | Description         |
|--------|-------------------|---------------------|
| id     | string            | Spotify artist ID   |
| name   | string            | Artist name         |
| images | SpotifyImage[]?   | Artist images       |
| genres | string[]?         | Genre tags          |

SpotifyAlbum:
| Field        | Type            | Description       |
|--------------|-----------------|-------------------|
| id           | string          | Album ID          |
| name         | string          | Album name        |
| images       | SpotifyImage[]  | Cover art         |
| release_date | string          | Release date      |

SpotifyImage:
| Field  | Type   | Description  |
|--------|--------|--------------|
| url    | string | Image URL    |
| width  | number | Width in px  |
| height | number | Height in px |

=== EXAMPLES ===
Example 1: Now playing display
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useSpotifyNowPlaying } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function NowPlaying({ token }: { token: string }) {
  const colors = useThemeColors();
  const { data, loading } = useSpotifyNowPlaying(token);
  const track = data?.item;

  if (loading) return <Text style={{ color: colors.muted }}>Loading...</Text>;
  if (!track) return <Text style={{ color: colors.muted }}>Nothing playing</Text>;

  const art = track.album.images[0]?.url;
  const artist = track.artists.map(a => a.name).join(', ');

  return (
    <Box style={{ flexDirection: 'row', gap: 12, alignItems: 'center' }}>
      {art && <Image src={art} style={{ width: 48, height: 48, borderRadius: 4 }} />}
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>{track.name}</Text>
        <Text style={{ color: colors.muted, fontSize: 12 }}>{artist}</Text>
      </Box>
    </Box>
  );
}
```

Example 2: Search with results
---
```tsx
import { useState } from 'react';
import { Box, Text } from '@reactjit/core';
import { useSpotifySearch } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function SpotifySearchPanel({ token }: { token: string }) {
  const colors = useThemeColors();
  const [query, setQuery] = useState<string | null>(null);
  const { data, loading } = useSpotifySearch(token, query);

  return (
    <Box style={{ gap: 8 }}>
      {loading && <Text style={{ color: colors.muted }}>Searching...</Text>}
      {data?.tracks?.items.map(track => (
        <Text key={track.id} style={{ color: colors.text, fontSize: 13 }}>
          {`${track.name} - ${track.artists[0]?.name}`}
        </Text>
      ))}
    </Box>
  );
}
```

Example 3: Playback controls
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { useSpotifyPlayback, useSpotifyNowPlaying } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function PlayerControls({ token }: { token: string }) {
  const colors = useThemeColors();
  const { data } = useSpotifyNowPlaying(token);
  const playback = useSpotifyPlayback(token);
  const isPlaying = data?.is_playing ?? false;

  return (
    <Box style={{ flexDirection: 'row', gap: 16, alignItems: 'center' }}>
      <Pressable onPress={playback.previous}>
        <Text style={{ color: colors.text, fontSize: 16 }}>Prev</Text>
      </Pressable>
      <Pressable onPress={isPlaying ? playback.pause : playback.play}>
        <Text style={{ color: colors.primary, fontSize: 18 }}>
          {isPlaying ? 'Pause' : 'Play'}
        </Text>
      </Pressable>
      <Pressable onPress={playback.next}>
        <Text style={{ color: colors.text, fontSize: 16 }}>Next</Text>
      </Pressable>
    </Box>
  );
}
```

Example 4: Top artists with key management
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useServiceKey, useSpotifyTopArtists } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TopArtists() {
  const colors = useThemeColors();
  const { key } = useServiceKey('spotify', 'token');
  const { data } = useSpotifyTopArtists(key, { timeRange: 'short_term', limit: 5 });

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>Top Artists (4 weeks)</Text>
      {data?.items.map((artist, i) => (
        <Text key={artist.id} style={{ color: colors.muted, fontSize: 13 }}>
          {`${i + 1}. ${artist.name}`}
        </Text>
      ))}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Spotify tokens expire after 1 hour. Refresh logic is outside the scope of these hooks -- manage token refresh in your app.
- useSpotifyNowPlaying returns null for `item` when nothing is playing. Always null-check.
- useSpotifyPlayback requires an active Spotify device. If no device is active, play/pause/next/previous will return HTTP 404.
- useSpotifySearch returns both tracks and artists by default. Set `type: 'track'` or `type: 'artist'` to narrow results.
