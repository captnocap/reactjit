=== OVERVIEW ===
The NASA hooks wrap four public NASA endpoints: Astronomy Picture of the Day (APOD), Mars rover photo archives, Near-Earth Object (NEO) tracking, and EPIC satellite imagery of Earth. All hooks are read-only and built on the reactive `useAPI` base — they start fetching on mount, surface `loading`/`error`/`data`, and expose a manual `refetch`.

Auth: All endpoints accept `DEMO_KEY` for low-rate testing (30 requests/hour). Register a free API key at https://api.nasa.gov for higher limits (1000 requests/hour). Pass `null` as `apiKey` and the hooks fall back to `DEMO_KEY` automatically.

=== API / SYNTAX ===

--- useNASAApod ---

Fetch the Astronomy Picture of the Day. Pass a specific date to get a historical image, or a count to get a random selection.

  import { useNASAApod } from '@reactjit/apis/nasa';

  const result: APIResult<NASAAPOD | NASAAPOD[]> = useNASAApod(
    apiKey: string | null,
    opts?:  { date?: string; count?: number },
  );

Parameters:
  apiKey      string | null   NASA API key, or null to use DEMO_KEY
  opts.date   string          ISO date string (e.g. '2024-07-04'). Returns a single NASAAPOD.
  opts.count  number          Return this many random APODs. Returns NASAAPOD[].

Returns: APIResult<NASAAPOD | NASAAPOD[]>
  data     NASAAPOD | NASAAPOD[] | null
  loading  boolean
  error    Error | null
  refetch  () => void

NASAAPOD fields:
  date           string               ISO date
  title          string
  explanation    string               Full text description
  url            string               Image or video URL
  hdurl          string | undefined   High-resolution image URL (images only)
  media_type     'image' | 'video'
  copyright      string | undefined
  thumbnail_url  string | undefined   Video thumbnail (videos only)

--- useNASAMarsPhotos ---

Fetch photos taken by a Mars rover on a given Martian sol (day) or Earth date. Supports all four rovers and optional camera filtering.

  import { useNASAMarsPhotos } from '@reactjit/apis/nasa';

  const result: APIResult<{ photos: NASAMarsPhoto[] }> = useNASAMarsPhotos(
    apiKey: string | null,
    opts?:  {
      rover?:     'curiosity' | 'opportunity' | 'spirit' | 'perseverance';
      sol?:       number;
      earthDate?: string;
      camera?:    string;
    },
  );

Parameters:
  apiKey          string | null   NASA API key, or null for DEMO_KEY
  opts.rover      string          Which rover to query. Defaults to 'curiosity'.
  opts.sol        number          Martian sol number. Defaults to 1000 when no earthDate is given.
  opts.earthDate  string          ISO date (e.g. '2023-01-15'). Takes precedence over sol.
  opts.camera     string          Filter by camera abbreviation (e.g. 'NAVCAM', 'FHAZ', 'MAST').

Returns: APIResult<{ photos: NASAMarsPhoto[] }>
  data.photos  NASAMarsPhoto[]

NASAMarsPhoto fields:
  id          number
  sol         number
  img_src     string               Direct image URL
  earth_date  string
  camera      { id, name, full_name }
  rover       { id, name, status, landing_date, launch_date }

--- useNASANeoFeed ---

Fetch Near-Earth Object (asteroid) close-approach data for a date range. Returns a date-keyed dictionary of asteroid arrays plus a total element count.

  import { useNASANeoFeed } from '@reactjit/apis/nasa';

  const result: APIResult<NASANeoFeed> = useNASANeoFeed(
    apiKey: string | null,
    opts?:  { startDate?: string; endDate?: string },
  );

Parameters:
  apiKey           string | null   NASA API key, or null for DEMO_KEY
  opts.startDate   string          ISO start date. Defaults to today.
  opts.endDate     string          ISO end date. Maximum 7-day window enforced by NASA.

Returns: APIResult<NASANeoFeed>
  data.element_count          number
  data.near_earth_objects     Record<string, NASANeoObject[]>   Keyed by ISO date string

NASANeoObject fields:
  id                                  string
  name                                string
  nasa_jpl_url                        string
  absolute_magnitude_h                number
  is_potentially_hazardous_asteroid   boolean
  estimated_diameter.kilometers       { estimated_diameter_min, estimated_diameter_max }
  estimated_diameter.meters           { estimated_diameter_min, estimated_diameter_max }
  close_approach_data                 Array of:
    close_approach_date                 string
    relative_velocity.kilometers_per_hour   string
    miss_distance.kilometers               string
    miss_distance.lunar                    string

--- useNASAEPIC ---

Fetch EPIC (Earth Polychromatic Imaging Camera) imagery. EPIC is a camera aboard the DSCOVR spacecraft that images the full sunlit face of Earth roughly every two hours.

  import { useNASAEPIC } from '@reactjit/apis/nasa';

  const result: APIResult<NASAEPICImage[]> = useNASAEPIC(
    apiKey: string | null,
    opts?:  { collection?: 'natural' | 'enhanced'; date?: string },
  );

Parameters:
  apiKey             string | null              NASA API key, or null for DEMO_KEY
  opts.collection    'natural' | 'enhanced'     Color collection. Defaults to 'natural'.
  opts.date          string                     ISO date. Omit for the most recent available.

Returns: APIResult<NASAEPICImage[]>
  data  NASAEPICImage[]

NASAEPICImage fields:
  identifier   string
  caption      string
  image        string   Image filename (without extension) — pass to nasaEPICImageUrl()
  date         string   ISO datetime string
  coords.centroid_coordinates   { lat: number; lon: number }

--- nasaEPICImageUrl (utility) ---

Build a full PNG URL for an EPIC image object. The EPIC archive URL structure encodes the date into directory segments, so this helper does the transformation for you.

  import { nasaEPICImageUrl } from '@reactjit/apis/nasa';

  const url: string = nasaEPICImageUrl(
    image:       NASAEPICImage,
    collection?: 'natural' | 'enhanced',
  );

Parameters:
  image       NASAEPICImage   An item from the useNASAEPIC result array
  collection  string          Defaults to 'natural'

Returns: string   Full PNG URL, e.g. 'https://epic.gsfc.nasa.gov/archive/natural/2024/07/04/png/epic_1b_20240704120022.png'

=== EXAMPLES ===

-- Astronomy Picture of the Day widget --

Fetch today's APOD and display the title and description. Show a note if it is a video.

  import { useNASAApod } from '@reactjit/apis/nasa';
  import { Box, Text, Image } from '@reactjit/core';

  function APODWidget({ apiKey }: { apiKey: string | null }) {
    const { data, loading, error } = useNASAApod(apiKey);

    if (loading) return <Text fontSize={14}>Loading APOD...</Text>;
    if (error) return <Text fontSize={14}>Failed: {error.message}</Text>;

    // When no count is passed, data is a single NASAAPOD object
    const apod = Array.isArray(data) ? data[0] : data;
    if (!apod) return null;

    return (
      <Box width="100%" gap={8} padding={16}>
        {apod.media_type === 'image' && (
          <Image src={apod.url} width={320} height={180} />
        )}
        {apod.media_type === 'video' && (
          <Box width={320} height={180} backgroundColor="#111" justifyContent="center" alignItems="center">
            <Text fontSize={12}>Video — open in browser</Text>
          </Box>
        )}
        <Text fontSize={16}>{apod.title}</Text>
        <Text fontSize={11}>{apod.date}</Text>
        <Text fontSize={12}>{apod.explanation.slice(0, 200)}...</Text>
      </Box>
    );
  }

-- Near-Earth object hazard list --

Show today's close-approaching asteroids and flag potentially hazardous ones.

  import { useNASANeoFeed } from '@reactjit/apis/nasa';
  import { Box, Text } from '@reactjit/core';

  function AsteroidToday({ apiKey }: { apiKey: string | null }) {
    const { data, loading } = useNASANeoFeed(apiKey);

    if (loading || !data) return <Text fontSize={14}>Scanning near-Earth objects...</Text>;

    const today = new Date().toISOString().slice(0, 10);
    const asteroids = data.near_earth_objects[today] ?? [];

    return (
      <Box width="100%" gap={6} padding={12}>
        <Text fontSize={16}>Near-Earth Objects: {today}</Text>
        <Text fontSize={12}>Total tracked today: {data.element_count}</Text>
        {asteroids.map((neo) => (
          <Box
            key={neo.id}
            flexDirection="row"
            gap={10}
            padding={8}
            backgroundColor={neo.is_potentially_hazardous_asteroid ? '#3d1010' : '#1a1a1a'}
          >
            <Text fontSize={13}>{neo.name}</Text>
            <Text fontSize={11}>
              Miss: {Number(neo.close_approach_data[0]?.miss_distance.lunar).toFixed(2)} LD
            </Text>
            {neo.is_potentially_hazardous_asteroid && (
              <Text fontSize={11}>HAZARDOUS</Text>
            )}
          </Box>
        ))}
      </Box>
    );
  }
