=== OVERVIEW ===
The CoinGecko hooks wrap the CoinGecko API v3 for cryptocurrency data. No authentication is required for the free tier. An optional API key (passed via opts.apiKey) is sent as the x-cg-demo-api-key header for higher rate limits.

Six hooks cover the most common use cases: simple price lookups, market rankings, detailed coin info, historical price charts, trending coins, and global market statistics.

Coin IDs use CoinGecko's slug format (e.g. "bitcoin", "ethereum", "solana"). Use the /coins/list endpoint or the CoinGecko website to find the correct ID for a coin.

=== API / SYNTAX ===
```tsx
import {
  useCoinPrice,
  useCoinMarkets,
  useCoinDetail,
  useCoinHistory,
  useCoinTrending,
  useCoinGlobal,
} from '@reactjit/apis';
```

---- useCoinPrice

```tsx
function useCoinPrice(
  coinIds: string | string[] | null,
  opts?: { vs?: string; include24hChange?: boolean; apiKey?: string | null }
): APIResult<CoinPrice>
```

| Param                | Type                        | Default | Description                            |
|----------------------|-----------------------------|---------|----------------------------------------|
| coinIds              | string \| string[] \| null  | required| Coin ID(s) to look up                  |
| opts.vs              | string                      | "usd"  | Target currency code                    |
| opts.include24hChange| boolean                     | true   | Include 24-hour price change percentage |

Returns a map of coin ID to price data: { bitcoin: { usd: 45000, usd_24h_change: 2.5 } }.

---- useCoinMarkets

```tsx
function useCoinMarkets(
  opts?: { vs?: string; perPage?: number; page?: number; sparkline?: boolean; apiKey?: string | null }
): APIResult<CoinMarket[]>
```

| Param          | Type    | Default | Description                              |
|----------------|---------|---------|------------------------------------------|
| opts.vs        | string  | "usd"  | Target currency                           |
| opts.perPage   | number  | 20     | Results per page                          |
| opts.page      | number  | 1      | Page number                               |
| opts.sparkline | boolean | false  | Include 7-day sparkline price array       |

Returns coins sorted by market cap descending.

---- useCoinDetail

```tsx
function useCoinDetail(
  coinId: string | null,
  opts?: { apiKey?: string | null }
): APIResult<CoinDetail>
```

Returns detailed information for a single coin including description, images, market data across currencies, links, and categories.

---- useCoinHistory

```tsx
function useCoinHistory(
  coinId: string | null,
  opts?: { vs?: string; days?: number | 'max'; apiKey?: string | null }
): APIResult<CoinHistory>
```

| Param     | Type              | Default | Description                              |
|-----------|-------------------|---------|------------------------------------------|
| coinId    | string \| null    | required| Coin ID                                   |
| opts.vs   | string            | "usd"  | Target currency                           |
| opts.days | number \| 'max'   | 7      | Days of history (1, 7, 30, 90, 365, max) |

Returns arrays of [timestamp, value] pairs for prices, market_caps, and total_volumes.

---- useCoinTrending

```tsx
function useCoinTrending(
  opts?: { apiKey?: string | null }
): APIResult<CoinTrending>
```

Returns the top-7 trending coins on CoinGecko in the last 24 hours.

---- useCoinGlobal

```tsx
function useCoinGlobal(
  opts?: { apiKey?: string | null }
): APIResult<CoinGlobal>
```

Returns global cryptocurrency market statistics: total market cap, volume, dominance percentages, active exchanges count.

---- Types

CoinMarket:
| Field                         | Type           | Description                          |
|-------------------------------|----------------|--------------------------------------|
| id                            | string         | CoinGecko coin ID                    |
| symbol                        | string         | Ticker symbol (btc, eth, etc.)       |
| name                          | string         | Full coin name                       |
| image                         | string         | Coin logo URL                        |
| current_price                 | number         | Current price in target currency     |
| market_cap                    | number         | Market capitalization                |
| market_cap_rank               | number         | Rank by market cap                   |
| price_change_percentage_24h   | number         | 24-hour price change (%)             |
| high_24h                      | number         | 24-hour high                         |
| low_24h                       | number         | 24-hour low                          |
| circulating_supply            | number         | Circulating supply                   |
| ath                           | number         | All-time high price                  |
| sparkline_in_7d               | object         | 7-day sparkline ({ price: number[] })|

=== EXAMPLES ===
Example 1: Simple price ticker
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useCoinPrice } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function PriceTicker() {
  const colors = useThemeColors();
  const { data, loading } = useCoinPrice(['bitcoin', 'ethereum', 'solana']);

  if (loading) return <Text style={{ color: colors.muted }}>Loading prices...</Text>;

  return (
    <Box style={{ flexDirection: 'row', gap: 16 }}>
      {data && Object.entries(data).map(([coin, prices]) => (
        <Box key={coin} style={{ alignItems: 'center' }}>
          <Text style={{ color: colors.text, fontWeight: 'bold' }}>{coin.toUpperCase()}</Text>
          <Text style={{ color: colors.text }}>{`$${prices.usd.toLocaleString()}`}</Text>
          <Text style={{ color: prices.usd_24h_change >= 0 ? '#22c55e' : '#ef4444', fontSize: 12 }}>
            {`${prices.usd_24h_change >= 0 ? '+' : ''}${prices.usd_24h_change.toFixed(2)}%`}
          </Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 2: Market rankings table
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useCoinMarkets } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function MarketTable() {
  const colors = useThemeColors();
  const { data: coins, loading } = useCoinMarkets({ perPage: 10 });

  if (loading) return <Text style={{ color: colors.muted }}>Loading markets...</Text>;

  return (
    <Box style={{ gap: 4 }}>
      {coins?.map(coin => (
        <Box key={coin.id} style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
          <Text style={{ color: colors.muted, width: 24 }}>{`${coin.market_cap_rank}`}</Text>
          <Image src={coin.image} style={{ width: 20, height: 20 }} />
          <Text style={{ color: colors.text, width: 100 }}>{coin.name}</Text>
          <Text style={{ color: colors.text, flexGrow: 1, textAlign: 'right' }}>
            {`$${coin.current_price.toLocaleString()}`}
          </Text>
          <Text style={{
            color: coin.price_change_percentage_24h >= 0 ? '#22c55e' : '#ef4444',
            width: 70,
            textAlign: 'right',
          }}>
            {`${coin.price_change_percentage_24h >= 0 ? '+' : ''}${coin.price_change_percentage_24h.toFixed(2)}%`}
          </Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 3: Price history for a chart
---
```tsx
import { Box, Text } from '@reactjit/core';
import { Sparkline } from '@reactjit/core';
import { useCoinHistory } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function CoinChart({ coinId }: { coinId: string }) {
  const colors = useThemeColors();
  const { data, loading } = useCoinHistory(coinId, { days: 30 });

  if (loading) return <Text style={{ color: colors.muted }}>Loading chart...</Text>;

  const prices = data?.prices?.map(([, price]) => price) ?? [];

  return (
    <Box style={{ gap: 8 }}>
      <Text style={{ color: colors.text, fontWeight: 'bold' }}>{`${coinId} - 30 Day Price`}</Text>
      <Sparkline data={prices} color={colors.primary} style={{ height: 100, width: '100%' }} />
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Free tier rate limit is approximately 10-30 requests/minute. Pass an API key for higher limits.
- Coin IDs are CoinGecko slugs, not ticker symbols. Use "bitcoin" not "BTC", "ethereum" not "ETH".
- useCoinPrice accepts a single string or an array of strings. Joins them with commas internally.
- useCoinHistory days parameter affects data granularity: 1 day = 5-minute intervals, 1-90 days = hourly, >90 days = daily.
