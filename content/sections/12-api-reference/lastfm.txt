=== OVERVIEW ===
Hooks for the Last.fm API. Auth uses an API key passed as a query parameter. Get one at https://www.last.fm/api/account/create.

All hooks require an API key and a Last.fm username. Pass null for either to disable.

useLastFMNowPlaying polls every 10 seconds by default. Other hooks fetch once.

The `lastfmImage()` helper extracts the best image URL from Last.fm's image array format.

=== API / SYNTAX ===
```tsx
import {
  useLastFMRecentTracks,
  useLastFMNowPlaying,
  useLastFMTopArtists,
  useLastFMTopTracks,
  useLastFMTopAlbums,
  useLastFMUser,
  lastfmImage,
} from '@reactjit/apis';
```

---- useLastFMRecentTracks

```tsx
function useLastFMRecentTracks(
  apiKey: string | null,
  user: string | null,
  opts?: { limit?: number; interval?: number }
): APIResult<{ recenttracks: { track: LastFMTrack[] } }>
```

Fetch recent scrobbles. Limit defaults to 20.

---- useLastFMNowPlaying

```tsx
function useLastFMNowPlaying(
  apiKey: string | null,
  user: string | null,
  opts?: { interval?: number }
): APIResult<{ recenttracks: { track: LastFMTrack[] } }>
```

Fetch the currently playing track (or most recent scrobble). Polls every 10 seconds by default. The now-playing track has `track[0]['@attr']?.nowplaying === 'true'`.

---- useLastFMTopArtists

```tsx
function useLastFMTopArtists(
  apiKey: string | null,
  user: string | null,
  opts?: { period?: 'overall' | '7day' | '1month' | '3month' | '6month' | '12month'; limit?: number }
): APIResult<{ topartists: { artist: LastFMArtist[] } }>
```

Fetch top artists for a time period. Limit defaults to 10.

---- useLastFMTopTracks

```tsx
function useLastFMTopTracks(
  apiKey: string | null,
  user: string | null,
  opts?: { period?: 'overall' | '7day' | '1month' | '3month' | '6month' | '12month'; limit?: number }
): APIResult<{ toptracks: { track: LastFMTrack[] } }>
```

Fetch top tracks for a time period. Limit defaults to 10.

---- useLastFMTopAlbums

```tsx
function useLastFMTopAlbums(
  apiKey: string | null,
  user: string | null,
  opts?: { period?: 'overall' | '7day' | '1month' | '3month' | '6month' | '12month'; limit?: number }
): APIResult<{ topalbums: { album: LastFMAlbum[] } }>
```

Fetch top albums for a time period. Limit defaults to 10.

---- useLastFMUser

```tsx
function useLastFMUser(
  apiKey: string | null,
  user: string | null
): APIResult<{ user: LastFMUser }>
```

Fetch a user's profile (playcount, registration date, country).

---- lastfmImage

```tsx
function lastfmImage(
  images: LastFMImage[] | undefined,
  preferSize?: string
): string | null
```

Extract the best image URL from a Last.fm image array. Prefers 'extralarge' by default. Returns null if no images available.

---- Types

LastFMTrack:
| Field     | Type                           | Description                          |
|-----------|--------------------------------|--------------------------------------|
| name      | string                         | Track name                           |
| artist    | LastFMArtist \| { '#text' }     | Artist name (format varies by endpoint) |
| album     | { '#text': string }?           | Album name                           |
| image     | LastFMImage[]?                 | Album art                            |
| url       | string                         | Last.fm track URL                    |
| date      | { uts, '#text' }?              | Scrobble timestamp (absent if now playing) |
| @attr     | { nowplaying: string }?        | Present if currently playing         |
| playcount | string?                        | Play count (top tracks only)         |

LastFMArtist:
| Field     | Type            | Description              |
|-----------|-----------------|--------------------------|
| name      | string          | Artist name              |
| mbid      | string?         | MusicBrainz ID           |
| url       | string          | Last.fm artist URL       |
| image     | LastFMImage[]?  | Artist images            |
| playcount | string?         | Scrobble count           |

LastFMAlbum:
| Field     | Type            | Description              |
|-----------|-----------------|--------------------------|
| name      | string          | Album name               |
| artist    | object          | Artist name/object       |
| image     | LastFMImage[]?  | Album art                |
| playcount | string?         | Scrobble count           |
| url       | string          | Last.fm album URL        |

LastFMUser:
| Field      | Type           | Description            |
|------------|----------------|------------------------|
| name       | string         | Username               |
| realname   | string         | Display name           |
| image      | LastFMImage[]  | Profile images         |
| playcount  | string         | Total scrobble count   |
| registered | { unixtime }   | Registration timestamp |
| url        | string         | Profile URL            |
| country    | string         | Country                |

LastFMImage:
| Field  | Type   | Description                                       |
|--------|--------|---------------------------------------------------|
| #text  | string | Image URL                                         |
| size   | string | 'small', 'medium', 'large', 'extralarge', 'mega'  |

=== EXAMPLES ===
Example 1: Now playing with album art
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useLastFMNowPlaying, lastfmImage } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function LastFMNow({ apiKey, user }: { apiKey: string; user: string }) {
  const colors = useThemeColors();
  const { data } = useLastFMNowPlaying(apiKey, user);
  const track = data?.recenttracks?.track?.[0];

  if (!track) return <Text style={{ color: colors.muted }}>No data</Text>;

  const isPlaying = track['@attr']?.nowplaying === 'true';
  const art = lastfmImage(track.image);
  const artist = typeof track.artist === 'string' ? track.artist
    : '#text' in track.artist ? track.artist['#text'] : track.artist.name;

  return (
    <Box style={{ flexDirection: 'row', gap: 10, alignItems: 'center' }}>
      {art && <Image src={art} style={{ width: 48, height: 48, borderRadius: 4 }} />}
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontSize: 13, fontWeight: 'bold' }}>{track.name}</Text>
        <Text style={{ color: colors.muted, fontSize: 11 }}>{artist}</Text>
        <Text style={{ color: isPlaying ? colors.primary : colors.muted, fontSize: 10 }}>
          {isPlaying ? 'Now Playing' : 'Last Played'}
        </Text>
      </Box>
    </Box>
  );
}
```

Example 2: Top artists list
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useLastFMTopArtists } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TopArtists({ apiKey, user }: { apiKey: string; user: string }) {
  const colors = useThemeColors();
  const { data } = useLastFMTopArtists(apiKey, user, { period: '1month', limit: 10 });

  const artists = data?.topartists?.artist ?? [];

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>Top Artists (1 month)</Text>
      {artists.map((artist, i) => (
        <Text key={artist.name} style={{ color: colors.text, fontSize: 12 }}>
          {`${i + 1}. ${artist.name} (${artist.playcount} plays)`}
        </Text>
      ))}
    </Box>
  );
}
```

Example 3: User profile stats
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useLastFMUser } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function UserStats({ apiKey, user }: { apiKey: string; user: string }) {
  const colors = useThemeColors();
  const { data } = useLastFMUser(apiKey, user);
  const profile = data?.user;

  if (!profile) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>{profile.realname || profile.name}</Text>
      <Text style={{ color: colors.muted, fontSize: 12 }}>
        {`${parseInt(profile.playcount).toLocaleString()} scrobbles | ${profile.country}`}
      </Text>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Last.fm returns artist names in different formats depending on the endpoint. In recent tracks, `track.artist` is `{ '#text': 'name' }`. In top artists, it is `{ name: 'name' }`. Always check the shape.
- The `@attr.nowplaying` field is a string `'true'`, not a boolean. Check with `=== 'true'`.
- playcount values are returned as strings, not numbers. Parse with `parseInt()` if you need arithmetic.
- Last.fm image arrays can have empty `'#text'` values. lastfmImage() handles this by returning null.
