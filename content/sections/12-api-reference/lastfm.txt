=== OVERVIEW ===
The Last.fm hooks wrap the Last.fm REST API (ws.audioscrobbler.com/2.0). Authentication uses an API key passed as a query parameter — get one at https://www.last.fm/api/account/create. No OAuth is required for read-only user data.

All hooks take the API key and a username as the first two arguments. Passing null for either skips the request. Hooks that display live "now playing" state accept an interval option and poll on a timer.

Last.fm's JSON responses use unconventional field names (artist fields may be objects or plain strings depending on context, images use '#text' as the URL key). The types reflect the actual API response shape. The package also exports a lastfmImage() utility to extract the best available image URL from an image array.

=== API / SYNTAX ===

--- useLastFMRecentTracks ---

Fetch a user's recently scrobbled tracks. The first item may be a currently-playing track (check for '@attr'.nowplaying).

  import { useLastFMRecentTracks } from '@reactjit/apis/lastfm';

  const result: APIResult<{ recenttracks: { track: LastFMTrack[] } }> =
    useLastFMRecentTracks(apiKey, user, opts?);

Parameters:
  apiKey   string | null
  user     string | null   Last.fm username. Pass null to skip.
  opts     object (optional)
    limit     number   Max tracks to return. Default: 20.
    interval  number   Polling interval in ms. No polling if omitted.

Returns: APIResult<{ recenttracks: { track: LastFMTrack[] } }>
  data.recenttracks.track   LastFMTrack[]

--- useLastFMNowPlaying ---

Poll for the track a user is currently scrobbling. Returns the same response shape as useLastFMRecentTracks (limit 1). The now-playing track has '@attr'.nowplaying === 'true'; a recently-played track has a date field instead.

  import { useLastFMNowPlaying } from '@reactjit/apis/lastfm';

  const result: APIResult<{ recenttracks: { track: LastFMTrack[] } }> =
    useLastFMNowPlaying(apiKey, user, opts?);

Parameters:
  apiKey   string | null
  user     string | null
  opts     object (optional)
    interval  number   Polling interval in ms. Default: 10000.

Returns: APIResult<{ recenttracks: { track: LastFMTrack[] } }>

--- useLastFMTopArtists ---

Fetch a user's top artists over a time period.

  import { useLastFMTopArtists } from '@reactjit/apis/lastfm';

  const result: APIResult<{ topartists: { artist: LastFMArtist[] } }> =
    useLastFMTopArtists(apiKey, user, opts?);

Parameters:
  apiKey   string | null
  user     string | null
  opts     object (optional)
    period  'overall' | '7day' | '1month' | '3month' | '6month' | '12month'   Default: none (API default: overall)
    limit   number   Default: 10.

Returns: APIResult<{ topartists: { artist: LastFMArtist[] } }>
  data.topartists.artist   LastFMArtist[]   Ordered by play count descending.

--- useLastFMTopTracks ---

Fetch a user's top tracks over a time period.

  import { useLastFMTopTracks } from '@reactjit/apis/lastfm';

  const result: APIResult<{ toptracks: { track: LastFMTrack[] } }> =
    useLastFMTopTracks(apiKey, user, opts?);

Parameters:
  apiKey   string | null
  user     string | null
  opts     object (optional)
    period  'overall' | '7day' | '1month' | '3month' | '6month' | '12month'
    limit   number   Default: 10.

Returns: APIResult<{ toptracks: { track: LastFMTrack[] } }>

--- useLastFMTopAlbums ---

Fetch a user's top albums over a time period.

    useLastFMTopAlbums(apiKey, user, opts?);

Parameters:
  apiKey   string | null
  user     string | null
  opts     object (optional)
    period  'overall' | '7day' | '1month' | '3month' | '6month' | '12month'
    limit   number   Default: 10.

Returns: APIResult<{ topalbums: { album: LastFMAlbum[] } }>

--- useLastFMUser ---

Fetch a user's profile information.

  import { useLastFMUser } from '@reactjit/apis/lastfm';

  const result: APIResult<{ user: LastFMUser }> = useLastFMUser(apiKey, user);

Parameters:
  apiKey   string | null
  user     string | null

Returns: APIResult<{ user: LastFMUser }>

LastFMUser fields:
  name        string
  realname    string
  image       LastFMImage[]
  playcount   string   Total scrobble count (returned as a string by the API)
  registered  { unixtime: string }
  url         string
  country     string

--- lastfmImage ---

Extract the best available image URL from a Last.fm image array. Last.fm returns images as an array of { '#text': url; size: ... } objects. This utility picks the preferred size and falls back to the last available entry.

  import { lastfmImage } from '@reactjit/apis/lastfm';

  const url: string | null = lastfmImage(images, preferSize?);

Parameters:
  images      LastFMImage[] | undefined
  preferSize  string   Image size to prefer. Default: 'extralarge'. Options: 'small', 'medium', 'large', 'extralarge', 'mega'.

Returns: string | null — null if no images or all URLs are empty.

=== SHARED TYPES ===

LastFMTrack fields:
  name     string
  artist   LastFMArtist | { '#text': string }   Shape depends on endpoint
  album    { '#text': string } (optional)
  image    LastFMImage[] (optional)
  url      string
  date     { uts: string; '#text': string } (optional)   Absent on now-playing tracks
  '@attr'  { nowplaying: string } (optional)   Present on currently-playing track
  playcount string (optional)

LastFMArtist fields:
  name      string
  mbid      string (optional)   MusicBrainz ID
  url       string
  image     LastFMImage[] (optional)
  playcount string (optional)

LastFMAlbum fields:
  name      string
  artist    LastFMArtist | { name: string }
  image     LastFMImage[] (optional)
  playcount string (optional)
  url       string

LastFMImage fields:
  '#text'  string   Image URL
  size     'small' | 'medium' | 'large' | 'extralarge' | 'mega'

=== EXAMPLES ===

-- Now playing display --

Show what a user is currently scrobbling, or the last played track if nothing is active:

  import { useLastFMNowPlaying, lastfmImage } from '@reactjit/apis/lastfm';
  import { Box, Text } from '@reactjit/core';

  const API_KEY = process.env.LASTFM_KEY ?? null;

  function NowPlaying({ username }: { username: string }) {
    const { data } = useLastFMNowPlaying(API_KEY, username, { interval: 10000 });

    const tracks = data?.recenttracks?.track ?? [];
    const track = tracks[0];

    if (!track) return <Text fontSize={13}>No recent activity</Text>;

    const isPlaying = track['@attr']?.nowplaying === 'true';
    const artistName = typeof track.artist === 'object' && '#text' in track.artist
      ? track.artist['#text']
      : (track.artist as any).name ?? '';

    return (
      <Box flexDirection="row" gap={12} alignItems="center" padding={12}>
        <Box width={48} height={48} backgroundColor="#222" />
        <Box gap={4}>
          <Text fontSize={14}>{track.name}</Text>
          <Text fontSize={12}>{artistName}</Text>
          {track.album && <Text fontSize={11}>{track.album['#text']}</Text>}
          <Text fontSize={11}>{isPlaying ? 'Now playing' : 'Last played'}</Text>
        </Box>
      </Box>
    );
  }

-- Top artists over the past month --

Render a ranked list of the user's top artists for the past month with scrobble counts:

  import { useLastFMTopArtists } from '@reactjit/apis/lastfm';
  import { Box, Text } from '@reactjit/core';

  function TopArtists({ apiKey, username }: { apiKey: string; username: string }) {
    const { data, loading, error } = useLastFMTopArtists(apiKey, username, {
      period: '1month',
      limit: 10,
    });

    if (loading) return <Text fontSize={13}>Loading...</Text>;
    if (error) return <Text fontSize={13}>Error: {error.message}</Text>;

    const artists = data?.topartists?.artist ?? [];

    return (
      <Box gap={6} padding={12}>
        <Text fontSize={15}>Top Artists — Past Month</Text>
        {artists.map((artist, i) => (
          <Box key={artist.name} flexDirection="row" gap={8} alignItems="center">
            <Text fontSize={12}>{i + 1}.</Text>
            <Box flexGrow={1}>
              <Text fontSize={13}>{artist.name}</Text>
            </Box>
            {artist.playcount && (
              <Text fontSize={12}>{artist.playcount} plays</Text>
            )}
          </Box>
        ))}
      </Box>
    );
  }
