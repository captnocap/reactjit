=== OVERVIEW ===
The Google hooks in @reactjit/apis cover two Google APIs: Calendar and Sheets. Both use OAuth2 Bearer tokens. Obtain tokens via Google's OAuth2 flow (https://developers.google.com/identity/protocols/oauth2) and store them in your environment or a secure config file.

Required OAuth scopes:
  Calendar read: https://www.googleapis.com/auth/calendar.readonly
  Calendar write: https://www.googleapis.com/auth/calendar
  Sheets read:    https://www.googleapis.com/auth/spreadsheets.readonly
  Sheets write:   https://www.googleapis.com/auth/spreadsheets

Read hooks are reactive and fetch on mount. Mutation hooks (useGoogleCalendarMutation, useGoogleSheetMutation) return imperative functions and do not fetch automatically. All hooks accept null as the token to suppress fetching safely.

=== API / SYNTAX ===

--- useGoogleCalendars ---

Fetch all calendars in the authenticated user's calendar list.

  import { useGoogleCalendars } from '@reactjit/apis';

  const result: APIResult<{ items: GoogleCalendar[] }> = useGoogleCalendars(
    token: string | null,
  );

Parameters:
  token  string | null  OAuth2 Bearer token

Returns: APIResult<{ items: GoogleCalendar[] }>

GoogleCalendar fields:
  id               string   Calendar ID (e.g. 'primary' or a full email-style ID)
  summary          string   Calendar name
  description      string | undefined
  backgroundColor  string | undefined   Hex color set in Google Calendar UI
  foregroundColor  string | undefined
  primary          boolean | undefined  True for the user's primary calendar
  timeZone         string   IANA timezone string

--- useGoogleCalendarEvents ---

Fetch events from a calendar with optional time window and result limit. Defaults to upcoming events starting from now.

  import { useGoogleCalendarEvents } from '@reactjit/apis';

  const result: APIResult<GoogleCalendarList> = useGoogleCalendarEvents(
    token: string | null,
    opts?: {
      calendarId?: string;
      timeMin?: string;
      timeMax?: string;
      maxResults?: number;
      singleEvents?: boolean;
      orderBy?: 'startTime' | 'updated';
    },
  );

Parameters:
  token  string | null   OAuth2 Bearer token
  opts   object | undefined
    calendarId    string    Calendar to query (default: 'primary')
    timeMin       string    ISO 8601 lower bound for events (default: now)
    timeMax       string    ISO 8601 upper bound for events
    maxResults    number    Max events to return (default: 50)
    singleEvents  boolean   Expand recurring events into instances (default: true)
    orderBy       string    'startTime' or 'updated' (default: 'startTime')

Returns: APIResult<GoogleCalendarList>

GoogleCalendarList fields:
  items          GoogleCalendarEvent[]
  nextPageToken  string | undefined  Present when more events exist
  summary        string              Calendar name
  timeZone       string

GoogleCalendarEvent fields:
  id           string
  summary      string              Event title
  description  string | undefined
  location     string | undefined
  start        { dateTime?: string; date?: string; timeZone?: string }
               dateTime for timed events, date for all-day events
  end          { dateTime?: string; date?: string; timeZone?: string }
  status       string              'confirmed', 'tentative', or 'cancelled'
  htmlLink     string              Link to event in Google Calendar
  creator      { email: string; displayName?: string } | undefined
  attendees    Array<{ email: string; displayName?: string; responseStatus: string }> | undefined
  recurrence   string[] | undefined   RRULE strings for recurring events
  colorId      string | undefined

--- useGoogleCalendarMutation ---

Get imperative operations for creating and deleting calendar events.

  import { useGoogleCalendarMutation } from '@reactjit/apis';

  const mutation = useGoogleCalendarMutation(token: string | null);

Parameters:
  token  string | null  OAuth2 Bearer token

Returns: object
  createEvent  (calendarId: string, event: { summary: string; start: any; end: any; description?: string; location?: string }) => Promise<any>
               Create a new calendar event. start and end follow the GoogleCalendarEvent format.
  deleteEvent  (calendarId: string, eventId: string) => Promise<any>
               Delete an event by ID.
  loading      boolean        True while a mutation is in flight
  error        Error | null   Last error, or null

--- useGoogleSheet ---

Fetch spreadsheet metadata: sheet names, IDs, and document properties.

  import { useGoogleSheet } from '@reactjit/apis';

  const result: APIResult<GoogleSheet> = useGoogleSheet(
    token: string | null,
    spreadsheetId: string | null,
  );

Parameters:
  token          string | null  OAuth2 Bearer token
  spreadsheetId  string | null  Spreadsheet ID from the URL

Returns: APIResult<GoogleSheet>

GoogleSheet fields:
  spreadsheetId  string
  properties     { title: string; locale: string; timeZone: string }
  sheets         Array<{ properties: { sheetId: number; title: string; index: number } }>

--- useGoogleSheetValues ---

Read cell values from a range in a spreadsheet. Values are returned as a 2D string array.

  import { useGoogleSheetValues } from '@reactjit/apis';

  const result: APIResult<GoogleSheetValues> = useGoogleSheetValues(
    token: string | null,
    spreadsheetId: string | null,
    range: string | null,
  );

Parameters:
  token          string | null  OAuth2 Bearer token
  spreadsheetId  string | null  Spreadsheet ID
  range          string | null  A1 notation range (e.g. 'Sheet1!A1:D20' or 'A:C')

Returns: APIResult<GoogleSheetValues>

GoogleSheetValues fields:
  range           string      Canonical range returned by the API
  majorDimension  string      'ROWS' (values[i] is row i) or 'COLUMNS'
  values          string[][]  2D array of cell values as strings

--- useGoogleSheetMutation ---

Get imperative operations for appending and updating spreadsheet data.

  import { useGoogleSheetMutation } from '@reactjit/apis';

  const mutation = useGoogleSheetMutation(token: string | null);

Parameters:
  token  string | null  OAuth2 Bearer token

Returns: object
  append   (spreadsheetId: string, range: string, values: string[][]) => Promise<any>
           Append rows after the last row of data in the given range.
           values is a 2D array where each inner array is a row of cell values.
           Uses USER_ENTERED input mode (formulas and dates are parsed).
  update   (spreadsheetId: string, range: string, values: string[][]) => Promise<any>
           Overwrite the specified range with new values. The range must be large enough
           to hold all provided values. Uses USER_ENTERED input mode.
  loading  boolean        True while a mutation is in flight
  error    Error | null   Last error, or null

=== EXAMPLES ===

-- Upcoming events list --

Show the next 5 events from the primary calendar:

  import { useGoogleCalendarEvents } from '@reactjit/apis';
  import { Box, Text } from '@reactjit/core';

  const TOKEN = process.env.GOOGLE_TOKEN;

  function UpcomingEvents() {
    const { data, loading } = useGoogleCalendarEvents(TOKEN, {
      calendarId: 'primary',
      maxResults: 5,
      singleEvents: true,
      orderBy: 'startTime',
    });

    if (loading) return <Text fontSize={13}>Loading calendar...</Text>;

    const events = data?.items ?? [];

    return (
      <Box gap={8} width="100%">
        <Text fontSize={14}>Upcoming</Text>
        {events.map((event) => {
          // All-day events use event.start.date; timed events use event.start.dateTime
          const startStr = event.start.dateTime
            ? new Date(event.start.dateTime).toLocaleString([], { hour: '2-digit', minute: '2-digit' })
            : event.start.date ?? '';

          return (
            <Box key={event.id} flexDirection="row" gap={12} padding={10} backgroundColor="#1a1a2e">
              <Text fontSize={12}>{startStr}</Text>
              <Text fontSize={13}>{event.summary}</Text>
            </Box>
          );
        })}
      </Box>
    );
  }

-- Read a budget sheet and log to dashboard --

Fetch values from a named range in a Google Sheet and render a simple table:

  import { useGoogleSheetValues } from '@reactjit/apis';
  import { Box, Text } from '@reactjit/core';

  const TOKEN = process.env.GOOGLE_TOKEN;
  const SHEET_ID = 'your-spreadsheet-id-here';

  function BudgetTable() {
    // Reads columns A through C starting at row 1
    const { data, loading } = useGoogleSheetValues(TOKEN, SHEET_ID, 'Sheet1!A1:C20');

    if (loading) return <Text fontSize={13}>Loading sheet...</Text>;

    const rows = data?.values ?? [];
    // First row is treated as header
    const [header, ...body] = rows;

    return (
      <Box width="100%">
        {header && (
          <Box flexDirection="row" gap={16} padding={8} backgroundColor="#222">
            {header.map((col, i) => (
              <Box key={i} flex={1}>
                <Text fontSize={12}>{col}</Text>
              </Box>
            ))}
          </Box>
        )}
        {body.map((row, rowIdx) => (
          <Box key={rowIdx} flexDirection="row" gap={16} padding={8} backgroundColor={rowIdx % 2 === 0 ? '#111' : '#161616'}>
            {row.map((cell, colIdx) => (
              <Box key={colIdx} flex={1}>
                <Text fontSize={12}>{cell}</Text>
              </Box>
            ))}
          </Box>
        ))}
      </Box>
    );
  }
