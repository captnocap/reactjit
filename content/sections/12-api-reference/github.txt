=== OVERVIEW ===
The GitHub hooks wrap the GitHub REST API v3. All hooks work without authentication (public data only) but accept an optional Bearer token for higher rate limits and access to private data. Tokens are personal access tokens created at https://github.com/settings/tokens.

All hooks return APIResult<T> with data, loading, error, and refetch. A null username or null owner/repo pair skips the request. The Accept header is set to application/vnd.github+json on every request.

=== API / SYNTAX ===

--- useGitHubUser ---

Fetch a public GitHub user's profile.

  import { useGitHubUser } from '@reactjit/apis/github';

  const result: APIResult<GitHubUser> = useGitHubUser(username, token?);

Parameters:
  username   string | null   GitHub username. Pass null to skip.
  token      string | null   Optional Bearer token for authenticated requests.

Returns: APIResult<GitHubUser>

GitHubUser fields:
  login         string
  id            number
  avatar_url    string
  name          string | null
  bio           string | null
  public_repos  number
  followers     number
  following     number
  created_at    string   ISO 8601 timestamp
  html_url      string

--- useGitHubAuthUser ---

Fetch the profile of the currently authenticated user (requires a valid token).

  import { useGitHubAuthUser } from '@reactjit/apis/github';

  const result: APIResult<GitHubUser> = useGitHubAuthUser(token);

Parameters:
  token   string | null   Bearer token. Pass null to skip.

Returns: APIResult<GitHubUser>   Same shape as useGitHubUser.

--- useGitHubRepos ---

Fetch the public repositories for a given user.

  import { useGitHubRepos } from '@reactjit/apis/github';

  const result: APIResult<GitHubRepo[]> = useGitHubRepos(username, opts?);

Parameters:
  username   string | null
  opts       object (optional)
    sort     'updated' | 'stars' | 'pushed'   Default: 'updated'
    perPage  number                             Default: 30
    token    string | null

Returns: APIResult<GitHubRepo[]>

GitHubRepo fields:
  id                  number
  name                string
  full_name           string
  description         string | null
  html_url            string
  stargazers_count    number
  forks_count         number
  language            string | null
  updated_at          string
  open_issues_count   number
  topics              string[]
  fork                boolean
  private             boolean

--- useGitHubRepo ---

Fetch a single repository by owner and repo name.

  import { useGitHubRepo } from '@reactjit/apis/github';

  const result: APIResult<GitHubRepo> = useGitHubRepo(owner, repo, token?);

Parameters:
  owner   string | null   Repository owner (user or org). Pass null to skip.
  repo    string | null   Repository name. Pass null to skip.
  token   string | null   Optional Bearer token.

Returns: APIResult<GitHubRepo>

--- useGitHubIssues ---

Fetch the open (or closed) issues for a repository. Note: pull requests appear in this list too; check for the presence of pull_request on each item to distinguish them.

  import { useGitHubIssues } from '@reactjit/apis/github';

  const result: APIResult<GitHubIssue[]> = useGitHubIssues(owner, repo, opts?);

Parameters:
  owner   string | null
  repo    string | null
  opts    object (optional)
    state    'open' | 'closed' | 'all'   Default: 'open'
    perPage  number                        Default: 30
    token    string | null

Returns: APIResult<GitHubIssue[]>

GitHubIssue fields:
  id           number
  number       number   Issue number within the repo
  title        string
  body         string | null
  state        'open' | 'closed'
  user         { login: string; avatar_url: string }
  labels       Array<{ name: string; color: string }>
  created_at   string
  updated_at   string
  html_url     string
  pull_request { url: string } | undefined   Present if this item is a PR.

--- useGitHubEvents ---

Fetch the recent public activity events for a user.

  import { useGitHubEvents } from '@reactjit/apis/github';

  const result: APIResult<GitHubEvent[]> = useGitHubEvents(username, opts?);

Parameters:
  username   string | null
  opts       object (optional)
    perPage  number          Default: 30
    token    string | null

Returns: APIResult<GitHubEvent[]>

GitHubEvent fields:
  id          string
  type        string   e.g. 'PushEvent', 'IssuesEvent', 'WatchEvent'
  repo        { name: string }
  created_at  string
  payload     any   Event-specific payload (differs by type)

--- useGitHubGists ---

Fetch public gists for a user.

  import { useGitHubGists } from '@reactjit/apis/github';

  const result: APIResult<GitHubGist[]> = useGitHubGists(username, token?);

Parameters:
  username   string | null
  token      string | null   Optional Bearer token.

Returns: APIResult<GitHubGist[]>

GitHubGist fields:
  id           string
  description  string | null
  html_url     string
  files        Record<string, { filename: string; language: string; size: number }>
  created_at   string
  updated_at   string

--- useGitHubStarred ---

Fetch the repositories that a user has starred.

  import { useGitHubStarred } from '@reactjit/apis/github';

  const result: APIResult<GitHubRepo[]> = useGitHubStarred(username, opts?);

Parameters:
  username   string | null
  opts       object (optional)
    perPage  number          Default: 30
    token    string | null

Returns: APIResult<GitHubRepo[]>   Same shape as useGitHubRepos.

=== EXAMPLES ===

-- Developer profile card --

Display a user's avatar, bio, and key stats:

  import { useGitHubUser } from '@reactjit/apis/github';
  import { Box, Text } from '@reactjit/core';

  function GitHubCard({ username }: { username: string }) {
    const { data, loading, error } = useGitHubUser(username);

    if (loading) return <Text fontSize={13}>Loading...</Text>;
    if (error || !data) return <Text fontSize={13}>Not found</Text>;

    return (
      <Box padding={16} gap={8} width={280}>
        <Text fontSize={18}>{data.name ?? data.login}</Text>
        {data.bio && <Text fontSize={13}>{data.bio}</Text>}
        <Box flexDirection="row" gap={16}>
          <Box alignItems="center" gap={2}>
            <Text fontSize={16}>{data.public_repos}</Text>
            <Text fontSize={11}>Repos</Text>
          </Box>
          <Box alignItems="center" gap={2}>
            <Text fontSize={16}>{data.followers}</Text>
            <Text fontSize={11}>Followers</Text>
          </Box>
          <Box alignItems="center" gap={2}>
            <Text fontSize={16}>{data.following}</Text>
            <Text fontSize={11}>Following</Text>
          </Box>
        </Box>
      </Box>
    );
  }

-- Repository list sorted by stars --

Show the top repositories for a user, sorted by star count:

  import { useGitHubRepos } from '@reactjit/apis/github';
  import { Box, Text } from '@reactjit/core';

  function RepoList({ username, token }: { username: string; token?: string }) {
    const { data, loading } = useGitHubRepos(username, {
      sort: 'stars',
      perPage: 10,
      token,
    });

    if (loading) return <Text fontSize={13}>Loading...</Text>;

    const repos = (data ?? []).filter(r => !r.fork);

    return (
      <Box gap={8}>
        <Text fontSize={15}>Top Repos</Text>
        {repos.map(repo => (
          <Box key={repo.id} flexDirection="row" gap={8} alignItems="center">
            <Box flexGrow={1}>
              <Text fontSize={13}>{repo.name}</Text>
              {repo.description && (
                <Text fontSize={11}>{repo.description}</Text>
              )}
            </Box>
            <Box alignItems="flex-end" gap={2}>
              <Text fontSize={12}>{repo.stargazers_count} stars</Text>
              {repo.language && <Text fontSize={11}>{repo.language}</Text>}
            </Box>
          </Box>
        ))}
      </Box>
    );
  }
