=== OVERVIEW ===
The TMDB hooks wrap The Movie Database (TMDB) REST API v3. Authentication uses an API key passed as a query parameter — get one at https://www.themoviedb.org/settings/api. Pass the key as the first argument to every hook. A null key skips the request.

All hooks return APIResult<T> with data, loading, error, and refetch. Paginated endpoints return TMDBPaginated<T> which includes the current page, total pages, and total results.

The package also exports a tmdbImage() utility for constructing full image URLs from the partial paths the API returns.

=== API / SYNTAX ===

--- useTMDBTrending ---

Fetch trending movies or TV shows for today or this week.

  import { useTMDBTrending } from '@reactjit/apis/tmdb';

  const result: APIResult<TMDBPaginated<TMDBMovie>> = useTMDBTrending(apiKey, opts?);

Parameters:
  apiKey   string | null
  opts     object (optional)
    mediaType   'movie' | 'tv' | 'all'    Default: 'movie'
    timeWindow  'day' | 'week'            Default: 'week'

Returns: APIResult<TMDBPaginated<TMDBMovie>>
  data.page           number
  data.results        TMDBMovie[]
  data.total_pages    number
  data.total_results  number

--- useTMDBSearch ---

Search the TMDB catalog by text query. Supports movies, TV shows, or a combined multi-search.

  import { useTMDBSearch } from '@reactjit/apis/tmdb';

  const result: APIResult<TMDBPaginated<TMDBMovie & TMDBSeries>> = useTMDBSearch(apiKey, query, opts?);

Parameters:
  apiKey   string | null
  query    string | null   Pass null to skip (e.g. while the user is typing).
  opts     object (optional)
    type   'movie' | 'tv' | 'multi'   Default: 'multi'
    page   number                      Default: none (API default: 1)

Returns: APIResult<TMDBPaginated<TMDBMovie & TMDBSeries>>

--- useTMDBMovie ---

Fetch full details for a single movie, including runtime, budget, genres, and production companies.

  import { useTMDBMovie } from '@reactjit/apis/tmdb';

  const result: APIResult<TMDBMovieDetails> = useTMDBMovie(apiKey, movieId);

Parameters:
  apiKey    string | null
  movieId   number | null   TMDB movie ID. Pass null to skip.

Returns: APIResult<TMDBMovieDetails>

TMDBMovieDetails extends TMDBMovie and adds:
  runtime               number
  budget                number
  revenue               number
  tagline               string
  genres                Array<{ id: number; name: string }>
  production_companies  Array<{ id: number; name: string; logo_path: string | null }>
  imdb_id               string | null

--- useTMDBSeries ---

Fetch full details for a single TV series, including season count, episode count, and networks.

  import { useTMDBSeries } from '@reactjit/apis/tmdb';

  const result: APIResult<TMDBSeriesDetails> = useTMDBSeries(apiKey, seriesId);

Parameters:
  apiKey     string | null
  seriesId   number | null   TMDB series ID. Pass null to skip.

Returns: APIResult<TMDBSeriesDetails>

TMDBSeriesDetails extends TMDBSeries and adds:
  number_of_seasons    number
  number_of_episodes   number
  episode_run_time     number[]
  genres               Array<{ id: number; name: string }>
  status               string
  networks             Array<{ id: number; name: string; logo_path: string | null }>

--- useTMDBPopular ---

Fetch the current popular movies or TV shows.

  import { useTMDBPopular } from '@reactjit/apis/tmdb';

  const result: APIResult<TMDBPaginated<TMDBMovie>> = useTMDBPopular(apiKey, opts?);

Parameters:
  apiKey   string | null
  opts     object (optional)
    type   'movie' | 'tv'   Default: 'movie'
    page   number           Default: none (API default: 1)

Returns: APIResult<TMDBPaginated<TMDBMovie>>

--- tmdbImage ---

Construct a full image URL from the partial path returned by TMDB.

  import { tmdbImage } from '@reactjit/apis/tmdb';

  const url: string | null = tmdbImage(path, size?);

Parameters:
  path   string | null   The poster_path or backdrop_path value from any TMDB type.
  size   string          TMDB image size code. Default: 'w500'. Common: 'w92', 'w185', 'w342', 'w500', 'w780', 'original'.

Returns: string | null — null if path is null or empty.

=== EXAMPLES ===

-- Trending movies list --

Display a horizontal strip of this week's trending movies with their ratings:

  import { useTMDBTrending, tmdbImage } from '@reactjit/apis/tmdb';
  import { Box, Text, Image } from '@reactjit/core';

  const API_KEY = process.env.TMDB_KEY ?? null;

  function TrendingMovies() {
    const { data, loading, error } = useTMDBTrending(API_KEY, {
      mediaType: 'movie',
      timeWindow: 'week',
    });

    if (loading) return <Text fontSize={13}>Loading...</Text>;
    if (error) return <Text fontSize={13}>Error: {error.message}</Text>;

    return (
      <Box gap={8}>
        <Text fontSize={16}>Trending This Week</Text>
        <Box flexDirection="row" gap={12}>
          {data?.results.slice(0, 5).map(movie => (
            <Box key={movie.id} width={120} gap={4}>
              {tmdbImage(movie.poster_path, 'w185') && (
                <Image
                  src={tmdbImage(movie.poster_path, 'w185')!}
                  width={120}
                  height={180}
                />
              )}
              <Text fontSize={12}>{movie.title}</Text>
              <Text fontSize={11}>{movie.vote_average.toFixed(1)}</Text>
            </Box>
          ))}
        </Box>
      </Box>
    );
  }

-- Movie detail card --

Fetch and display full details for a specific movie ID:

  import { useTMDBMovie, tmdbImage } from '@reactjit/apis/tmdb';
  import { Box, Text } from '@reactjit/core';

  function MovieCard({ movieId }: { movieId: number }) {
    const { data, loading } = useTMDBMovie(process.env.TMDB_KEY ?? null, movieId);

    if (loading) return <Text fontSize={13}>Loading...</Text>;
    if (!data) return null;

    return (
      <Box padding={16} gap={6} width={320}>
        <Text fontSize={18}>{data.title}</Text>
        {data.tagline ? <Text fontSize={13}>{data.tagline}</Text> : null}
        <Text fontSize={12}>{data.genres.map(g => g.name).join(' / ')}</Text>
        <Text fontSize={12}>{data.runtime} min</Text>
        <Text fontSize={12}>{data.vote_average.toFixed(1)} / 10 ({data.vote_count} votes)</Text>
        <Text fontSize={13}>{data.overview}</Text>
      </Box>
    );
  }
