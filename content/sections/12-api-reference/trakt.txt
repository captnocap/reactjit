=== OVERVIEW ===
The Trakt hooks in @reactjit/apis connect to the Trakt.tv REST API. Most endpoints require a Client ID (public) and a user OAuth token (private). The now-watching hook is especially useful for building media dashboards because it reflects the current scrobble state in near real-time.

Credentials: create an application at https://trakt.tv/oauth/applications to get a Client ID. User tokens are obtained through OAuth — store them in your environment or config file.

All hooks return null data until credentials are provided. Passing null for clientId or token suppresses the fetch and is the recommended way to conditionally disable a hook without breaking the component tree.

=== API / SYNTAX ===

--- useTraktWatching ---

Fetch what a user is currently watching. Returns null data when nothing is playing. Supports polling.

  import { useTraktWatching } from '@reactjit/apis';

  const result: APIResult<TraktWatching | null> = useTraktWatching(
    clientId: string | null,
    token: string | null,
    username?: string,
    opts?: { interval?: number },
  );

Parameters:
  clientId  string | null   Trakt application Client ID
  token     string | null   OAuth Bearer token for the user
  username  string          Trakt username (default: 'me' — the authenticated user)
  opts      object | undefined
    interval  number        Polling interval in milliseconds (default: 30000)

Returns: APIResult<TraktWatching | null>
  data is null when nothing is currently playing.

TraktWatching fields:
  expires_at   string              ISO 8601 datetime when the scrobble expires
  started_at   string              ISO 8601 datetime when playback started
  action       string              Typically 'scrobble'
  type         'movie' | 'episode'
  movie        TraktMovie | undefined   Present when type is 'movie'
  show         TraktShow | undefined    Present when type is 'episode'
  episode      TraktEpisode | undefined Present when type is 'episode'

--- useTraktHistory ---

Fetch a user's watch history, optionally filtered by media type.

  import { useTraktHistory } from '@reactjit/apis';

  const result: APIResult<TraktHistoryItem[]> = useTraktHistory(
    clientId: string | null,
    token: string | null,
    opts?: { username?: string; type?: 'movies' | 'shows' | 'episodes'; limit?: number },
  );

Parameters:
  clientId  string | null   Trakt Client ID
  token     string | null   OAuth Bearer token
  opts      object | undefined
    username  string        Trakt username (default: 'me')
    type      string        Filter to 'movies', 'shows', or 'episodes'
    limit     number        Max items to return (default: 20)

Returns: APIResult<TraktHistoryItem[]>

TraktHistoryItem fields:
  id          number
  watched_at  string              ISO 8601 datetime
  action      string              'watch' or 'scrobble'
  type        'movie' | 'episode'
  movie       TraktMovie | undefined
  show        TraktShow | undefined
  episode     TraktEpisode | undefined

--- useTraktWatchlist ---

Fetch a user's watchlist, optionally filtered to movies or shows only.

  import { useTraktWatchlist } from '@reactjit/apis';

  const result: APIResult<TraktWatchlistItem[]> = useTraktWatchlist(
    clientId: string | null,
    token: string | null,
    opts?: { username?: string; type?: 'movies' | 'shows' },
  );

Parameters:
  clientId  string | null   Trakt Client ID
  token     string | null   OAuth Bearer token
  opts      object | undefined
    username  string        Trakt username (default: 'me')
    type      string        'movies' or 'shows' — omit for both

Returns: APIResult<TraktWatchlistItem[]>

TraktWatchlistItem fields:
  listed_at  string            ISO 8601 datetime added to list
  type       'movie' | 'show'
  movie      TraktMovie | undefined
  show       TraktShow | undefined

--- useTraktTrending ---

Fetch currently trending movies or shows across all Trakt users. Does not require a user token.

  import { useTraktTrending } from '@reactjit/apis';

  const result: APIResult<Array<{ watchers: number; movie?: TraktMovie; show?: TraktShow }>> = useTraktTrending(
    clientId: string | null,
    type?: 'movies' | 'shows',
    opts?: { limit?: number },
  );

Parameters:
  clientId  string | null   Trakt Client ID
  type      string          'movies' or 'shows' (default: 'movies')
  opts      object | undefined
    limit   number          Max items to return (default: 10)

Returns: APIResult<Array<{ watchers: number; movie?: TraktMovie; show?: TraktShow }>>
  watchers — number of Trakt users watching this item right now

--- useTraktStats ---

Fetch aggregate statistics for a user: total plays, time watched, shows and episodes seen.

  import { useTraktStats } from '@reactjit/apis';

  const result: APIResult<TraktStats> = useTraktStats(
    clientId: string | null,
    token: string | null,
    username?: string,
  );

Parameters:
  clientId  string | null   Trakt Client ID
  token     string | null   OAuth Bearer token
  username  string          Trakt username (default: 'me')

Returns: APIResult<TraktStats>

TraktStats fields:
  movies    { plays: number; watched: number; minutes: number }
  shows     { watched: number; collected: number }
  episodes  { plays: number; watched: number; minutes: number }

--- Shared Types ---

TraktMovie fields:
  title  string
  year   number
  ids    { trakt: number; slug: string; imdb?: string; tmdb?: number }

TraktShow fields:
  title  string
  year   number
  ids    { trakt: number; slug: string; imdb?: string; tmdb?: number; tvdb?: number }

TraktEpisode fields:
  season  number
  number  number
  title   string
  ids     { trakt: number; tvdb?: number; imdb?: string; tmdb?: number }

=== EXAMPLES ===

-- Now-watching banner --

Show what the authenticated user is currently watching, polling every 30 seconds:

  import { useTraktWatching } from '@reactjit/apis';
  import { Box, Text } from '@reactjit/core';

  const CLIENT_ID = process.env.TRAKT_CLIENT_ID;
  const TOKEN = process.env.TRAKT_TOKEN;

  function NowWatching() {
    const { data, loading } = useTraktWatching(CLIENT_ID, TOKEN, 'me', { interval: 30000 });

    if (loading) return null;
    if (!data) return <Text fontSize={13}>Not watching anything</Text>;

    const title =
      data.type === 'movie'
        ? data.movie?.title
        : `${data.show?.title} S${data.episode?.season}E${data.episode?.number}`;

    return (
      <Box padding={12} backgroundColor="#1b2838">
        <Text fontSize={12}>Watching now</Text>
        <Text fontSize={16}>{title}</Text>
      </Box>
    );
  }

-- Watchlist and trending side by side --

Render the user's movie watchlist next to the global trending list:

  import { useTraktWatchlist, useTraktTrending } from '@reactjit/apis';
  import { Box, Text } from '@reactjit/core';

  function TraktDashboard({ clientId, token }: { clientId: string; token: string }) {
    const watchlist = useTraktWatchlist(clientId, token, { type: 'movies' });
    const trending = useTraktTrending(clientId, 'movies', { limit: 5 });

    return (
      <Box flexDirection="row" gap={24} width="100%">
        <Box flex={1}>
          <Text fontSize={14}>My Watchlist</Text>
          {(watchlist.data ?? []).map((item) => (
            <Text key={item.movie?.ids.trakt} fontSize={12}>
              {item.movie?.title} ({item.movie?.year})
            </Text>
          ))}
        </Box>

        <Box flex={1}>
          <Text fontSize={14}>Trending Now</Text>
          {(trending.data ?? []).map((item) => (
            <Text key={item.movie?.ids.trakt} fontSize={12}>
              {item.movie?.title} — {item.watchers} watching
            </Text>
          ))}
        </Box>
      </Box>
    );
  }
