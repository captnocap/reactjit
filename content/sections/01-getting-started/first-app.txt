=== OVERVIEW ===
This tutorial walks you through building an interactive counter application from
scratch. You will learn how to import primitives, manage state with React hooks, handle
user input with Pressable, and style your UI with flexbox properties. By the end you
will have a working app with increment and decrement buttons, styled with theme colors
and proper spacing.

Remember: React is just the layout declaration layer here. It describes what the UI
looks like, Lua computes the geometry, and Love2D draws the pixels.

Example 2: Step 2 - Display the Count (src/App.tsx)
---
```tsx
import { useState } from 'react';
import { Box, Text } from '@reactjit/core';

export function App() {
  const [count, setCount] = useState(0);

  return (
    <Box style={{
      width: '100%',
      height: '100%',
      backgroundColor: '#0f172a',
      justifyContent: 'center',
      alignItems: 'center',
    }}>
      <Text style={{ fontSize: 48, color: '#f8fafc' }}>
        {`${count}`}
      </Text>
    </Box>
  );
}
```

Key points:
- The root Box uses width/height '100%' to fill the viewport.
- justifyContent and alignItems center the text in both axes.
- Every Text node has an explicit fontSize (the linter enforces this).
- Dynamic text uses a template literal: {`${count}`}, not mixed children.

Example 3: Step 3 - Add Buttons
---
```tsx
import { useState } from 'react';
import { Box, Text, Pressable } from '@reactjit/core';

function CounterButton({ label, onPress }: { label: string; onPress: () => void }) {
  return (
    <Pressable onPress={onPress}>
      {({ pressed }) => (
        <Box style={{
          width: 60,
          height: 60,
          backgroundColor: pressed ? '#1d4ed8' : '#3b82f6',
          borderRadius: 12,
          justifyContent: 'center',
          alignItems: 'center',
        }}>
          <Text style={{ fontSize: 28, color: '#ffffff' }}>{label}</Text>
        </Box>
      )}
    </Pressable>
  );
}

export function App() {
  const [count, setCount] = useState(0);

  return (
    <Box style={{
      width: '100%',
      height: '100%',
      backgroundColor: '#0f172a',
      justifyContent: 'center',
      alignItems: 'center',
      gap: 24,
    }}>
      <Text style={{ fontSize: 48, color: '#f8fafc' }}>
        {`${count}`}
      </Text>
      <Box style={{
        flexDirection: 'row',
        gap: 16,
      }}>
        <CounterButton label="-" onPress={() => setCount(c => c - 1)} />
        <CounterButton label="+" onPress={() => setCount(c => c + 1)} />
      </Box>
    </Box>
  );
}
```

Key concepts:
- Pressable provides press state via render props: ({ pressed }) => ...
- gap: 24 adds vertical space between the count and the button row.
- gap: 16 adds horizontal space between the two buttons.
- Each button has fixed width/height for a square shape.
---
Platforms: love2d

Example 4: Step 4 - Add Hover States
---
```tsx
function CounterButton({ label, onPress }: { label: string; onPress: () => void }) {
  return (
    <Pressable onPress={onPress}>
      {({ pressed, hovered }) => (
        <Box style={{
          width: 60,
          height: 60,
          backgroundColor: pressed ? '#1d4ed8' : hovered ? '#2563eb' : '#3b82f6',
          borderRadius: 12,
          justifyContent: 'center',
          alignItems: 'center',
        }}>
          <Text style={{ fontSize: 28, color: '#ffffff' }}>{label}</Text>
        </Box>
      )}
    </Pressable>
  );
}
```

Pressable's render props include both `pressed` and `hovered`. The ternary
chain gives visual feedback: default blue, lighter on hover, darker on press.
---
Platforms: love2d

Example 5: Step 5 - Using Theme Colors
---
```tsx
import { useState } from 'react';
import { Box, Text, Pressable } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

function CounterButton({ label, onPress }: { label: string; onPress: () => void }) {
  const c = useThemeColors();
  return (
    <Pressable onPress={onPress}>
      {({ pressed, hovered }) => (
        <Box style={{
          width: 60,
          height: 60,
          backgroundColor: pressed ? c.primaryPressed : hovered ? c.primaryHover : c.primary,
          borderRadius: 12,
          justifyContent: 'center',
          alignItems: 'center',
        }}>
          <Text style={{ fontSize: 28, color: c.textOnPrimary }}>{label}</Text>
        </Box>
      )}
    </Pressable>
  );
}

export function App() {
  const c = useThemeColors();
  const [count, setCount] = useState(0);

  return (
    <Box style={{
      width: '100%',
      height: '100%',
      backgroundColor: c.bg,
      justifyContent: 'center',
      alignItems: 'center',
      gap: 24,
    }}>
      <Text style={{ fontSize: 48, color: c.text }}>
        {`${count}`}
      </Text>
      <Box style={{ flexDirection: 'row', gap: 16 }}>
        <CounterButton label="-" onPress={() => setCount(c => c - 1)} />
        <CounterButton label="+" onPress={() => setCount(c => c + 1)} />
      </Box>
    </Box>
  );
}
```

useThemeColors() returns the current theme's color tokens. This makes
the app respond to theme changes automatically. The @reactjit/theme
package is included by default when scaffolding with `reactjit init`.

=== CRITICAL RULES ===
- Every Text node MUST have an explicit fontSize.
- The root container MUST use width: '100%', height: '100%' -- not flexGrow: 1.
- Use template literals for dynamic text: {`${count}`}, not mixed children like
  <Text>Count: {count}</Text> which creates multiple text nodes that stack vertically.
- Always run `reactjit lint` after modifying components.
- Do not use paddingHorizontal or paddingVertical. Use paddingLeft, paddingRight,
  paddingTop, paddingBottom individually.
