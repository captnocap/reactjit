=== METADATA ===
title: useHotkey
description: Register a global keyboard shortcut that fires a handler when a key combo is pressed.
category: Hooks
platforms: love2d, web, terminal, cc, nvim, hs, awesome
keywords: keyboard, hotkey, shortcut, keydown, combo, ctrl, shift, alt
related: useLoveEvent, useClipboard
difficulty: beginner

=== OVERVIEW ===
useHotkey binds a callback to a keyboard combination string such as "ctrl+z" or "ctrl+shift+s". The combo is parsed once on mount and matched against every keydown event received from the bridge. The handler reference is always kept current without re-subscribing, so passing an inline arrow function is safe. Pass `enabled: false` in options to conditionally disable the binding without unmounting the component.

=== API / SYNTAX ===
```tsx
import { useHotkey } from '@reactjit/core';
```

Parameters:
| Param   | Type                       | Default   | Description |
|---------|----------------------------|-----------|-------------|
| combo   | string                     | required  | Key combination string. Use "+" as separator. Modifiers: ctrl, control, shift, alt, meta, cmd, gui. Key names are lowercased (e.g. "s", "z", "escape", "f1"). |
| handler | (event: LoveEvent) => void | required  | Called when the combo is matched. Receives the raw LoveEvent payload. |
| options | { enabled?: boolean }      | undefined | Optional config object. Set enabled: false to disable the binding temporarily. |

Returns:
void — this hook has no return value.

=== EXAMPLES ===
Example 1: Undo and redo shortcuts
---
```tsx
import { useHotkey } from '@reactjit/core';

function Editor() {
  useHotkey('ctrl+z', () => undo());
  useHotkey('ctrl+shift+z', () => redo());

  return <EditorCanvas />;
}
```
---
Platforms: All

Example 2: Escape closes a modal only while it is open
---
```tsx
import { useState } from 'react';
import { useHotkey, Box, Text } from '@reactjit/core';

function App() {
  const [open, setOpen] = useState(false);

  useHotkey('escape', () => setOpen(false), { enabled: open });

  return (
    <Box width="100%" height="100%">
      {open && (
        <Box backgroundColor="#222" padding={16}>
          <Text fontSize={14}>Press Escape to close</Text>
        </Box>
      )}
    </Box>
  );
}
```
---
Platforms: All

Example 3: Save with Ctrl+S and Ctrl+Shift+S for save-as
---
```tsx
import { useHotkey } from '@reactjit/core';

function Document() {
  useHotkey('ctrl+s', () => save());
  useHotkey('ctrl+shift+s', () => saveAs());

  return <DocumentView />;
}
```
---
Platforms: All

Example 4: Access the raw LoveEvent in the handler
---
```tsx
import { useHotkey } from '@reactjit/core';
import type { LoveEvent } from '@reactjit/core';

function DevTools() {
  useHotkey('f12', (event: LoveEvent) => {
    console.log('key:', event.key, 'shift:', event.shift);
    toggleDebugPanel();
  });

  return <App />;
}
```
