=== OVERVIEW ===
The @reactjit/rss package turns any RSS/Atom feed URL into a reactive data source. It ships a zero-dependency XML parser that runs in QuickJS (Love2D), Node.js, and browsers without requiring DOMParser. The two main hooks — useRSSFeed and useRSSAggregate — handle fetching, polling, deduplication, and limit enforcement. Imperative helpers (fetchFeed, parseFeed) and OPML import/export utilities round out the package for building complete feed readers.

=== API / SYNTAX ===

--- useRSSFeed ---

Subscribe to a single RSS, Atom, or RSS 1.0 feed. Fetches immediately on mount, re-fetches on url change or manual refetch, and polls if an interval is configured. New items are merged on top of the existing list with deduplication by item ID.

  import { useRSSFeed } from '@reactjit/rss';

  const result: RSSFeedResult = useRSSFeed(url: string | null, options?: RSSFeedOptions);

Parameters:
  url      string | null     Feed URL to fetch. Pass null to disable and reset state.
  options  RSSFeedOptions    Optional configuration (see below)

RSSFeedOptions fields:
  interval   number    Polling interval in milliseconds. 0 or omitted = no polling.
  limit      number    Maximum items to keep (newest first). Default: keep all.
  dedupe     boolean   Deduplicate items by their id/guid across polls. Default: true.
  corsProxy  string    CORS proxy prefix prepended to the URL (e.g. 'https://corsproxy.io/?'). Useful for the web target.

Returns: RSSFeedResult
  feed     Feed | null   Parsed feed metadata, or null while loading/on error
  items    FeedItem[]    Items from the feed (empty array until first successful fetch)
  loading  boolean       True during the initial fetch and each re-fetch
  error    Error | null  Last fetch or parse error, or null on success
  refetch  () => void    Trigger an immediate re-fetch outside the polling interval

Feed fields:
  title          string
  description    string
  link           string
  language       string | null
  lastBuildDate  string | null   ISO date string
  image          string | null   Feed logo URL
  type           'rss2' | 'atom' | 'rss1' | 'unknown'
  items          FeedItem[]

FeedItem fields:
  id           string              Unique identifier (guid/id, falls back to link or title)
  title        string
  link         string
  description  string              Summary or full HTML content
  content      string | null       Full content (content:encoded or Atom content element)
  pubDate      string | null       ISO date string
  author       string | null
  categories   string[]
  enclosure    { url, type, length } | null   Podcast audio, attached files, etc.
  thumbnail    string | null       Image URL from media:thumbnail, media:content, or itunes:image

--- useRSSAggregate ---

Subscribe to multiple feeds simultaneously and receive a single merged, chronologically sorted item list. All feeds are fetched in parallel. Each feed failure is isolated — one broken feed does not prevent the others from rendering.

  import { useRSSAggregate } from '@reactjit/rss';

  const result: RSSAggregateResult = useRSSAggregate(urls: string[], options?: RSSAggregateOptions);

Parameters:
  urls     string[]              Array of feed URLs to subscribe to
  options  RSSAggregateOptions   Optional configuration (see below)

RSSAggregateOptions fields:
  interval   number   Polling interval in milliseconds. Applied to all feeds. Default: no polling.
  limit      number   Maximum total items after merge. Default: keep all.
  corsProxy  string   CORS proxy prefix applied to every URL.

Returns: RSSAggregateResult
  items    Array<FeedItem & { feedTitle: string; feedUrl: string }>
           Merged item list sorted by pubDate descending. Each item carries feedTitle and feedUrl
           from its originating feed.
  feeds    Array<{ url: string; feed: Feed | null; error: Error | null }>
           Per-feed status — inspect individual failures without losing the rest.
  loading  boolean     True while any feed is still being fetched on the current cycle.
  refetch  () => void  Trigger an immediate re-fetch of all feeds.

--- fetchFeed (imperative) ---

Fetch and parse a feed outside of React. Returns a Promise<Feed>. Use this in event handlers, background tasks, or server-side scripts where hooks are not available.

  import { fetchFeed } from '@reactjit/rss';

  const feed: Feed = await fetchFeed(url: string, corsProxy?: string);

Parameters:
  url        string   Feed URL to fetch
  corsProxy  string   Optional CORS proxy prefix

Returns: Promise<Feed>
Throws on network errors or non-2xx HTTP responses.

--- parseFeed (utility) ---

Parse a raw XML string into a normalized Feed object. Supports RSS 2.0, Atom 1.0, and RSS 1.0 (RDF). No DOMParser — works in QuickJS.

  import { parseFeed } from '@reactjit/rss';

  const feed: Feed = parseFeed(xml: string);

Parameters:
  xml  string  Raw XML string (RSS or Atom document)

Returns: Feed — never throws; returns an empty feed with type 'unknown' on unrecognized input.

--- parseOPML (utility) ---

Parse an OPML subscription list into an array of feed descriptors. Useful for importing browser bookmark exports or shared subscription bundles.

  import { parseOPML } from '@reactjit/rss';

  const outlines: OPMLOutline[] = parseOPML(xml: string);

Parameters:
  xml  string  Raw OPML XML string

Returns: OPMLOutline[]

OPMLOutline fields:
  text      string   Display name
  title     string   Feed title (falls back to text)
  xmlUrl    string   RSS/Atom feed URL
  htmlUrl   string   Website URL
  type      string   Usually 'rss'
  category  string   Optional category tag

--- generateOPML (utility) ---

Serialize a list of subscriptions as an OPML 2.0 document. Use this to export a user's feed list for sharing or backup.

  import { generateOPML } from '@reactjit/rss';

  const xml: string = generateOPML(
    title: string,
    feeds: Array<{ title: string; xmlUrl: string; htmlUrl?: string; category?: string }>,
  );

Parameters:
  title  string   Title for the OPML document
  feeds  Array    Subscription entries

Returns: string — valid OPML 2.0 XML

=== EXAMPLES ===

-- Hacker News front page reader --

Display the latest stories from Hacker News with live polling every 5 minutes:

  import { useRSSFeed } from '@reactjit/rss';
  import { Box, Text } from '@reactjit/core';

  function HNReader() {
    const { feed, items, loading, error } = useRSSFeed(
      'https://hnrss.org/frontpage',
      { interval: 300_000, limit: 30 },
    );

    if (loading && !feed) {
      return (
        <Box width="100%" height="100%" justifyContent="center" alignItems="center">
          <Text fontSize={14}>Loading...</Text>
        </Box>
      );
    }

    if (error) {
      return (
        <Box width="100%" padding={16}>
          <Text fontSize={13}>Failed to load feed: {error.message}</Text>
        </Box>
      );
    }

    return (
      <Box width="100%" height="100%" flexDirection="column" gap={2}>
        <Box padding={12} backgroundColor="#ff6600">
          <Text fontSize={16}>{feed?.title ?? 'Hacker News'}</Text>
        </Box>
        {items.map((item) => (
          <Box key={item.id} padding={10} backgroundColor="#1a1a1a">
            <Text fontSize={13}>{item.title}</Text>
            {item.author && (
              <Text fontSize={11}>{item.author}</Text>
            )}
          </Box>
        ))}
      </Box>
    );
  }

-- Multi-source news aggregator --

Merge three feeds into one chronological stream and show which source each story comes from:

  import { useRSSAggregate } from '@reactjit/rss';
  import { Box, Text } from '@reactjit/core';

  const FEEDS = [
    'https://hnrss.org/frontpage',
    'https://lobste.rs/rss',
    'https://www.reddit.com/r/programming/.rss',
  ];

  function NewsFeed() {
    const { items, feeds, loading } = useRSSAggregate(FEEDS, {
      interval: 120_000,
      limit: 50,
    });

    // Show per-feed error badges
    const failedFeeds = feeds.filter((f) => f.error);

    return (
      <Box width="100%" height="100%" flexDirection="column">
        {failedFeeds.length > 0 && (
          <Box padding={8} backgroundColor="#550000">
            <Text fontSize={11}>
              {failedFeeds.length} feed(s) unavailable
            </Text>
          </Box>
        )}
        {loading && items.length === 0 && (
          <Box padding={16}>
            <Text fontSize={13}>Fetching feeds...</Text>
          </Box>
        )}
        {items.map((item) => (
          <Box
            key={item.id}
            padding={10}
            backgroundColor="#111"
            flexDirection="row"
            gap={8}
          >
            <Box backgroundColor="#333" padding={4}>
              <Text fontSize={10}>{item.feedTitle}</Text>
            </Box>
            <Text fontSize={13}>{item.title}</Text>
          </Box>
        ))}
      </Box>
    );
  }

-- OPML import flow --

Let the user paste an OPML export from their old feed reader and subscribe to all feeds at once:

  import { parseOPML, useRSSAggregate } from '@reactjit/rss';
  import { useState } from 'react';
  import { Box, Text, TextInput } from '@reactjit/core';

  function OPMLImporter() {
    const [urls, setUrls] = useState<string[]>([]);

    // In a real app, load OPML text from a file picker or paste box
    function handleOPML(opmlText: string) {
      const outlines = parseOPML(opmlText);
      setUrls(outlines.map((o) => o.xmlUrl));
    }

    const { items, loading } = useRSSAggregate(urls, { limit: 100 });

    return (
      <Box width="100%" height="100%" flexDirection="column" gap={8} padding={16}>
        <Text fontSize={16}>Imported {urls.length} feed(s)</Text>
        {loading && <Text fontSize={13}>Fetching...</Text>}
        {items.map((item) => (
          <Box key={item.id} padding={8} backgroundColor="#1c1c1c">
            <Text fontSize={13}>{item.title}</Text>
            <Text fontSize={11}>{item.feedTitle}</Text>
          </Box>
        ))}
      </Box>
    );
  }
