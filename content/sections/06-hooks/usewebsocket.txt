=== OVERVIEW ===
useWebSocket manages a WebSocket connection with reactive status, send function, and last-received message. In Love2D it uses the WebSocket polyfill backed by lua-websocket via the bridge.

Pass `null` as the URL to close the connection. The hook automatically cleans up on unmount.

=== API / SYNTAX ===
```tsx
import { useWebSocket } from '@reactjit/core';
```

Signature:
```tsx
function useWebSocket(url: string | null): {
  status: 'connecting' | 'open' | 'closed' | 'error';
  send: (data: string) => void;
  lastMessage: string | null;
  error: string | null;
}
```

Parameters:
| Param | Type           | Default  | Description                         |
|-------|----------------|----------|-------------------------------------|
| url   | string \| null | required | WebSocket URL, or null to close     |

Returns:
| Field       | Type                                          | Description                         |
|-------------|-----------------------------------------------|-------------------------------------|
| status      | 'connecting' \| 'open' \| 'closed' \| 'error'| Current connection state            |
| send        | (data: string) => void                        | Send a string message               |
| lastMessage | string \| null                                | Most recently received message      |
| error       | string \| null                                | Error description if status='error' |

=== EXAMPLES ===
Example 1: Basic WebSocket connection
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useWebSocket } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function LiveFeed() {
  const colors = useThemeColors();
  const { status, lastMessage } = useWebSocket('ws://localhost:9050');

  return (
    <Box style={{ gap: 8, padding: 16 }}>
      <Text style={{ color: colors.muted }}>{`Status: ${status}`}</Text>
      {lastMessage && (
        <Text style={{ color: colors.text }}>{`Last: ${lastMessage}`}</Text>
      )}
    </Box>
  );
}
```
---
Platforms: love2d

Example 2: Send and receive messages
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { useWebSocket } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function Chat() {
  const colors = useThemeColors();
  const { status, send, lastMessage } = useWebSocket('ws://localhost:9050');

  return (
    <Box style={{ gap: 8, padding: 16 }}>
      <Text style={{ color: colors.muted }}>{`Status: ${status}`}</Text>
      <Pressable onPress={() => send('hello')}>
        <Text style={{ color: colors.primary }}>Send Hello</Text>
      </Pressable>
      {lastMessage && (
        <Text style={{ color: colors.text }}>{lastMessage}</Text>
      )}
    </Box>
  );
}
```
---
Platforms: love2d

=== CRITICAL RULES ===
- send() is a no-op if the connection is not in 'open' state.
- Only string messages are supported. Serialize objects with JSON.stringify before sending.
- lastMessage only holds the single most recent message. For message history, accumulate in your own state.
- Passing null closes the connection and resets all state.
