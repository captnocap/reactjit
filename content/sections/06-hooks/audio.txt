=== OVERVIEW ===
The @reactjit/audio package exposes a modular synthesis system as React hooks. Audio modules (oscillators, filters, envelopes, mixers, samplers, clocks, sequencers) are wired together into a signal graph called the rack. State flows from the Lua audio engine into React via bridge events; commands flow back via RPC. MIDI input is also surfaced through this package, including device enumeration, CC-to-parameter mapping, and a learn mode for real-time controller assignment. The package also provides sample file loading, microphone recording, a sample-accurate BPM clock, and a step sequencer for building drum machines and other rhythmic applications.

=== API / SYNTAX ===

--- useAudioInit ---

Initialize the audio engine. Call this once at the root of your app before using any other audio hooks. Returns true when the engine is ready. Resolves even if audio hardware is unavailable, so the UI always renders.

  import { useAudioInit } from '@reactjit/audio';

  const ready: boolean = useAudioInit();

Parameters: none
Returns: boolean — true once the engine has initialized (or failed gracefully)

--- useRack ---

Access the full audio rack: the list of active modules, the connections between them, and operations to mutate the graph.

  import { useRack } from '@reactjit/audio';

  const rack: UseRackResult = useRack();

Parameters: none

Returns: UseRackResult
  modules       ModuleState[]   All modules currently in the rack
  connections   Connection[]    All active port connections
  addModule     (type: string, id: string, params?: Record<string, any>) => Promise<any>
  removeModule  (id: string) => Promise<any>
  connect       (fromId: string, fromPort: string, toId: string, toPort: string) => Promise<any>
  disconnect    (fromId: string, fromPort: string, toId: string, toPort: string) => Promise<any>

ModuleState fields:
  id           string
  type         string
  params       Record<string, any>
  ports        Record<string, { type: PortType; direction: PortDirection }>
  activeNotes  Record<string, { note: number; envelope: number }> | undefined

Connection fields:
  fromId    string
  fromPort  string
  toId      string
  toPort    string
  type      PortType   ('audio' | 'control' | 'midi')

--- useModule ---

Access and control a single module by its ID. Subscribes to rack state updates and surfaces only the named module's params and ports.

  import { useModule } from '@reactjit/audio';

  const mod: UseModuleResult = useModule(moduleId: string);

Parameters:
  moduleId  string  The ID assigned to the module when it was added

Returns: UseModuleResult
  id           string
  type         string
  params       Record<string, any>
  ports        Record<string, { type: PortType; direction: PortDirection }>
  activeNotes  Record<string, { note: number; envelope: number }> | undefined
  setParam     (name: string, value: any) => Promise<any>

--- useParam ---

Read and write a single parameter on a module. Provides an optimistic update: the local value is set immediately while the RPC call propagates to Lua.

  import { useParam } from '@reactjit/audio';

  const [value, setParam] = useParam(moduleId: string, paramName: string);

Parameters:
  moduleId   string  Module to target
  paramName  string  Parameter name on that module

Returns: [any, (value: any) => Promise<any>]
  [0]  Current parameter value (null until first rack state event)
  [1]  Setter — sends RPC and applies optimistic update

--- useMIDI ---

Access connected MIDI devices, current CC mappings, learn mode state, and operations to add or remove mappings.

  import { useMIDI } from '@reactjit/audio';

  const midi: UseMIDIResult = useMIDI();

Parameters: none

Returns: UseMIDIResult
  available  boolean          Whether a MIDI subsystem is available
  devices    MIDIDevice[]     Connected devices
  mappings   MIDIMapping[]    Active CC-to-parameter mappings
  learning   { moduleId: string; param: string } | null   Active learn target, or null
  learn      (moduleId: string, param: string) => Promise<any>   Arm learn mode
  map        (moduleId: string, param: string, channel: number, cc: number) => Promise<any>
  unmap      (moduleId: string, param: string) => Promise<any>

MIDIDevice fields:
  id         string
  name       string
  connected  boolean

MIDIMapping fields:
  channel   number
  cc        number
  moduleId  string
  param     string

--- useMIDINote ---

Subscribe to raw MIDI note on/off events from any connected device. The handler is stable — reassigning it between renders does not re-subscribe.

  import { useMIDINote } from '@reactjit/audio';

  useMIDINote(handler: (event: MIDINoteEvent) => void);

Parameters:
  handler  (event: MIDINoteEvent) => void

Returns: void

MIDINoteEvent fields:
  note      number   MIDI note number (0–127)
  velocity  number   Velocity (0–127; 0 on note-off)
  on        boolean  true = note on, false = note off
  channel   number   MIDI channel (1–16)
  device    string   Device ID

--- useMIDICC ---

Subscribe to raw MIDI CC (continuous controller) events from any connected device.

  import { useMIDICC } from '@reactjit/audio';

  useMIDICC(handler: (event: MIDICCEvent) => void);

Parameters:
  handler  (event: MIDICCEvent) => void

Returns: void

MIDICCEvent fields:
  cc       number   CC number (0–127)
  value    number   CC value (0–127)
  channel  number   MIDI channel (1–16)
  device   string   Device ID

--- useClock ---

Access a clock module's beat position and transport controls. The clock runs sample-accurate in the Lua audio engine. Position updates arrive via the 30fps state push.

  import { useClock } from '@reactjit/audio';

  const clock: UseClockResult = useClock(moduleId: string);

Parameters:
  moduleId  string  The clock module's ID

Returns: UseClockResult
  beat         number   Current beat within the bar (0–3)
  bar          number   Current bar number
  step         number   Current step (depends on division)
  phase        number   Fractional position within current step (0–1)
  running      boolean  Whether the clock is ticking
  bpm          number   Current tempo
  start        () => Promise<any>   Start the clock
  stop         () => Promise<any>   Stop the clock
  setBpm       (bpm: number) => Promise<any>   Set tempo
  setDivision  (division: string) => Promise<any>   Set division ("1/4", "1/8", "1/16", "1/32")
  setSwing     (swing: number) => Promise<any>   Set swing amount (0–1)

--- useClockEvent ---

Subscribe to clock tick events. Fires on each tick at the audio engine's rate. The handler is stable across re-renders.

  import { useClockEvent } from '@reactjit/audio';

  useClockEvent(handler: (event: ClockTickEvent) => void);

Parameters:
  handler  (event: ClockTickEvent) => void

Returns: void

ClockTickEvent fields:
  beat  number  Beat within bar
  bar   number  Bar number
  step  number  Step number
  bpm   number  Current tempo

--- useSampler ---

Manage a sampler module's slots: load audio files, clear slots, trigger playback. Samples are loaded from the Love2D filesystem. MIDI note 36 = slot 1, 37 = slot 2, etc. (GM drum map starting at C2).

  import { useSampler } from '@reactjit/audio';

  const sampler: UseSamplerResult = useSampler(moduleId: string);

Parameters:
  moduleId  string  The sampler module's ID

Returns: UseSamplerResult
  slots        Record<number, SampleSlot | null>  Loaded sample info per slot (1–16)
  voices       SamplerVoice[]                     Currently playing voices
  loadSample   (slot: number, path: string, mode?: 'oneshot' | 'loop') => Promise<any>
  clearSample  (slot: number) => Promise<any>
  trigger      (slot: number, velocity?: number) => Promise<any>

SampleSlot fields:
  name        string   Filename of the loaded sample
  duration    number   Duration in seconds
  sampleRate  number   Original sample rate
  mode        string   "oneshot" or "loop"

SamplerVoice fields:
  slot      number  Which slot is playing
  position  number  Current playback position in seconds
  duration  number  Total duration of the sample

--- useRecorder ---

Record audio from a microphone or input device into a sampler slot. Uses Love2D's RecordingDevice API. Call listDevices() to enumerate available inputs, startRecording() to begin capture, and stopRecording() to finalize the sample into its target slot.

  import { useRecorder } from '@reactjit/audio';

  const recorder: UseRecorderResult = useRecorder();

Parameters: none

Returns: UseRecorderResult
  devices         AudioRecordingDevice[]  Available recording devices
  recording       RecordingState          Current recording status
  listDevices     () => Promise<any>      Enumerate available devices
  startRecording  (moduleId: string, slot: number, deviceIndex?: number) => Promise<any>
  stopRecording   () => Promise<any>

AudioRecordingDevice fields:
  index  number  Device index (1-based)
  name   string  Device name

RecordingState fields:
  active    boolean        Whether recording is in progress
  moduleId  string | null  Target sampler module
  slot      number | null  Target slot
  device    string | null  Recording device name
  duration  number         Elapsed recording time in seconds

--- useSequencer ---

Control a step sequencer module's pattern and track assignments. The sequencer advances on rising edges from a connected clock module's gate output. Each track targets a different module for note triggering.

  import { useSequencer } from '@reactjit/audio';

  const seq: UseSequencerResult = useSequencer(moduleId: string);

Parameters:
  moduleId  string  The sequencer module's ID

Returns: UseSequencerResult
  pattern         Record<string, Record<string, StepData>>  Pattern grid [track][step]
  currentStep     number                                     Current step position
  trackTargets    Record<string, string>                     Track → target module ID
  setStep         (track: number, step: number, active: boolean, note?: number, velocity?: number) => Promise<any>
  setTrackTarget  (track: number, targetModuleId: string) => Promise<any>
  clearPattern    () => Promise<any>

StepData fields:
  active    boolean  Whether this step fires
  note      number   MIDI note number (default 36)
  velocity  number   Velocity (default 100)

=== EXAMPLES ===

-- Minimal synth setup --

Initialize the engine, build a one-oscillator patch, and render a waveform selector:

  import { useAudioInit, useRack, useParam } from '@reactjit/audio';
  import { Box, Text } from '@reactjit/core';

  function SynthApp() {
    const ready = useAudioInit();
    const rack = useRack();

    // Build the patch once on first ready
    const patchBuilt = useRef(false);
    useEffect(() => {
      if (!ready || patchBuilt.current) return;
      patchBuilt.current = true;
      rack.addModule('oscillator', 'osc1', { waveform: 'saw', frequency: 440 });
      rack.addModule('amplifier', 'amp1', { gain: 0.8 });
      rack.addModule('output', 'audio_out', {});
      rack.connect('osc1', 'audio_out', 'amp1', 'input');
      rack.connect('amp1', 'audio_out', 'audio_out', 'input');
    }, [ready]);

    return (
      <Box width="100%" height="100%">
        {ready ? <WaveformSelector moduleId="osc1" /> : <Text fontSize={14}>Starting audio...</Text>}
      </Box>
    );
  }

  const WAVEFORMS = ['sine', 'saw', 'square', 'triangle'];

  function WaveformSelector({ moduleId }: { moduleId: string }) {
    const [waveform, setWaveform] = useParam(moduleId, 'waveform');

    return (
      <Box flexDirection="row" gap={8}>
        {WAVEFORMS.map((w) => (
          <Box
            key={w}
            backgroundColor={waveform === w ? '#4CAF50' : '#333'}
            padding={8}
            onPress={() => setWaveform(w)}
          >
            <Text fontSize={12}>{w}</Text>
          </Box>
        ))}
      </Box>
    );
  }

-- MIDI learn workflow --

Arm a parameter for learn, then watch the mapping appear:

  import { useMIDI } from '@reactjit/audio';
  import { Box, Text } from '@reactjit/core';

  function MIDILearnButton({ moduleId, param }: { moduleId: string; param: string }) {
    const midi = useMIDI();

    const isLearning =
      midi.learning?.moduleId === moduleId && midi.learning?.param === param;

    const existingMap = midi.mappings.find(
      (m) => m.moduleId === moduleId && m.param === param,
    );

    return (
      <Box flexDirection="row" gap={8} alignItems="center">
        <Box
          backgroundColor={isLearning ? '#FF5722' : '#555'}
          padding={8}
          onPress={() => midi.learn(moduleId, param)}
        >
          <Text fontSize={12}>{isLearning ? 'Listening...' : 'MIDI Learn'}</Text>
        </Box>
        {existingMap && (
          <Text fontSize={11}>
            CH{existingMap.channel} CC{existingMap.cc}
          </Text>
        )}
      </Box>
    );
  }

-- MIDI note display --

Show which notes are currently held:

  import { useMIDINote } from '@reactjit/audio';
  import { useState } from 'react';
  import { Box, Text } from '@reactjit/core';

  function NoteMonitor() {
    const [held, setHeld] = useState<number[]>([]);

    useMIDINote((event) => {
      setHeld((prev) =>
        event.on ? [...prev, event.note] : prev.filter((n) => n !== event.note),
      );
    });

    return (
      <Box>
        <Text fontSize={14}>Held notes: {held.join(', ') || 'none'}</Text>
      </Box>
    );
  }

-- Drum machine with clock + sequencer + sampler --

Wire a clock, sequencer, and sampler together for a basic drum machine:

  import { useAudioInit, useRack, useClock, useSequencer, useSampler } from '@reactjit/audio';
  import { Box, Text } from '@reactjit/core';
  import { useRef, useEffect } from 'react';

  function DrumMachine() {
    const ready = useAudioInit();
    const rack = useRack();
    const clock = useClock('clock1');
    const seq = useSequencer('seq1');
    const sampler = useSampler('sampler1');

    const patchBuilt = useRef(false);
    useEffect(() => {
      if (!ready || patchBuilt.current) return;
      patchBuilt.current = true;

      // Build the rack
      rack.addModule('clock', 'clock1', { bpm: 120, division: '1/16' });
      rack.addModule('sequencer', 'seq1', { steps: 16, tracks: 4 });
      rack.addModule('sampler', 'sampler1');

      // Clock drives the sequencer
      rack.connect('clock1', 'gate_out', 'seq1', 'clock_in');

      // Sequencer track 0 triggers sampler1
      seq.setTrackTarget(0, 'sampler1');

      // Load samples into slots
      sampler.loadSample(1, 'samples/kick.wav');
      sampler.loadSample(2, 'samples/snare.wav');
      sampler.loadSample(3, 'samples/hihat.wav');

      // Set a simple 4-on-the-floor pattern
      // Track 0: kick on beats 0, 4, 8, 12
      seq.setStep(0, 0, true, 36, 100);
      seq.setStep(0, 4, true, 36, 100);
      seq.setStep(0, 8, true, 36, 100);
      seq.setStep(0, 12, true, 36, 100);
    }, [ready]);

    return (
      <Box width="100%" height="100%">
        <Text fontSize={16}>BPM: {clock.bpm}</Text>
        <Text fontSize={14}>Step: {seq.currentStep}</Text>
        <Box
          backgroundColor={clock.running ? '#4CAF50' : '#555'}
          padding={8}
          onPress={() => clock.running ? clock.stop() : clock.start()}
        >
          <Text fontSize={14}>{clock.running ? 'Stop' : 'Play'}</Text>
        </Box>
      </Box>
    );
  }

-- Recording a sample from microphone --

Record audio from the default input device into a sampler slot:

  import { useRecorder, useSampler } from '@reactjit/audio';
  import { Box, Text } from '@reactjit/core';

  function RecordButton({ moduleId, slot }: { moduleId: string; slot: number }) {
    const recorder = useRecorder();
    const sampler = useSampler(moduleId);

    const isRecording = recorder.recording.active;

    return (
      <Box>
        <Box
          backgroundColor={isRecording ? '#F44336' : '#555'}
          padding={8}
          onPress={() => {
            if (isRecording) {
              recorder.stopRecording();
            } else {
              recorder.startRecording(moduleId, slot);
            }
          }}
        >
          <Text fontSize={14}>{isRecording ? 'Stop Recording' : 'Record'}</Text>
        </Box>
        {isRecording && (
          <Text fontSize={12}>{recorder.recording.duration.toFixed(1)}s</Text>
        )}
        {sampler.slots[slot] && (
          <Text fontSize={12}>Slot {slot}: {sampler.slots[slot]?.name} ({sampler.slots[slot]?.duration.toFixed(1)}s)</Text>
        )}
      </Box>
    );
  }
