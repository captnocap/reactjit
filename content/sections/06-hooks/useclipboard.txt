=== OVERVIEW ===
useClipboard provides copy and paste functions that interact with the system clipboard via Lua RPCs. It also exposes a `copied` boolean that stays true for 2 seconds after a copy -- useful for "Copied!" feedback in UI.

=== API / SYNTAX ===
```tsx
import { useClipboard } from '@reactjit/core';
```

Signature:
```tsx
function useClipboard(): {
  copy: (text: string) => Promise<void>;
  paste: () => Promise<string>;
  copied: boolean;
}
```

Returns:
| Field  | Type                          | Description                                     |
|--------|-------------------------------|-------------------------------------------------|
| copy   | (text: string) => Promise     | Write text to the system clipboard              |
| paste  | () => Promise<string>         | Read text from the system clipboard             |
| copied | boolean                       | true for 2 seconds after a successful copy      |

=== EXAMPLES ===
Example 1: Copy button with feedback
---
```tsx
import { useClipboard } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

function CopyButton({ text }: { text: string }) {
  const colors = useThemeColors();
  const { copy, copied } = useClipboard();

  return (
    <Pressable onPress={() => copy(text)}>
      <Text style={{ color: colors.primary, fontSize: 14 }}>
        {copied ? 'Copied!' : 'Copy'}
      </Text>
    </Pressable>
  );
}
```
---
Platforms: love2d

Example 2: Copy and paste workflow
---
```tsx
import { useClipboard, useHotkey } from '@reactjit/core';

function TextEditor() {
  const { copy, paste } = useClipboard();
  const [text, setText] = useState('');

  useHotkey('ctrl+c', () => copy(text));
  useHotkey('ctrl+v', async () => {
    const pasted = await paste();
    setText(prev => prev + pasted);
  });

  return <Text>{text}</Text>;
}
```
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Clipboard operations go through `clipboard:write` and `clipboard:read` RPCs which call `love.system.setClipboardText()` and `love.system.getClipboardText()`. These interact with the OS clipboard, so copied text is available in other applications and vice versa.

=== CRITICAL RULES ===
- The `copied` flag auto-resets after 2 seconds. Multiple rapid copies reset the timer.
- The copy/paste functions are async (return Promises) because they cross the bridge. Await them if you need confirmation.
