=== OVERVIEW ===
usePeerServer hosts a WebSocket server from within your application. Peers connect to the specified port, and you can send messages to individual clients or broadcast to all. Useful for local multiplayer, device pairing, or tool communication.

Pass `null` as the port to shut down the server.

=== API / SYNTAX ===
```tsx
import { usePeerServer } from '@reactjit/core';
```

Signature:
```tsx
function usePeerServer(port: number | null): {
  ready: boolean;
  peers: number[];
  broadcast: (data: string) => void;
  send: (clientId: number, data: string) => void;
  lastMessage: PeerMessage | null;
  error: string | null;
}
```

Types:
```tsx
interface PeerMessage {
  clientId: number;
  data: string;
}
```

Parameters:
| Param | Type           | Default  | Description                        |
|-------|----------------|----------|------------------------------------|
| port  | number \| null | required | Port to listen on, or null to stop |

Returns:
| Field       | Type                | Description                           |
|-------------|---------------------|---------------------------------------|
| ready       | boolean             | True when server is listening         |
| peers       | number[]            | Array of connected client IDs         |
| broadcast   | (data: string) => void | Send to all connected peers        |
| send        | (clientId, data) => void | Send to a specific peer           |
| lastMessage | PeerMessage \| null | Most recent message from any peer     |
| error       | string \| null      | Error if the server failed to start   |

=== EXAMPLES ===
Example 1: Host a server and broadcast
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { usePeerServer } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function GameHost() {
  const colors = useThemeColors();
  const server = usePeerServer(8080);

  return (
    <Box style={{ gap: 8, padding: 16 }}>
      <Text style={{ color: colors.muted }}>
        {server.ready ? `Listening on :8080` : 'Starting...'}
      </Text>
      <Text style={{ color: colors.text }}>
        {`${server.peers.length} peer(s) connected`}
      </Text>
      <Pressable onPress={() => server.broadcast('ping')}>
        <Text style={{ color: colors.primary }}>Broadcast Ping</Text>
      </Pressable>
      {server.lastMessage && (
        <Text style={{ color: colors.text }}>
          {`Client ${server.lastMessage.clientId}: ${server.lastMessage.data}`}
        </Text>
      )}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- The server requires the `__wsListen` host function to be available in the runtime. If it is missing, the hook silently does nothing.
- Client IDs are assigned by the Lua runtime and are stable for the lifetime of a connection.
- lastMessage only holds the single most recent message across all peers. For per-peer message history, accumulate in your own state.
- Passing null as port shuts down the server and disconnects all peers.
