=== OVERVIEW ===
usePorts monitors all listening TCP and UDP ports on the system. Each port entry includes the port number, host, PID, process name, protocol, and state. The hook also provides a kill function to terminate processes by PID.

Polls via useLuaInterval. Returns a loading state and supports structured logging via toSysLog.

=== API / SYNTAX ===
```tsx
import { usePorts } from '@reactjit/core';
```

Signature:
```tsx
function usePorts(interval?: number): PortMonitor
```

Types:
```tsx
interface PortMonitor {
  list: PortInfo[];
  loading: boolean;
  kill: (pid: number, signal?: string) => Promise<boolean>;
  toSysLog: (path: string) => void;
}

interface PortInfo {
  port: number;
  host: string;
  pid: number;
  process: string;
  protocol: 'tcp' | 'udp';
  state: string;
  toSysLog: (path: string) => void;
}
```

Parameters:
| Param    | Type   | Default | Description                 |
|----------|--------|---------|-----------------------------|
| interval | number | 2000    | Ms between refreshes        |

=== EXAMPLES ===
Example 1: List open ports
---
```tsx
import { Box, Text } from '@reactjit/core';
import { usePorts } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function PortList() {
  const colors = useThemeColors();
  const ports = usePorts(2000);

  return (
    <Box style={{ gap: 4, padding: 16 }}>
      {ports.list.map(p => (
        <Text key={`${p.protocol}-${p.port}`} style={{ color: colors.text }}>
          {`${p.protocol}:${p.port} - ${p.process} (PID ${p.pid})`}
        </Text>
      ))}
    </Box>
  );
}
```

Platforms: love2d

Example 2: Kill a process by PID
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { usePorts } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function PortManager() {
  const colors = useThemeColors();
  const ports = usePorts(2000);

  const handleKill = async (pid: number) => {
    const success = await ports.kill(pid);
    if (!success) console.log('Kill failed');
  };

  return (
    <Box style={{ gap: 4, padding: 16 }}>
      {ports.list.map(p => (
        <Box key={`${p.protocol}-${p.port}`} style={{ flexDirection: 'row', gap: 8 }}>
          <Text style={{ color: colors.text, flexGrow: 1 }}>
            {`${p.protocol}:${p.port} ${p.process}`}
          </Text>
          <Pressable onPress={() => handleKill(p.pid)}>
            <Text style={{ color: colors.primary }}>Kill</Text>
          </Pressable>
        </Box>
      ))}
    </Box>
  );
}
```
---
Platforms: love2d

=== CRITICAL RULES ===
- Requires the Lua bridge. Returns empty list without it.
- kill() sends SIGTERM by default. Pass 'KILL' for SIGKILL.
- Port data comes from the Lua side parsing system tools (ss, netstat). Root/sudo may be needed to see all processes.
