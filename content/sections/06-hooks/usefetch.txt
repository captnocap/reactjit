=== OVERVIEW ===
useFetch wraps the global fetch() API in a React hook that manages loading, data, and error state. It works across all ReactJIT targets: on Love2D the fetch polyfill routes HTTP requests through LuaSocket and local file paths through love.filesystem; on web and Node targets the platform's native fetch is used. Pass null as the URL to clear state and skip any in-flight request, which is useful for conditional fetching. The options argument accepts standard RequestInit and is kept in a ref so changing it does not trigger a re-fetch.

=== API / SYNTAX ===
```tsx
import { useFetch } from '@reactjit/core';
```

Parameters:
| Param   | Type           | Default   | Description |
|---------|----------------|-----------|-------------|
| url     | string \| null | required  | The URL or local file path to fetch. Pass null to skip fetching and clear state. |
| options | RequestInit    | undefined | Standard fetch options (method, headers, body, etc.). Changes to this object do not trigger a re-fetch. |

Returns:
| Field   | Type        | Description |
|---------|-------------|-------------|
| data    | T \| null   | The parsed JSON response body, or null while loading or on error. |
| error   | Error \| null | The fetch or HTTP error, or null when successful or still loading. |
| loading | boolean     | True while the request is in flight. Starts as true when url is non-null. |

=== EXAMPLES ===
Example 1: Fetch a list of users from an API
---
```tsx
import { useFetch, Box, Text } from '@reactjit/core';

interface User {
  id: number;
  name: string;
}

function UserList() {
  const { data, loading, error } = useFetch<User[]>('https://api.example.com/users');

  if (loading) return <Text fontSize={14}>Loading...</Text>;
  if (error) return <Text fontSize={14}>Error: {error.message}</Text>;

  return (
    <Box flexDirection="column" gap={4} padding={12}>
      {data?.map(user => (
        <Text key={user.id} fontSize={14}>{user.name}</Text>
      ))}
    </Box>
  );
}
```

Example 2: Load a local config file bundled with the Love2D project
---
```tsx
import { useFetch, Box, Text } from '@reactjit/core';

interface Config {
  version: string;
  theme: string;
}

function ConfigDisplay() {
  const { data: config, loading } = useFetch<Config>('data/config.json');

  if (loading || !config) return <Text fontSize={14}>Loading config...</Text>;

  return (
    <Box flexDirection="column" gap={4} padding={8}>
      <Text fontSize={14}>Version: {config.version}</Text>
      <Text fontSize={14}>Theme: {config.theme}</Text>
    </Box>
  );
}
```
---
Platforms: love2d

Example 3: Conditional fetch — only fetch when an ID is selected
---
```tsx
import { useState } from 'react';
import { useFetch, Box, Text, Pressable } from '@reactjit/core';

interface Item {
  id: number;
  title: string;
}

function ItemViewer() {
  const [selectedId, setSelectedId] = useState<number | null>(null);
  const url = selectedId != null ? `https://api.example.com/items/${selectedId}` : null;
  const { data, loading } = useFetch<Item>(url);

  return (
    <Box flexDirection="column" gap={8} padding={12} width="100%">
      <Box flexDirection="row" gap={8} width="100%">
        {[1, 2, 3].map(id => (
          <Pressable key={id} onPress={() => setSelectedId(id)}>
            <Box backgroundColor="#333" padding={8}>
              <Text fontSize={13}>Item {id}</Text>
            </Box>
          </Pressable>
        ))}
      </Box>
      {loading && <Text fontSize={14}>Loading...</Text>}
      {data && <Text fontSize={15}>{data.title}</Text>}
    </Box>
  );
}
```

Platforms: All

Example 4: POST request with custom options
---
```tsx
import { useFetch, Text } from '@reactjit/core';

interface CreateResponse {
  id: number;
}

function Creator() {
  const { data, loading, error } = useFetch<CreateResponse>(
    'https://api.example.com/items',
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ name: 'New Item' }),
    }
  );

  if (loading) return <Text fontSize={14}>Creating...</Text>;
  if (error) return <Text fontSize={14}>Failed: {error.message}</Text>;
  if (data) return <Text fontSize={14}>Created with id: {data.id}</Text>;
  return null;
}
```
