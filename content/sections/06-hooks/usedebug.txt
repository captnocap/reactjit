title: useDebug
description: Register live component data in a global registry accessible from the inspector console.
category: Hooks

=== OVERVIEW ===
useDebug registers arbitrary component data under a named key on `globalThis.__debug`, making it available for live evaluation in the Love2D inspector console (F12) or any JavaScript REPL attached to the runtime. Data is written on every render so values are always current, and the entry is automatically removed when the component unmounts. This hook adds zero visible output — it is purely a debugging side-channel, not an overlay or HUD.

=== API / SYNTAX ===
```tsx
import { useDebug, getDebugData, registerDebug } from '@reactjit/core';
```

Hook signature:
```tsx
useDebug(key: string, data: any): void
```

Parameters:

| Parameter | Type   | Description                                                           |
|-----------|--------|-----------------------------------------------------------------------|
| key       | string | Unique identifier for the debug entry. Usually the component name.    |
| data      | any    | Any value to expose. Updated every render. Cleaned up on unmount.     |

Returns: void

Companion utilities:
```tsx
// Read all currently registered debug entries (non-hook, for tooling use).
getDebugData(): Record<string, any>

// Register debug data imperatively outside of React (e.g. in a game loop).
// Returns an unregister function — call it when done.
registerDebug(key: string, data: any): () => void
```

Console access patterns:
```
> __debug                         // all registered entries
> __debug['PlayerHUD']            // specific component
> __debug['PlayerHUD'].health     // specific field
:watch __debug['PlayerHUD'].fps   // live-updating watch expression
```

=== EXAMPLES ===
Example 1: Basic component state inspection
---
```tsx
import { useDebug } from '@reactjit/core';

function PlayerHUD({ player }: { player: Player }) {
  const [health, setHealth] = useState(player.maxHealth);
  const [mana, setMana]     = useState(player.maxMana);

  useDebug('PlayerHUD', { health, mana, level: player.level });

  return (
    <Box>
      <Text fontSize={16}>HP: {health}</Text>
      <Text fontSize={16}>MP: {mana}</Text>
    </Box>
  );
}

// Inspector console (F12):
// > __debug['PlayerHUD']
// { health: 85, mana: 40, level: 12 }
```

Example 2: FPS and frame counter via watch expression
---
```tsx
import { useRef } from 'react';
import { useDebug } from '@reactjit/core';

function GameLoop({ deltaTime }: { deltaTime: number }) {
  const frameCount = useRef(0);
  frameCount.current++;

  const fps = deltaTime > 0 ? Math.round(1 / deltaTime) : 0;

  useDebug('GameLoop', { fps, frameCount: frameCount.current, deltaTime });

  return null;
}

// Inspector console:
// :watch __debug['GameLoop'].fps
// :watch __debug['GameLoop'].frameCount
```

Platforms: All

Example 3: Multiple components sharing the debug registry
---
```tsx
import { useDebug } from '@reactjit/core';

function Inventory({ items }: { items: Item[] }) {
  useDebug('Inventory', {
    count: items.length,
    totalWeight: items.reduce((s, i) => s + i.weight, 0),
  });
  return <Box>...</Box>;
}

function QuestLog({ quests }: { quests: Quest[] }) {
  useDebug('QuestLog', {
    active:    quests.filter(q => q.active).length,
    completed: quests.filter(q => q.completed).length,
  });
  return <Box>...</Box>;
}

// Inspector console:
// > __debug
// { Inventory: { count: 23, totalWeight: 45.2 }, QuestLog: { active: 3, completed: 17 } }
```
---
Platforms: All

Example 4: Imperative registration from a game service
---
```tsx
import { registerDebug } from '@reactjit/core';

const gameState = { running: true, paused: false, tickCount: 0 };

// Register once at startup; update the reference object in place each tick.
const unregister = registerDebug('GameState', gameState);

function tick() {
  gameState.tickCount++;
  // The object reference is already registered — mutations are visible immediately.
  requestAnimationFrame(tick);
}

tick();

// On teardown:
unregister();
```

Example 5: Read all debug data from tooling code
---
```tsx
import { getDebugData } from '@reactjit/core';

function DebugSnapshot() {
  const snapshot = getDebugData();
  const keys = Object.keys(snapshot);

  return (
    <Box style={{ padding: 8, gap: 4 }}>
      {keys.map(k => (
        <Text key={k} fontSize={12}>{k}: {JSON.stringify(snapshot[k])}</Text>
      ))}
    </Box>
  );
}
```
