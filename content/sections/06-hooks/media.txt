=== OVERVIEW ===
The @reactjit/media package provides two sets of hooks: archive hooks for working with compressed files (RAR, ZIP, 7z, TAR, and others), and media library hooks for scanning directories of media files. The archive hooks delegate to a Lua-side libarchive integration; the scan hooks walk the filesystem. useMediaIndex combines both — it looks inside archives to surface their contents as first-class MediaFile entries alongside regular files.

=== API / SYNTAX ===

--- useArchive ---

List all entries inside an archive. Fetches automatically when filepath changes. Provides a refresh function for manual re-fetching.

  import { useArchive } from '@reactjit/media';

  const { entries, loading, error, refresh } = useArchive(filepath: string | null);

Parameters:
  filepath  string | null  Absolute path to the archive file. Pass null to idle.

Returns:
  entries   ArchiveEntry[]  All entries in the archive (files, directories, symlinks)
  loading   boolean
  error     string | null
  refresh   () => Promise<void>   Re-fetch the entry list

ArchiveEntry fields:
  path       string                        Path within the archive
  size       number                        Size in bytes
  type       'file' | 'directory' | 'symlink'
  mtime      number                        Modification timestamp (Unix seconds)
  encrypted  boolean

--- useArchiveInfo ---

Fetch a summary of an archive without listing every entry. Returns counts, total size, and an extension histogram.

  import { useArchiveInfo } from '@reactjit/media';

  const { info, loading, error } = useArchiveInfo(filepath: string | null);

Parameters:
  filepath  string | null

Returns:
  info     ArchiveInfo | null
  loading  boolean
  error    string | null

ArchiveInfo fields:
  totalEntries  number
  totalSize     number                        Total uncompressed bytes
  fileCount     number
  dirCount      number
  extensions    Record<string, number>        Counts per extension, e.g. { mp4: 3, srt: 3 }

--- useArchiveRead ---

Returns an imperative function that reads a single file from inside an archive. Call it on demand rather than on mount. Throws on error.

  import { useArchiveRead } from '@reactjit/media';

  const read = useArchiveRead();
  const { content, size } = await read(filepath, entryPath, maxBytes?);

Parameters (of the returned function):
  filepath   string   Absolute path to the archive
  entryPath  string   Path of the entry within the archive
  maxBytes   number   (optional) Byte limit on the returned content

Returns: Promise<{ content: string; size: number }>
  content  string   File content as a string
  size     number   Full entry size in bytes (may exceed maxBytes if truncated)

Throws: Error with message from Lua if the entry cannot be read.

--- useArchiveSearch ---

Search archive entries by a Lua pattern. Fetches automatically when filepath or pattern changes.

  import { useArchiveSearch } from '@reactjit/media';

  const { matches, loading } = useArchiveSearch(filepath: string | null, pattern: string);

Parameters:
  filepath  string | null  Archive to search. Pass null to idle.
  pattern   string         Lua pattern matched against each entry path (e.g. '%.mp4$')

Returns:
  matches  ArchiveEntry[]
  loading  boolean

--- useMediaLibrary ---

Scan a directory for media files. Returns typed MediaFile entries with source set to 'filesystem'. Does not look inside archives; use useMediaIndex for that.

  import { useMediaLibrary } from '@reactjit/media';

  const { files, stats, loading, error, rescan } = useMediaLibrary(
    directory: string | null,
    options?: MediaLibraryOptions,
  );

Parameters:
  directory  string | null
  options    MediaLibraryOptions (optional)

MediaLibraryOptions fields:
  recursive  boolean      Scan subdirectories (default true)
  maxDepth   number       Maximum recursion depth (default 10)
  autoScan   boolean      Scan on mount (default true)
  filter     MediaType[]  Only return entries of these types

Returns:
  files    MediaFile[]   All scanned files matching the filter
  stats    DirStats      Computed from the returned files (see below)
  loading  boolean
  error    string | null
  rescan   () => Promise<void>

MediaFile fields:
  path          string                        Full filesystem path
  name          string                        Filename only
  size          number                        Bytes
  mtime         number | undefined            Modification timestamp (Unix seconds)
  type          MediaType                     Classified type
  source        'filesystem' | 'archive'
  archivePath   string | undefined            Set when source is 'archive'
  archiveEntry  string | undefined            Set when source is 'archive'

MediaType values:
  'video' | 'audio' | 'image' | 'subtitle' | 'document' | 'archive' | 'metadata' | 'unknown'

DirStats fields:
  total        number
  byType       Partial<Record<MediaType, number>>
  totalSize    number
  largestFile  MediaFile | null

--- useMediaStats ---

Fetch quick directory statistics without loading every file. Lighter than useMediaLibrary when you only need counts and sizes.

  import { useMediaStats } from '@reactjit/media';

  const { stats, loading } = useMediaStats(directory: string | null);

Parameters:
  directory  string | null

Returns:
  stats    DirStats | null
  loading  boolean

--- useMediaIndex ---

Deep-index a directory, scanning both filesystem files and the contents of archives. Archive contents appear as MediaFile entries with source set to 'archive'. This is the archive walker.

  import { useMediaIndex } from '@reactjit/media';

  const { index, loading, error, rescan } = useMediaIndex(
    directory: string | null,
    options?: MediaIndexOptions,
  );

Parameters:
  directory  string | null
  options    MediaIndexOptions (optional)

MediaIndexOptions fields:
  indexArchives    boolean   Look inside archives (default true)
  archivePattern   string    Only index archives whose filename matches this Lua pattern
  filter           MediaType[]   Filter results by type

Returns:
  index    MediaFile[]   Flat list of all discovered files including archive contents
  loading  boolean
  error    string | null
  rescan   () => Promise<void>

=== EXAMPLES ===

-- Browse an archive --

List entries in a RAR file and let the user pick one to read:

  import { useArchive, useArchiveRead } from '@reactjit/media';
  import { useState } from 'react';
  import { Box, Text } from '@reactjit/core';

  function ArchiveBrowser({ filepath }: { filepath: string }) {
    const { entries, loading, error } = useArchive(filepath);
    const read = useArchiveRead();
    const [content, setContent] = useState<string | null>(null);

    async function openEntry(path: string) {
      try {
        const result = await read(filepath, path, 65536);
        setContent(result.content);
      } catch (e: any) {
        setContent('Error: ' + e.message);
      }
    }

    if (loading) return <Text fontSize={13}>Loading...</Text>;
    if (error) return <Text fontSize={13}>Error: {error}</Text>;

    const files = entries.filter((e) => e.type === 'file');

    return (
      <Box width="100%" height="100%" flexDirection="row">
        <Box width={300}>
          {files.map((e) => (
            <Box key={e.path} padding={4} onPress={() => openEntry(e.path)}>
              <Text fontSize={12}>{e.path}</Text>
            </Box>
          ))}
        </Box>
        <Box flexGrow={1} padding={8}>
          {content && <Text fontSize={11}>{content}</Text>}
        </Box>
      </Box>
    );
  }

-- Media library with type filter --

Show only video files in a directory, with a count breakdown by type:

  import { useMediaLibrary, formatSize } from '@reactjit/media';
  import { Box, Text } from '@reactjit/core';

  function VideoLibrary({ dir }: { dir: string }) {
    const { files, stats, loading, rescan } = useMediaLibrary(dir, {
      filter: ['video'],
      recursive: true,
    });

    if (loading) return <Text fontSize={13}>Scanning...</Text>;

    return (
      <Box width="100%" height="100%">
        <Box flexDirection="row" justifyContent="space-between" padding={8}>
          <Text fontSize={14}>{stats.total} videos</Text>
          <Text fontSize={14}>{formatSize(stats.totalSize)}</Text>
          <Box padding={4} onPress={rescan}>
            <Text fontSize={12}>Rescan</Text>
          </Box>
        </Box>
        {files.map((f) => (
          <Box key={f.path} padding={6} flexDirection="row" justifyContent="space-between">
            <Text fontSize={12}>{f.name}</Text>
            <Text fontSize={11}>{formatSize(f.size)}</Text>
          </Box>
        ))}
      </Box>
    );
  }

-- Deep index with archive contents --

Surface video files both on disk and packed inside archives:

  import { useMediaIndex, formatSize } from '@reactjit/media';
  import { Box, Text } from '@reactjit/core';

  function DeepIndex({ dir }: { dir: string }) {
    const { index, loading, error } = useMediaIndex(dir, {
      indexArchives: true,
      filter: ['video'],
    });

    if (loading) return <Text fontSize={13}>Indexing (looking inside archives)...</Text>;
    if (error) return <Text fontSize={13}>Error: {error}</Text>;

    const onDisk = index.filter((f) => f.source === 'filesystem');
    const inArchives = index.filter((f) => f.source === 'archive');

    return (
      <Box width="100%" height="100%">
        <Text fontSize={14}>On disk: {onDisk.length}</Text>
        <Text fontSize={14}>Inside archives: {inArchives.length}</Text>
        {inArchives.map((f) => (
          <Box key={f.path} padding={4}>
            <Text fontSize={11}>{f.name}</Text>
            <Text fontSize={10}>{f.archivePath}</Text>
          </Box>
        ))}
      </Box>
    );
  }

-- Archive info card --

Show a summary of a ZIP without listing every entry:

  import { useArchiveInfo, formatSize } from '@reactjit/media';
  import { Box, Text } from '@reactjit/core';

  function ArchiveCard({ filepath }: { filepath: string }) {
    const { info, loading } = useArchiveInfo(filepath);

    if (loading || !info) return <Text fontSize={12}>Loading...</Text>;

    const topExts = Object.entries(info.extensions)
      .sort((a, b) => b[1] - a[1])
      .slice(0, 3);

    return (
      <Box padding={12} gap={4}>
        <Text fontSize={14}>{info.fileCount} files, {info.dirCount} dirs</Text>
        <Text fontSize={12}>{formatSize(info.totalSize)} uncompressed</Text>
        {topExts.map(([ext, count]) => (
          <Text key={ext} fontSize={11}>.{ext}: {count}</Text>
        ))}
      </Box>
    );
  }
