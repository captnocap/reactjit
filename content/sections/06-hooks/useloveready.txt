=== OVERVIEW ===
useLoveReady returns true when the bridge transport is initialized and ready for communication. Use it to gate rendering until the bridge is available -- any hook that calls bridge methods before it is ready will fail.

In practice, the bridge initializes fast enough that most apps never see `false`. This hook is mainly useful for splash screens or when you need to guarantee that RPC calls will succeed on first render.

=== API / SYNTAX ===
```tsx
import { useLoveReady } from '@reactjit/core';
```

Signature:
```tsx
function useLoveReady(): boolean
```

Returns:
| Type    | Description                                |
|---------|--------------------------------------------|
| boolean | true when the bridge is ready for use      |

=== EXAMPLES ===
Example 1: Loading gate
---
```tsx
import { useLoveReady } from '@reactjit/core';

function App() {
  const ready = useLoveReady();

  if (!ready) {
    return <Text>Initializing...</Text>;
  }

  return <MainApp />;
}
```
