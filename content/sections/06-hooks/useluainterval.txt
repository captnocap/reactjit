=== METADATA ===
title: useLuaInterval
description: Register a Lua-side interval timer that fires callbacks on a steady cadence
category: Hooks
platforms: love2d
keywords: timer, interval, polling, tick, lua, periodic, setinterval
related: useIFTTT, useLoveEvent
difficulty: intermediate

=== OVERVIEW ===
useLuaInterval creates a timer that ticks inside love.update(dt) and pushes a bridge event each interval. This replaces JavaScript's setInterval for hooks that need periodic execution -- the timer lives in Lua, so it ticks even when React is between renders.

The timer is automatically cancelled on unmount or when the interval changes to 0/null/undefined.

=== API / SYNTAX ===
```tsx
import { useLuaInterval } from '@reactjit/core';
```

Signature:
```tsx
function useLuaInterval(
  intervalMs: number | null | undefined,
  callback: () => void
): void
```

Parameters:
| Param      | Type                         | Default  | Description                        |
|------------|------------------------------|----------|------------------------------------|
| intervalMs | number \| null \| undefined  | required | Interval in ms. 0/null disables.   |
| callback   | () => void                   | required | Called each tick                    |

=== EXAMPLES ===
Example 1: Poll for updates every 2 seconds
---
```tsx
import { useState } from 'react';
import { Box, Text } from '@reactjit/core';
import { useLuaInterval, useLoveRPC } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function StatusPoller() {
  const colors = useThemeColors();
  const [status, setStatus] = useState('unknown');
  const rpc = useLoveRPC<string>('app:status');

  useLuaInterval(2000, () => {
    rpc().then(setStatus);
  });

  return <Text style={{ color: colors.text }}>{`Status: ${status}`}</Text>;
}
```
---
Platforms: love2d

Example 2: Conditional timer
---
```tsx
import { useState } from 'react';
import { useLuaInterval } from '@reactjit/core';

export function useCountdown(seconds: number, active: boolean) {
  const [remaining, setRemaining] = useState(seconds);

  useLuaInterval(active && remaining > 0 ? 1000 : null, () => {
    setRemaining(r => r - 1);
  });

  return remaining;
}
```
---
Platforms: love2d

=== CRITICAL RULES ===
- EVERY useLuaInterval in your app MUST have a unique interval value. When multiple timers share the same interval, they can fire on the same Lua frame and cause frame spikes. Stagger by at least 1ms (e.g. 2000, 2001, 2003).
- The callback is captured by ref -- it always sees the latest closure without needing to be a dependency.
- Pass null or 0 to disable the timer. Do not pass a negative number.
- The timer fires in love.update, which means it has frame-level precision (typically ~16ms at 60fps). Do not rely on sub-frame timing.
