=== OVERVIEW ===
The Text primitive has its own set of style properties for controlling typography. Text
dimensions are measured from font metrics -- the layout engine uses Love2D's font
rasterizer to compute exact pixel widths and heights for text content. This measurement
respects font size, font family, line height, letter spacing, and the available wrapping
width from the parent container.

The default font size is 14px when no fontSize is specified. While the layout engine
handles missing fontSize gracefully, setting it explicitly produces more predictable
results and clearer code.

Text properties set on a parent Text node are inherited by nested __TEXT__ child nodes.
This includes fontSize, fontFamily, fontWeight, lineHeight, letterSpacing, and text
wrapping properties.

=== API / SYNTAX ===
Text style properties:

  color: Color
    Text color. Accepts CSS strings ("#cdd6f4") or Love2D arrays ([0.8, 0.84, 0.96, 1]).

  fontSize: number
    Font size in pixels. Default: 14. Controls both rendering size and layout measurement.

  fontFamily: string
    Font family name. Must be a .ttf or .otf font file available in the project's assets.
    The default font (DejaVu Sans) has full Unicode coverage for arrows, geometric shapes,
    dingbats, block elements, and math symbols.

  fontWeight: 'normal' | 'bold' | number
    Font weight. 'normal' is typically 400, 'bold' is 700. Numeric values (100-900) are
    supported if the font file includes weight variants.

  textAlign: 'left' | 'center' | 'right'
    Horizontal text alignment within the Text element's bounds.
    Default: 'left'.

  textOverflow: 'clip' | 'ellipsis'
    How to handle text that overflows its container.
    'clip': text is cut off at the boundary.
    'ellipsis': text is truncated with "..." at the boundary.

  textDecorationLine: 'none' | 'underline' | 'line-through'
    Text decoration. 'underline' draws a line below the text, 'line-through' draws a
    line through the middle.

  lineHeight: number
    Line height in pixels. Controls vertical spacing between lines of wrapped text.
    When not set, uses the font's natural line height. Scales with textScale.

  letterSpacing: number
    Additional horizontal space between characters, in pixels. Affects both rendering
    and width measurement.

  numberOfLines: number (prop, not style)
    Maximum number of lines to display. Text is truncated after this many lines.
    Set as a prop on Text, not in the style object.

  whiteSpace: 'normal' | 'nowrap'
    Controls text wrapping behavior. 'nowrap' prevents text from wrapping at the
    container boundary.

  textWrap: 'wrap' | 'nowrap'
    Alias for whiteSpace. 'nowrap' is equivalent to whiteSpace: 'nowrap'.

  textScale: number
    Overrides the global text scale for this subtree. Set to 1 to render at native
    size regardless of global scaling.

  userSelect: 'none' | 'text' | 'auto'
    Controls whether text can be selected.

Text shadow properties:

  textShadowColor: Color
    Color of the text shadow. No shadow is rendered when unset.

  textShadowOffsetX: number
    Horizontal offset of the text shadow.

  textShadowOffsetY: number
    Vertical offset of the text shadow.

Text shorthand props on the Text component:
  size: number -- maps to style.fontSize
  color: Color -- maps to style.color
  bold: boolean -- maps to fontWeight: 'bold'
  italic: boolean -- maps to fontStyle: 'italic'
  align: 'left' | 'center' | 'right' -- maps to textAlign
  font: string -- maps to fontFamily
  lines: number -- maps to numberOfLines prop

=== EXAMPLES ===
Example 1: Basic text
---
import { Box, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

function BasicText() {
  const c = useThemeColors();
  return (
    <Box style={{ width: '100%', height: '100%', padding: 16, backgroundColor: c.bg }}>
      <Text style={{ fontSize: 16, color: c.text }}>Hello, world</Text>
    </Box>
  );
}
---
Platforms: love2d

Example 2: Styled heading with shorthand props
---
const c = useThemeColors();
<Text size={32} bold color={c.text}>Page Title</Text>
---
Platforms: love2d

Example 3: Text with line height and letter spacing
---
const c = useThemeColors();
<Text style={{
  fontSize: 16,
  color: c.text,
  lineHeight: 24,
  letterSpacing: 0.5,
}}>
  {`This text has increased line height and subtle letter spacing for readability.`}
</Text>
---
Platforms: love2d

Example 4: Centered text with decoration
---
const c = useThemeColors();
<Text style={{
  fontSize: 20,
  color: c.primary,
  textAlign: 'center',
  textDecorationLine: 'underline',
}}>
  {`Centered and underlined`}
</Text>
---
Platforms: love2d

Example 5: Text with shadow
---
const c = useThemeColors();
<Text style={{
  fontSize: 28,
  color: c.text,
  textShadowColor: 'rgba(0, 0, 0, 0.5)',
  textShadowOffsetX: 2,
  textShadowOffsetY: 2,
}}>
  {`Shadow text`}
</Text>
---
Platforms: love2d

Example 6: Truncated text with numberOfLines
---
const c = useThemeColors();
<Box style={{ width: 200 }}>
  <Text numberOfLines={2} style={{ fontSize: 14, color: c.text }}>
    {`This is a long paragraph that will be truncated after two lines of text, preventing overflow.`}
  </Text>
</Box>
---
Platforms: love2d

Example 7: No-wrap text
---
const c = useThemeColors();
<Box style={{ width: 150, overflow: 'hidden' }}>
  <Text style={{ fontSize: 14, color: c.text, whiteSpace: 'nowrap' }}>
    {`This text will not wrap and may overflow its container`}
  </Text>
</Box>
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
Text measurement uses Love2D's font rasterizer via the Measure module (lua/measure.lua).
The layout engine calls measureText() with the available wrap width and gets back exact
pixel dimensions. Text nodes are re-measured after flex distribution if their width changed,
ensuring correct wrapped height. The default font is DejaVu Sans with full Unicode coverage.
Custom fonts must be .ttf or .otf files accessible via Love2D's filesystem.

=== CRITICAL RULES ===
- Do not mix text and expressions in Text -- {`Score: ${score}`} not <>Score: {score}</>
- Template literals produce a single __TEXT__ node; mixed children produce multiple nodes that stack vertically
- fontSize defaults to 14 but should be set explicitly for clarity
- lineHeight and letterSpacing scale with the textScale property
- numberOfLines is a prop on Text, not a style property
