=== OVERVIEW ===
ReactJIT provides a comprehensive set of visual styling properties for Box elements.
These include background colors, gradients, border styling (width, color, radius with
per-corner and per-side control), box shadows, opacity, overflow clipping, z-index
stacking, outlines, and visibility toggling.

Colors can be specified as CSS strings ("#ff0000", "rgba(255, 0, 0, 0.5)") or as Love2D-style
number arrays [r, g, b, a] where each component is in the 0-1 range. The Color type accepts
both formats and the renderer converts as needed.

Love2D also supports vector shape overrides: arcShape, polygonPoints, and strokePaths
let you draw arcs, polygons, and polyline paths instead of plain rectangles, all driven
by the same backgroundColor and style properties.

=== API / SYNTAX ===
Color type:
  type Color = string | [number, number, number, number?]

  String format: "#ff0000", "#ff000080" (with alpha), "rgba(255, 0, 0, 0.5)"
  Array format: [1, 0, 0, 1] (Love2D-style, 0-1 range, alpha optional)

Background:

  backgroundColor: Color
    Fills the element background with a solid color.

  backgroundGradient: {
    direction: 'horizontal' | 'vertical' | 'diagonal',
    colors: [Color, Color]
  }
    Linear gradient background. 'horizontal' goes left to right, 'vertical' top to
    bottom, 'diagonal' top-left to bottom-right.

Border:

  borderWidth: number
    Border width on all sides. Requires borderColor to be visible.

  borderColor: Color
    Color of the border. Only visible when borderWidth > 0.

  borderRadius: number
    Rounds all four corners uniformly.

  Per-side border width:
    borderTopWidth, borderRightWidth, borderBottomWidth, borderLeftWidth: number

  Per-side border color:
    borderTopColor, borderRightColor, borderBottomColor, borderLeftColor: Color

  Per-corner border radius:
    borderTopLeftRadius, borderTopRightRadius,
    borderBottomLeftRadius, borderBottomRightRadius: number

Shadow:

  shadowColor: Color
    Shadow color. No shadow is rendered when unset.

  shadowOffsetX: number
    Horizontal shadow offset (default: 0).

  shadowOffsetY: number
    Vertical shadow offset (default: 0).

  shadowBlur: number
    Shadow blur radius (default: 0). Higher values produce softer shadows.

Outline:

  outlineColor: Color
    Color of an outline drawn outside the border.

  outlineWidth: number
    Width of the outline.

  outlineOffset: number
    Distance between the outline and the border edge.

Overflow:

  overflow: 'visible' | 'hidden' | 'scroll' | 'auto'
    Default: 'visible'
    'hidden': clips content that extends beyond the element's bounds.
    'scroll': clips content and enables scrolling (requires explicit height).
    'auto': auto-sizes to content but enables scroll when externally constrained.

Opacity and visibility:

  opacity: number
    Element opacity from 0 (transparent) to 1 (opaque). Affects the entire subtree.

  visibility: 'visible' | 'hidden'
    'hidden' makes the element invisible but it still occupies layout space.
    Unlike display: 'none', hidden elements are laid out and affect sibling positions.

Stacking:

  zIndex: number
    Controls paint order. Higher values render on top of lower values. Elements with
    the same zIndex render in tree order.

Vector shapes (Love2D only):

  arcShape: { startAngle: number, endAngle: number, innerRadius?: number }
    Draws an arc or donut slice instead of a rectangle. Angles in radians, 0 = right.
    Set innerRadius > 0 for a donut shape.

  polygonPoints: number[]
    Flat array [x0, y0, x1, y1, ...] of polygon vertices relative to box top-left.
    backgroundColor fills this polygon instead of a rectangle.

  strokePaths: number[][]
    Array of polylines for vector icons. Each sub-array is [x0, y0, x1, y1, ...] in
    a 24x24 viewBox, scaled to fit the element's width/height.

  strokeWidth: number (default: 2)
  strokeColor: Color (defaults to color / text color)

Stroke dash animation:

  strokeDasharray: number[]
    Dash pattern [dash, gap] or [dash, gap, dash, gap, ...]. Works on borders
    (borderWidth > 0) and strokePaths.

  strokeDashoffset: number
    Shifts the dash pattern start. Animate this for marching ants or draw-on effects.

CSS transitions (Lua-side interpolation):

  transition: { [propertyName]: { duration?: number, easing?: string, delay?: number } }
    Declares which properties should animate smoothly when changed. Duration in ms
    (default 300). Easing: 'linear', 'easeIn', 'easeOut', 'easeInOut', 'bounce', 'elastic'.

=== EXAMPLES ===
Example 1: Background color and border radius
---
import { Box } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

function Card() {
  const c = useThemeColors();
  return (
    <Box style={{
      width: 200,
      height: 120,
      backgroundColor: c.surface,
      borderRadius: 12,
    }} />
  );
}
---
Platforms: love2d

Example 2: Border with per-corner radius
---
const c = useThemeColors();
<Box style={{
  width: 200,
  height: 100,
  backgroundColor: c.surface,
  borderWidth: 2,
  borderColor: c.border,
  borderTopLeftRadius: 16,
  borderTopRightRadius: 16,
  borderBottomLeftRadius: 0,
  borderBottomRightRadius: 0,
}} />
---
Platforms: love2d

Example 3: Box shadow
---
const c = useThemeColors();
<Box style={{
  width: 200,
  height: 120,
  backgroundColor: c.surface,
  borderRadius: 8,
  shadowColor: 'rgba(0, 0, 0, 0.3)',
  shadowOffsetX: 0,
  shadowOffsetY: 4,
  shadowBlur: 12,
}} />
---
Platforms: love2d

Example 4: Gradient background
---
const c = useThemeColors();
<Box style={{
  width: '100%',
  height: 200,
  borderRadius: 12,
  backgroundGradient: {
    direction: 'horizontal',
    colors: [c.primary, c.surface],
  },
}} />
---
Platforms: love2d

Example 5: Overflow hidden for clipping
---
const c = useThemeColors();
<Box style={{
  width: 150,
  height: 80,
  overflow: 'hidden',
  backgroundColor: c.surface,
  borderRadius: 8,
}}>
  <Box style={{
    width: 300,
    height: 300,
    backgroundColor: c.primary,
    borderRadius: 150,
  }} />
</Box>
---
Platforms: love2d

Example 6: CSS transition on hover
---
const c = useThemeColors();
<Box
  style={{
    width: 120,
    height: 48,
    backgroundColor: c.surface,
    borderRadius: 8,
    transition: {
      backgroundColor: { duration: 200, easing: 'easeInOut' },
    },
  }}
  hoverStyle={{ backgroundColor: c.primary }}
/>
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
All visual properties are rendered via the Lua painter (lua/painter.lua). Border radius
uses Love2D's arc-based rounded rectangle drawing. Shadows use offset + blur rendering.
Gradients are drawn with vertex-colored meshes. Vector shapes (arcShape, polygonPoints,
strokePaths) are Love2D-specific and use love.graphics drawing primitives directly.

=== CRITICAL RULES ===
- Always use useThemeColors() for colors -- never hardcode hex values in style props
- borderColor requires borderWidth > 0 to be visible
- overflow: 'hidden' uses the scissor rule internally -- see critical-rules.txt for details
- opacity affects the entire subtree, not just the element itself
- zIndex only affects paint order, not layout position
