=== OVERVIEW ===
@reactjit/physics wraps Love2D's built-in Box2D physics engine in declarative React components. You compose a physics scene the same way you compose layout -- nest components, set props, let the framework handle the rest.

The core pattern is a PhysicsWorld container with RigidBody children. Each RigidBody can contain Collider shapes that define its physical geometry and Sensor regions that detect overlaps without generating contact forces. Joints connect bodies with constraints (revolute, distance, prismatic, weld, rope, mouse).

All physics simulation runs in Lua inside Love2D's update loop. React declares the scene graph; Lua owns the Box2D world and steps it every frame. The components route through `<Native>` host elements -- they are one-liner wrappers with no JS-side logic.

Three hooks provide imperative control over bodies: useForce (continuous per-frame force), useImpulse (one-shot velocity change), and useTorque (continuous rotation force). All three operate via bridge RPCs to the Lua physics module.

=== API / SYNTAX ===
```tsx
import {
  PhysicsWorld, RigidBody, Collider, Sensor,
  RevoluteJoint, DistanceJoint, PrismaticJoint, WeldJoint, RopeJoint, MouseJoint,
  useForce, useImpulse, useTorque,
} from '@reactjit/physics';
```

PhysicsWorld props:
| Prop      | Type             | Default    | Description                          |
|-----------|------------------|------------|--------------------------------------|
| gravity   | [number, number] | [0, 9.81]  | World gravity vector (x, y)          |
| debug     | boolean          | false      | Draw collision shapes and joints     |
| timeScale | number           | 1          | Simulation speed multiplier          |
| sleeping  | boolean          | undefined  | Allow bodies to sleep when at rest   |
| style     | Style            | undefined  | Container style                      |

RigidBody props:
| Prop           | Type                          | Default   | Description                        |
|----------------|-------------------------------|-----------|------------------------------------|
| id             | string                        | undefined | Stable identifier for hooks/joints |
| bodyId         | string                        | undefined | Alias for id                       |
| type           | 'dynamic' | 'static' | 'kinematic' | 'dynamic' | Body type                   |
| x              | number                        | 0         | Initial x position                 |
| y              | number                        | 0         | Initial y position                 |
| angle          | number                        | 0         | Initial rotation (radians)         |
| linearDamping  | number                        | 0         | Linear velocity damping            |
| angularDamping | number                        | 0         | Angular velocity damping           |
| fixedRotation  | boolean                       | false     | Prevent rotation                   |
| bullet         | boolean                       | false     | Enable CCD for fast-moving bodies  |
| gravityScale   | number                        | 1         | Per-body gravity multiplier        |
| onCollide      | (event: CollisionEvent) => void | undefined | Collision start callback         |
| onCollideEnd   | (event: CollisionEvent) => void | undefined | Collision end callback           |

=== EXAMPLES ===
Example 1: Basic physics scene
---
```tsx
import { PhysicsWorld, RigidBody, Collider } from '@reactjit/physics';
import { Box } from '@reactjit/core';

export function PhysicsDemo() {
  return (
    <Box style={{ width: '100%', height: '100%' }}>
      <PhysicsWorld gravity={[0, 9.81]} debug>
        <RigidBody type="dynamic" x={200} y={50}>
          <Collider shape="circle" radius={20} restitution={0.7} />
        </RigidBody>
        <RigidBody type="static" x={200} y={400}>
          <Collider shape="rectangle" width={400} height={20} />
        </RigidBody>
      </PhysicsWorld>
    </Box>
  );
}
```

Example 2: Applying forces with hooks
---
```tsx
import { useState } from 'react';
import { PhysicsWorld, RigidBody, Collider, useForce, useImpulse } from '@reactjit/physics';
import { Box, Pressable, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function ForceDemo() {
  const c = useThemeColors();
  const [thrust, setThrust] = useState<[number, number]>([0, 0]);

  useForce('ship', thrust);

  return (
    <Box style={{ width: '100%', height: '100%' }}>
      <PhysicsWorld gravity={[0, 0]} debug>
        <RigidBody id="ship" type="dynamic" x={200} y={200} linearDamping={0.5}>
          <Collider shape="circle" radius={15} />
        </RigidBody>
      </PhysicsWorld>
      <Box style={{ flexDirection: 'row', gap: 8, padding: 8 }}>
        <Pressable onPress={() => setThrust([0, -200])} onPressOut={() => setThrust([0, 0])}>
          <Text style={{ color: c.text, fontSize: 14 }}>Thrust Up</Text>
        </Pressable>
        <Pressable onPress={() => setThrust([200, 0])} onPressOut={() => setThrust([0, 0])}>
          <Text style={{ color: c.text, fontSize: 14 }}>Thrust Right</Text>
        </Pressable>
      </Box>
    </Box>
  );
}
```

Example 4: Collision detection
---
```tsx
import { useState } from 'react';
import { PhysicsWorld, RigidBody, Collider, Sensor } from '@reactjit/physics';
import { Box, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function CollisionDemo() {
  const c = useThemeColors();
  const [hits, setHits] = useState(0);

  return (
    <Box style={{ width: '100%', height: '100%' }}>
      <Text style={{ color: c.text, fontSize: 14, padding: 8 }}>
        {`Collisions: ${hits}`}
      </Text>
      <PhysicsWorld gravity={[0, 9.81]} debug>
        <RigidBody type="dynamic" x={200} y={50} onCollide={() => setHits(h => h + 1)}>
          <Collider shape="circle" radius={15} restitution={0.8} />
        </RigidBody>
        <RigidBody type="static" x={200} y={350}>
          <Collider shape="rectangle" width={300} height={20} />
        </RigidBody>
      </PhysicsWorld>
    </Box>
  );
}
```

=== PLATFORM NOTES ===
Love2D:
  Physics simulation uses Love2D's built-in love.physics module (Box2D 2.4). All simulation runs in the Lua update loop. Debug rendering draws wireframe shapes, joints, and contact points directly via the painter. Body type 'kinematic' is useful for moving platforms -- it has velocity but is not affected by forces or collisions from dynamic bodies.

=== CRITICAL RULES ===
- useForce applies force continuously each frame. Set force to [0, 0] to stop. Do not call useForce conditionally -- always call it and pass [0, 0] when no force is needed.
- useImpulse fires once when the impulse vector changes. It resets its internal flag when the vector returns to [0, 0].
- Body IDs passed to hooks and joints must match the `id` prop on the corresponding RigidBody. If the body doesn't exist yet (hasn't mounted), the hook silently no-ops.
- Collider points for polygon shapes are flat arrays: [x1, y1, x2, y2, ...]. Polygons must be convex with vertices in counter-clockwise order.
- PhysicsWorld is a surface element rendered via `<Native>`. It needs explicit dimensions or flexGrow for sizing.
