=== OVERVIEW ===
@reactjit/imaging provides a non-destructive image processing pipeline backed by Love2D's GPU shader infrastructure. Operations are declared in TypeScript and executed on the Lua side, where Love2D can use GLSL shaders for per-pixel processing.

The package has five major subsystems:

1. **Operations pipeline** (useImaging) -- apply chains of operations (brightness, contrast, blur, sharpen, edge detect, color adjustments, blend modes) to images. Operations execute sequentially on the Lua side.
2. **Layer composition** (useImagingComposer) -- non-destructive multi-layer image composition with blend modes, per-layer transforms (scale, rotation, crop, pivot), opacity, and nested layer hierarchies.
3. **Selection system** (useImagingSelection) -- geometric shape selections (rect, ellipse, polygon) with replace/add/subtract/intersect modes. Selections are rasterized to GPU mask canvases on the Lua side for masked operations.
4. **Draw canvas** (useDrawCanvas) -- a mutable Love2D canvas that accumulates paint/erase/fill operations via bridge RPCs. Can be composited as a layer in the composition graph.
5. **Golden image testing** (runGoldenFixtures) -- hash-based visual regression testing using FNV-1a checksums on RGBA buffers.

All hooks return stable references and manage async state (processing, error) internally.

=== API / SYNTAX ===
```tsx
import {
  // Core hooks
  useImaging, useImagingComposer, useImagingOps, useBlendModes,
  useImagingHistory, useImagingSelection, useImagingTools,
  useDrawCanvas,
  // Golden testing
  hashRGBA, diffRGBA, runGoldenFixtures, captureGoldenHashes, DEFAULT_GOLDEN_FIXTURES,
} from '@reactjit/imaging';
```

=== EXAMPLES ===
Example 1: Apply operations to an image
---
```tsx
import { useImaging } from '@reactjit/imaging';
import { Box, Image, Pressable, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function ImageFilter() {
  const c = useThemeColors();
  const { apply, processing } = useImaging();

  const enhance = async () => {
    await apply({
      src: 'photos/landscape.png',
      operations: [
        { op: 'brightness', amount: 0.1 },
        { op: 'contrast', factor: 1.2 },
        { op: 'gaussian_blur', radius: 2 },
      ],
      output: 'photos/landscape_enhanced.png',
    });
  };

  return (
    <Box style={{ padding: 16, gap: 8 }}>
      <Image src="photos/landscape.png" style={{ width: 300, height: 200 }} />
      <Pressable onPress={enhance}>
        <Text style={{ color: c.primary, fontSize: 14 }}>
          {processing ? 'Processing...' : 'Enhance'}
        </Text>
      </Pressable>
    </Box>
  );
}
```

Example 3: Draw canvas with paint and erase
---
```tsx
import { useDrawCanvas } from '@reactjit/imaging';
import { Native, Box, Pressable, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function DrawDemo() {
  const c = useThemeColors();
  const dc = useDrawCanvas(400, 300);

  return (
    <Box style={{ width: '100%', height: '100%', gap: 8 }}>
      <Native type="DrawCanvas" canvasId={dc.canvasId} width={400} height={300} />
      <Box style={{ flexDirection: 'row', gap: 8 }}>
        <Pressable onPress={() => dc.paint([[50,50],[200,150]], [1,0,0,1], 8)}>
          <Text style={{ color: c.text, fontSize: 12 }}>Draw Red</Text>
        </Pressable>
        <Pressable onPress={() => dc.erase([[100,80],[200,120]], 20)}>
          <Text style={{ color: c.text, fontSize: 12 }}>Erase</Text>
        </Pressable>
        <Pressable onPress={() => dc.clear()}>
          <Text style={{ color: c.text, fontSize: 12 }}>Clear</Text>
        </Pressable>
      </Box>
    </Box>
  );
}
```

Example 4: Selection masking
---
```tsx
import { useImaging, useImagingSelection } from '@reactjit/imaging';

export function SelectionExample() {
  const { apply } = useImaging();
  const { addShape, rasterize } = useImagingSelection();

  const applyToSelection = async () => {
    addShape({ type: 'rect', x: 50, y: 50, width: 200, height: 150 });
    const maskId = await rasterize(400, 300);
    if (maskId) {
      await apply({
        src: 'photo.png',
        operations: [{ op: 'gaussian_blur', radius: 8 }],
        maskId,
        output: 'photo_blurred_selection.png',
      });
    }
  };

}
```

Example 5: Undo/redo history
---
```tsx
import { useImagingHistory } from '@reactjit/imaging';
import type { ImagingComposition } from '@reactjit/imaging';

const initialComp: ImagingComposition = { width: 800, height: 600, layers: [] };

export function HistoryDemo() {
  const { history, canUndo, canRedo, commit, undo, redo } = useImagingHistory(initialComp);

  const addLayer = () => {
    const current = history.present?.state ?? initialComp;
    const next = {
      ...current,
      layers: [...current.layers, { id: `layer_${Date.now()}`, src: 'new.png' }],
    };
    commit(next, 'Add layer');
  };

  // canUndo, canRedo are booleans for button state
  // undo(), redo() restore previous/next state
}
```

=== PLATFORM NOTES ===
Love2D:
  All imaging operations execute on the Lua side using Love2D's canvas and shader pipeline. Shaders run on the GPU for pixel-level operations (blur, edge detect, color adjustments). Layer composition uses Love2D's render-to-texture with proper blend mode stacking. DrawCanvas is a persistent Love2D canvas that accumulates strokes -- it persists across frames until explicitly cleared. Selection masks are rasterized to temporary Love2D canvases and referenced by handle ID.

=== CRITICAL RULES ===
- useImaging.apply is async. Always await it and check the result's `ok` field.
- Operations are serialized as JSON and sent to Lua via RPC. Large operation chains are fine but avoid sending very large pixel data through the bridge.
- DrawCanvas requires a matching `<Native type="DrawCanvas" canvasId={dc.canvasId}>` element in the tree for the canvas to be visible. The hook creates the RPC interface; the Native element creates the renderable surface.
- Selection rasterize returns a maskId string handle. Pass it to apply's maskId field to restrict operations to the selected region. Call clearMask when done to free GPU memory.
- Golden testing (hashRGBA, runGoldenFixtures) operates on raw Uint8Array RGBA buffers. The executor function must render the fixture and return pixel data.
