=== OVERVIEW ===
ReactJIT ships optional packages that layer specialized functionality on top of the core framework. Each package follows the one-liner philosophy: complex domain logic lives in Lua or pure TS, React components are thin wrappers, and hooks handle the bridge plumbing.

All packages are installed separately and imported from their own namespace (e.g. `@reactjit/3d`, `@reactjit/ai`). They depend on `@reactjit/core` and follow the same patterns -- declarative components, bridge RPCs for heavy computation, and capability registration for 60fps Lua-painted surfaces.

Framework packages:

- **@reactjit/3d** -- Declarative 3D scenes: Scene, Camera, Mesh, DirectionalLight, AmbientLight via Love2D + g3d
- **@reactjit/ai** -- LLM integration: useChat, useCompletion, useModels, MCP servers, streaming, tool execution, drop-in chat UIs
- **@reactjit/audio** -- Modular audio: rack management, oscillators, samplers, sequencers, clocks, MIDI, recording
- **@reactjit/controls** -- Hardware-style UI controls: Knob, Fader, Meter, PadButton, StepSequencer, PianoKeyboard, XYPad
- **@reactjit/theme** -- Theming: ThemeProvider, useThemeColors, createTheme, 16+ built-in themes, ThemeSwitcher

Domain packages:

- **@reactjit/chemistry** -- 118 elements, molecular data, reactions, reagent tests, spectra, phase diagrams, PubChem API, chemical notation
- **@reactjit/physics** -- 2D physics via Love2D's Box2D bindings: rigid bodies, colliders, sensors, joints, force/impulse/torque hooks
- **@reactjit/finance** -- Technical indicators, portfolio management, live price feeds, chart components, WalletConnect relay
- **@reactjit/imaging** -- Image processing pipeline, layer composition, selection tools, draw canvas, undo/redo history, golden image testing
- **@reactjit/math** -- Vector/matrix/quaternion math, geometry, interpolation, noise fields, FFT, bezier curves
- **@reactjit/time** -- Wall clock, stopwatch, countdown, precision scheduling, date arithmetic, formatting, timezone utilities
- **@reactjit/convert** -- Fluent unit/color/encoding/number-base conversions with an extensible registry and live currency rates
