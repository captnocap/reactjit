=== OVERVIEW ===
@reactjit/data provides a full spreadsheet with a formula engine, cell selection, column resizing, formula bar, and status bar. It works in both controlled and uncontrolled modes.

The formula engine supports standard cell references (A1, B2), ranges (A1:C10), and 25+ built-in functions including SUM, AVERAGE, MIN, MAX, COUNT, IF, AND, OR, ROUND, ABS, SQRT, CLAMP, LERP, REMAP, SMOOTHSTEP, DIST2D, and CONVERT (unit conversion). Custom functions can be added via the functionMap prop.

The useSpreadsheet hook provides headless access to the formula engine without the visual component -- useful for data processing or building custom spreadsheet UIs.

=== API / SYNTAX ===
```tsx
import { Spreadsheet, useSpreadsheet } from '@reactjit/data';
```

---- Spreadsheet Component ----

| Prop                   | Type                      | Default   | Description                              |
|------------------------|---------------------------|-----------|------------------------------------------|
| rows                   | number                    | 20        | Number of rows                           |
| cols                   | number                    | 8         | Number of columns                        |
| initialCells           | SpreadsheetCellMap        | {}        | Initial cell values (uncontrolled)       |
| cells                  | SpreadsheetCellMap        | undefined | Cell values (controlled mode)            |
| onCellsChange          | (cells) => void           | undefined | Callback when cells change               |
| functionMap            | SpreadsheetFunctionMap    | undefined | Custom formula functions                 |
| readOnly               | boolean                   | false     | Disable editing                          |
| showFormulaBar         | boolean                   | true      | Show formula bar above grid              |
| showStatusBar          | boolean                   | true      | Show status bar below grid               |
| columnWidth            | number                    | 118       | Default column width                     |
| columnWidths           | number[]                  | undefined | Per-column widths (controlled)           |
| resizableColumns       | boolean                   | true      | Allow drag-to-resize columns             |
| rowHeight              | number                    | 30        | Row height in pixels                     |
| viewportHeight         | number \| string          | auto      | Scroll viewport height                   |
| selectedAddress        | string                    | undefined | Selected cell (controlled)               |
| onSelectedAddressChange| (address) => void         | undefined | Selection change callback                |
| style                  | Style                     | undefined | Container style                          |

---- useSpreadsheet Hook ----

```tsx
function useSpreadsheet(options?: UseSpreadsheetOptions): UseSpreadsheetResult
```

Options:

| Option          | Type                   | Default   | Description                |
|-----------------|------------------------|-----------|----------------------------|
| initialCells    | SpreadsheetCellMap     | {}        | Initial cell values        |
| functions       | SpreadsheetFunctionMap | undefined | Custom formula functions   |
| maxRangeCells   | number                 | 10000     | Max cells in a range       |

Returns:

| Field         | Type                            | Description                    |
|---------------|---------------------------------|--------------------------------|
| cells         | SpreadsheetCellMap              | Current raw cell inputs        |
| values        | SpreadsheetValueMap             | Evaluated cell values          |
| errors        | SpreadsheetErrorMap             | Cells with formula errors      |
| setCell       | (address, input) => void        | Set a single cell              |
| setCells      | (cells) => void                 | Replace all cells              |
| getCellInput  | (address) => string             | Get raw input for a cell       |
| getCellValue  | (address) => SpreadsheetScalar  | Get evaluated value            |
| getCellError  | (address) => string \| undefined| Get error message if any       |

---- Built-in Formula Functions ----

| Function    | Example                       | Description                    |
|-------------|-------------------------------|--------------------------------|
| SUM         | =SUM(A1:A10)                  | Sum of values                  |
| AVERAGE     | =AVERAGE(B1:B5)               | Arithmetic mean                |
| MIN / MAX   | =MIN(A1:A10)                  | Minimum / maximum value        |
| COUNT       | =COUNT(A1:A10)                | Count of numeric values        |
| COUNTA      | =COUNTA(A1:A10)               | Count of non-empty values      |
| IF          | =IF(A1>10, "high", "low")     | Conditional                    |
| AND / OR    | =AND(A1>0, B1>0)              | Logical operators              |
| ROUND       | =ROUND(A1, 2)                 | Round to N digits              |
| ABS / SQRT  | =SQRT(ABS(A1))                | Math functions                 |
| CLAMP       | =CLAMP(A1, 0, 100)            | Clamp value to range           |
| LERP        | =LERP(A1, B1, 0.5)            | Linear interpolation           |
| REMAP       | =REMAP(A1, 0, 100, 0, 1)      | Map value between ranges       |
| CONVERT     | =CONVERT(A1, "mi", "km")      | Unit conversion                |
| DIST2D      | =DIST2D(A1, B1, A2, B2)       | 2D distance                    |

=== EXAMPLES ===
Example 1: Basic spreadsheet
---
```tsx
import { Box } from '@reactjit/core';
import { Spreadsheet } from '@reactjit/data';
import { useThemeColors } from '@reactjit/theme';

export function BasicSheet() {
  const colors = useThemeColors();

  return (
    <Box style={{ width: '100%', height: '100%', backgroundColor: colors.bg }}>
      <Spreadsheet
        rows={10}
        cols={5}
        initialCells={{
          A1: 'Item',   B1: 'Price', C1: 'Qty',  D1: 'Total',
          A2: 'Widget', B2: '10',    C2: '5',     D2: '=B2*C2',
          A3: 'Gadget', B3: '25',    C3: '3',     D3: '=B3*C3',
          A4: 'Total',  D4: '=SUM(D2:D3)',
        }}
      />
    </Box>
  );
}
```

Example 2: Headless formula engine
---
```tsx
import { useSpreadsheet } from '@reactjit/data';
import { Box, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function FormulaDemo() {
  const colors = useThemeColors();
  const sheet = useSpreadsheet({
    initialCells: {
      A1: '100', A2: '200', A3: '300',
      B1: '=SUM(A1:A3)',
      B2: '=AVERAGE(A1:A3)',
      B3: '=CONVERT(A1, "mi", "km")',
    },
  });

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text }}>{`Sum: ${sheet.getCellValue('B1')}`}</Text>
      <Text style={{ color: colors.text }}>{`Average: ${sheet.getCellValue('B2')}`}</Text>
      <Text style={{ color: colors.text }}>{`100 mi = ${sheet.getCellValue('B3')} km`}</Text>
    </Box>
  );
}
```

Example 3: Custom functions
---
```tsx
import { Spreadsheet } from '@reactjit/data';

const customFunctions = {
  TAX: (amount: unknown, rate: unknown) => {
    const a = Number(amount) || 0;
    const r = Number(rate) || 0;
    return a * (r / 100);
  },
  DISCOUNT: (price: unknown, pct: unknown) => {
    const p = Number(price) || 0;
    const d = Number(pct) || 0;
    return p * (1 - d / 100);
  },
};

export function CustomFunctionSheet() {
  return (
    <Spreadsheet
      rows={5}
      cols={4}
      functionMap={customFunctions}
      initialCells={{
        A1: 'Price', B1: '100',
        A2: 'Tax',   B2: '=TAX(B1, 8.5)',
        A3: 'After Discount', B3: '=DISCOUNT(B1, 15)',
      }}
    />
  );
}
```

=== CRITICAL RULES ===
- Formulas start with `=`. Plain text without `=` is treated as a literal value.
- Cell addresses are case-insensitive and normalized to uppercase (a1 becomes A1).
- Circular references are detected and produce an error. The cell displays #ERR.
- The ^ operator maps to exponentiation, & to string concatenation, and <> to inequality.
- Range expressions (A1:C3) expand to all cells in the rectangle. The maxRangeCells option caps expansion to prevent OOM on huge ranges.
- Spreadsheet uses useThemeColorsOptional internally with dark fallback colors.
