=== OVERVIEW ===
@reactjit/icons provides a Lucide-compatible icon library rendered as stroke paths. Icons are polyline data (arrays of [x, y] coordinate pairs) drawn via the strokePaths style property. No SVG, no images -- pure geometry.

Importing the package auto-registers all icons into core's icon registry, so they work with Image src lookups. The Icon component is a convenience wrapper around Box with strokePaths.

The package includes 1400+ icons covering common UI needs: arrows, file types, media controls, navigation, social, weather, devices, and more.

=== API / SYNTAX ===
```tsx
import { Icon, iconNames } from '@reactjit/icons';
import { Heart, Search, Settings, ChevronRight } from '@reactjit/icons';
```

Icon component props:

| Prop        | Type      | Default   | Description                              |
|-------------|-----------|-----------|------------------------------------------|
| icon        | number[][]| required  | Polyline path data from a named export   |
| size        | number    | 24        | Width and height in pixels               |
| color       | Color     | inherited | Stroke color                             |
| strokeWidth | number    | 2         | Line thickness before scaling            |
| style       | Style     | undefined | Container style overrides                |

Utility exports:

| Export    | Type      | Description                          |
|----------|-----------|--------------------------------------|
| iconNames| string[]  | Array of all available icon names    |

=== EXAMPLES ===
Example 1: Basic icon usage
---
```tsx
import { Box, Text } from '@reactjit/core';
import { Icon } from '@reactjit/icons';
import { Heart, Search, Settings } from '@reactjit/icons';
import { useThemeColors } from '@reactjit/theme';

export function IconDemo() {
  const colors = useThemeColors();

  return (
    <Box style={{ flexDirection: 'row', gap: 16, alignItems: 'center' }}>
      <Icon icon={Heart} size={24} color={colors.primary} />
      <Icon icon={Search} size={20} color={colors.text} />
      <Icon icon={Settings} size={24} color={colors.muted} strokeWidth={1.5} />
    </Box>
  );
}
```

Example 2: Icon button with label
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { Icon } from '@reactjit/icons';
import { Plus } from '@reactjit/icons';
import { useThemeColors } from '@reactjit/theme';

export function AddButton({ onPress }: { onPress: () => void }) {
  const colors = useThemeColors();

  return (
    <Pressable onPress={onPress}>
      <Box style={{ flexDirection: 'row', gap: 8, alignItems: 'center', paddingLeft: 12, paddingRight: 12, paddingTop: 8, paddingBottom: 8, backgroundColor: colors.primary, borderRadius: 6 }}>
        <Icon icon={Plus} size={16} color={colors.bg} />
        <Text style={{ color: colors.bg, fontSize: 14 }}>Add Item</Text>
      </Box>
    </Pressable>
  );
}
```

=== CRITICAL RULES ===
- Importing @reactjit/icons runs registerIcons() as a side effect. All icons become available immediately.
- Icon data is number[][] (polyline coordinates on a 24x24 grid). The Icon component scales them to the requested size.
- Use the named export (e.g. Heart, Search) not a string name. The icon prop expects the actual path data.
- The strokeWidth prop controls the line thickness at native resolution. It does not scale with the size prop.
