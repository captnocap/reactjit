=== OVERVIEW ===
StackedBarChart renders vertical bars where each bar is divided into colored segments representing different series. Each column shows how multiple categories contribute to a total.

Built with Box and Text primitives (not a Chart2D surface). Series are rendered bottom-to-top within each column. When `interactive` is enabled, hovering a column highlights it and shows a tooltip with per-series breakdowns.

If `barWidth` is set, bars have fixed width and the chart computes its own total width. If `barWidth` is omitted, bars expand with `flexGrow` to fill the available container width.

=== API / SYNTAX ===
```tsx
import { StackedBarChart } from '@reactjit/core';
```

Props:
| Prop        | Type                      | Default   | Description                              |
|-------------|---------------------------|-----------|------------------------------------------|
| series      | StackedBarChartSeries[]   | required  | Array of series (each with data array)   |
| labels      | string[]                  | undefined | Category labels shown below bars         |
| height      | number                    | 120       | Chart area height in pixels              |
| barWidth    | number                    | undefined | Fixed bar width (omit for flex fill)     |
| gap         | number                    | 8         | Gap between columns                      |
| showLabels  | boolean                   | true      | Show category labels below bars          |
| interactive | boolean                   | false     | Enable hover highlight and tooltips      |
| style       | Style                     | undefined | Full style object                        |

StackedBarChartSeries:
| Field | Type     | Default  | Description                        |
|-------|----------|----------|------------------------------------|
| label | string   | required | Series name (shown in tooltip)     |
| color | Color    | required | Color for this series              |
| data  | number[] | required | Values per category (one per bar)  |

=== EXAMPLES ===
Example 1: Basic stacked bar chart
---
```tsx
const colors = useThemeColors();

<StackedBarChart
  series={[
    { label: 'Desktop', color: '#3b82f6', data: [30, 25, 40, 35] },
    { label: 'Mobile', color: '#22c55e', data: [20, 30, 15, 25] },
    { label: 'Tablet', color: '#f59e0b', data: [10, 5, 8, 12] },
  ]}
  labels={['Q1', 'Q2', 'Q3', 'Q4']}
  height={150}
/>
```

=== CRITICAL RULES ===
- All series must have the same `data` array length. The number of columns equals `series[0].data.length`.
- If `series` is empty or the first series has no data, an empty Box is rendered.
- Segment heights are proportional to the maximum column total across all columns.
- Not a Chart2D surface -- built from Box/Text primitives.
