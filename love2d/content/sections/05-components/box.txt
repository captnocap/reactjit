=== OVERVIEW ===
Box is the fundamental building block. It renders as a Lua `View` node — a rectangle that can have background color, borders, border radius, shadows, gradients, and children. Every layout in ReactJIT starts with Box.

Box also comes in two convenience variants:
- `Row` — shorthand for `<Box direction="row">`, sets `flexDirection: 'row'`
- `Col` — shorthand for `<Box direction="col">`, with optional 12-column grid support via breakpoint-responsive span props

Box supports both a `style` prop (full Style object) and shorthand props for common layout patterns. Shorthands are lower priority — `style={}` wins when both are set.

=== API / SYNTAX ===
```tsx
import { Box, Row, Col } from '@reactjit/core';
```

=== EXAMPLES ===
Example 1: Basic container with padding and background
---
```tsx
const colors = useThemeColors();

<Box style={{ padding: 16, backgroundColor: colors.surface, borderRadius: 8 }}>
  <Text style={{ color: colors.text }}>Hello, world</Text>
</Box>
```
---
Platforms: love2d

Example 2: Using shorthand props
---
```tsx
const colors = useThemeColors();

<Box padding={16} bg={colors.surface} radius={8}>
  <Text style={{ color: colors.text }}>Same result, less typing</Text>
</Box>
```
---
Platforms: love2d

Example 3: Row layout with gap
---
```tsx
const colors = useThemeColors();

<Row gap={12} align="center" padding={16}>
  <Box w={48} h={48} bg={colors.primary} radius={24} />
  <Text style={{ color: colors.text, fontSize: 18 }}>Username</Text>
</Row>
```
---
Platforms: love2d

Example 4: Full-height layout with flexGrow
---
```tsx
const colors = useThemeColors();

<Box fill>
  <Box padding={12} bg={colors.bgElevated}>
    <Text style={{ color: colors.text, fontSize: 20 }}>Header</Text>
  </Box>
  <Box grow padding={16}>
    <Text style={{ color: colors.text }}>Content fills remaining space</Text>
  </Box>
  <Box padding={12} bg={colors.bgElevated}>
    <Text style={{ color: colors.muted }}>Footer</Text>
  </Box>
</Box>
```
---
Platforms: love2d

Example 5: Responsive grid with Col
---
```tsx
const colors = useThemeColors();

<Row gap={16} wrap style={{ padding: 16 }}>
  <Col responsive>
    <Box padding={16} bg={colors.surface} radius={8}>
      <Text style={{ color: colors.text }}>Card 1</Text>
    </Box>
  </Col>
  <Col responsive>
    <Box padding={16} bg={colors.surface} radius={8}>
      <Text style={{ color: colors.text }}>Card 2</Text>
    </Box>
  </Col>
  <Col responsive>
    <Box padding={16} bg={colors.surface} radius={8}>
      <Text style={{ color: colors.text }}>Card 3</Text>
    </Box>
  </Col>
</Row>
```
---
Platforms: love2d

Example 6: Hover and active styles
---
```tsx
const colors = useThemeColors();

<Box
  style={{ padding: 12, backgroundColor: colors.surface, borderRadius: 8 }}
  hoverStyle={{ backgroundColor: colors.bgElevated }}
  activeStyle={{ opacity: 0.7 }}
  onClick={() => console.log('clicked')}
>
  <Text style={{ color: colors.text }}>Hover me</Text>
</Box>
```
---

=== CRITICAL RULES ===
- Root containers need `width: '100%', height: '100%'` (or `fill`) to fill the viewport. The proportional fallback does not apply at the root.
- Use `flexGrow: 1` (or `grow`) for space-filling elements. Do not hardcode pixel heights to fill a known window size.
- Empty Box with no children and no explicit dimensions falls back to 1/4 of parent size (proportional surface fallback).
- Never use `paddingHorizontal` or `paddingVertical` — the linter rejects them. Use `px`/`py` shorthands or `paddingLeft`/`paddingRight`/`paddingTop`/`paddingBottom`.
- Event handlers use an explicit whitelist in Box. If you add a new event type at the framework level, you must add it to the destructure list in `Box()` and to the `createElement` props, or the handler will silently disappear.
