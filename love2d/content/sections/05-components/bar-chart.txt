=== OVERVIEW ===
BarChart renders a vertical bar chart as a Lua `Chart2D` node. Each bar represents a labeled value. The chart is rendered entirely by `lua/chart.lua` -- React declares the data and options, Lua does all the drawing.

When `interactive` is enabled, hovering a bar shows a tooltip with label/value pairs. Press events can be handled via `onBarPress`.

BarChart is a surface element. If no explicit dimensions are set, it falls back to proportional surface sizing (1/4 of parent). Set `width`/`height` or use `flexGrow` for precise control.

=== API / SYNTAX ===
```tsx
import { BarChart } from '@reactjit/core';
```

Props:
| Prop       | Type                                               | Default   | Description                          |
|------------|----------------------------------------------------|-----------|--------------------------------------|
| data       | BarChartBar[]                                      | required  | Array of bars to render              |
| height     | number                                             | undefined | Chart height in pixels               |
| barWidth   | number                                             | undefined | Width of each bar in pixels          |
| gap        | number                                             | undefined | Gap between bars                     |
| showLabels | boolean                                            | undefined | Show labels below bars               |
| showValues | boolean                                            | undefined | Show values above bars               |
| color      | Color                                              | undefined | Default bar color                    |
| interactive| boolean                                            | false     | Enable hover tooltips and press      |
| onBarHover | (index: number \| null, bar: BarChartBar \| null) => void | undefined | Called on bar hover/unhover  |
| onBarPress | (index: number, bar: BarChartBar) => void          | undefined | Called on bar press                   |
| style      | Style                                              | undefined | Full style object                    |

BarChartBar:
| Field | Type   | Default   | Description                |
|-------|--------|-----------|----------------------------|
| label | string | required  | Label text for the bar     |
| value | number | required  | Numeric value              |
| color | Color  | undefined | Per-bar color override     |

=== EXAMPLES ===
Example 1: Basic bar chart
---
```tsx
const colors = useThemeColors();

<BarChart
  data={[
    { label: 'Mon', value: 12 },
    { label: 'Tue', value: 19 },
    { label: 'Wed', value: 8 },
    { label: 'Thu', value: 25 },
    { label: 'Fri', value: 15 },
  ]}
  color={colors.primary}
  height={120}
  showLabels
/>
```
---
Platforms: love2d

Example 2: Interactive with per-bar colors
---
```tsx
const colors = useThemeColors();

<BarChart
  data={[
    { label: 'React', value: 45, color: '#61dafb' },
    { label: 'Vue', value: 30, color: '#42b883' },
    { label: 'Svelte', value: 18, color: '#ff3e00' },
  ]}
  height={150}
  showLabels
  showValues
  interactive
  onBarPress={(index, bar) => console.log(bar.label, bar.value)}
/>
```
---
Platforms: love2d

Example 3: Filling available space
---
```tsx
const colors = useThemeColors();

<Box style={{ width: '100%', height: '100%' }}>
  <Text style={{ color: colors.text, fontSize: 16 }}>Weekly Sales</Text>
```

Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  Rendered by lua/chart.lua as a Chart2D surface. The bar layout, scaling, and label rendering are handled entirely in Lua. When interactive is true, a table-layout tooltip is generated from the data array and rendered by the Lua tooltip system on hover.
