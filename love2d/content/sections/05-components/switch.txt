=== OVERVIEW ===
Switch is a boolean toggle control. It renders as a `Switch` host element with on/off states, thumb animation, and track color transitions all handled in Lua.

Supports both controlled (`value` prop) and uncontrolled (`defaultValue` prop) modes. The track shows different colors for the on and off states.

=== API / SYNTAX ===
```tsx
import { Switch } from '@reactjit/core';
```

Props:
| Prop         | Type                          | Default                                  | Description                    |
|--------------|-------------------------------|------------------------------------------|--------------------------------|
| value        | boolean                       | undefined                                | Controlled toggle state        |
| defaultValue | boolean                       | false                                    | Initial state for uncontrolled |
| onValueChange| (value: boolean) => void      | undefined                                | Fires when toggled             |
| disabled     | boolean                       | false                                    | Disable interaction            |
| style        | Style                         | undefined                                | Container style                |
| trackColor   | { true: Color; false: Color } | { true: '#81b0ff', false: '#767577' }    | Track colors for each state    |
| thumbColor   | Color                         | '#f4f3f4'                                | Thumb color                    |
| width        | number                        | 50                                       | Switch width in pixels         |
| height       | number                        | 28                                       | Switch height in pixels        |

=== EXAMPLES ===
Example 1: Basic toggle
---
```tsx
const colors = useThemeColors();
const [enabled, setEnabled] = useState(false);

<Box direction="row" align="center" gap={12}>
  <Text style={{ color: colors.text }}>Notifications</Text>
  <Switch value={enabled} onValueChange={setEnabled} />
</Box>
```
---
Platforms: love2d

Example 2: Custom colors
---
```tsx
const colors = useThemeColors();
const [darkMode, setDarkMode] = useState(true);

<Switch
  value={darkMode}
  onValueChange={setDarkMode}
  trackColor={{ true: colors.primary, false: colors.muted }}
  thumbColor="#ffffff"
/>
```
---
Platforms: love2d

Example 3: Disabled switch
---
```tsx
const colors = useThemeColors();

<Switch
  value={true}
  disabled
  trackColor={{ true: colors.muted, false: colors.muted }}
/>
```
---
Platforms: love2d

=== PLATFORM NOTES ===
Love2D:
  The thumb animation (sliding from off to on position) is handled by Lua on every frame. Toggle happens on click -- Lua updates the visual state immediately, then fires the onValueChange event back to React.

=== CRITICAL RULES ===
- `trackColor` is an object with `true` and `false` keys, not a single color. Each key maps to the track color for that toggle state.
- Switch sizes itself from the `width` and `height` props, not from style. The style prop is for positioning and layout (margin, flexGrow, etc.).
