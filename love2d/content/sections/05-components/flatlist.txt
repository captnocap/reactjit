=== OVERVIEW ===
FlatList renders large lists efficiently by only mounting items visible within the scroll viewport plus a configurable buffer zone. Spacer elements above and below the visible window maintain correct scroll height without mounting every item.

FlatList wraps ScrollView internally, so all scrolling behavior (clipping, indicators, wheel events) is inherited. It requires a `data` array and a `renderItem` function. Item height should be provided for accurate virtualization — without it, an `estimatedItemSize` of 50px is used.

FlatList supports grid layouts via `numColumns`, inverted lists, headers/footers, separators, and an imperative ref for programmatic scrolling.

=== API / SYNTAX ===
```tsx
import { FlatList } from '@reactjit/core';
```

Props:
| Prop                   | Type                                        | Default   | Description                                      |
|------------------------|---------------------------------------------|-----------|--------------------------------------------------|
| data                   | T[]                                         | required  | Array of items to render                         |
| renderItem             | (info: { item: T; index: number }) => ReactNode | required | Render function for each item               |
| keyExtractor           | (item: T, index: number) => string          | index     | Unique key for each item                         |
| horizontal             | boolean                                     | false     | Horizontal scroll direction                      |
| itemHeight             | number                                      | undefined | Fixed height per item (vertical lists)           |
| itemWidth              | number                                      | undefined | Fixed width per item (horizontal lists)          |
| estimatedItemSize      | number                                      | 50        | Estimated item size when exact size unknown       |
| onEndReached           | () => void                                  | undefined | Fires near the end of the list                   |
| onEndReachedThreshold  | number                                      | 0.5       | How far from end to trigger (fraction of viewport)|
| onScroll               | (event: ScrollEvent) => void                | undefined | Fires during scroll                              |
| ListHeaderComponent    | ReactNode                                   | undefined | Rendered before the first item                   |
| ListFooterComponent    | ReactNode                                   | undefined | Rendered after the last item                     |
| ListEmptyComponent     | ReactNode                                   | undefined | Rendered when data is empty                      |
| ItemSeparatorComponent | ReactNode                                   | undefined | Rendered between items                           |
| style                  | Style                                       | undefined | Style for the outer ScrollView                   |
| contentContainerStyle  | Style                                       | undefined | Style for the inner content container            |
| initialNumToRender     | number                                      | 10        | Items to render on first mount                   |
| windowSize             | number                                      | 5         | Viewports to keep mounted (buffer)               |
| inverted               | boolean                                     | false     | Reverse the list order                           |
| numColumns             | number                                      | undefined | Grid mode: items per row                         |

Imperative API (via ref):
| Method         | Signature                                                 | Description                |
|----------------|-----------------------------------------------------------|----------------------------|
| scrollToIndex  | (params: { index: number; animated?: boolean }) => void   | Scroll to item by index    |
| scrollToOffset | (params: { offset: number; animated?: boolean }) => void  | Scroll to pixel offset     |

=== EXAMPLES ===
Example 1: Basic list
---
```tsx
const colors = useThemeColors();

<FlatList
  data={items}
  itemHeight={60}
  renderItem={({ item }) => (
    <Box padding={16} style={{ height: 60 }}>
      <Text style={{ color: colors.text }}>{item.name}</Text>
    </Box>
  )}
  keyExtractor={(item) => item.id}
  style={{ flexGrow: 1 }}
/>
```

Example 4: Empty state and separators
---
```tsx
const colors = useThemeColors();

<FlatList
  data={results}
  itemHeight={48}
  renderItem={({ item }) => (
    <Box padding={12}>
      <Text style={{ color: colors.text }}>{item.title}</Text>
    </Box>
  )}
  keyExtractor={(item) => item.id}
  ListEmptyComponent={
    <Box padding={32} align="center">
      <Text style={{ color: colors.muted }}>No results found</Text>
    </Box>
  }
  ItemSeparatorComponent={
    <Box style={{ height: 1, backgroundColor: colors.border }} />
  }
  style={{ flexGrow: 1 }}
/>
```

=== CRITICAL RULES ===
- Always provide `itemHeight` (or `itemWidth` for horizontal) for accurate virtualization. Without it, the list uses `estimatedItemSize` (50px) and visible range calculations may be off.
- FlatList inherits ScrollView's sizing requirement — it needs explicit height or `flexGrow` to define its viewport.
- The `keyExtractor` should return stable, unique keys. Using array index (the default) causes incorrect recycling when items are reordered or removed.
