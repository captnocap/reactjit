=== OVERVIEW ===
Video renders a video file as a Lua `Video` node, powered by libmpv on the backend. It provides declarative control over playback state — paused, loop, muted, volume — and fires lifecycle events for time updates, play/pause, end, and errors.

VideoPlayer extends Video with built-in transport controls (play/pause, seek bar, time display). Use Video when you want full custom UI control over playback. Use VideoPlayer when you want ready-made controls.

Both are surface elements — without explicit dimensions, they fall back to 1/4 of parent size (proportional surface fallback).

=== API / SYNTAX ===
```tsx
import { Video, VideoPlayer } from '@reactjit/core';
```

Video Props:
| Prop         | Type                                | Default   | Description                  |
|--------------|-------------------------------------|-----------|------------------------------|
| src          | string                              | required  | Video file path              |
| paused       | boolean                             | false     | Whether playback is paused   |
| loop         | boolean                             | false     | Loop playback                |
| muted        | boolean                             | false     | Mute audio                   |
| volume       | number                              | 1         | Volume level (0 to 1)        |
| w            | number \| string                    | undefined | Width shorthand              |
| h            | number \| string                    | undefined | Height shorthand             |
| radius       | number                              | undefined | Border radius shorthand      |
| style        | Style                               | undefined | Full style object            |
| onTimeUpdate | (event: VideoTimeEvent) => void     | undefined | Fires periodically with time |
| onEnded      | () => void                          | undefined | Playback reached the end     |
| onPlay       | () => void                          | undefined | Playback started             |
| onPause      | () => void                          | undefined | Playback paused              |
| onReady      | () => void                          | undefined | Video loaded and ready       |
| onError      | (event: { message: string }) => void| undefined | Error occurred               |
| onClick      | (event: LoveEvent) => void          | undefined | Click handler                |

VideoTimeEvent:
| Field       | Type   | Description                |
|-------------|--------|----------------------------|
| currentTime | number | Current playback position  |
| duration    | number | Total video duration       |

VideoPlayer Props:
All Video props, plus:
| Prop     | Type    | Default | Description                     |
|----------|---------|---------|---------------------------------|
| controls | boolean | true    | Show built-in transport controls|

=== EXAMPLES ===
Example 1: Basic video playback
---
```tsx
<Video src="videos/intro.mp4" style={{ flexGrow: 1 }} />
```
---
Platforms: love2d

Example 2: Video with playback control
---
```tsx
const [paused, setPaused] = useState(false);

<Video
  src="videos/demo.mp4"
  paused={paused}
  loop
  volume={0.8}
  onClick={() => setPaused(p => !p)}
  style={{ flexGrow: 1 }}
/>
```
---
Platforms: love2d

Example 3: VideoPlayer with built-in controls
---
```tsx
<VideoPlayer
  src="videos/tutorial.mp4"
  controls
  style={{ width: '100%', height: 400 }}
/>
```
---
Platforms: love2d

Example 4: Tracking playback position
---
```tsx
const [time, setTime] = useState(0);
const [duration, setDuration] = useState(0);

<Video
  src="videos/clip.mp4"
  onTimeUpdate={(e) => {
    setTime(e.currentTime);
    setDuration(e.duration ?? 0);
  }}
  onEnded={() => console.log('finished')}
  style={{ flexGrow: 1 }}
/>
```
---
Platforms: love2d
