=== OVERVIEW ===
RadarChart renders a spider/radar chart from an array of axis definitions and corresponding values. Each axis radiates from the center. The data polygon connects normalized values on each axis, filled with a semi-transparent color.

Grid rings are drawn at 25%, 50%, 75%, and 100% to provide scale reference. Axis lines extend from center to the outer ring.

Built with Box primitives and the `polygonPoints` style property (not a Chart2D surface). The data polygon is rendered as a filled polygon overlay. When `interactive` is enabled, hovering shows a table-layout tooltip with per-axis values and percentages.

=== API / SYNTAX ===
```tsx
import { RadarChart } from '@reactjit/core';
```

Props:
| Prop        | Type             | Default   | Description                              |
|-------------|------------------|-----------|------------------------------------------|
| axes        | RadarChartAxis[] | required  | Axis definitions                         |
| data        | number[]         | required  | Value for each axis (same order as axes) |
| size        | number           | 120       | Chart diameter in pixels                 |
| color       | Color            | '#3b82f6' | Data polygon fill color                  |
| gridColor   | Color            | '#1e293b' | Grid ring and axis line color            |
| interactive | boolean          | false     | Enable hover tooltip                     |
| style       | Style            | undefined | Full style object                        |

RadarChartAxis:
| Field | Type   | Default                    | Description                        |
|-------|--------|----------------------------|------------------------------------|
| label | string | required                   | Axis label (shown in tooltip)      |
| max   | number | max value in data array    | Maximum value for normalization    |

=== CRITICAL RULES ===
- The `data` and `axes` arrays must have the same length. Each data value corresponds to the axis at the same index.
- If `max` is omitted from an axis, the maximum value in the entire `data` array is used for normalization.
- Values are clamped to [0, max]. Values above max display at the outer ring.
- The data polygon is rendered at 0.6 opacity by default.
- Grid rings are decorative Box elements with `borderRadius`, not Lua-drawn circles. The data polygon uses the `polygonPoints` style property.
