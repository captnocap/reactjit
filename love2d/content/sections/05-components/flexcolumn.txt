=== OVERVIEW ===
FlexColumn is a convenience wrapper around Box that sets `flexDirection: 'column'`. It provides shorthand props for gap, main-axis justification, and cross-axis alignment.

Since `flexDirection: 'column'` is already the default in ReactJIT's layout engine, FlexColumn is most useful as a self-documenting component that makes vertical intent explicit and provides the `gap`, `justify`, and `align` shorthands without needing a full `style` object.

=== API / SYNTAX ===
```tsx
import { FlexColumn } from '@reactjit/core';
```

Props:
| Prop     | Type                          | Default   | Description                              |
|----------|-------------------------------|-----------|------------------------------------------|
| gap      | number                        | undefined | Space between children in pixels         |
| justify  | Style['justifyContent']       | undefined | Main-axis alignment (start, center, end, space-between, space-around, space-evenly) |
| align    | Style['alignItems']           | undefined | Cross-axis alignment (start, center, end, stretch) |
| style    | Style                         | undefined | Additional style overrides               |
| children | ReactNode                     | required  | Child elements                           |

=== EXAMPLES ===
Example 1: Stacked labels with gap
---
```tsx
const colors = useThemeColors();

<FlexColumn gap={8}>
  <Text style={{ color: colors.text, fontSize: 16, fontWeight: 'bold' }}>Title</Text>
  <Text style={{ color: colors.muted, fontSize: 12 }}>Subtitle goes here</Text>
  <Text style={{ color: colors.muted, fontSize: 12 }}>Additional detail</Text>
</FlexColumn>
```
---
Platforms: love2d

Example 2: Centered content
---
```tsx
const colors = useThemeColors();

<FlexColumn align="center" justify="center" style={{ flexGrow: 1 }}>
  <Text style={{ color: colors.text, fontSize: 24 }}>Empty State</Text>
  <Text style={{ color: colors.muted, fontSize: 14 }}>Nothing to show yet</Text>
</FlexColumn>
```
---
Platforms: love2d

Example 3: Form layout
---
```tsx
const colors = useThemeColors();

<FlexColumn gap={16} style={{ padding: 16 }}>
  <FlexColumn gap={4}>
    <Text style={{ color: colors.muted, fontSize: 11 }}>NAME</Text>
    <TextInput placeholder="Enter your name" style={{ color: colors.text }} />
  </FlexColumn>
  <FlexColumn gap={4}>
    <Text style={{ color: colors.muted, fontSize: 11 }}>EMAIL</Text>
    <TextInput placeholder="Enter your email" style={{ color: colors.text }} />
  </FlexColumn>
</FlexColumn>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- FlexColumn sets `flexDirection: 'column'` explicitly, though column is already the engine default. This makes layout direction visible in code.
- The `style` prop merges after the shorthand props. Conflicting values in `style` override the shorthands.
