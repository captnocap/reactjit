=== OVERVIEW ===
Sparkline renders a compact mini bar chart from an array of numbers. Each value becomes a vertical bar scaled relative to the min/max of the dataset. Designed for inline use alongside text -- dashboards, table cells, status rows.

Built with Box primitives (not a Chart2D surface). When `interactive` is enabled, hovering a bar highlights it and dims others. A tooltip shows the value and index. Press events are available via `onPointPress`.

Defaults to 80x24 pixels -- small enough to sit inline.

=== API / SYNTAX ===
```tsx
import { Sparkline } from '@reactjit/core';
```

Props:
| Prop         | Type                                           | Default   | Description                          |
|--------------|-------------------------------------------------|-----------|--------------------------------------|
| data         | number[]                                       | required  | Array of numeric values              |
| width        | number                                         | 80        | Total width in pixels                |
| height       | number                                         | 24        | Total height in pixels               |
| color        | Color                                          | '#3b82f6' | Bar color                            |
| interactive  | boolean                                        | false     | Enable hover highlight and tooltips  |
| onPointHover | (index: number \| null, value: number \| null) => void | undefined | Called on hover/unhover     |
| onPointPress | (index: number, value: number) => void         | undefined | Called on bar press                   |
| style        | Style                                          | undefined | Full style object                    |

=== EXAMPLES ===
Example 1: Basic sparkline
---
```tsx
const colors = useThemeColors();

<Sparkline data={[3, 7, 4, 9, 2, 6, 8]} color={colors.primary} />
```
---
Platforms: love2d

Example 2: Inline with text
---
```tsx
const colors = useThemeColors();

<Box style={{ flexDirection: 'row', alignItems: 'center', gap: 8 }}>
  <Text style={{ color: colors.text, fontSize: 14 }}>CPU</Text>
  <Sparkline data={cpuHistory} color={colors.primary} width={60} height={16} />
  <Text style={{ color: colors.muted, fontSize: 12 }}>{`${cpuHistory[cpuHistory.length - 1]}%`}</Text>
</Box>
```
---
Platforms: love2d

Example 3: Interactive sparkline
---
```tsx
const colors = useThemeColors();

<Sparkline
  data={[10, 25, 18, 42, 35, 28, 50, 38]}
  color={colors.primary}
  width={120}
  height={32}
  interactive
  onPointPress={(index, value) => console.log(`Point ${index}: ${value}`)}
/>
```
---
Platforms: love2d

=== CRITICAL RULES ===
- An empty `data` array renders an empty Box at the specified dimensions.
- Bar widths are computed automatically from the data length, width, and a 1px gap between bars.
- Sparkline is not a Chart2D surface -- it is built from Box primitives. It sizes from its `width`/`height` props (defaulting to 80x24), not from proportional surface fallback.
