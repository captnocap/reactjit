=== OVERVIEW ===
Divider renders a thin line to visually separate content. It defaults to horizontal (full width, 1px tall) but can be set to vertical (full height, 1px wide).

The divider is a plain Box with a background color. It has no margins by default — use `style` or parent gap to control spacing around it.

=== API / SYNTAX ===
```tsx
import { Divider } from '@reactjit/core';
```

Props:
| Prop      | Type                        | Default      | Description                          |
|-----------|-----------------------------|--------------|--------------------------------------|
| direction | 'horizontal' \| 'vertical' | 'horizontal' | Orientation of the divider           |
| color     | string                      | '#334155'    | Line color                           |
| thickness | number                      | 1            | Line thickness in pixels             |
| style     | Style                       | undefined    | Additional style overrides           |

=== EXAMPLES ===
Example 1: Horizontal divider
---
```tsx
const colors = useThemeColors();

<FlexColumn gap={12} style={{ padding: 16 }}>
  <Text style={{ color: colors.text, fontSize: 14 }}>Section One</Text>
  <Divider color={colors.border} />
  <Text style={{ color: colors.text, fontSize: 14 }}>Section Two</Text>
</FlexColumn>
```
---
Platforms: love2d

Example 2: Vertical divider in a row
---
```tsx
const colors = useThemeColors();

<FlexRow gap={12} align="center" style={{ height: 40 }}>
  <Text style={{ color: colors.text, fontSize: 12 }}>Left</Text>
  <Divider direction="vertical" color={colors.border} />
  <Text style={{ color: colors.text, fontSize: 12 }}>Right</Text>
</FlexRow>
```
---
Platforms: love2d

Example 3: Thick accent divider
---
```tsx
const colors = useThemeColors();

<Divider color={colors.primary} thickness={3} />
```
---
Platforms: love2d

=== CRITICAL RULES ===
- A horizontal divider sets `width: '100%'`. It needs a parent with a defined width to be visible.
- A vertical divider sets `height: '100%'`. It needs a parent with a defined height to be visible. In a FlexRow, give the row an explicit height or use `align="stretch"` (the default) so the divider inherits height from taller siblings.
