=== OVERVIEW ===
Three components for building chat interfaces: MessageBubble for individual messages, ChatInput for the text entry bar, and MessageList for the scrollable conversation container.

MessageBubble renders a styled message with configurable alignment and colors. The `variant` prop controls left-aligned (incoming), right-aligned (outgoing), or center-aligned (system) styling. Each variant has distinct background colors and border radius patterns. Optional label (sender name) and timestamp slots are included.

ChatInput is a text input bar with a send button. It supports controlled and uncontrolled modes, multiline input, and left/right content slots for icons or action buttons. The send button is automatically disabled when the input is empty.

MessageList is a scrollable container that wraps its children in a ScrollView. By default it is inverted (justifyContent: 'end') so messages anchor to the bottom, matching typical chat UI behavior. An optional emptyContent slot displays when there are no messages.

=== API / SYNTAX ===
```tsx
import { MessageBubble, ChatInput, MessageList } from '@reactjit/core';
```

|--------------|-------------------------------|-----------|----------------------------------|

| timestamp    | string                        | undefined | Time label below the bubble      |

| style        | Style                         | undefined | Container style                  |
| contentStyle | Style                         | undefined | Content text style               |

=== EXAMPLES ===
Example 1: Basic chat layout
---
```tsx
const [messages, setMessages] = useState([
  { id: '1', text: 'Hello!', from: 'them' },
  { id: '2', text: 'Hi there!', from: 'me' },
]);

<Box style={{ flexGrow: 1 }}>
  <MessageList>
    {messages.map(m => (
      <MessageBubble
        key={m.id}
        variant={m.from === 'me' ? 'right' : 'left'}
        label={m.from === 'me' ? 'You' : 'Alice'}
      >
        {m.text}
      </MessageBubble>
    ))}
  </MessageList>
  <ChatInput onSend={(text) => {
    setMessages(prev => [...prev, { id: String(Date.now()), text, from: 'me' }]);
  }} />
</Box>
```

=== PLATFORM NOTES ===
Love2D:
  MessageBubble uses hardcoded dark-theme colors (#1e293b for left, #2563eb for right, #334155 for center). Override with the `bg` and `color` props for custom themes. ChatInput uses TextInput internally and submits on Enter via onSubmit. MessageList wraps content in a ScrollView that needs flexGrow or explicit height from its parent.

=== CRITICAL RULES ===
- MessageList uses ScrollView internally. Its parent must provide height via flexGrow or explicit dimensions, or the scroll viewport will have zero height.
- ChatInput supports both controlled (value + onChangeText) and uncontrolled modes. In uncontrolled mode, the input clears itself after send.
- String children in MessageBubble are automatically wrapped in a Text component. For rich content (images, formatted text), pass ReactNode children directly.
