=== OVERVIEW ===
VideoPlayer extends Video with built-in transport controls — play/pause, seek bar, and time display. It renders a Video surface with a control overlay managed by the Lua side.

Use VideoPlayer when you want ready-made playback controls without building custom UI. Use Video directly when you need full control over the transport interface.

VideoPlayer is a surface element — without explicit dimensions, it falls back to 1/4 of parent size (proportional surface fallback).

=== API / SYNTAX ===
```tsx
import { VideoPlayer } from '@reactjit/core';
```

Props (all Video props plus):
| Prop         | Type                                | Default   | Description                     |
|--------------|-------------------------------------|-----------|---------------------------------|
| src          | string                              | required  | Video file path                 |
| controls     | boolean                             | true      | Show built-in transport controls|
| paused       | boolean                             | false     | Whether playback is paused      |
| loop         | boolean                             | false     | Loop playback                   |
| muted        | boolean                             | false     | Mute audio                      |
| volume       | number                              | 1         | Volume level (0 to 1)           |
| w            | number \| string                    | undefined | Width shorthand                 |
| h            | number \| string                    | undefined | Height shorthand                |
| radius       | number                              | undefined | Border radius shorthand         |
| style        | Style                               | undefined | Full style object               |
| onTimeUpdate | (event: VideoTimeEvent) => void     | undefined | Periodic time update            |
| onEnded      | () => void                          | undefined | Playback reached end            |
| onPlay       | () => void                          | undefined | Playback started                |
| onPause      | () => void                          | undefined | Playback paused                 |
| onReady      | () => void                          | undefined | Video loaded and ready          |
| onError      | (event: { message: string }) => void| undefined | Error occurred                  |
| onClick      | (event: LoveEvent) => void          | undefined | Click handler                   |

=== PLATFORM NOTES ===
Love2D:
  VideoPlayer renders as a 'VideoPlayer' host element. The Lua side draws built-in transport controls (play/pause button, seek bar, elapsed/total time) over the video surface. Controls auto-hide after a period of inactivity and reappear on mouse movement. Set `controls={false}` to suppress the overlay entirely.

=== CRITICAL RULES ===
- VideoPlayer is a surface element. Without explicit dimensions, it uses the proportional surface fallback (1/4 of parent). Always set width/height, w/h shorthand, or flexGrow.
- The `controls` prop defaults to true. Set it to false to remove the Lua-drawn transport overlay.
- The shorthand props (w, h, radius) are merged into the style object. If both w and style.width are set, style.width takes precedence due to spread order.
