=== OVERVIEW ===
@reactjit/privacy provides a comprehensive cryptographic privacy toolkit. The usePrivacy hook bundles all modules into a single API object with namespaced methods. Individual functions can also be imported directly.

All cryptographic operations delegate to the Lua runtime via RPC (libsodium FFI, GPG CLI, OS filesystem). React never handles raw key material -- keys live in Lua process memory or the OS keyring.

Modules: GPG (sign/verify/encrypt/decrypt), keyrings (create/rotate/revoke keys), Shamir secret sharing (split/combine), HKDF key derivation, file encryption (XChaCha20-Poly1305 or AES-256-GCM), PII detection and redaction, Noise protocol sessions, steganography (image LSB and whitespace encoding), encrypted key-value stores, GDPR-style policy enforcement, and tamper-evident audit logs.

=== API / SYNTAX ===
```tsx
import { usePrivacy } from '@reactjit/privacy';
```

=== EXAMPLES ===
Example 1: Shamir secret sharing
---
```tsx
import { usePrivacy } from '@reactjit/privacy';
import { Box, Text, Pressable } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';
import { useState } from 'react';

export function ShamirDemo() {
  const colors = useThemeColors();
  const privacy = usePrivacy();
  const [shares, setShares] = useState<Array<{ index: number; hex: string }>>([]);

  const split = async () => {
    const result = await privacy.shamir.split('deadbeef', 5, 3);
    setShares(result);
  };

  const recover = async () => {
    const subset = shares.slice(0, 3);
    const secret = await privacy.shamir.combine(subset);
    console.log('Recovered:', secret);
  };

  return (
    <Box style={{ gap: 8 }}>
      <Pressable onPress={split}>
        <Text style={{ color: colors.primary }}>Split Secret (5 shares, threshold 3)</Text>
      </Pressable>
      {shares.map(s => (
        <Text key={s.index} style={{ color: colors.text, fontSize: 12 }}>
          {`Share ${s.index}: ${s.hex.slice(0, 16)}...`}
        </Text>
      ))}
      {shares.length > 0 && (
        <Pressable onPress={recover}>
          <Text style={{ color: colors.primary }}>Recover from 3 shares</Text>
        </Pressable>
      )}
    </Box>
  );
}
```

Example 2: PII redaction
---
```tsx
import { detectPII, redactPII, maskValue } from '@reactjit/privacy';

const text = 'Contact john@example.com or call 555-123-4567';
const matches = detectPII(text);
// [{ type: 'email', value: 'john@example.com', ... }, { type: 'phone', value: '555-123-4567', ... }]

const redacted = redactPII(text);
// 'Contact [REDACTED] or call [REDACTED]'

const masked = maskValue('4111111111111111', { visibleEnd: 4, maskChar: '*' });
// '************1111'
```

Example 3: Encrypted key-value store
---
```tsx
import { usePrivacy } from '@reactjit/privacy';
import { useEffect } from 'react';

export function SecureStorage() {
  const privacy = usePrivacy();

  useEffect(() => {
    (async () => {
      const store = await privacy.store.create({
        path: 'secrets.db',
        password: 'hunter2',
        kdf: 'argon2id',
      });
      await store.set('api-key', 'sk-abc123');
      const key = await store.get('api-key');
      console.log(key); // 'sk-abc123'
      await store.close();
    })();
  }, []);

  return null;
}
```

Example 4: Tamper-evident audit log
---
```tsx
import { usePrivacy } from '@reactjit/privacy';

export function useAudit() {
  const privacy = usePrivacy();

  const log = async (event: string, data?: any) => {
    await privacy.audit.append(event, data);
  };

  const verify = async () => {
    const result = await privacy.audit.verify();
    if (!result.valid) {
      console.error(`Audit chain broken at entry ${result.brokenAt}`);
    }
    return result;
  };

  return { log, verify };
}
```

=== PLATFORM NOTES ===
Love2D:
  GPG operations require gpg/gpg2 installed on the system.
  File encryption uses libsodium FFI (bundled with the framework).
  Secure memory uses mlock/mprotect syscalls -- requires appropriate OS permissions.
  Tor status checks the local Tor daemon via control port.
  Steganography image embedding requires a PNG input and produces a PNG output.

=== CRITICAL RULES ===
- All crypto operations are async (return Promises). Await them.
- PII detection (detectPII, redactPII) runs synchronously in JS -- no RPC needed.
- Keyring files are encrypted at rest with the master password. Losing the password means losing all keys.
- Shamir shares must be from the same split operation. Mixing shares from different splits will produce garbage.
- The audit log uses hash chaining (each entry hashes the previous). Tampering with any entry breaks the chain from that point forward.
- Noise protocol sessions are stateful. Call noiseClose when done to free Lua-side resources.
