=== OVERVIEW ===
useSemanticTerminal extends usePTY with semantic classification. The Lua-side classifier tags each terminal row with a token type (prompt, output, error, user_input, etc.), and this hook tracks those classifications, graph state, and playback position.

Supports two modes:
- `live` -- real PTY session with classification tracking
- `playback` -- replay a recorded .rec.lua session with transport controls (play/pause/seek/step)

The hook returns a `terminalProps` object to spread onto a `<SemanticTerminal>` element.

=== API / SYNTAX ===
```tsx
import { useSemanticTerminal } from '@reactjit/core';
```

Signature:
```tsx
function useSemanticTerminal(opts?: UseSemanticTerminalOptions): UseSemanticTerminalResult
```

Options:
| Option        | Type               | Default  | Description                          |
|---------------|--------------------|----------|--------------------------------------|
| mode          | 'live' \| 'playback'| 'live' | Session mode                         |
| command       | string             | 'bash'   | Command to run in live mode          |
| args          | string             | undefined| Command arguments                    |
| cwd           | string             | undefined| Working directory                    |
| rows          | number             | 40       | Terminal rows                        |
| cols          | number             | 120      | Terminal columns                     |
| classifier    | string             | 'basic'  | Classifier name from lua/classifiers/|
| recording     | boolean            | false    | Record output for later playback     |
| playbackSrc   | string             | undefined| Path to .rec.lua file for playback   |
| playbackSpeed | number             | 1.0      | Playback speed multiplier            |
| showTokens    | boolean            | false    | Show token type badges on rows       |
| showGraph     | boolean            | false    | Show semantic graph panel            |
| showTimeline  | boolean            | false    | Show timeline scrubber in playback   |

Returns:
| Field          | Type               | Description                           |
|----------------|--------------------|---------------------------------------|
| classifiedRows | ClassifiedRow[]    | All classified rows so far            |
| graphState     | GraphState \| null | Current semantic graph state          |
| playerState    | PlayerState \| null| Playback transport state              |
| play           | () => void         | Resume/start playback                 |
| pause          | () => void         | Pause playback                        |
| seek           | (seconds) => void  | Seek to time position                 |
| step           | () => void         | Advance one frame                     |
| stepBack       | () => void         | Go back one frame                     |
| setSpeed       | (speed) => void    | Set playback speed                    |
| saveRecording  | (path) => void     | Save recording to file                |
| terminalProps  | SemanticTerminalProps | Spread onto <SemanticTerminal>     |

Types:
```tsx
interface ClassifiedRow {
  row: number;
  token: string;    // "prompt", "output", "error", "user_input", etc.
  text: string;
}

interface GraphState {
  nodeCount: number;
  turnCount: number;
  mode: string;
  streaming: boolean;
}

interface PlayerState {
  playing: boolean;
  time: number;
  duration: number;
  progress: number;    // 0-1
  frame: number;
  totalFrames: number;
  speed: number;
}
```

=== EXAMPLES ===
Example 1: Live session with classification
---
```tsx
import { Box } from '@reactjit/core';
import { useSemanticTerminal, SemanticTerminal } from '@reactjit/core';

export function ClassifiedShell() {
  const { classifiedRows, terminalProps } = useSemanticTerminal({
    mode: 'live',
    command: 'bash',
    classifier: 'claude',
    showTokens: true,
  });

  return (
    <Box style={{ flexGrow: 1 }}>
      <SemanticTerminal {...terminalProps} style={{ flexGrow: 1 }} />
    </Box>
  );
}
```
---
Platforms: love2d

Example 2: Session playback with controls
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { useSemanticTerminal, SemanticTerminal } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function SessionPlayer({ src }: { src: string }) {
  const colors = useThemeColors();
  const { playerState, play, pause, terminalProps } = useSemanticTerminal({
    mode: 'playback',
    playbackSrc: src,
    showTimeline: true,
  });

  return (
    <Box style={{ flexGrow: 1, gap: 8 }}>
      <SemanticTerminal {...terminalProps} style={{ flexGrow: 1 }} />
      <Box style={{ flexDirection: 'row', gap: 8, padding: 8 }}>
        <Pressable onPress={playerState?.playing ? pause : play}>
          <Text style={{ color: colors.primary }}>
            {playerState?.playing ? 'Pause' : 'Play'}
          </Text>
        </Pressable>
        <Text style={{ color: colors.muted }}>
          {`${playerState?.time.toFixed(1)}s / ${playerState?.duration.toFixed(1)}s`}
        </Text>
      </Box>
    </Box>
  );
}
```
---
Platforms: love2d

=== CRITICAL RULES ===
- You MUST render `<SemanticTerminal {...terminalProps} />` in your component tree.
- Classifier names map to Lua modules in lua/classifiers/. The 'basic' classifier is always available.
- classifiedRows accumulates over the session lifetime. Rows are replaced (not appended) when their row number already exists.
- Playback controls (play/pause/seek/step) only work when mode is 'playback'.
