=== OVERVIEW ===
useLoveOverlays maps Lua-side entity coordinates to screen positions. It subscribes to the `overlays` bridge event and converts normalized coordinates (0-1) to pixel positions based on a canvas reference element.

This hook is primarily useful in WASM builds where React controls a DOM layer on top of the Love2D canvas. In native Love2D mode, overlays are typically rendered directly by the painter. Most applications will not need this hook.

=== API / SYNTAX ===
```tsx
import { useLoveOverlays } from '@reactjit/core';
```

Signature:
```tsx
function useLoveOverlays(canvasRef: React.RefObject<HTMLCanvasElement | null>): Overlay[]
```

Overlay interface:
| Field  | Type    | Description                           |
|--------|---------|---------------------------------------|
| id     | string  | Unique overlay identifier             |
| nx     | number  | Normalized X coordinate (0-1)         |
| ny     | number  | Normalized Y coordinate (0-1)         |
| px     | number  | Pixel X in game coordinates           |
| py     | number  | Pixel Y in game coordinates           |
| domX   | number  | Computed X position for DOM overlay   |
| domY   | number  | Computed Y position for DOM overlay   |

=== EXAMPLES ===
Example 1: WASM overlay positioning
---
```tsx
import { useLoveOverlays } from '@reactjit/core';

function GameOverlays({ canvasRef }: { canvasRef: React.RefObject<HTMLCanvasElement> }) {
  const overlays = useLoveOverlays(canvasRef);

  return (
    <Box>
      {overlays.map((o) => (
        <Box
          key={o.id}
          style={{
            position: 'absolute',
            left: o.domX,
            top: o.domY,
          }}
        >
          <Text>{o.id}</Text>
        </Box>
      ))}
    </Box>
  );
}
```

=== PLATFORM NOTES ===
Love2D:
  In native Love2D mode (QuickJS FFI), this hook is rarely needed because overlays can be rendered directly in the Lua painter. It exists primarily for WASM builds where a DOM layer sits on top of the canvas element and needs pixel-accurate positioning from game coordinates.
