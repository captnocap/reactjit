=== OVERVIEW ===
Welcome to ReactJIT. This section covers everything you need to go from zero to a
running application.

ReactJIT is a Love2D rendering framework that uses React as its layout declaration
layer. There is no DOM, no CSS, no browser. React declares the layout tree, Lua
computes geometry, and Love2D draws pixels via OpenGL 2.1.

Start with the philosophy document to understand the design principles. Then follow the
installation guide, run through the quick start, and build your first interactive app.
