=== OVERVIEW ===
The Google hooks cover two services: Google Calendar and Google Sheets. Both use OAuth2 Bearer tokens for authentication. All read hooks return the standard APIResult shape.

Google Calendar hooks let you list calendars, fetch events with time-range filtering, and create or delete events. Google Sheets hooks read spreadsheet metadata, fetch cell values by range, and append or update rows.

Both mutation hooks (useGoogleCalendarMutation, useGoogleSheetMutation) return named action functions plus loading/error state.

=== API / SYNTAX ===
```tsx
import {
  // Calendar
  useGoogleCalendars,
  useGoogleCalendarEvents,
  useGoogleCalendarMutation,
  // Sheets
  useGoogleSheet,
  useGoogleSheetValues,
  useGoogleSheetMutation,
} from '@reactjit/apis';
```

---- useGoogleCalendars

```tsx
function useGoogleCalendars(token: string | null): APIResult<{ items: GoogleCalendar[] }>
```

Lists all calendars for the authenticated user.

---- useGoogleCalendarEvents

```tsx
function useGoogleCalendarEvents(
  token: string | null,
  opts?: {
    calendarId?: string;
    timeMin?: string;
    timeMax?: string;
    maxResults?: number;
    singleEvents?: boolean;
    orderBy?: 'startTime' | 'updated';
  }
): APIResult<GoogleCalendarList>
```

| Param             | Type                       | Default              | Description                                |
|-------------------|----------------------------|----------------------|--------------------------------------------|
| token             | string \| null              | required             | OAuth2 token, or null to disable            |
| opts.calendarId   | string                     | "primary"            | Calendar ID (URL-encoded automatically)     |
| opts.timeMin      | string                     | now (ISO string)     | Start of time range                         |
| opts.timeMax      | string                     | undefined            | End of time range                           |
| opts.maxResults   | number                     | 50                   | Max events to return                        |
| opts.singleEvents | boolean                    | true                 | Expand recurring events into instances       |
| opts.orderBy      | 'startTime' \| 'updated'   | 'startTime'          | Sort order                                  |

---- useGoogleCalendarMutation

```tsx
function useGoogleCalendarMutation(token: string | null): {
  createEvent: (calendarId: string, event: { summary: string; start: any; end: any; description?: string; location?: string }) => Promise<any>;
  deleteEvent: (calendarId: string, eventId: string) => Promise<any>;
  loading: boolean;
  error: Error | null;
}
```

---- useGoogleSheet

```tsx
function useGoogleSheet(
  token: string | null,
  spreadsheetId: string | null
): APIResult<GoogleSheet>
```

Fetches spreadsheet metadata including sheet names and properties.

---- useGoogleSheetValues

```tsx
function useGoogleSheetValues(
  token: string | null,
  spreadsheetId: string | null,
  range: string | null
): APIResult<GoogleSheetValues>
```

Reads cell values from a named range (e.g. "Sheet1!A1:D10"). All three parameters must be non-null to fetch.

---- useGoogleSheetMutation

```tsx
function useGoogleSheetMutation(token: string | null): {
  append: (spreadsheetId: string, range: string, values: string[][]) => Promise<any>;
  update: (spreadsheetId: string, range: string, values: string[][]) => Promise<any>;
  loading: boolean;
  error: Error | null;
}
```

Both append and update use USER_ENTERED value input option, so formulas and number formats are interpreted.

---- Types

GoogleCalendar:
| Field           | Type    | Description                          |
|-----------------|---------|--------------------------------------|

| summary         | string  | Calendar name                        |

| Field     | Type   | Description                                  |
|-----------|--------|----------------------------------------------|

| start     | object | Start time ({ dateTime, date, timeZone })    |

| status    | string | Event status (confirmed, tentative, etc.)    |

=== EXAMPLES ===
Example 1: Today's calendar events
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useGoogleCalendarEvents } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TodayEvents({ token }: { token: string }) {
  const colors = useThemeColors();
  const today = new Date();
  const tomorrow = new Date(today);
  tomorrow.setDate(tomorrow.getDate() + 1);

  const { data, loading } = useGoogleCalendarEvents(token, {
    timeMin: today.toISOString(),
    timeMax: tomorrow.toISOString(),
    maxResults: 10,
  });

  if (loading) return <Text style={{ color: colors.muted }}>Loading events...</Text>;

  return (
    <Box style={{ gap: 4 }}>
      {data?.items?.map(event => (
        <Box key={event.id} style={{ flexDirection: 'row', gap: 8 }}>
          <Text style={{ color: colors.muted, width: 60 }}>
            {event.start.dateTime ? new Date(event.start.dateTime).toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' }) : 'All day'}
          </Text>
          <Text style={{ color: colors.text }}>{event.summary}</Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 2: Read spreadsheet data
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useGoogleSheetValues } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function SheetTable({ token, sheetId }: { token: string; sheetId: string }) {
  const colors = useThemeColors();
  const { data, loading } = useGoogleSheetValues(token, sheetId, 'Sheet1!A1:D20');

  if (loading) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  const rows = data?.values ?? [];
  const [header, ...body] = rows;

  return (
    <Box style={{ gap: 2 }}>
      <Box style={{ flexDirection: 'row', gap: 8 }}>
        {header?.map((col, i) => (
          <Text key={i} style={{ color: colors.primary, fontWeight: 'bold', width: 100 }}>{col}</Text>
        ))}
      </Box>
      {body.map((row, i) => (
        <Box key={i} style={{ flexDirection: 'row', gap: 8 }}>
          {row.map((cell, j) => (
            <Text key={j} style={{ color: colors.text, width: 100 }}>{cell}</Text>
          ))}
        </Box>
      ))}
    </Box>
  );
}
```

Platforms: love2d

Example 3: Append rows to a sheet
---
```tsx
import { useState } from 'react';
import { Box, Text, Pressable, Input } from '@reactjit/core';
import { useGoogleSheetMutation } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function AddRow({ token, sheetId }: { token: string; sheetId: string }) {
  const colors = useThemeColors();
  const { append, loading } = useGoogleSheetMutation(token);
  const [name, setName] = useState('');
  const [value, setValue] = useState('');

  const handleSubmit = async () => {
    await append(sheetId, 'Sheet1!A:B', [[name, value]]);
    setName('');
    setValue('');
  };

  return (
    <Box style={{ gap: 8 }}>
      <Input value={name} onChangeText={setName} placeholder="Name" />
      <Input value={value} onChangeText={setValue} placeholder="Value" />
      <Pressable onPress={handleSubmit}>
        <Text style={{ color: colors.primary }}>{loading ? 'Saving...' : 'Add Row'}</Text>
      </Pressable>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Calendar IDs are URL-encoded automatically. Pass the raw ID (e.g. "primary" or an email address).
- timeMin defaults to the current time. If you want all events, pass an explicit past date.
- Sheet ranges use A1 notation: "Sheet1!A1:D10", "Sheet1!A:B" (full columns), or just "Sheet1" (all data).
- Sheet mutations use USER_ENTERED value input option, meaning formulas (=SUM(A1:A10)) and number formats are interpreted by Google.
