=== OVERVIEW ===
Hooks for the Trakt.tv API. Auth uses a Client ID header plus an optional OAuth Bearer token for user-specific endpoints. Get credentials at https://trakt.tv/oauth/applications.

The Client ID is required for all hooks. The OAuth token is required for user-specific data (watching, history, watchlist, stats). Trending content only needs the Client ID.

useTraktWatching polls every 30 seconds by default.

Trakt does not serve images. Pair with TMDB using the `tmdb` ID from `ids.tmdb` to fetch poster art.

=== API / SYNTAX ===
```tsx
import {
  useTraktWatching,
  useTraktHistory,
  useTraktWatchlist,
  useTraktTrending,
  useTraktStats,
} from '@reactjit/apis';
```

---- useTraktWatching

```tsx
function useTraktWatching(
  clientId: string | null,
  token: string | null,
  username?: string,
  opts?: { interval?: number }
): APIResult<TraktWatching | null>
```

Check what a user is currently watching. Returns null when not watching anything. Username defaults to 'me' (requires OAuth token). Polls every 30 seconds by default.

---- useTraktHistory

```tsx
function useTraktHistory(
  clientId: string | null,
  token: string | null,
  opts?: { username?: string; type?: 'movies' | 'shows' | 'episodes'; limit?: number }
): APIResult<TraktHistoryItem[]>
```

Fetch watch history. Optionally filter by type. Username defaults to 'me'. Limit defaults to 20.

---- useTraktWatchlist

```tsx
function useTraktWatchlist(
  clientId: string | null,
  token: string | null,
  opts?: { username?: string; type?: 'movies' | 'shows' }
): APIResult<TraktWatchlistItem[]>
```

Fetch the user's watchlist. Optionally filter by type. Username defaults to 'me'.

---- useTraktTrending

```tsx
function useTraktTrending(
  clientId: string | null,
  type?: 'movies' | 'shows',
  opts?: { limit?: number }
): APIResult<Array<{ watchers: number; movie?: TraktMovie; show?: TraktShow }>>
```

Fetch trending movies or shows. Does not require an OAuth token. Type defaults to 'movies'. Limit defaults to 10.

---- useTraktStats

```tsx
function useTraktStats(
  clientId: string | null,
  token: string | null,
  username?: string
): APIResult<TraktStats>
```

Fetch user watch statistics. Username defaults to 'me'.

---- Types

TraktMovie:
| Field      | Type   | Description                           |
|------------|--------|---------------------------------------|
| title      | string | Movie title                           |
| year       | number | Release year                          |
| ids.trakt  | number | Trakt ID                              |
| ids.slug   | string | URL slug                              |
| ids.imdb   | string?| IMDb ID                               |
| ids.tmdb   | number?| TMDB ID (use with tmdbImage)          |

TraktShow:
| Field      | Type    | Description                          |
|------------|---------|--------------------------------------|
| title      | string  | Show title                           |
| year       | number  | First air year                       |
| ids.trakt  | number  | Trakt ID                             |
| ids.slug   | string  | URL slug                             |
| ids.imdb   | string? | IMDb ID                              |
| ids.tmdb   | number? | TMDB ID                              |
| ids.tvdb   | number? | TVDB ID                              |

TraktEpisode:
| Field      | Type    | Description                          |
|------------|---------|--------------------------------------|
| season     | number  | Season number                        |
| number     | number  | Episode number                       |
| title      | string  | Episode title                        |
| ids.trakt  | number  | Trakt ID                             |

TraktWatching:
| Field      | Type               | Description                |
|------------|--------------------|----------------------------|
| expires_at | string             | When the check-in expires  |
| started_at | string             | When playback started      |
| action     | string             | 'watching', 'scrobble'     |
| type       | 'movie'\|'episode' | What is being watched      |
| movie      | TraktMovie?        | Present if type is 'movie' |
| show       | TraktShow?         | Present if type is 'episode'|
| episode    | TraktEpisode?      | Present if type is 'episode'|

TraktHistoryItem:
| Field      | Type               | Description                |
|------------|--------------------|----------------------------|
| id         | number             | History entry ID           |
| watched_at | string             | Watch timestamp            |
| action     | string             | 'watch', 'scrobble'       |
| type       | 'movie'\|'episode' | Watched content type       |
| movie      | TraktMovie?        | Present if type is 'movie' |
| show       | TraktShow?         | Present if type is 'episode'|
| episode    | TraktEpisode?      | Present if type is 'episode'|

TraktWatchlistItem:
| Field     | Type              | Description            |
|-----------|-------------------|------------------------|
| listed_at | string            | When added to list     |
| type      | 'movie'\|'show'   | Item type              |
| movie     | TraktMovie?       | Present if movie       |
| show      | TraktShow?        | Present if show        |

TraktStats:
| Field            | Type   | Description                    |
|------------------|--------|--------------------------------|
| movies.plays     | number | Total movie plays              |
| movies.watched   | number | Unique movies watched          |
| movies.minutes   | number | Total movie watch time         |
| shows.watched    | number | Unique shows watched           |
| shows.collected  | number | Shows in collection            |
| episodes.plays   | number | Total episode plays            |
| episodes.watched | number | Unique episodes watched        |
| episodes.minutes | number | Total episode watch time       |

=== EXAMPLES ===
Example 1: Currently watching
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useTraktWatching } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function Watching({ clientId, token }: { clientId: string; token: string }) {
  const colors = useThemeColors();
  const { data } = useTraktWatching(clientId, token);

  if (!data) return <Text style={{ color: colors.muted }}>Not watching anything</Text>;

  const title = data.type === 'movie'
    ? data.movie?.title
    : `${data.show?.title} S${data.episode?.season}E${data.episode?.number}`;

  return (
    <Box style={{ gap: 2 }}>
      <Text style={{ color: colors.primary, fontSize: 10 }}>NOW WATCHING</Text>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>{title}</Text>
    </Box>
  );
}
```

Example 2: Trending movies
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useTraktTrending } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TrendingMovies({ clientId }: { clientId: string }) {
  const colors = useThemeColors();
  const { data } = useTraktTrending(clientId, 'movies', { limit: 5 });

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>Trending Movies</Text>
      {data?.map((item, i) => (
        <Text key={item.movie?.ids.trakt} style={{ color: colors.text, fontSize: 12 }}>
          {`${i + 1}. ${item.movie?.title} (${item.watchers} watching)`}
        </Text>
      ))}
    </Box>
  );
}
```

Example 3: Watch stats dashboard
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useTraktStats } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function WatchStats({ clientId, token }: { clientId: string; token: string }) {
  const colors = useThemeColors();
  const { data } = useTraktStats(clientId, token);

  if (!data) return <Text style={{ color: colors.muted }}>Loading stats...</Text>;

  const movieDays = Math.round(data.movies.minutes / 1440);
  const tvDays = Math.round(data.episodes.minutes / 1440);

  return (
    <Box style={{ gap: 6 }}>
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>Movies</Text>
        <Text style={{ color: colors.muted, fontSize: 12 }}>
          {`${data.movies.watched} watched | ${movieDays} days`}
        </Text>
      </Box>
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>TV Shows</Text>
        <Text style={{ color: colors.muted, fontSize: 12 }}>
          {`${data.episodes.watched} episodes | ${tvDays} days`}
        </Text>
      </Box>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Trakt does not serve images. Use `ids.tmdb` with tmdbImage() from the TMDB hooks to get poster art.
- useTraktWatching returns null (not an empty object) when nothing is playing. Null-check before accessing fields.
- User-specific hooks (watching, history, watchlist, stats) require both clientId and token. Without the token, the hook is disabled.
- useTraktTrending is the only hook that works with just a clientId (no OAuth token needed).
