=== OVERVIEW ===
Hooks for the GitHub REST API. Auth uses a Bearer token (personal access token). Get one at https://github.com/settings/tokens.

The token is optional for public data -- pass null for unauthenticated requests (lower rate limits). All hooks return `{ data, loading, error, refetch }`.

GitHub headers include `Accept: application/vnd.github+json` automatically.

=== API / SYNTAX ===
```tsx
import {
  useGitHubUser,
  useGitHubAuthUser,
  useGitHubRepos,
  useGitHubRepo,
  useGitHubIssues,
  useGitHubEvents,
  useGitHubGists,
  useGitHubStarred,
} from '@reactjit/apis';
```

---- useGitHubUser

```tsx
function useGitHubUser(
  username: string | null,
  token?: string | null
): APIResult<GitHubUser>
```

Fetch a public user profile by username. Token is optional (for higher rate limits).

---- useGitHubAuthUser

```tsx
function useGitHubAuthUser(
  token: string | null
): APIResult<GitHubUser>
```

Fetch the authenticated user's profile. Requires a valid token.

---- useGitHubRepos

```tsx
function useGitHubRepos(
  username: string | null,
  opts?: { sort?: 'updated' | 'stars' | 'pushed'; perPage?: number; token?: string | null }
): APIResult<GitHubRepo[]>
```

List a user's repositories. Defaults to sort by 'updated', 30 per page.

---- useGitHubRepo

```tsx
function useGitHubRepo(
  owner: string | null,
  repo: string | null,
  token?: string | null
): APIResult<GitHubRepo>
```

Fetch a single repository by owner and name.

---- useGitHubIssues

```tsx
function useGitHubIssues(
  owner: string | null,
  repo: string | null,
  opts?: { state?: 'open' | 'closed' | 'all'; perPage?: number; token?: string | null }
): APIResult<GitHubIssue[]>
```

List issues for a repository. Defaults to 'open' state, 30 per page. Includes pull requests (check the `pull_request` field to filter).

---- useGitHubEvents

```tsx
function useGitHubEvents(
  username: string | null,
  opts?: { perPage?: number; token?: string | null }
): APIResult<GitHubEvent[]>
```

List recent public events for a user. Defaults to 30 per page.

---- useGitHubGists

```tsx
function useGitHubGists(
  username: string | null,
  token?: string | null
): APIResult<GitHubGist[]>
```

List a user's public gists.

---- useGitHubStarred

```tsx
function useGitHubStarred(
  username: string | null,
  opts?: { perPage?: number; token?: string | null }
): APIResult<GitHubRepo[]>
```

List repositories starred by a user. Defaults to 30 per page.

---- Types

GitHubUser:
| Field        | Type           | Description              |
|--------------|----------------|--------------------------|
| login        | string         | Username                 |
| id           | number         | User ID                  |
| avatar_url   | string         | Avatar image URL         |
| name         | string \| null | Display name             |
| bio          | string \| null | Bio text                 |
| public_repos | number         | Public repository count  |
| followers    | number         | Follower count           |
| following    | number         | Following count          |
| created_at   | string         | Account creation date    |
| html_url     | string         | Profile URL              |

GitHubRepo:
| Field             | Type           | Description            |
|-------------------|----------------|------------------------|
| id                | number         | Repository ID          |
| name              | string         | Repository name        |
| full_name         | string         | owner/repo format      |
| description       | string \| null | Description            |
| html_url          | string         | Repository URL         |
| stargazers_count  | number         | Star count             |
| forks_count       | number         | Fork count             |
| language          | string \| null | Primary language       |
| updated_at        | string         | Last update timestamp  |
| open_issues_count | number         | Open issues + PRs      |
| topics            | string[]       | Topic tags             |
| fork              | boolean        | Whether it is a fork   |
| private           | boolean        | Whether it is private  |

GitHubIssue:
| Field      | Type                        | Description           |
|------------|-----------------------------|-----------------------|
| id         | number                      | Issue ID              |
| number     | number                      | Issue number          |
| title      | string                      | Issue title           |
| body       | string \| null              | Issue body (markdown) |
| state      | 'open' \| 'closed'          | Current state         |
| user       | { login, avatar_url }       | Author                |
| labels     | Array<{ name, color }>      | Label list            |
| created_at | string                      | Creation timestamp    |
| html_url   | string                      | Issue URL             |

GitHubEvent:
| Field      | Type   | Description              |
|------------|--------|--------------------------|
| id         | string | Event ID                 |
| type       | string | PushEvent, etc.          |
| repo       | object | `{ name: 'owner/repo' }` |
| created_at | string | Event timestamp          |
| payload    | any    | Event-specific data      |

GitHubGist:
| Field       | Type           | Description                          |
|-------------|----------------|--------------------------------------|
| id          | string         | Gist ID                              |
| description | string \| null | Gist description                     |
| html_url    | string         | Gist URL                             |
| files       | Record         | Map of filename to `{ filename, language, size }` |
| created_at  | string         | Creation timestamp                   |

=== EXAMPLES ===
Example 1: User profile card
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useGitHubUser } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function GitHubCard({ username }: { username: string }) {
  const colors = useThemeColors();
  const { data, loading } = useGitHubUser(username);

  if (loading || !data) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  return (
    <Box style={{ flexDirection: 'row', gap: 12, alignItems: 'center' }}>
      <Image src={data.avatar_url} style={{ width: 48, height: 48, borderRadius: 24 }} />
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>{data.name ?? data.login}</Text>
        <Text style={{ color: colors.muted, fontSize: 12 }}>
          {`${data.public_repos} repos | ${data.followers} followers`}
        </Text>
      </Box>
    </Box>
  );
}
```

Example 2: Repository list
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useGitHubRepos } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function RepoList({ username }: { username: string }) {
  const colors = useThemeColors();
  const { data } = useGitHubRepos(username, { sort: 'stars', perPage: 10 });

  return (
    <Box style={{ gap: 6 }}>
      {data?.map(repo => (
        <Box key={repo.id} style={{ gap: 2 }}>
          <Text style={{ color: colors.primary, fontSize: 13, fontWeight: 'bold' }}>{repo.name}</Text>
          <Text style={{ color: colors.muted, fontSize: 11 }}>
            {`${repo.language ?? 'Unknown'} | ${repo.stargazers_count} stars`}
          </Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 3: Issue tracker
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useGitHubIssues } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function IssueList({ owner, repo, token }: { owner: string; repo: string; token?: string }) {
  const colors = useThemeColors();
  const { data } = useGitHubIssues(owner, repo, { state: 'open', token });

  const issues = data?.filter(i => !i.pull_request) ?? [];

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>
        {`Open Issues (${issues.length})`}
      </Text>
      {issues.map(issue => (
        <Text key={issue.id} style={{ color: colors.text, fontSize: 12 }}>
          {`#${issue.number} ${issue.title}`}
        </Text>
      ))}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Unauthenticated requests are rate limited to 60 per hour. With a token, the limit is 5000 per hour.
- useGitHubIssues includes pull requests. Filter by checking `!issue.pull_request` to get issues only.
- GitHub tokens must have the appropriate scopes (e.g., `repo` for private repos, `read:user` for user data).
- Pass null as the username/owner to disable any hook.
