=== OVERVIEW ===
The YNAB hooks wrap the You Need A Budget API v1 for reading budget data. All hooks accept a Bearer token and return the standard APIResult shape. YNAB responses are nested under a data key (e.g. data.budgets, data.accounts), which the hooks return as-is.

YNAB amounts are in "milliunits" -- divide by 1000 to get dollars. The ynabAmount() helper handles this conversion.

Most hooks accept an optional budgetId. When omitted, they default to "last-used", which resolves to the most recently opened budget.

Get your API token from https://app.ynab.com/settings/developer.

=== API / SYNTAX ===
```tsx
import {
  useYNABBudgets,
  useYNABAccounts,
  useYNABCategories,
  useYNABTransactions,
  useYNABMonths,
  ynabAmount,
} from '@reactjit/apis';
```

---- useYNABBudgets

```tsx
function useYNABBudgets(token: string | null): APIResult<{ data: { budgets: YNABBudget[] } }>
```

---- useYNABAccounts

```tsx
function useYNABAccounts(
  token: string | null,
  budgetId: string | null
): APIResult<{ data: { accounts: YNABAccount[] } }>
```

budgetId defaults to "last-used" when null.

---- useYNABCategories

```tsx
function useYNABCategories(
  token: string | null,
  budgetId?: string | null
): APIResult<{ data: { category_groups: YNABCategoryGroup[] } }>
```

Returns category groups, each containing a categories array.

---- useYNABTransactions

```tsx
function useYNABTransactions(
  token: string | null,
  opts?: { budgetId?: string; sinceDate?: string; type?: 'uncategorized' | 'unapproved' }
): APIResult<{ data: { transactions: YNABTransaction[] } }>
```

| Param          | Type                               | Default     | Description                             |
|----------------|------------------------------------|-----------  |-----------------------------------------|
| token          | string \| null                     | required    | YNAB API token, or null to disable       |
| opts.budgetId  | string                             | "last-used" | Budget ID                                |
| opts.sinceDate | string                             | undefined   | ISO date to filter transactions from     |
| opts.type      | 'uncategorized' \| 'unapproved'    | undefined   | Filter by transaction status             |

---- useYNABMonths

```tsx
function useYNABMonths(
  token: string | null,
  budgetId?: string | null
): APIResult<{ data: { months: YNABMonth[] } }>
```

---- ynabAmount

```tsx
function ynabAmount(milliunits: number): number
```

Converts YNAB milliunits to dollars. Example: ynabAmount(1234560) returns 1234.56.

---- Types

YNABBudget:
| Field            | Type   | Description                        |
|------------------|--------|------------------------------------|
| id               | string | Budget ID                          |
| name             | string | Budget name                        |
| last_modified_on | string | ISO timestamp of last modification |
| first_month      | string | First month in the budget          |
| last_month       | string | Last month in the budget           |
| currency_format  | object | Currency formatting info           |

YNABTransaction:
| Field           | Type                                      | Description                |
|-----------------|-------------------------------------------|----------------------------|
| id              | string                                    | Transaction ID             |
| date            | string                                    | Transaction date           |
| amount          | number                                    | Amount in milliunits       |
| memo            | string \| null                             | Transaction memo           |
| cleared         | 'cleared' \| 'uncleared' \| 'reconciled'  | Cleared status             |
| approved        | boolean                                   | Whether approved           |
| payee_name      | string \| null                             | Payee name                 |
| category_name   | string \| null                             | Category name              |
| account_name    | string                                    | Source account name         |
| subtransactions | Array                                     | Split transaction entries   |

=== EXAMPLES ===
Example 1: Display account balances
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useYNABAccounts, ynabAmount } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function AccountBalances({ token }: { token: string }) {
  const colors = useThemeColors();
  const { data, loading } = useYNABAccounts(token, null);
  const accounts = data?.data?.accounts?.filter(a => !a.closed) ?? [];

  if (loading) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  return (
    <Box style={{ gap: 4 }}>
      {accounts.map(acct => (
        <Box key={acct.id} style={{ flexDirection: 'row', justifyContent: 'space-between' }}>
          <Text style={{ color: colors.text }}>{acct.name}</Text>
          <Text style={{ color: ynabAmount(acct.balance) >= 0 ? colors.text : colors.primary }}>
            {`$${ynabAmount(acct.balance).toFixed(2)}`}
          </Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 2: Recent transactions
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useYNABTransactions, ynabAmount } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function RecentTransactions({ token }: { token: string }) {
  const colors = useThemeColors();
  const thirtyDaysAgo = new Date(Date.now() - 30 * 86400000).toISOString().slice(0, 10);
  const { data } = useYNABTransactions(token, { sinceDate: thirtyDaysAgo });
  const txns = data?.data?.transactions ?? [];

  return (
    <Box style={{ gap: 4 }}>
      {txns.slice(0, 20).map(txn => (
        <Box key={txn.id} style={{ flexDirection: 'row', gap: 12 }}>
          <Text style={{ color: colors.muted, width: 80 }}>{txn.date}</Text>
          <Text style={{ color: colors.text, flexGrow: 1 }}>{txn.payee_name ?? 'Unknown'}</Text>
          <Text style={{ color: colors.text }}>{`$${ynabAmount(txn.amount).toFixed(2)}`}</Text>
        </Box>
      ))}
    </Box>
  );
}
```
---
Platforms: love2d

=== CRITICAL RULES ===
- All YNAB amounts are in milliunits (1/1000 of a dollar). Always use ynabAmount() to convert before display.
- Response data is nested under a data key: access it as result.data?.data?.budgets, not result.data?.budgets.
- When budgetId is omitted or null, hooks default to "last-used" which resolves to the most recently opened budget.
