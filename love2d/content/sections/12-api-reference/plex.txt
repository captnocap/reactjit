=== OVERVIEW ===
Hooks for Plex and Jellyfin/Emby media servers. Both require a server base URL and an authentication token or API key.

Plex uses the `X-Plex-Token` header. Find your token at https://support.plex.tv/articles/204059436-finding-an-authentication-token-x-plex-token/.

Jellyfin/Emby uses the `MediaBrowser Token` header. Generate an API key in your Jellyfin dashboard under Administration > API Keys.

Session hooks (usePlexSessions, useJellyfinSessions) poll every 10 seconds by default.

=== API / SYNTAX ===
```tsx
// Plex
import {
  usePlexLibraries,
  usePlexLibrary,
  usePlexRecentlyAdded,
  usePlexSessions,
} from '@reactjit/apis';

// Jellyfin / Emby
import {
  useJellyfinLibraries,
  useJellyfinItems,
  useJellyfinSessions,
  useJellyfinLatest,
  jellyfinImage,
} from '@reactjit/apis';
```

---- usePlexLibraries

```tsx
function usePlexLibraries(
  baseUrl: string | null,
  token: string | null
): APIResult<{ MediaContainer: { Directory: PlexLibrary[] } }>
```

List all libraries on the Plex server.

---- usePlexLibrary

```tsx
function usePlexLibrary(
  baseUrl: string | null,
  token: string | null,
  sectionId: string | null
): APIResult<{ MediaContainer: { Metadata: PlexMediaItem[] } }>
```

List all items in a specific library section.

---- usePlexRecentlyAdded

```tsx
function usePlexRecentlyAdded(
  baseUrl: string | null,
  token: string | null,
  opts?: { limit?: number }
): APIResult<{ MediaContainer: { Metadata: PlexMediaItem[] } }>
```

Fetch recently added media across all libraries. Limit defaults to 20.

---- usePlexSessions

```tsx
function usePlexSessions(
  baseUrl: string | null,
  token: string | null,
  opts?: { interval?: number }
): APIResult<{ MediaContainer: { Metadata?: PlexSession[] } }>
```

Fetch active playback sessions. Polls every 10 seconds by default.

---- useJellyfinLibraries

```tsx
function useJellyfinLibraries(
  baseUrl: string | null,
  apiKey: string | null
): APIResult<{ Items: JellyfinLibrary[] }>
```

List all virtual folders (libraries) on the Jellyfin server.

---- useJellyfinItems

```tsx
function useJellyfinItems(
  baseUrl: string | null,
  apiKey: string | null,
  opts?: { parentId?: string; type?: string; limit?: number; sortBy?: string }
): APIResult<{ Items: JellyfinItem[]; TotalRecordCount: number }>
```

Query items from the library. Defaults to 20 items sorted by DateCreated descending. Set `type` to filter (e.g., 'Movie', 'Series', 'Episode').

---- useJellyfinSessions

```tsx
function useJellyfinSessions(
  baseUrl: string | null,
  apiKey: string | null,
  opts?: { interval?: number }
): APIResult<JellyfinSession[]>
```

Fetch active sessions. Polls every 10 seconds by default.

---- useJellyfinLatest

```tsx
function useJellyfinLatest(
  baseUrl: string | null,
  apiKey: string | null,
  userId: string | null,
  opts?: { limit?: number }
): APIResult<JellyfinItem[]>
```

Fetch latest items for a specific user. Requires a Jellyfin user ID. Limit defaults to 20.

---- jellyfinImage

```tsx
function jellyfinImage(
  baseUrl: string,
  itemId: string,
  tag: string,
  type?: string,
  maxWidth?: number
): string
```

Build a Jellyfin image URL. Type defaults to 'Primary'. maxWidth defaults to 400.

---- Plex Types

PlexMediaItem:
| Field        | Type    | Description                  |
|--------------|---------|------------------------------|
| ratingKey    | string  | Unique item identifier       |
| title        | string  | Item title                   |
| type         | string  | 'movie', 'show', 'episode'   |
| year         | number? | Release year                 |
| thumb        | string? | Thumbnail path               |
| summary      | string? | Plot summary                 |
| duration     | number? | Duration in milliseconds     |
| addedAt      | number  | Added timestamp (Unix)       |
| viewCount    | number? | Number of views              |

PlexLibrary:
| Field   | Type   | Description           |
|---------|--------|-----------------------|
| key     | string | Section ID            |
| title   | string | Library name          |
| type    | string | 'movie', 'show', etc. |
| agent   | string | Metadata agent        |
| scanner | string | Scanner type          |

PlexSession:
| Field      | Type   | Description                          |
|------------|--------|--------------------------------------|
| sessionKey | string | Session identifier                   |
| title      | string | Media title                          |
| type       | string | Media type                           |
| thumb      | string? | Thumbnail path                      |
| User.title | string | Username                             |
| Player     | object | `{ title, state, platform }`         |
| viewOffset | number | Current position (ms)                |
| duration   | number | Total duration (ms)                  |

---- Jellyfin Types

JellyfinItem:
| Field            | Type     | Description                     |
|------------------|----------|---------------------------------|

=== EXAMPLES ===
Example 1: Plex now playing
---
```tsx
import { Box, Text } from '@reactjit/core';
import { usePlexSessions } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function PlexNowPlaying({ baseUrl, token }: { baseUrl: string; token: string }) {
  const colors = useThemeColors();
  const { data } = usePlexSessions(baseUrl, token);
  const sessions = data?.MediaContainer?.Metadata ?? [];

  if (sessions.length === 0) return <Text style={{ color: colors.muted }}>Nothing playing</Text>;

  return (
    <Box style={{ gap: 6 }}>
      {sessions.map(s => (
        <Box key={s.sessionKey} style={{ gap: 2 }}>
          <Text style={{ color: colors.text, fontSize: 13, fontWeight: 'bold' }}>{s.title}</Text>
          <Text style={{ color: colors.muted, fontSize: 11 }}>
            {`${s.User.title} on ${s.Player.title} (${s.Player.state})`}
          </Text>
        </Box>
      ))}
    </Box>
  );
}
```

Example 2: Recently added media
---
```tsx
import { Box, Text } from '@reactjit/core';
import { usePlexRecentlyAdded } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function RecentlyAdded({ baseUrl, token }: { baseUrl: string; token: string }) {
  const colors = useThemeColors();
  const { data } = usePlexRecentlyAdded(baseUrl, token, { limit: 5 });
  const items = data?.MediaContainer?.Metadata ?? [];

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>Recently Added</Text>
      {items.map(item => (
        <Text key={item.ratingKey} style={{ color: colors.text, fontSize: 12 }}>
          {`${item.title}${item.year ? ` (${item.year})` : ''}`}
        </Text>
      ))}
    </Box>
  );
}
```

Example 3: Jellyfin library browser
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useJellyfinItems, jellyfinImage } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function JellyfinMovies({ baseUrl, apiKey }: { baseUrl: string; apiKey: string }) {
  const colors = useThemeColors();
  const { data } = useJellyfinItems(baseUrl, apiKey, { type: 'Movie', limit: 10, sortBy: 'SortName' });

  return (
    <Box style={{ gap: 6 }}>
      {data?.Items.map(item => {
        const tag = item.ImageTags?.Primary;
        const imgUrl = tag ? jellyfinImage(baseUrl, item.Id, tag) : null;

        return (
          <Box key={item.Id} style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
            {imgUrl && <Image src={imgUrl} style={{ width: 40, height: 60, borderRadius: 4 }} />}
            <Box style={{ gap: 2 }}>
              <Text style={{ color: colors.text, fontSize: 13 }}>{item.Name}</Text>
              <Text style={{ color: colors.muted, fontSize: 11 }}>
                {`${item.ProductionYear ?? '?'} | ${item.CommunityRating?.toFixed(1) ?? 'N/A'}`}
              </Text>
            </Box>
          </Box>
        );
      })}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Plex wraps responses in `MediaContainer`. Always access `data.MediaContainer.Metadata` or `data.MediaContainer.Directory`.
- Plex session `Metadata` can be undefined (empty array when no sessions). Always use `?? []`.
- Jellyfin RunTimeTicks uses .NET ticks (10,000,000 = 1 second). Divide by 10000000 for seconds.
- Jellyfin image URLs require the image tag. Check `item.ImageTags?.Primary` before calling jellyfinImage().
- Plex thumbnail paths are relative to the server. Prepend your baseUrl to use them.
