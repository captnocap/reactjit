=== OVERVIEW ===
The Steam hooks wrap the Steam Web API and Steam Store API. Authentication uses an API key passed as a query parameter. Get one at https://steamcommunity.com/dev/apikey.

Most hooks require both an API key and a Steam ID (the 64-bit numeric ID, not the vanity URL). useSteamAppDetails queries the public Store API and only needs an app ID.

Two utility functions build Steam CDN URLs for game icons and header images.

=== API / SYNTAX ===
```tsx
import {
  useSteamUser,
  useSteamOwnedGames,
  useSteamRecentGames,
  useSteamFriends,
  useSteamAchievements,
  useSteamAppDetails,
  steamGameIcon,
  steamHeaderImage,
} from '@reactjit/apis';
```

---- useSteamUser

```tsx
function useSteamUser(
  apiKey: string | null,
  steamId: string | null
): APIResult<{ response: { players: SteamPlayer[] } }>
```

Returns player profile data. The player is in response.players[0].

---- useSteamOwnedGames

```tsx
function useSteamOwnedGames(
  apiKey: string | null,
  steamId: string | null,
  opts?: { includeAppInfo?: boolean }
): APIResult<{ response: { game_count: number; games: SteamOwnedGame[] } }>
```

includeAppInfo defaults to true (includes game names and icons). Also includes free-to-play games.

---- useSteamRecentGames

```tsx
function useSteamRecentGames(
  apiKey: string | null,
  steamId: string | null,
  opts?: { count?: number }
): APIResult<{ response: { total_count: number; games: SteamRecentGame[] } }>
```

count defaults to 10.

---- useSteamFriends

```tsx
function useSteamFriends(
  apiKey: string | null,
  steamId: string | null
): APIResult<{ friendslist: { friends: SteamFriend[] } }>
```

---- useSteamAchievements

```tsx
function useSteamAchievements(
  apiKey: string | null,
  steamId: string | null,
  appId: number | null
): APIResult<{ playerstats: { achievements: SteamAchievement[] } }>
```

Requires all three parameters (apiKey, steamId, appId) to be non-null.

---- useSteamAppDetails

```tsx
function useSteamAppDetails(
  appId: number | null
): APIResult<Record<string, { success: boolean; data: SteamAppDetails }>>
```

Queries the public Store API. No API key needed. Response is keyed by app ID string.

---- steamGameIcon / steamHeaderImage

```tsx
function steamGameIcon(appId: number, iconHash: string): string
function steamHeaderImage(appId: number): string
```

Build CDN URLs. steamGameIcon needs the icon hash from SteamOwnedGame.img_icon_url.

---- Types

SteamPlayer:
| Field                    | Type    | Description                          |
|--------------------------|---------|--------------------------------------|
| steamid                  | string  | 64-bit Steam ID                     |
| personaname              | string  | Display name                         |
| profileurl               | string  | Profile URL                          |
| avatar                   | string  | 32x32 avatar URL                     |
| avatarmedium             | string  | 64x64 avatar URL                     |
| avatarfull               | string  | 184x184 avatar URL                   |
| personastate             | number  | Online status (0=offline, 1=online)  |
| gameextrainfo            | string  | Currently playing game (optional)    |
| gameid                   | string  | App ID of current game (optional)    |
| timecreated              | number  | Account creation timestamp           |

SteamOwnedGame:
| Field                       | Type    | Description                        |
|-----------------------------|---------|----------------------------------  |
| appid                       | number  | Steam app ID                       |
| name                        | string  | Game name                          |
| playtime_forever            | number  | Total playtime in minutes          |
| playtime_2weeks             | number  | Playtime in last 2 weeks (minutes) |
| img_icon_url                | string  | Icon hash for steamGameIcon()      |
| rtime_last_played           | number  | Unix timestamp of last session     |

SteamAppDetails:
| Field             | Type    | Description                            |
|-------------------|---------|----------------------------------------|
| name              | string  | Game name                              |
| steam_appid       | number  | App ID                                 |
| is_free           | boolean | Whether the game is free               |
| short_description | string  | Brief description                      |
| header_image      | string  | Header image URL                       |
| developers        | string[]| Developer names                        |
| publishers        | string[]| Publisher names                        |
| price_overview    | object  | Price info (final_formatted, discount) |
| metacritic        | object  | Metacritic score and URL (optional)    |

=== EXAMPLES ===
Example 1: Player profile card
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useSteamUser } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function SteamProfile({ apiKey, steamId }: { apiKey: string; steamId: string }) {
  const colors = useThemeColors();
  const { data, loading } = useSteamUser(apiKey, steamId);
  const player = data?.response?.players?.[0];

  if (loading) return <Text style={{ color: colors.muted }}>Loading profile...</Text>;
  if (!player) return <Text style={{ color: colors.primary }}>Player not found</Text>;

  const statusLabels = ['Offline', 'Online', 'Busy', 'Away', 'Snooze', 'Trade', 'Play'];

  return (
    <Box style={{ flexDirection: 'row', gap: 12, alignItems: 'center' }}>
      <Image src={player.avatarfull} style={{ width: 64, height: 64, borderRadius: 8 }} />
      <Box style={{ gap: 2 }}>
        <Text style={{ color: colors.text, fontWeight: 'bold', fontSize: 16 }}>{player.personaname}</Text>
        <Text style={{ color: colors.muted }}>{statusLabels[player.personastate] ?? 'Unknown'}</Text>
        {player.gameextrainfo && (
          <Text style={{ color: colors.primary }}>{`Playing: ${player.gameextrainfo}`}</Text>
        )}
      </Box>
    </Box>
  );
}
```

Example 2: Most played games
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useSteamOwnedGames, steamGameIcon } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TopGames({ apiKey, steamId }: { apiKey: string; steamId: string }) {
  const colors = useThemeColors();
  const { data, loading } = useSteamOwnedGames(apiKey, steamId);
  const games = (data?.response?.games ?? [])
    .sort((a, b) => b.playtime_forever - a.playtime_forever)
    .slice(0, 10);

  if (loading) return <Text style={{ color: colors.muted }}>Loading games...</Text>;

  return (
    <Box style={{ gap: 4 }}>
      {games.map(game => (
        <Box key={game.appid} style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
          <Image src={steamGameIcon(game.appid, game.img_icon_url)} style={{ width: 32, height: 32 }} />
          <Text style={{ color: colors.text, flexGrow: 1 }}>{game.name}</Text>
          <Text style={{ color: colors.muted }}>{`${(game.playtime_forever / 60).toFixed(1)}h`}</Text>
        </Box>
      ))}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Steam IDs must be the 64-bit numeric format (e.g. "76561198012345678"), not vanity URLs.
- useSteamUser returns an array of players (even for a single ID). Access with response.players[0].
- Playtime values are in minutes. Divide by 60 for hours.
- useSteamAppDetails response is keyed by the string version of the app ID: data[String(appId)].data.
- The friends list API requires the target profile to be public. Private profiles return an error.
