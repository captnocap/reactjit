=== OVERVIEW ===
Hooks for the Home Assistant REST API. Auth uses a long-lived access token (Bearer). Create one at Settings > Profile > Long-Lived Access Tokens in your HA instance.

All hooks require both a `baseUrl` (your HA instance URL, e.g., `http://homeassistant.local:8123`) and a `token`. Pass null for either to disable the hook.

useHACallService provides imperative service invocations (turn on lights, lock doors, trigger automations).

=== API / SYNTAX ===
```tsx
import {
  useHAStates,
  useHAEntity,
  useHAConfig,
  useHAServices,
  useHAHistory,
  useHALogbook,
  useHACallService,
} from '@reactjit/apis';
```

---- useHAStates

```tsx
function useHAStates(
  baseUrl: string | null,
  token: string | null,
  opts?: { interval?: number }
): APIResult<HAState[]>
```

Fetch all entity states. Optional polling interval in milliseconds.

---- useHAEntity

```tsx
function useHAEntity(
  baseUrl: string | null,
  token: string | null,
  entityId: string | null,
  opts?: { interval?: number }
): APIResult<HAState>
```

Fetch a single entity's state by ID (e.g., 'light.living_room'). Optional polling.

---- useHAConfig

```tsx
function useHAConfig(
  baseUrl: string | null,
  token: string | null
): APIResult<HAConfig>
```

Fetch the HA instance configuration (location, timezone, version).

---- useHAServices

```tsx
function useHAServices(
  baseUrl: string | null,
  token: string | null
): APIResult<HAService[]>
```

List all available services grouped by domain.

---- useHAHistory

```tsx
function useHAHistory(
  baseUrl: string | null,
  token: string | null,
  entityId: string | null,
  opts?: { hours?: number }
): APIResult<HAState[][]>
```

Fetch state history for an entity. Returns an array of arrays (one per entity). Lookback defaults to 24 hours.

---- useHALogbook

```tsx
function useHALogbook(
  baseUrl: string | null,
  token: string | null,
  opts?: { entityId?: string; hours?: number }
): APIResult<HALogEntry[]>
```

Fetch logbook entries. Optionally filter by entity ID. Lookback defaults to 24 hours.

---- useHACallService

```tsx
function useHACallService(
  baseUrl: string | null,
  token: string | null
): { call: (domain: string, service: string, data?: Record<string, any>) => Promise; loading: boolean; error: Error | null }
```

Imperative service caller. Returns a `call` function that POSTs to `/api/services/{domain}/{service}`.

---- Types

HAState:
| Field         | Type                   | Description                    |
|---------------|------------------------|--------------------------------|
| entity_id     | string                 | Entity ID (e.g., 'light.desk') |
| state         | string                 | Current state ('on', '22.5')   |
| attributes    | Record<string, any>    | Entity attributes              |
| last_changed  | string                 | Last state change timestamp    |
| last_updated  | string                 | Last update timestamp          |

HAConfig:
| Field         | Type                   | Description               |
|---------------|------------------------|---------------------------|
| location_name | string                 | Instance name              |
| latitude      | number                 | Location latitude          |
| longitude     | number                 | Location longitude         |
| elevation     | number                 | Elevation in meters        |
| unit_system   | Record<string, string> | Unit configuration         |
| time_zone     | string                 | Timezone string            |
| version       | string                 | HA version                 |

HAService:
| Field    | Type                                    | Description         |
|----------|-----------------------------------------|---------------------|
| domain   | string                                  | Service domain      |
| services | Record<string, { description, fields }> | Available services  |

HALogEntry:
| Field     | Type    | Description           |
|-----------|---------|-----------------------|
| name      | string  | Entity display name   |
| message   | string  | Log message           |
| entity_id | string? | Related entity ID     |
| when      | string  | Timestamp             |

=== EXAMPLES ===
Example 1: Entity state display
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useHAEntity } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function LightStatus({ baseUrl, token }: { baseUrl: string; token: string }) {
  const colors = useThemeColors();
  const { data } = useHAEntity(baseUrl, token, 'light.living_room', { interval: 5000 });

  if (!data) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  const isOn = data.state === 'on';
  const brightness = data.attributes.brightness;

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14 }}>
        {`Living Room: ${isOn ? 'On' : 'Off'}`}
      </Text>
      {isOn && brightness != null && (
        <Text style={{ color: colors.muted, fontSize: 12 }}>
          {`Brightness: ${Math.round((brightness / 255) * 100)}%`}
        </Text>
      )}
    </Box>
  );
}
```

Example 2: Toggle a light
---
```tsx
import { Box, Text, Pressable } from '@reactjit/core';
import { useHAEntity, useHACallService } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function LightToggle({ baseUrl, token }: { baseUrl: string; token: string }) {
  const colors = useThemeColors();
  const { data, refetch } = useHAEntity(baseUrl, token, 'light.desk');
  const { call } = useHACallService(baseUrl, token);

  const toggle = async () => {
    await call('light', 'toggle', { entity_id: 'light.desk' });
    refetch();
  };

  return (
    <Pressable onPress={toggle}>
      <Text style={{ color: colors.primary, fontSize: 14 }}>
        {`Desk Light: ${data?.state ?? '...'} (tap to toggle)`}
      </Text>
    </Pressable>
  );
}
```

Platforms: love2d

Example 3: Temperature sensor history
---
```tsx
import { Box, Text } from '@reactjit/core';
import { useHAHistory } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TempHistory({ baseUrl, token }: { baseUrl: string; token: string }) {
  const colors = useThemeColors();
  const { data } = useHAHistory(baseUrl, token, 'sensor.temperature', { hours: 12 });

  const states = data?.[0] ?? [];
  const latest = states[states.length - 1];

  return (
    <Box style={{ gap: 4 }}>
      <Text style={{ color: colors.text, fontSize: 14, fontWeight: 'bold' }}>Temperature (12h)</Text>
      <Text style={{ color: colors.text, fontSize: 24 }}>
        {latest ? `${latest.state}\u00B0C` : '...'}
      </Text>
      <Text style={{ color: colors.muted, fontSize: 11 }}>
        {`${states.length} readings`}
      </Text>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Both baseUrl and token must be non-null for any hook to fire. Pass null for either to disable.
- useHACallService.call requires the full entity_id in the data object (e.g., `{ entity_id: 'light.desk' }`).
- useHAHistory returns an array of arrays. For a single entity, access `data[0]` to get the state array.
- Long-lived access tokens do not expire but can be revoked. Store them securely via useServiceKey.
