=== OVERVIEW ===
Hooks for The Movie Database (TMDB) API. Auth uses an API key passed as a query parameter. Get one at https://www.themoviedb.org/settings/api.

Pass the API key as the first argument. Pass null to disable fetching. Every hook returns `{ data, loading, error, refetch }`.

TMDB image paths (poster_path, backdrop_path) are relative. Use the `tmdbImage()` helper to construct full URLs.

=== API / SYNTAX ===
```tsx
import {
  useTMDBTrending,
  useTMDBSearch,
  useTMDBMovie,
  useTMDBSeries,
  useTMDBPopular,
  tmdbImage,
} from '@reactjit/apis';
```

---- useTMDBTrending

```tsx
function useTMDBTrending(
  apiKey: string | null,
  opts?: { mediaType?: 'movie' | 'tv' | 'all'; timeWindow?: 'day' | 'week' }
): APIResult<TMDBPaginated<TMDBMovie>>
```

Returns trending content. Defaults to movies trending this week.

---- useTMDBSearch

```tsx
function useTMDBSearch(
  apiKey: string | null,
  query: string | null,
  opts?: { type?: 'movie' | 'tv' | 'multi'; page?: number }
): APIResult<TMDBPaginated<TMDBMovie & TMDBSeries>>
```

Search for movies, TV shows, or both. Defaults to 'multi' (both). Pass null as query to disable.

---- useTMDBMovie

```tsx
function useTMDBMovie(
  apiKey: string | null,
  movieId: number | null
): APIResult<TMDBMovieDetails>
```

Fetch full details for a specific movie by ID.

---- useTMDBSeries

```tsx
function useTMDBSeries(
  apiKey: string | null,
  seriesId: number | null
): APIResult<TMDBSeriesDetails>
```

Fetch full details for a specific TV series by ID.

---- useTMDBPopular

```tsx
function useTMDBPopular(
  apiKey: string | null,
  opts?: { type?: 'movie' | 'tv'; page?: number }
): APIResult<TMDBPaginated<TMDBMovie>>
```

Returns popular movies or TV shows. Defaults to movies.

---- tmdbImage

```tsx
function tmdbImage(path: string | null, size?: string): string | null
```

Constructs a full TMDB image URL from a relative path. Returns null if path is null. Size defaults to 'w500'. Common sizes: 'w92', 'w154', 'w185', 'w342', 'w500', 'w780', 'original'.

---- Types

TMDBMovie:
| Field         | Type           | Description              |
|---------------|----------------|--------------------------|
| id            | number         | TMDB movie ID            |
| title         | string         | Movie title              |
| overview      | string         | Plot summary             |
| poster_path   | string \| null | Relative poster path     |
| backdrop_path | string \| null | Relative backdrop path   |
| release_date  | string         | Release date (YYYY-MM-DD)|
| vote_average  | number         | Rating out of 10         |
| vote_count    | number         | Number of votes          |
| genre_ids     | number[]       | Genre ID array           |
| popularity    | number         | Popularity score         |

TMDBSeries:
| Field          | Type           | Description                 |
|----------------|----------------|-----------------------------|
| id             | number         | TMDB series ID              |
| name           | string         | Series name                 |
| overview       | string         | Plot summary                |
| poster_path    | string \| null | Relative poster path        |
| backdrop_path  | string \| null | Relative backdrop path      |
| first_air_date | string         | First air date (YYYY-MM-DD) |
| vote_average   | number         | Rating out of 10            |
| genre_ids      | number[]       | Genre ID array              |

TMDBMovieDetails extends TMDBMovie with:
| Field                | Type     | Description                    |
|----------------------|----------|--------------------------------|
| runtime              | number   | Runtime in minutes             |
| budget               | number   | Production budget              |
| revenue              | number   | Box office revenue             |
| tagline              | string   | Movie tagline                  |
| genres               | Array    | `{ id, name }` genre objects   |
| production_companies | Array    | `{ id, name, logo_path }` list |
| imdb_id              | string?  | IMDb ID                        |

TMDBSeriesDetails extends TMDBSeries with:
| Field              | Type     | Description                 |
|--------------------|----------|-----------------------------|
| number_of_seasons  | number   | Total seasons               |
| number_of_episodes | number   | Total episodes              |
| episode_run_time   | number[] | Episode runtimes in minutes |
| genres             | Array    | `{ id, name }` genre list   |
| status             | string   | 'Returning Series', etc.    |
| networks           | Array    | `{ id, name, logo_path }`   |

TMDBPaginated:
| Field         | Type   | Description           |
|---------------|--------|-----------------------|
| page          | number | Current page number   |
| results       | T[]    | Array of results      |
| total_pages   | number | Total number of pages |
| total_results | number | Total result count    |

=== EXAMPLES ===
Example 1: Trending movies
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useTMDBTrending, tmdbImage } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function TrendingMovies({ apiKey }: { apiKey: string }) {
  const colors = useThemeColors();
  const { data, loading } = useTMDBTrending(apiKey);

  if (loading) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  return (
    <Box style={{ gap: 8 }}>
      {data?.results.slice(0, 5).map(movie => (
        <Box key={movie.id} style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
          <Image
            src={tmdbImage(movie.poster_path, 'w92') ?? ''}
            style={{ width: 46, height: 69, borderRadius: 4 }}
          />
          <Box style={{ gap: 2 }}>
            <Text style={{ color: colors.text, fontSize: 13, fontWeight: 'bold' }}>{movie.title}</Text>
            <Text style={{ color: colors.muted, fontSize: 11 }}>{`${movie.vote_average.toFixed(1)}/10`}</Text>
          </Box>
        </Box>
      ))}
    </Box>
  );
}
```

Example 2: Movie detail view
---
```tsx
import { Box, Text, Image } from '@reactjit/core';
import { useTMDBMovie, tmdbImage } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function MovieDetail({ apiKey, movieId }: { apiKey: string; movieId: number }) {
  const colors = useThemeColors();
  const { data } = useTMDBMovie(apiKey, movieId);

  if (!data) return <Text style={{ color: colors.muted }}>Loading...</Text>;

  return (
    <Box style={{ gap: 8 }}>
      <Image
        src={tmdbImage(data.backdrop_path, 'w780') ?? ''}
        style={{ width: '100%', height: 200 }}
      />
      <Text style={{ color: colors.text, fontSize: 18, fontWeight: 'bold' }}>{data.title}</Text>
      {data.tagline ? <Text style={{ color: colors.muted, fontSize: 12 }}>{data.tagline}</Text> : null}
      <Text style={{ color: colors.text, fontSize: 13 }}>{data.overview}</Text>
      <Text style={{ color: colors.muted, fontSize: 11 }}>
        {`${data.runtime} min | ${data.genres.map(g => g.name).join(', ')}`}
      </Text>
    </Box>
  );
}
```

Example 3: Search with conditional fetch
---
```tsx
import { useState } from 'react';
import { Box, Text } from '@reactjit/core';
import { useTMDBSearch } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function MovieSearch({ apiKey }: { apiKey: string }) {
  const colors = useThemeColors();
  const [query, setQuery] = useState<string | null>(null);
  const { data, loading } = useTMDBSearch(apiKey, query, { type: 'movie' });

  return (
    <Box style={{ gap: 4 }}>
      {loading && <Text style={{ color: colors.muted }}>Searching...</Text>}
      {data?.results.map(movie => (
        <Text key={movie.id} style={{ color: colors.text, fontSize: 13 }}>
          {`${movie.title} (${movie.release_date?.slice(0, 4) ?? '?'})`}
        </Text>
      ))}
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Image paths from TMDB are relative (e.g., '/abc123.jpg'). Always use tmdbImage() to build a full URL.
- poster_path and backdrop_path can be null. Always null-check before passing to tmdbImage().
- useTMDBSearch with type 'multi' returns a union of movie and TV fields. Check for `title` (movies) vs `name` (TV shows) to distinguish.
- TMDB rate limits to ~40 requests per 10 seconds. Use the rateLimit option if you have many concurrent hooks.
