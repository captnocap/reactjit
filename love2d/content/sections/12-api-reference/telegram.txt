=== OVERVIEW ===
The Telegram hooks wrap the Telegram Bot API for building bot dashboards and notification systems. Authentication uses a bot token embedded in the URL path. Get one from @BotFather on Telegram.

useTelegramBot fetches bot identity. useTelegramUpdates polls for incoming messages and callback queries with configurable interval. useTelegramSend provides functions for sending text messages and photos.

=== API / SYNTAX ===
```tsx
import {
  useTelegramBot,
  useTelegramUpdates,
  useTelegramSend,
} from '@reactjit/apis';
```

---- useTelegramBot

```tsx
function useTelegramBot(
  botToken: string | null
): APIResult<{ ok: boolean; result: TelegramBotInfo }>
```

Calls the getMe endpoint. Returns bot identity including username and capability flags.

---- useTelegramUpdates

```tsx
function useTelegramUpdates(
  botToken: string | null,
  opts?: { offset?: number; limit?: number; interval?: number }
): APIResult<{ ok: boolean; result: TelegramUpdate[] }>
```

| Param         | Type   | Default | Description                              |
|---------------|--------|---------|------------------------------------------|
| opts.offset   | number | undefined| Update ID offset (for pagination)       |
| opts.limit    | number | 20     | Max updates to return                     |
| opts.interval | number | 5000   | Polling interval in milliseconds          |

Polls for new messages, edited messages, and callback queries.

---- useTelegramSend

```tsx
function useTelegramSend(botToken: string | null): {
  sendMessage: (chatId: number | string, text: string, opts?: {
    parseMode?: 'HTML' | 'Markdown' | 'MarkdownV2';
    replyToMessageId?: number;
  }) => Promise<any>;
  sendPhoto: (chatId: number | string, photoUrl: string, opts?: {
    caption?: string;
  }) => Promise<any>;
  loading: boolean;
  error: Error | null;
}
```

---- Types

TelegramBotInfo:
| Field                        | Type    | Description                              |
|------------------------------|---------|------------------------------------------|
| id                           | number  | Bot user ID                              |
| is_bot                       | boolean | Always true                              |
| first_name                   | string  | Bot display name                         |
| username                     | string  | Bot username (without @)                 |
| can_join_groups              | boolean | Whether the bot can join group chats     |
| can_read_all_group_messages  | boolean | Whether group privacy mode is disabled   |
| supports_inline_queries      | boolean | Whether inline mode is enabled           |

TelegramMessage:
| Field            | Type             | Description                          |
|------------------|------------------|--------------------------------------|
| message_id       | number           | Message ID                           |
| from             | TelegramUser     | Sender (optional for channels)       |
| chat             | TelegramChat     | Chat the message belongs to          |
| date             | number           | Unix timestamp                       |
| text             | string           | Message text (optional)              |
| photo            | Array            | Photo sizes (optional)               |
| document         | object           | Attached document (optional)         |
| reply_to_message | TelegramMessage  | Original message if this is a reply  |

TelegramUpdate:
| Field          | Type            | Description                          |
|----------------|-----------------|--------------------------------------|
| update_id      | number          | Sequential update ID                 |
| message        | TelegramMessage | New message (optional)               |
| edited_message | TelegramMessage | Edited message (optional)            |
| callback_query | object          | Inline keyboard callback (optional)  |

=== EXAMPLES ===
Example 1: Bot message monitor
---
```tsx
import { useState, useEffect, useRef } from 'react';
import { Box, Text, ScrollView } from '@reactjit/core';
import { useTelegramBot, useTelegramUpdates } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function BotMonitor({ botToken }: { botToken: string }) {
  const colors = useThemeColors();
  const { data: bot } = useTelegramBot(botToken);
  const offsetRef = useRef<number | undefined>(undefined);
  const { data: updates } = useTelegramUpdates(botToken, {
    offset: offsetRef.current,
    interval: 3000,
  });

  const messages = updates?.result
    ?.filter(u => u.message?.text)
    .map(u => u.message!) ?? [];

  useEffect(() => {
    if (updates?.result?.length) {
      offsetRef.current = updates.result[updates.result.length - 1].update_id + 1;
    }
  }, [updates]);

  return (
    <Box style={{ gap: 8 }}>
      <Text style={{ color: colors.text, fontWeight: 'bold' }}>
        {`@${bot?.result?.username ?? '...'}`}
      </Text>
      <ScrollView style={{ height: 300 }}>
        <Box style={{ gap: 4 }}>
          {messages.map(msg => (
            <Box key={msg.message_id} style={{ flexDirection: 'row', gap: 8 }}>
              <Text style={{ color: colors.primary }}>{msg.from?.first_name ?? 'Unknown'}</Text>
              <Text style={{ color: colors.text }}>{msg.text}</Text>
            </Box>
          ))}
        </Box>
      </ScrollView>
    </Box>
  );
}
```

Example 2: Send a message
---
```tsx
import { useState } from 'react';
import { Box, Text, Pressable, Input } from '@reactjit/core';
import { useTelegramSend } from '@reactjit/apis';
import { useThemeColors } from '@reactjit/theme';

export function SendMessage({ botToken, chatId }: { botToken: string; chatId: number }) {
  const colors = useThemeColors();
  const { sendMessage, loading } = useTelegramSend(botToken);
  const [text, setText] = useState('');

  const handleSend = async () => {
    if (!text.trim()) return;
    await sendMessage(chatId, text, { parseMode: 'HTML' });
    setText('');
  };

  return (
    <Box style={{ flexDirection: 'row', gap: 8 }}>
      <Input
        value={text}
        onChangeText={setText}
        onSubmit={handleSend}
        placeholder="Type a message..."
        style={{ flexGrow: 1 }}
      />
      <Pressable onPress={handleSend}>
        <Text style={{ color: colors.primary }}>{loading ? 'Sending...' : 'Send'}</Text>
      </Pressable>
    </Box>
  );
}
```

=== CRITICAL RULES ===
- Bot tokens must never be exposed in client-side code for public apps. These hooks are for local/private dashboards.
- useTelegramUpdates polls by default every 5 seconds. Adjust opts.interval for faster or slower polling.
- Track the offset to avoid re-processing old updates. Set offset to the last update_id + 1.
- sendPhoto accepts a URL, not a file upload. The image must be publicly accessible.
- Chat IDs can be negative for group chats.
