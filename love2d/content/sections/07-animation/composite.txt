=== OVERVIEW ===
The animation system provides four composition functions that combine Animation objects into more complex motion patterns. Each returns a new Animation with the same start/stop interface, so they can be nested arbitrarily.

parallel() runs all animations at the same time and completes when every animation finishes.

sequence() runs animations one after another. Each animation starts only after the previous one completes.

stagger(delay, animations) starts each animation with an incremental delay between them. The first animation starts immediately, the second starts after delay ms, the third after delay*2 ms, and so on.

loop(animation, config?) repeats an animation. Pass { iterations: N } to repeat N times, or omit for infinite looping. An infinite loop never calls its completion callback.

All four functions accept and return the Animation interface, so they compose freely: you can loop a sequence of staggered parallels if the situation calls for it.

=== API / SYNTAX ===
```tsx
import { parallel, sequence, stagger, loop } from '@reactjit/core';
```

| Function | Signature                                                  | Description                             |
|----------|------------------------------------------------------------|-----------------------------------------|
| parallel | (animations: Animation[]) => Animation                     | Run all at once, finish when all finish |
| sequence | (animations: Animation[]) => Animation                     | Run one after another in order          |
| stagger  | (delay: number, animations: Animation[]) => Animation      | Staggered parallel start                |
| loop     | (animation: Animation, config?: { iterations?: number }) => Animation | Repeat N times or infinitely  |

Loop config:
| Field      | Type   | Default | Description                        |
|------------|--------|---------|------------------------------------|
| iterations | number | -1      | Number of repeats (-1 = infinite)  |

=== EXAMPLES ===
Example 1: Parallel fade + slide
---
```tsx
import { useEffect } from 'react';
import { Box, useAnimation, parallel, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function FadeSlideIn() {
  const c = useThemeColors();
  const [opacityAnim, opacity] = useAnimation(0);
  const [slideAnim, translateY] = useAnimation(30);

  useEffect(() => {
    const anim = parallel([
      opacityAnim.timing({ toValue: 1, duration: 400, easing: Easing.easeOut }),
      slideAnim.timing({ toValue: 0, duration: 400, easing: Easing.easeOut }),
    ]);
    anim.start();
    return () => anim.stop();
  }, [opacityAnim, slideAnim]);

  return (
    <Box style={{ opacity, transform: { translateY }, backgroundColor: c.surface, padding: 16 }}>
      {/* content */}
    </Box>
  );
}
```

Example 2: Sequence for multi-step animation
---
```tsx
import { useEffect } from 'react';
import { useAnimation, sequence, Easing } from '@reactjit/core';

export function usePopIn() {
  const [anim, scale] = useAnimation(0);

  useEffect(() => {
    const pop = sequence([
      anim.timing({ toValue: 1.2, duration: 200, easing: Easing.easeOut }),
      anim.timing({ toValue: 1.0, duration: 150, easing: Easing.easeInOut }),
    ]);
    pop.start();
    return () => pop.stop();
  }, [anim]);

  return scale;
}
```

Example 3: Staggered list entrance
---
```tsx
import { useEffect } from 'react';
import { Box, Text, useAnimation, stagger, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function StaggeredList({ items }: { items: string[] }) {
  const c = useThemeColors();
  const anims = items.map(() => useAnimation(0));

  useEffect(() => {
    const animations = anims.map(([anim]) =>
      anim.timing({ toValue: 1, duration: 300, easing: Easing.easeOut })
    );
    const staggered = stagger(80, animations);
    staggered.start();
    return () => staggered.stop();
  }, []);

  return (
    <Box style={{ gap: 4 }}>
      {items.map((item, i) => {
        const [, opacity] = anims[i];
        return (
          <Box key={item} style={{ opacity, padding: 8, backgroundColor: c.surface, borderRadius: 4 }}>
            <Text style={{ color: c.text, fontSize: 14 }}>{item}</Text>
          </Box>
        );
      })}
    </Box>
  );
}
```

Platforms: love2d

Example 4: Infinite pulse with loop + sequence
---
```tsx
import { useEffect } from 'react';
import { Box, useAnimation, loop, sequence, Easing } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function PulsingDot() {
  const c = useThemeColors();
  const [anim, scale] = useAnimation(1);

  useEffect(() => {
    const pulse = loop(sequence([
      anim.timing({ toValue: 1.4, duration: 600, easing: Easing.easeInOut }),
      anim.timing({ toValue: 1.0, duration: 600, easing: Easing.easeInOut }),
    ]));
    pulse.start();
    return () => pulse.stop();
  }, [anim]);

  return (
    <Box style={{
      width: 12,
      height: 12,
      borderRadius: 6,
      backgroundColor: c.primary,
      transform: { scaleX: scale, scaleY: scale },
    }} />
  );
}
```

Platforms: love2d

Example 5: Finite loop
---
```tsx
import { useEffect } from 'react';
import { useAnimation, loop, Easing } from '@reactjit/core';

export function useFlash(times: number = 3) {
  const [anim, opacity] = useAnimation(1);

  useEffect(() => {
    const flash = loop(
      anim.timing({ toValue: 0, duration: 150, easing: Easing.linear }),
      { iterations: times },
    );
    flash.start(() => {
      anim.setValue(1); // reset after flashing
    });
    return () => flash.stop();
  }, [anim, times]);

  return opacity;
}
```

=== CRITICAL RULES ===
- sequence() stops if any animation in the chain finishes with { finished: false }. The remaining animations in the sequence are skipped.
- stagger() uses setTimeout for delays between starts. In an infinite loop, prefer using Lua-driven animation with keyframe delay instead for lower overhead.
- loop() with no iterations config runs forever and never calls its completion callback. Always stop it in a useEffect cleanup.
- An empty array passed to parallel(), sequence(), or stagger() completes immediately with { finished: true }.
- Composition functions return the standard Animation interface, so they can be nested: loop(sequence([parallel([a, b]), c])).
