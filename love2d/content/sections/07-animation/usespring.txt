=== OVERVIEW ===
useSpring automatically springs to a target value whenever it changes. It uses a physics simulation with stiffness, damping, and mass to produce natural-feeling motion that overshoots and settles. No duration is needed — the spring determines its own timing based on the physics parameters.

The hook skips animation on the first render and jumps straight to the target value. On subsequent changes to targetValue, it starts a spring animation from the current position.

useSpring is the most commonly used animation hook. It works well for:
- Counters and numeric displays that animate between values
- Toggle states (0/1) with natural bounce
- Position and size changes that need to feel physical
- Any value that changes in response to user interaction

For timing-based animations with explicit duration, use useTransition instead.

=== API / SYNTAX ===
```tsx
import { useSpring } from '@reactjit/core';

const currentValue = useSpring(targetValue, config?);
```

Parameters:
| Parameter   | Type   | Description                                   |
|-------------|--------|-----------------------------------------------|
| targetValue | number | The value to spring toward                    |
| config      | object | Optional spring physics configuration         |

Config:
| Field     | Type   | Default | Description                          |
|-----------|--------|---------|--------------------------------------|
| stiffness | number | 100     | Spring stiffness (higher = snappier) |
| damping   | number | 10      | Damping factor (lower = bouncier)    |
| mass      | number | 1       | Mass of the object                   |

Returns: number — the current interpolated value. Updates each frame during motion.

Common presets:
- Stiff: { stiffness: 300, damping: 20 } — quick snap with minimal overshoot
- Bouncy: { stiffness: 120, damping: 8 } — visible bounce before settling
- Sloppy: { stiffness: 80, damping: 5 } — heavy overshoot, multiple bounces

=== EXAMPLES ===
Example 1: Animated counter
---
```tsx
import { useState } from 'react';
import { Box, Pressable, Text, useSpring } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function SpringCounter() {
  const c = useThemeColors();
  const [count, setCount] = useState(0);
  const display = useSpring(count, { stiffness: 200, damping: 15 });

  return (
    <Box style={{ flexDirection: 'row', gap: 8, alignItems: 'center' }}>
      <Pressable onPress={() => setCount(n => n + 100)}>
        <Box style={{ padding: 8, backgroundColor: c.primary, borderRadius: 4 }}>
          <Text style={{ color: '#fff', fontSize: 14, fontWeight: 'bold' }}>{'+100'}</Text>
        </Box>
      </Pressable>
      <Text style={{ color: c.text, fontSize: 28, fontWeight: 'bold' }}>
        {`${Math.round(display)}`}
      </Text>
    </Box>
  );
}
```

Example 2: Toggle with bounce
---
```tsx
import { useState } from 'react';
import { Box, Pressable, Text, useSpring } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

export function BouncyToggle() {
  const c = useThemeColors();
  const [on, setOn] = useState(false);
  const progress = useSpring(on ? 1 : 0, { stiffness: 180, damping: 12 });

  return (
    <Pressable onPress={() => setOn(v => !v)}>
      <Box style={{
        width: 50,
        height: 28,
        borderRadius: 14,
        backgroundColor: c.surface,
        padding: 2,
      }}>
        <Box style={{
          width: 24,
          height: 24,
          borderRadius: 12,
          backgroundColor: c.primary,
          transform: { translateX: progress * 22 },
        }} />
      </Box>
    </Pressable>
  );
}
```

Example 3: Spring-driven bar chart
---
```tsx
import { useSpring, Box, Text } from '@reactjit/core';
import { useThemeColors } from '@reactjit/theme';

function Bar({ value, max, color }: { value: number; max: number; color: string }) {
  const width = useSpring((value / max) * 200, { stiffness: 120, damping: 14 });

  return (
    <Box style={{ height: 20, backgroundColor: 'rgba(255,255,255,0.05)', borderRadius: 4 }}>
      <Box style={{
        height: 20,
        width: Math.max(0, width),
        backgroundColor: color,
        borderRadius: 4,
      }} />
    </Box>
  );
}
```

=== PLATFORM NOTES ===
Love2D:
  The spring simulation uses Verlet integration with a clamped dt (max 64ms) to prevent instability during frame drops. The spring settles when both velocity and displacement drop below the restThreshold (0.001 by default).

=== CRITICAL RULES ===
- useSpring skips animation on the first render — the component starts at the target value instantly.
- The component re-renders every frame while the spring is in motion. For pure visual effects without layout impact, prefer Lua-driven style.transition instead.
- Very low damping values (below 3) can cause springs that oscillate for a long time before settling.
